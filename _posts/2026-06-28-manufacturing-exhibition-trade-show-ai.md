展示会は製造業の重要な営業機会です。AIを使って出展計画・ブース説明資料・来場者フォローメール・出展効果レポートを効率化することで、展示会ROIを最大化できます。

---

## 展示会出展計画書を整理するプロンプト

出展の目的・予算・準備計画を文書化します。

```
以下の情報をもとに、展示会出展計画書を整理してください。

【出展する展示会の概要】
展示会名: ものづくりワールド2026 東京
主催: リード エグジビション ジャパン
会期: 2026年10月14日（水）〜16日（金）3日間
会場: 東京ビッグサイト 東ホール
出展料金: ¥320,000（9m²・1小間）

【出展の目的・目標】
  ・新規取引先開拓（目標: 来場者名刺50枚獲得）
  ・医療機器・半導体装置分野への新規参入の足がかり作り
  ・自社技術（5軸加工・難材質加工）のブランド認知向上

【出展する技術・製品】
  ・5軸CNCマシニングセンタによる精密加工サンプル（3点）
  ・SUS316L・チタン加工の実物サンプル
  ・加工事例パネル（医療機器向け・半導体装置向け）
  ・会社パンフレット

【準備期間・予算】
準備期間: 出展4ヶ月前（6月）から
予算: 出展料¥320,000 + 装飾・印刷・交通費等¥150,000 = 約¥470,000

出力形式:
・「出展計画書の構成と各項目の記載内容」
・「準備タスクとスケジュール（出展4ヶ月前〜当日まで）」
・「ブース設計の考え方（9m²の効果的な配置案）」
・「必要な制作物リスト（パネル・チラシ・名刺入れ・
  デモ機・のぼり等）と発注先・費用感」
・「社内承認（予算申請）のための費用対効果の説明資料骨子」
```

## ブース説明・来場者対応スクリプトプロンプト

展示会当日の来場者への説明を標準化します。

```
以下の情報をもとに、展示会ブースでの
来場者対応スクリプトと質問想定Q&Aを整理してください。

【ブースで伝えたいメッセージ（優先順位順）】
  1. 「5軸加工で複雑形状を1工程で仕上げる技術力」
  2. 「SUS316L・チタン・インコネルの難材質加工実績」
  3. 「試作1個から量産まで一貫対応・最短3日納期」
  4. 「ISO 9001取得・不良率0.08%の品質保証」

【想定される来場者の属性】
  A: 購買・調達担当者（取引先候補として評価中）
  B: 技術開発者・設計者（加工の可能性を探っている）
  C: 同業他社・学生（情報収集目的）

出力形式:
・「ブースへの声がけ・アイキャッチのトーク（15秒版）」
・「来場者属性A（購買担当）への説明スクリプト（3分版）」
・「来場者属性B（技術者）への説明スクリプト（3分版）」
・「よくある質問（10問）と回答（Q&A集）」
  （例: 「最小ロットは？」「納期はどのくらい？」
  「見積もりはどうすれば？」）
・「名刺交換時にヒアリングすべき情報（5項目）」
```

## 展示会後のフォローメールと効果測定プロンプト

展示会後の営業フォローと振り返りを整理します。

```
以下の情報をもとに、展示会終了後の
フォローメールと効果測定レポートを整理してください。

【展示会の結果】
  ・来場者名刺獲得数: 68枚（目標50枚→達成）
  ・うち「見積もり依頼あり」: 8件
  ・うち「後日詳しく聞きたい」: 15件
  ・うち「情報収集のみ」: 45件

名刺の内訳（業種別）:
  ・医療機器メーカー購買部: 12枚
  ・半導体製造装置メーカー設計部: 8枚
  ・産業機械メーカー調達部: 22枚
  ・その他（同業・学生等）: 26枚

出展費用合計: ¥485,000

出力形式:
・「フォローメールの文案（3パターン）」
  パターン1: 見積もり依頼があった人向け（即アクション）
  パターン2: 「後で詳しく」と言った人向け（追客）
  パターン3: 情報収集だった人向け（ライトフォロー）
・「展示会効果測定レポートのフォーマット」
  （来場者数・名刺数・見積もり件数・
  受注に至った件数・売上への貢献・コスト対効果）
・「次回出展の改善ポイント（今回の反省点と次回の対策）」
・「展示会リードをCRM/Excelで管理するフォーマット
  （氏名・会社・役職・関心度・フォロー日・ステータス）」
```
