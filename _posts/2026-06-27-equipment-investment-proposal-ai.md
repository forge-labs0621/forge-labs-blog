製造現場で設備の更新・増設・新規購入を申請する「稟議書（設備投資申請書）」は、金額が大きいほど経営者や上長を説得する論理構成が求められます。AIを使うと、散在する情報を整理して、承認されやすい稟議書の骨子を素早く作れます。

## 設備投資稟議書に必要な要素

経営者・工場長が承認を判断するために見る主な項目:

1. **申請の背景・現状の問題**: なぜ今この設備が必要なのか
2. **投資の目的・効果**: 何を改善するか、どのくらい改善するか
3. **投資金額・支払い計画**: いくらかかるか、いつ払うか
4. **回収期間の目安**: 何年で元が取れるか（費用対効果）
5. **代替案との比較**: 修繕・外注・リースとの比較
6. **リスクと対策**: 導入しないリスク・導入するリスク

## 設備更新（老朽化設備の入れ替え）の稟議書を作るプロンプト

```
以下の情報をもとに、設備更新の稟議書を作成してください。

【申請者・部署】
（例：製造部 部長 田中○○）

【申請する設備の概要】
更新する既存設備: （例：旋盤 Model-A（2004年導入、築21年））
導入を希望する新設備: （例：NC旋盤 Model-X（△△機械工業製））
希望導入時期: （例：2026年9月末まで）

【申請金額】
（例：
・機械本体: ¥4,800,000（税別）
・設置・据え付け工事: ¥200,000
・初期教育・立ち上げ費用: ¥100,000
・合計: ¥5,100,000（税別）
）

【現状の問題（更新が必要な理由）】
（例：
・2024年に主軸ベアリングを交換したが、今年に入って再び振動が増加している
・修繕見積もりでは¥600,000かかる見込み。また数年後に再度修繕が必要になる可能性が高い
・加工精度が低下し、不良品発生率が月平均3.2%まで上昇（目標1.0%以下）
・部品の供給終了が2027年に見込まれており、次の故障時には修繕不可能になる
）

【導入後の効果（できるだけ数値で）】
（例：
・加工精度の向上: 不良品発生率 3.2% → 0.8%（年間不良損失額 約¥350,000削減見込み）
・稼働率の向上: 修繕ダウンタイム 月平均8時間 → 0時間
・電力コスト削減: 新型は省エネ設計で電力消費約20%削減
）

出力形式:
・「設備更新申請書」として正式な稟議書の体裁で
・申請金額・申請者・申請日・承認者欄
・現状の問題（数値あり）
・更新内容と選定理由
・費用対効果（概算の投資回収期間を含む）
・代替案（修繕継続）との比較
・スケジュール案（発注〜稼働開始）
・リスクと対策（1〜2点）
```

## AI導入の稟議書を作るプロンプト

ChatGPT等のAIツールを会社で使い始める際の稟議書です。

```
以下の情報をもとに、AI（ChatGPT等）の社内導入申請書を作成してください。

【申請者・部署】
（例：製造部 係長 鈴木○○）

【導入を希望するサービス】
（例：ChatGPT Plus（OpenAI） / ¥3,000〜4,000/月（1ライセンス））

【利用目的・対象業務】
（例：
・日報・報告書の下書き作成
・クレーム対応文の作成
・安全教育資料の骨子作成
・月次報告書のコメント整理
）

【現状の問題・課題】
（例：
・報告書の作成に1件あたり平均30〜45分かかっており、月に計15〜20時間を
  書類作業に費やしている
・忙しい時期は残業して書類作業をすることもある
）

【導入後の効果（見込み）】
（例：
・報告書作成時間を約50%削減（15〜20時間 → 7〜10時間/月）
・削減工数を段取り・段取り替えの改善活動に充当できる
）

【情報管理・リスク対応】
（例：
・顧客名・個人情報・設計図面はAIに入力しない
・「〇〇社の図面について」などの入力も禁止し、ルールを明文化する
・学習オプトアウト設定を利用する（ChatGPT Plusは設定可能）
）

出力形式:
・「AI活用ツール導入申請書」として
・導入の目的・現状の課題・期待効果
・月額費用・年間費用
・情報管理ルール（リスク対策）
・試用期間の設定（例：3ヶ月間 試用後に継続可否を判断）
・申請者・承認者・日付欄
```

## 設備増設（生産能力増強）の稟議書を作るプロンプト

受注増加・新規取引先獲得のための設備増設申請に使えます。

```
以下の情報をもとに、設備増設の稟議書（投資判断材料）を作成してください。

【申請者・部署】
（例：製造部 工場長 山田○○）

【増設を希望する設備・内容】
（例：プレスブレーキ（曲げ加工機）1台の追加導入）

【申請金額】（例：¥6,500,000（機械本体 + 設置工事込み））

【申請の背景・受注見込み】
（例：
・今年4月から新規取引先（○○電機株式会社）から板金加工の引き合いがあり、
  月産500個の受注が見込まれる（先方から内示を得ている）
・現在の設備（プレスブレーキ2台）は既存受注でほぼフル稼働しており、
  新規受注を受け入れる余力がない
・外注で対応した場合、原価が上がりすぎて採算が合わない（外注単価試算済み）
）

【増設後の売上・利益見込み】
（例：
・月産500個 × 単価¥2,800 = 月次売上¥1,400,000（年間¥16,800,000）
・原価率60%（外注なし）として、年間粗利約¥6,720,000見込み
・設備投資¥6,500,000に対し、概ね1年以内での回収が見込める
）

出力形式:
・「設備増設投資申請書」として
・申請背景（受注見込み・現状の能力不足）
・投資金額と資金調達方法（自己資金・リース・補助金の選択肢）
・売上・利益見込みと投資回収期間
・スケジュール（発注・納品・稼働開始）
・リスク（受注が見込み通りにならなかった場合等）と対応策
・承認申請額・承認者欄
```

## 稟議書に「費用対効果」の計算コメントを付けるプロンプト

数値計算はAIに頼みすぎず、自社で計算した数値をAIに整理させます。

```
以下の数値をもとに、稟議書の「費用対効果」セクションのコメントを
わかりやすく整理してください。

【投資金額】（例：¥5,100,000）

【削減・改善の数値（自分で計算・推計したもの）】
（例：
・不良品削減による損失防止: 年間約¥350,000
・修繕ダウンタイム解消による生産増: 年間約¥480,000
・電力コスト削減: 年間約¥90,000
・合計年間改善効果: 約¥920,000/年
）

出力形式:
・「費用対効果（試算）」の見出しで
・各改善項目を表形式で整理
・投資回収期間の計算式（¥5,100,000 ÷ ¥920,000 ≒ 5.5年）
・「あくまで試算であり、実際の効果は運用後に確認する」の一文を付ける
```

## AIを使う際の注意

- 稟議書の金額・効果試算・回収期間はAIの計算結果を鵜呑みにせず、必ず担当者・経理が確認してください
- 設備投資の見積書・カタログはAIに貼り付ける前に、機密情報（顧客名等）が含まれていないか確認してください
- 稟議書の承認後、実際の発注・契約手続きは別途行う必要があります
