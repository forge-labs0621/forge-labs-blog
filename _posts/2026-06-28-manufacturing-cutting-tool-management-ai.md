切削工具のコストは加工コストの大きな割合を占めます。AIを使って工具台帳・工具寿命基準・工具費用分析・再研磨管理を体系化し、工具費削減につなげられます。

---

## 切削工具管理台帳と工具寿命基準を整理するプロンプト

```
以下の情報をもとに、切削工具（エンドミル・ドリル等）の
管理台帳と工具寿命基準を整理してください。

【切削工具管理の現状と課題】
使用工具の種類（主なもの）:
  ・超硬エンドミル（φ10・φ6・φ4）各種
  ・超硬ドリル（φ3〜φ12）各種
  ・インサートチップ（ターニング用・正面フライス用）

管理上の問題:
  ①工具の在庫数が把握できていない
    → 「なくなったら注文」で急ぎ調達コストが発生
  ②工具の寿命基準がなく、作業者の目視で交換判断
    → 折損・加工不良・過剰使用（もったいない）がランダムに発生
  ③再研磨品と新品が混在して管理が曖昧

目標: 工具費を現状比 20%削減（月間工具費 約12万円）

出力形式:
・「工具台帳フォーマット（工具名称 / 規格 / メーカー / 単価 / 在庫数 /
  発注点 / 発注ロット / 保管場所 / 再研磨可否）」
・「工具寿命基準の設定方法（加工個数基準 / 距離基準 / 状態基準）と
  基準を決めるためのデータ収集方法」
・「工具使用記録フォーマット（工具番号 / 使用機械 / 使用開始日 / 
  加工個数 / 摩耗状態 / 交換・再研磨記録）」
・「工具費月次集計フォーマット（工具種別ごとの費用 / 前月比 / 工具費/個）」
・「再研磨品の管理方法（再研磨回数の上限 / 再研磨後の検収手順）」
```
