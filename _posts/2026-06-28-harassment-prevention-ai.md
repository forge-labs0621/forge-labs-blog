2020年6月から大企業、2022年4月から中小企業にもパワーハラスメント防止措置が義務化されています（労働施策総合推進法）。製造業の現場では上下関係が明確なため、ハラスメントリスクへの対応書類の整備が重要です。

> **注意**: ハラスメントの法的判断・対応は必ず社会保険労務士・弁護士に確認してください。

---

## ハラスメント防止方針・社内規程を作るプロンプト

会社のハラスメント防止に関する方針と規程の骨子を作ります。

```
以下の情報をもとに、ハラスメント防止方針・規程の骨子を
作成してください。

【自社の状況】
業種・規模: （例：金属加工業 / 従業員55名）
現状: （例：就業規則の懲戒規定にハラスメント禁止の記載はあるが、
  具体的な防止方針・相談窓口の設置・対応手順がない）

【含めたい主な内容】
・ハラスメントの定義（パワハラ・セクハラ・マタハラ等）
・会社の基本方針（ハラスメントを許さない旨の明確な宣言）
・禁止行為の具体的な例示
・相談窓口の設置（担当者・連絡先）
・相談があった場合の対応の流れ（事実確認→対処→フォロー）
・相談者・被害者のプライバシー保護・不利益取り扱い禁止
・違反した場合の懲戒処分

出力形式:
・ハラスメント防止規程の骨子（目次と各条の概要）
・「パワハラの6類型（身体的攻撃・精神的攻撃・人間関係からの切り離し・
  過大な要求・過小な要求・個の侵害）」の説明（社内周知用）
・全従業員向けの「ハラスメント防止宣言文」（トップメッセージ）
・※最終的な規程は社労士に確認することを明記
```

## 相談窓口の周知文・案内ポスターを作るプロンプト

従業員にハラスメント相談窓口を周知する文書を作ります。

```
以下の情報をもとに、ハラスメント相談窓口の
社内周知文と掲示用案内を作成してください。

【自社の相談窓口設定】
社内相談窓口: （例：人事部 ○○（内線○○ / メール: ○○@○○.co.jp））
社外相談窓口（任意）: （例：○○社労士事務所 / または
  外部EAP（従業員支援プログラム）の電話相談）
公的な相談先: 都道府県労働局雇用環境・均等部（室）

【伝えたいポイント】
・「相談すること」を恐れないでほしい
・相談内容は秘密にする（相談者の許可なく第三者に開示しない）
・相談しても不利益な扱いはしない
・「これってハラスメント？」と思ったら気軽に相談してほしい

出力形式:
・A4掲示用の「相談窓口案内ポスター文案」（見出し・本文・連絡先）
・従業員への周知メール本文
・「よくある質問（Q&A）」3〜5件
  （例：「相談したら加害者に知られる？」「証拠がなくても相談できる？」）
```

## ハラスメント研修の資料（管理職向け）を作るプロンプト

班長・リーダー・管理職向けのハラスメント防止研修テキストを作ります。

```
以下の情報をもとに、製造業の管理職（班長・リーダー）向けの
ハラスメント防止研修テキストを作成してください。

【対象者の特徴】
（例：製造現場の班長（リーダー層）/ 平均年齢42歳 /
  技術は一流だがマネジメント経験は少ない人が多い）

【研修で伝えたいこと】
・パワハラの定義と6類型（事例を使って具体的に）
・「厳しい指導」と「パワハラ」の境界線
・「部下への叱り方」の正しい方法（行動を指摘する・感情的にならない等）
・部下から相談を受けた場合の対応（聞き方・記録・報告）
・管理職自身が「やってしまいがちなNG行動」チェックリスト

【製造現場特有のリスク】
（例：
・大きな声で怒鳴る文化が残っている
・「俺が若い頃は〜」という昔話で過度な残業を正当化する
・ミスをした部下を皆の前で叱責する
・特定の作業者だけ雑用を押しつける
）

出力形式:
・研修テキストのアウトライン（45分〜1時間の研修を想定）
・グループワーク用の「事例問題（これはパワハラ？）」3ケース
・管理職向けの「NG行動チェックリスト」（自己チェック用）
・研修の締めのメッセージ（「職場は部下と一緒に作るもの」）
```

## ハラスメント相談の事実確認記録書を作るプロンプト

相談を受けた後の事実確認・対応経緯を記録する書類を作ります。

```
以下の情報をもとに、ハラスメント相談の事実確認記録書の
フォーマットを作成してください。

【記録すべき内容】
相談受付:
  ・相談日時・相談者・相談内容の要旨
  ・相談者の希望（解決方法・プライバシー保護の範囲）
事実確認:
  ・相談者へのヒアリング内容・日時
  ・行為者（指摘された側）へのヒアリング内容・日時
  ・第三者（目撃者等）へのヒアリング（行った場合）
  ・客観的な証拠の確認（メール・記録等）
判断・対応:
  ・ハラスメントの有無の判断とその根拠
  ・会社としての対応方針（指導・懲戒・配置転換等）
  ・相談者へのフィードバック内容・日時
フォローアップ:
  ・対応後の状況確認（1か月後等）

出力形式:
・事実確認記録書のフォーマット（表形式で記入しやすい構成）
・「ヒアリング時の留意点」（誘導せず・中立的に・記録を取る）
・「秘密保持の確約書」の文言（ヒアリング対象者に署名してもらう用）
・※法的判断・懲戒処分の判断は弁護士・社労士に確認することを明記
```
