製造業・工場で人を採用する際、「労働条件通知書」は法律で交付が義務づけられた重要な書類です。書き方・含めるべき項目・注意事項の確認をAIが補助できます。

> **重要**: 労働条件通知書・雇用契約書の内容は、労働基準法・パートタイム労働法・労働契約法等の法令に基づいて作成する必要があります。ここで紹介するプロンプトは文書の整理・下書きの補助であり、最終的な内容は社会保険労務士や法的専門家に確認することを推奨します。AIは法的判断を行いません。

---

## 労働条件通知書の骨子を整理するプロンプト

採用者ごとの労働条件をまとめる労働条件通知書の下書きを整理します。

```
以下の採用条件をもとに、労働条件通知書の内容を整理してください。

【採用する従業員の情報】
氏名: （例：山田太郎）
雇用形態: （正社員 / パートタイム / 有期雇用 から選択）
今回の例: 正社員

【労働条件の内容】
業務内容: （例：製造部 機械加工オペレーター業務）
就業場所: （例：○○製作所 本社工場（○○県○○市））
労働時間: （例：8:00〜17:00（休憩60分）/ 所定労働時間 8時間）
休日: （例：土日祝・年末年始・夏季休暇（年間休日108日））
有給休暇: （例：入社後6か月経過後、法定通り10日付与）
給与: （例：月給 ○○万円（基本給）/ 残業手当別途）
試用期間: （例：3か月）
社会保険: （例：雇用保険・健康保険・厚生年金・労災保険に加入）

出力形式:
・労働条件通知書のフォーマット（法定必須記載事項をカバー）
・「以下の事項を変更する場合は事前に通知します」の文言
・法定必須記載事項のチェックメモ（各項目が含まれているか確認用）
・署名欄（会社名・代表者名・交付日）
・「受領しました」の受領欄（従業員のサイン欄）
```

## パートタイム・有期雇用の労働条件通知書を整理するプロンプト

パート社員・アルバイト・派遣契約満了後の直接雇用などに対応します。

```
以下の情報をもとに、パートタイム・有期雇用労働者の
労働条件通知書を整理してください。

【採用内容】
雇用形態: （例：パートタイム有期雇用）
契約期間: （例：2026年9月1日〜2027年3月31日（7か月）/ 更新ありの可能性あり）
業務内容: （例：製造ライン作業（食品パッケージング工程））
就業場所: （例：第2工場 食品ライン）
労働時間: （例：9:00〜14:00（休憩なし）/ 1日5時間）
勤務日: （例：月〜金 ただし水曜は除く（週4日・月〜火・木〜金））
時給: （例：¥1,100 / 残業時は1.25倍）
社会保険: （例：雇用保険加入。健康保険・厚生年金は勤務時間の関係で加入なし）

出力形式:
・パートタイム労働法・労働契約法に対応した労働条件通知書形式
・「更新の有無・更新の基準」の欄（有期雇用の場合は必須）
・「正社員との待遇の違いとその理由の説明」の欄（パートタイム労働法対応）
・受領欄（サイン・日付）
```

## 外国人労働者向け労働条件通知書を整理するプロンプト

技能実習生・特定技能・就労ビザを持つ外国人スタッフの雇用書類を整理します。

```
以下の情報をもとに、外国人労働者への労働条件通知書の文章を
整理してください。

【採用者の情報】
国籍・在留資格: （例：ベトナム国籍・技能実習2号）
在留期限: （例：2028年3月31日）
日本語能力: （例：日常会話レベル（N4程度）/ 書類は母国語サポート必要）

【労働条件（日本語版と母国語サポートの両方が必要）】
（日本語版の労働条件は通常と同じ内容）

出力形式:
・日本語版（正式書類）と、ベトナム語版（参考・説明用）の両方
  （ベトナム語版は「翻訳は参考用であり、正式な法的効力は日本語版のみ」の注記を付ける）
・「労働時間外・時間外労働・休日出勤の条件」を特に明確に（認識相違が多い箇所）
・「相談窓口（社内の日本語サポート担当者・電話番号）」の記載欄
```

## 入社書類の案内状を作るプロンプト

内定者・新入社員に渡す入社手続き書類の案内文を作ります。

```
以下の情報をもとに、入社手続き書類の案内状を作成してください。

【入社日】（例：2026年9月1日）
【会社名・担当者】（例：○○製作所 総務部 ○○）

【入社時に必要な書類と持参物】
提出書類:
- 住民票（本籍地入り）1通（発行後3か月以内）
- マイナンバーカードのコピー（または通知カード＋身分証）
- 雇用保険被保険者証（お持ちの方）
- 源泉徴収票（前職がある方・今年分）
- 卒業証明書（新卒の方）
持参物:
- 通帳（給与振込先の口座番号がわかるもの）
- 印鑑（認め印で可）

出力形式:
・A4 1枚の入社手続き案内状
・書類リストをチェックボックス形式で（忘れ物防止）
・「提出先・提出期限（○月○日まで）」を明記
・不明な点の問い合わせ先（担当者名・電話・メールのプレースホルダー）
```
