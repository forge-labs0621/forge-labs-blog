受注生産の製造業（個別受注・多品種少量）では、「この仕事に何時間かかったか」「予定原価と実際原価はどれくらい違ったか」を把握することが利益管理の基本です。AIを使うと、工数メモや生産実績の集計・差異分析コメントの文書化を効率化できます。

## 工数管理・原価管理でAIが役立つ場面

- **工数日報の集計整理**: 複数人・複数工程のメモを集計表に変換
- **受注案件ごとの実際原価集計**: 工数・材料費・外注費のメモを整理
- **予定原価と実際原価の差異分析コメント**: 「なぜ予定より時間がかかったか」の分析文書
- **見積書へのフィードバック**: 実績データをもとに次回見積の改善メモを作る

## 工数日報の集計・整理プロンプト

複数人・複数案件の工数メモから集計表を作ります。

---

```
以下の工数メモを集計し、案件別・工程別の工数一覧表を作成してください。

【工数メモ（日付・氏名・案件番号・工程・時間のメモ）】
（例：
7/3 鈴木 A-101 旋盤加工 3.5h
7/3 田中 A-101 検査 1.0h
7/3 鈴木 B-205 旋盤加工 2.0h
7/3 佐藤 B-205 溶接 4.0h
7/4 鈴木 A-101 旋盤加工 2.0h（完了）
7/4 田中 B-205 検査 1.5h
）

出力形式:
・案件別に「氏名」「工程」「日付別の工数」「小計」を一覧化
・案件ごとの合計工数も集計する
・未完了の案件と完了済みの案件を区別する
（プロンプト内の数値・氏名はサンプルです。実際の値に差し替えてください）
```

## 受注案件の実際原価を集計するプロンプト

材料費・工数（賃率換算）・外注費をまとめて原価集計します。

```
以下の情報をもとに、受注案件の実際原価計算書を作成してください。

【案件情報】
案件番号: （例：A-101）
品名・品番: （例：カバープレート 100個）

【材料費の実績】
（例：
・SPCC鋼板 t1.2mm 　購入金額: ¥12,500（仕入れ値。仕損分含む）
・表面処理（外注）: ¥15,000（100個分）
）

【工数の実績と賃率】
（例：
・機械加工: 5.5時間 × ¥2,500/h = ¥13,750
・組立・梱包: 2.0時間 × ¥1,800/h = ¥3,600
→ 賃率はあくまで目安。実際の賃率は社内の計算値を使用のこと
）

【その他直接費】
（例：治具の摩耗・専用工具購入など、あれば）

出力形式:
・「受注案件実際原価計算書」として
・費目ごとの集計表（材料費/加工費/外注費/その他）
・合計実際原価
・見積原価との対比欄（見積原価を入力すれば差異も表示）
```

## 予定原価と実際原価の差異分析コメントを作るプロンプト

「予定より原価がかかった」理由を分析・記録するのに使えます。

```
以下の情報をもとに、受注案件の原価差異分析コメントを作成してください。

【案件番号・品名】
（例：A-101 カバープレート 100個）

【見積（予定）原価】
材料費: ¥10,000
加工費: ¥14,000（5.6h × ¥2,500）
外注費: ¥13,000
合計: ¥37,000

【実際原価】
材料費: ¥12,500
加工費: ¥17,350（6.94h × ¥2,500）
外注費: ¥15,000
合計: ¥44,850

【超過・不足の理由として思い当たること】
（例：
・材料を多めに使ったのは試作段階で板取りミスが1枚あったため
・加工時間が増えたのはプログラムを途中で修正したため（新しい品番だった）
・外注費が増えたのは急ぎで依頼したため割増になった
）

出力形式:
・差異の金額・率（予定比）
・費目ごとの差異と原因の分析
・「次回見積への反映事項」（何をどう修正すべきか）
・報告者・確認日欄
```

## 案件収益性の簡易チェックプロンプト

受注金額と実際原価から粗利率を計算し、案件の収益性を確認します。

```
以下の情報をもとに、受注案件の収益性を確認してください。

【受注金額】（税抜）: （例：¥62,000）
【実際原価合計】: （例：¥44,850）
【外注先への支払】: （例：¥15,000（上記実際原価に含む））

計算してほしい内容:
1. 粗利額（受注金額 - 実際原価）
2. 粗利率（粗利 ÷ 受注金額 × 100%）
3. 加工費部分の収益性評価（材料費・外注費を除いた自社加工部分の利益）
4. 収益性についての一言コメント（十分か・改善が必要かの目安）

※ 賃率・固定費配分は社内の基準値を使用のこと。
   AIは計算の枠組みと数値の整理を補助します。
```

---

## AIを使う際の注意

- **原価計算の精度は入力値の精度に依存します**: 賃率・間接費配分などは社内で決めた数値を使ってください。AIが賃率を勝手に設定しないよう、プロンプトに具体的な値を入れてください
- **単価・仕入れ値などの機密情報の取り扱いに注意**: 外部AIサービスに送信する場合、機密性の高い価格情報は入力しないか、仮の数値に置き換えてから整形後に実際の値に差し替えてください
- **正式な財務・税務計算にはAIを使わない**: 原価計算のサポートとしての活用を想定しています。正式な決算・税務申告は専門家が確認してください
