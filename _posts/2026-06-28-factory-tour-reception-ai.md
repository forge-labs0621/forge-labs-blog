取引先の品質監査、学校の職場体験、行政機関・金融機関の工場視察など、製造業ではさまざまな来社・工場見学対応が必要です。見学案内資料・当日の流れ・ガイドのトーク内容などをAIで準備できます。

---

## 工場見学の事前案内文・受入れ準備表を作るプロンプト

見学者への事前連絡文と、当日のスケジュール・準備チェックリストを作ります。

```
以下の情報をもとに、工場見学の事前案内文と当日スケジュールを
作成してください。

【見学の概要】
来社日時: （例：2026年7月15日（火）午後2時〜4時）
来訪者: （例：○○大学 工学部 製造システム専攻 学生20名・引率教員2名）
目的: （例：製造業の現場理解を深めるための工場見学（授業の一環））

【自社の紹介内容】
会社名: （例：○○製作所）
主な製品・事業: （例：自動車部品の精密プレス加工・溶接組立）
従業員数: （例：85名）
見学可能なエリア: （例：プレス工程→溶接工程→組立工程→検査工程（順路）））
見学不可エリア・注意事項: （例：試作エリアは守秘義務のため立入不可。
  工場内は安全靴・帽子着用必須。撮影禁止エリアあり）

出力形式:
・来訪者への事前案内メール（日程・集合場所・当日の持ち物・注意事項）
・当日スケジュール表（時間ごとの内容・担当者）
・社内向け準備チェックリスト（会議室準備・安全装備の用意・案内担当の確認等）
```

---

## 工場案内・説明トークの台本を作るプロンプト

ガイドが工場を案内する際の説明文（トーク台本）を作ります。

```
以下の情報をもとに、工場見学ガイドのトーク台本を作成してください。

【対象者・目的】
来訪者の属性: （例：取引先の品質担当者2名・生産技術担当者1名）
見学の目的: （例：サプライヤー評価のための現場確認・品質管理体制の確認）
見学担当: （例：品質保証部長 ○○が案内）

【各工程の紹介内容】
[プレス工程]
・設備台数・規模: （例：プレス機 15台 / 最大800ton）
・主な加工品: （例：自動車ボディ部品 / 板厚0.5〜3.2mm）
・品質管理のポイント: （例：全数外観検査・サンプリング寸法検査、
  不適合品は赤タグで即時識別）

[溶接工程]
・設備・工法: （例：CO2溶接・スポット溶接）
・特殊工程への対応: （例：WPS（溶接施工仕様書）管理・資格保有者のみ担当）
・品質保証: （例：溶接ビード外観検査・定期的な断面検査）

[検査工程]
・検査体制: （例：全数外観検査＋抜取寸法検査・三次元測定機保有）
・出荷判定基準: （例：検査成績書（CoC）を全ロット発行）

出力形式:
・工程ごとの説明文（1〜2分程度で読める長さ）
・「取引先担当者がよく聞くこと」への回答を各工程に添える
・「品質に対する我々のこだわり」を伝える締めのトーク
・ガイドが実際に喋るイメージで自然な口語体で
```

## 取引先品質監査の事前説明資料を作るプロンプト

取引先から品質監査（サプライヤー監査）が来る際の事前準備資料を作ります。

```
以下の情報をもとに、取引先品質監査の受入れ準備資料を作成してください。

【監査の概要】
監査実施日: （例：2026年8月20日（木））
監査者: （例：○○自動車工業 品質保証部 ○○様・○○様（2名））
監査の種類: （例：定期サプライヤー監査（年1回））
重点確認事項（事前に先方から連絡があった場合）:
  （例：4M変更管理の記録 / 不適合品管理の手順 / 最近の不良率データ）

【自社の準備内容】
用意する書類:
  （例：品質マニュアル / 工程FMEA / 管理計画書（コントロールプラン）/
  過去1年の不良率推移データ / 是正措置報告書（直近3件））
確認してもらう現場:
  （例：プレス工程の作業標準書・検査基準書の掲示状況 /
  不適合品の識別・隔離エリア / 測定器の校正記録）

出力形式:
・監査受入れの社内準備リスト（書類準備・現場整備・担当者アサイン）
・監査当日のアジェンダ（時間割・担当者）
・監査前日のリマインドチェックリスト（最終確認事項）
・「監査でよく聞かれる質問と模範回答」のQ&A（10問程度）
```

## 行政・金融機関向けの工場説明資料を作るプロンプト

補助金審査官・銀行担当者・行政機関向けの会社紹介・工場説明資料を作ります。

```
以下の情報をもとに、行政機関（または金融機関）向けの
工場説明・会社概要資料を作成してください。

【来訪者と目的】
来訪者: （例：○○銀行 法人営業部 ○○様・○○様）
目的: （例：設備投資融資のための現地確認・事業内容の理解）
または
来訪者: （例：○○市経済部 産業振興課 ○○様）
目的: （例：地元製造業の実態把握・補助金施策への活用）

【説明したい内容】
会社のアピールポイント:
  （例：設立35年・地域雇用85名 / 自動車1次サプライヤーへの供給実績 /
  ISO 9001認証 / 自動化設備の投資計画中）
現在の課題・将来計画:
  （例：熟練作業者の退職が続き、技能伝承が急務。
  省人化・自動化設備の導入を予定しており、今回の資金調達で実現したい）

出力形式:
・会社概要説明の流れ（5〜10分で説明できる構成）
・「強みと課題・将来計画」をセットで説明する段落
・金融機関が気にする「返済能力・成長性・業界動向」に答える内容
・「本日はお越しいただきありがとうございます」「今後ともよろしくお願いします」の締め
```
