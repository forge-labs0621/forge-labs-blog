MRP（Material Requirements Planning / 資材所要量計画）は、製品の生産計画からBOMを展開して部品・材料の必要量と手配時期を計算する仕組みです。AIを使って所要量の展開ロジック整理・生産計画書類・手配計画の作成を効率化できます。

---

## MRPの所要量展開ロジックを整理するプロンプト

受注・需要計画からMRPの計算方法を整理します。

```
以下の情報をもとに、MRP（資材所要量計画）の
所要量展開の手順と計算ロジックを整理してください。

【生産計画の前提】
製品: （例: 産業用ブラケット PB-2025）
来月の生産計画: 500個

BOM（主要構成部品）:
  ・P-101 本体プレート（1枚/個）: 在庫80枚 / リードタイム7日
  ・P-102 リブ（4枚/個）: 在庫300枚 / リードタイム5日
  ・P-103 ボルトM10×25（4本/個）: 在庫1,200本 / リードタイム3日
  ・P-201 取付板（2枚/個）: 在庫50枚 / リードタイム14日

生産ロットサイズ:
  ・P-101: ロット100枚（100枚単位で発注）
  ・P-102: ロット500枚
  ・P-103: ロット2,000本
  ・P-201: ロット200枚

安全在庫:
  ・P-101: 50枚（長リードタイムのため多め）
  ・P-201: 100枚（長リードタイムのため多め）

出力形式:
・「MRPの所要量展開表（部品ごとに: 総所要量→
  正味所要量（在庫控除）→発注量（ロット調整後）→発注日の計算）」
・「各部品の所要量計算の説明（ステップバイステップ）」
・「部品別の発注計画（何月何日に何個発注するかのタイムライン）」
・「MRPの結果を現場（購買・製造）に伝える書類のフォーマット」
・「MRP計算に必要な5つの入力情報（マスターデータ整備チェック）」
```

## 生産週次・月次計画表を作るプロンプト

週次・月次の生産計画を視覚的に整理します。

```
以下の情報をもとに、製造業の月次・週次生産計画表を
整理してください。

【生産計画の前提】
対象月: 来月（20稼働日）
1日あたりの生産能力:
  ・プレス工程: 300個/日
  ・組立工程: 250個/日（ボトルネック）
  ・検査工程: 400個/日

受注・内示状況:
  品番A: 2,000個（確定受注: 1,200個 + 内示: 800個）
  品番B: 1,500個（確定受注: 800個 + 内示: 700個）
  品番C: 600個（新規急ぎ受注: 顧客要求は月15日まで）

制約:
  ・品番Cは月15日（第3週前半）までに全量出荷必要
  ・月第4週は設備定期点検（1.5日分の能力が減る）
  ・品番A・Bは月末出荷（品番Aを優先）

出力形式:
・「週次生産計画表（週1〜週4の品番別生産数量の割り当て）」
・「日程計画の合理性の確認（各工程の能力内に収まっているか）」
・「品番Cの優先生産スケジュール（月15日納期の実現可能性）」
・「定期点検期間の影響と対策（前後への積み上げ計画）」
・「生産計画の変動があった場合の見直し手順（内示変更・急受注時）」
```

## 資材手配・仕入先指示書を作るプロンプト

MRP計算結果をもとに購買部門への指示書を作ります。

```
以下の情報をもとに、資材手配指示書と
仕入先への発注書の骨子を整理してください。

【手配が必要な資材（MRP展開結果）】
① P-101 本体プレート:
  発注数量: 200枚（在庫80→目標130（安全在庫50）= 正味50枚、ロット100→200枚発注）
  希望納期: 7日後（プレス工程の使用開始日から逆算）
  仕入先: A鋼材（株）

② P-201 取付板:
  発注数量: 400枚（在庫50→目標600（安全在庫100+使用分500）= 正味550→ロット200×2=400枚）
  希望納期: 14日後（注意: リードタイムが長い！）
  仕入先: B鉄板（有）

【発注条件】
  P-101: 電話+FAXで発注（A鋼材は電子発注未対応）
  P-201: B鉄板専用ポータルからネット発注可能

出力形式:
・「購買部門への資材手配指示書のフォーマット
  （品番・品名・数量・希望納期・仕入先・緊急度）」
・「仕入先向け発注書（FAX・メール用）の文面テンプレート」
・「手配の優先順位（リードタイムが長い品目を先行手配する理由）」
・「手配漏れを防ぐためのチェックリスト（手配完了・納期確認の手順）」
・「手配状況の管理表（発注済・納期・入荷予定・完了の4ステータス管理）」
```
