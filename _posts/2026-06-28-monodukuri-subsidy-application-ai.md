ものづくり補助金（中小企業技術革新補助金）の事業計画書は毎回の採択率が40〜50%程度で、書き方で差がつきます。AIを使って、審査で評価されやすい表現と構成を効率化できます。

---

## 「革新性・技術的課題」の記述を強化するプロンプト

審査で最も重要視される「革新性」の文章を整理します。

```
以下の情報をもとに、ものづくり補助金の事業計画書における
「革新的サービス・試作品開発の内容」欄の文章を作成してください。

【自社の取り組みの概要】
（例:
  現在の課題: 手作業で行っているプレス金型の段取り交換が
    平均65分かかり、段取り替えによる機会損失が年間○○時間発生している
  
  取り組む内容: AIビジョンカメラと自社開発のアルゴリズムを組み合わせた
    「段取り替え自動検知システム」を試作する
  
  革新性: これまで段取り替えの開始・完了を作業者の申告で管理していたが、
    カメラ画像から自動判定することで、人が管理しなくても工程データが
    リアルタイムで可視化される
  
  目標: 段取り時間の30%短縮（65分→45分）、工程の見える化による
    改善活動の定量化
）

【技術的な差別化ポイント】
（例:
  既存システムとの違い: 一般的な工程管理システムは人が入力する前提だが、
    本システムはカメラが自動検知するため、入力漏れ・遅延がゼロになる
  
  自社の強み: 20年間の金型製造の経験から段取り替え工程のパターンを
    熟知しており、そのドメイン知識をAI学習データに反映できる
）

出力形式:
・「革新性・技術的課題」欄の文章（400〜600字、常体）
・「既存技術・サービスと比べた優位性」の表現（具体的な差分を強調）
・「技術的課題と解決方法」の構成（問題→要因→解決策→達成目標）
・「審査員に刺さりやすいキーワード」と使い方のアドバイス
```

## 「事業化計画・収益計画」の数字を整理するプロンプト

審査員が最も見る収益性・実現可能性の数字を整理します。

```
以下の情報をもとに、ものづくり補助金の「事業化計画」に記載する
収益計画・実施体制を整理してください。

【補助対象の取り組みによる効果（数字ベース）】
投資内容:
  ・AIシステム開発費: ○百万円（補助申請額）
  ・試作機器・センサー購入: ○百万円
  合計: ○百万円（うち補助対象: ○百万円 × 補助率1/2 = 補助額○百万円）

導入効果（数量化）:
  ・段取り時間短縮: 月○時間 × 時間単価¥○ = 月○万円の労務費削減
  ・稼働率向上による売上増: 月○件の受注増 × 単価○万円 = 月○万円の増収
  ・年間合計効果: ○百万円/年

投資回収:
  ・回収期間: ○年

市場背景（なぜ今か）:
  ・人手不足・高齢化でノウハウ継承が困難になってきており、
    デジタル化による属人性排除が急務
  ・競合他社もDX投資を加速しており、遅れると競争力を失うリスク

出力形式:
・事業化計画の文章骨子（投資内容・効果・市場背景の論理構成）
・「収益計画の説明文」（3〜5年の数字を根拠付きで示す書き方）
・「実施スケジュール（ガントチャート的な表）」のフォーマット
・「審査で問われやすい質問」と回答例（加点ポイントの観点から）
・「事業計画書全体のチェックポイント（採択率を上げるために見直す点）」
```

## 補助金申請の「自社の強み・経営課題」を整理するプロンプト

事業計画書の冒頭で差をつける「自社の現状分析」を整理します。

```
以下の情報をもとに、ものづくり補助金申請書の
「自社の強み・経営課題」の文章を作成してください。

【自社の基本情報】
業種: （例: 金属プレス加工業 / 設立○年 / 従業員○名）
主要製品: （例: 自動車部品・精密部品のプレス加工 / 月産○万個）
主な顧客: （例: 自動車Tier1・Tier2 / ○社との取引）
売上規模: （例: 年商○百万円 / うち○○客先が○%を占める）

【自社の強み（他社にない点）】
（例:
  ・プレス金型の設計〜製作〜量産まで一貫対応できる
  ・50年以上の歴史で培った職人の技術（薄板0.2mm精密プレスの実績）
  ・ISO 9001認証・IATF 16949認証取得済み
  ・顧客の設計段階からVE・VA提案を行い採用実績○件
）

【自社の経営課題（今回の取り組みで解決する課題）】
（例:
  ・熟練者の定年退職が続き、暗黙知の技術伝承が困難になっている
  ・人手不足で段取り替え工程の属人化が進み、品質がバラつく
  ・顧客からコスト削減要求が続き、利益率が年々低下している
）

出力形式:
・「自社の強み」の文章（箇条書き3〜5点 + まとめ文100字）
・「経営課題と取り組みの必然性」の文章（課題→なぜ今→この取り組みを
  選んだ理由 という論理展開）
・「SWOT分析の簡易版」（強み・弱み・機会・脅威の整理）
・申請書の各項目で「審査員が加点しやすい表現のコツ」5点
```

## ものづくり補助金の採択事例を参考に表現を整理するプロンプト

採択された事業計画書の特徴を踏まえた表現改善を行います。

```
以下の事業計画書のドラフト文章を改善してください。

【改善前の文章（例）】
「当社は今回、新しい機械を導入します。この機械により作業時間が
短縮できます。また、品質も向上します。売上は3年後に○○万円に
なる予定です。」

【改善のポイント（採択事例から学んだこと）】
・「革新性」を数字・比較・独自性で具体化する
・課題→解決→効果の因果関係を明確にする
・市場環境（外部環境変化）と自社の必要性をつなげる
・「事業者の意欲・覚悟」が伝わる表現にする
・「なぜ補助金が必要か（自己資金では実施できない理由）」を示す

出力形式:
・「改善後の文章」（400〜800字 / 具体的な数字・比較を入れた形）
・「改善のポイント解説」（どこをどう変えたか・なぜその表現が良いか）
・「審査で避けるべき表現・NG例」5点
・「ものづくり補助金で毎回指摘される弱い申請書の特徴」3点
```

---

### 申請の注意点

ものづくり補助金の申請は毎回の公募要領が変わるため、AIが生成した文章は**最新の公募要領・審査基準を確認の上で修正**してください。また、AIが生成した文章には事実と異なる数字が入る可能性があるため、自社の実績データと必ず照合してください。
