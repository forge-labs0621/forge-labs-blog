製造業では、完成品を出荷する前の「出荷検査」と、仕入先から届いた材料・部品の「受入検査（購入品検査）」が品質管理の重要な工程です。検査結果を記録として残すことはISOの要求でもありますが、毎回の記録作成が手間になりがちです。AIを使うと、検査結果のメモから整形された検査記録・成績書を素早く作れます。

## 出荷検査記録とは

完成品を顧客に出荷する前に、品質・数量・梱包を最終確認する記録です。

**出荷検査記録が必要な場面:**
- 製品ごとに検査成績書を添付する必要がある場合（自動車・医療・精密機器関連）
- 顧客から検査記録の提出を求められている場合
- ISO 9001で「プロセスのアウトプットの検証」として記録を保管する場合

## 出荷検査記録・検査成績書をAIで作るプロンプト

---

```
以下の情報をもとに、出荷検査記録（検査成績書）を作成してください。

【製品情報】
品名・品番: （例：カバープレート A-101）
ロット番号・製造番号: （例：L2607-001）
数量: （例：100個）
出荷先: （プレースホルダー可）
出荷予定日: （例：2026年7月10日）

【検査項目と結果】
（例：
1. 外観検査（全数）: キズ・打痕・錆なし → 合格 100/100個
2. 寸法検査（抜取：n=5）:
   外径φ25.0±0.05 → 測定値: 25.01, 25.00, 24.99, 25.02, 25.01 → 合格
   長さ 50.0±0.1   → 測定値: 50.0, 50.0, 50.1, 50.0, 49.9 → 合格
3. 表面粗さ（抜取：n=3）: Ra1.6以下 → Ra0.9, 1.1, 1.0 → 合格
4. 梱包確認: 保護材あり・品番ラベル正確 → 合格
）

【特記事項】
（例：なし / または、外観にごく軽微な加工目が2個あったが規格内と判定）

出力形式:
・「出荷検査成績書」として
・製品情報・ロット情報・検査日・検査者
・検査項目ごとに「検査内容」「判定基準」「測定値・結果」「合否」を表形式で
・総合合否判定（合格/不合格）
・検査者・承認者署名欄（プレースホルダー）
```

## 受入検査記録（購入品検査記録）を作るプロンプト

外注先・仕入先から納品された材料・部品を検査した際の記録も作れます。

```
以下の情報をもとに、受入検査記録を作成してください。

【購入品情報】
品名・品番: （例：旋盤加工品 B-205）
仕入先・外注先名: （プレースホルダー可）
納品日: （例：2026年7月5日）
数量: （例：50個）
納品書No.: （例：D-2607-015）

【検査内容と結果】
（例：
1. 数量確認: 50個（発注数量と一致）→ 合格
2. 外観検査（全数目視）: キズ・バリなし → 合格 50/50個
3. 寸法確認（抜取 n=5）:
   外径φ20.0±0.05 → 19.99, 20.01, 20.00, 20.02, 20.00 → 合格
   全長 80.0±0.1   → 79.9, 80.0, 80.1, 80.0, 80.0    → 合格
4. 検査成績書の添付: あり（仕入先発行）→ 確認済み
）

【判定結果】
（例：全項目合格 → 受入可 / または 一部不合格 → 保留・返品・特採）

出力形式:
・「受入検査記録」として
・購入品情報・仕入先・納品日
・検査項目ごとに表形式で
・受入判定（合格/不合格/保留）
・不合格の場合は「処置内容」欄（返品・代品要求・特採など）
・検査者・承認者欄（プレースホルダー）
```

## 検査不合格・保留品の処置記録を作るプロンプト

受入検査で不合格になった場合の処置も記録しておく必要があります。

```
以下の受入検査不合格品の処置について、処置記録を作成してください。

【不合格品の情報】
品名・品番・仕入先・納品日・数量
（例：旋盤加工品 B-205 / ○○加工様 / 7月5日納品 / 50個中3個不合格）

【不合格の内容】
（例：外径φ20.0±0.05に対して、20.08〜20.11mmと規格上限を超えていた（3個）。
      原因は外注先の工具摩耗と推定。外注先に連絡済み）

【処置の内容】
（例：
・不合格3個は返品扱いとし、外注先に代品を緊急手配依頼した（7/7納期）
・残り47個は合格品として受入
・外注先に原因分析と再発防止策の提出を依頼した（7/12提出期限）
）

出力形式:
・不合格内容・数量・処置方法・対応期日
・担当者・承認者・処置完了日
・仕入先評価への反映有無（仕入先評価記録と連動させる場合）
```

## 抜取検査の計画を整理するプロンプト

「何個中何個を検査すればよいか」という抜取数の基準を整理する際に使えます。

```
以下の条件で、製品検査の抜取計画を整理してください。

【ロットサイズ（1回の納品・製造数量）】
（例：50個 / 100個 / 500個）

【製品のリスクレベル】
□ 高（安全に関わる部品、顧客からの要求が厳しい）
□ 中（一般的な品質要求）
□ 低（内製部品で工程内品質が安定している）

【現在の不良率の実績（わかる範囲で）】
（例：過去6ヶ月の平均不良率 約1%）

参考にしてほしい考え方:
・JIS Z 9015（計数値サンプリング検査）の考え方で整理する
・抜取検査で「全数検査と同等のリスク管理」はできないことも説明してほしい

出力形式:
・推奨する抜取数の目安（ロットサイズ別）
・全数検査が必要なケースの目安
・抜取検査の限界（見落としリスク）についての一言
```

## AIを使う際の注意

- 検査成績書は正式な品質文書です。AIが整形した内容は必ず検査実施者が確認・修正してください
- 測定値はAIに計算させることができますが、**実際に測定した数値が根拠**です。数値の入力間違いに注意してください
- 顧客提出用の検査成績書には、必ず社内で定めた書式・承認フローを経てから送付してください
