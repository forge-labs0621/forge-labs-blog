電気機器・機械製品を国内外で販売するには、各種安全規格の認証（CE・PSEマーク・UL認証等）が必要です。AIを使って認証試験の申請書類整理・試験報告書の解釈・認証維持管理を効率化できます。

---

## 製品認証の取得計画を整理するプロンプト

どの認証が必要か・どの順番で取得するかを整理します。

```
以下の製品情報をもとに、必要な製品認証の
一覧と取得計画を整理してください。

【製品の概要】
製品名: （例: 産業用電動アクチュエータ）
用途: 製造設備の自動化ラインに組み込む駆動ユニット
主要市場: 国内（日本）+ EU（ドイツ・フランス）+ 北米（アメリカ）
電源: AC100〜240V 50/60Hz 単相
出力: 0.4kW

【想定される認証（確認が必要）】
国内販売向け:
  ・PSEマーク（電気用品安全法）
  ・産業機械の安全規格（JIS B 9700系 機械類の安全性）

EU販売向け:
  ・CEマーキング（機械指令・低電圧指令・EMC指令）

北米販売向け:
  ・UL/CSA認証（北米電気安全規格）

出力形式:
・「必要な認証一覧（市場別・認証名・根拠法令/指令・
  適用規格・認証機関の例）」
・「認証取得の優先順位と順序の推奨理由
  （並行取得可能なものの整理）」
・「各認証の概算費用・取得期間（一般的な情報として）」
・「第三者試験機関への初回問い合わせ時に必要な情報リスト」
・「認証取得プロジェクトのスケジュール骨子（製品完成→試験申請→
  認証取得→市場投入までの流れ）」
```

## CE認証の技術文書（Technical File）を整理するプロンプト

CEマーキングに必要なDoC・技術文書を整理します。

```
以下の情報をもとに、CEマーキングのための
技術文書（Technical File）と適合宣言書（DoC）の
骨子を整理してください。

【製品情報】
製品名: Electric Actuator Model EA-200
適用指令:
  ・機械指令（Machinery Directive 2006/42/EC）
  ・低電圧指令（Low Voltage Directive 2014/35/EU）
  ・EMC指令（2014/30/EU）

適用規格（例）:
  ・EN ISO 12100:2010（機械安全の基本原則）
  ・EN 60335-1（電気機器の安全性）
  ・EN 61000系（EMC規格）

試験結果:
  ・電気安全試験: 合格（認定試験機関 TUV Rheinland にて実施）
  ・EMC試験（放射エミッション・イミュニティ）: 合格
  ・機能安全評価: 実施完了

製造者情報:
  会社名: Forge Labs Manufacturing Co., Ltd.
  住所: ○○県○○市...（日本）
  EU代理人（EU Representative）: 未定（取得前に選定が必要）

出力形式:
・「適合宣言書（DoC / Declaration of Conformity）のフォーマット
  （EC指令番号・規格番号・製造者情報・署名欄）」
・「技術文書（Technical File）に含めるべき書類一覧」
・「EU代理人の必要性と選定方法（EU内に拠点がない場合の対応）」
・「CEマーキングの表示要件（マークのサイズ・位置の規定）」
・「技術文書の保管義務（10年間）の管理方法」
```

## 試験報告書の読み方・社内管理を整理するプロンプト

第三者試験機関からの試験報告書を理解・管理します。

```
以下の情報をもとに、製品試験報告書の
読み方と社内管理方法を整理してください。

【受け取った試験報告書の概要（例）】
試験機関: （例: TÜV SÜD Japan）
試験報告書番号: 2026-XXXX-001
試験日: ○年○月○日
テストサンプル: EA-200 シリアル番号: SN001〜003

試験結果サマリー（報告書記載）:
  項目1: Withstand Voltage Test（耐電圧試験）→ PASS
  項目2: Insulation Resistance Test（絶縁抵抗試験）→ PASS
  項目3: Temperature Rise Test（温度上昇試験）→ PASS
  項目4: EMC放射エミッション試験 → PASS（限界値から3dBのマージンあり）
  項目5: サージ耐性試験 → CONDITIONAL PASS
    （条件: 外部サージ保護デバイスの追加推奨）

出力形式:
・「試験報告書の主要項目の読み方と用語解説」
・「CONDITIONAL PASSへの対応方針（製品設計変更の要否判断）」
・「試験報告書の社内管理フォーマット（版数・試験日・
  結果・有効期限・更新トリガーの一覧）」
・「製品変更（設計変更・材料変更）時に再試験が必要かどうかの
  判断基準」
・「試験機関との継続的な関係管理のポイント
  （コスト削減・スケジュール調整の交渉方法）」
```
