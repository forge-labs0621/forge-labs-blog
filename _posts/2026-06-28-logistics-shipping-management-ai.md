製造業の出荷・物流業務では、輸配送計画・運送業者との契約管理・輸送コスト分析・輸送事故への対応など多くの書類が必要です。AIを使って物流管理書類の整備と報告書の文書化を効率化できます。

---

## 輸配送計画・出荷業務の管理書類を作るプロンプト

月次・週次の出荷計画と輸配送の管理書類を整理します。

```
以下の情報をもとに、製造業の輸配送計画と
出荷管理書類を作成してください。

【自社の出荷状況】
出荷先: （例: 国内Tier1メーカー3社・国内販売店10社）
出荷頻度: （例: 主力顧客へは週3〜5回定期便 / 小ロット顧客は随時）
使用する運送会社: （例: 契約運送業者A（チャーター便） / 宅配業者B（小口）
  / 混載業者C（遠方向け））
出荷形態: 直送 / 倉庫経由 / 顧客指定倉庫への入庫

現在の課題:
  ・月末の出荷集中（全月間出荷の60%が月末最終週に集中）
  ・出荷ミス（誤品・数量違い・送り先間違い）が月に○件程度
  ・輸送コストの全体像が把握できていない（運賃の可視化がない）

出力形式:
・「週次出荷計画フォーマット（客先別・品番別・数量・出荷日・
  運送会社・特記事項）」
・「出荷指示書（ピッキングリスト）の標準フォーマット」
・「出荷前最終確認チェックリスト（誤品・数量・梱包・送り状）」
・「月末出荷集中を緩和するための工夫（顧客交渉・前倒し生産等）」
  の施策アイデア
・「輸送コスト月次集計表（客先別・輸送手段別）」のフォーマット
```

## 運送業者の管理・輸送コスト交渉書類を作るプロンプト

複数の運送業者を管理し、コスト最適化を検討します。

```
以下の情報をもとに、運送業者管理・輸送コスト
最適化の書類を作成してください。

【運送業者の管理状況】
業者A（定期チャーター）:
  ・月間利用料: ○万円 / 月間輸送量: ○トン・○件
  ・運賃変更の通知: 年1回（燃油サーチャージ別途変動）
  ・実績: 遅延発生率 ○%・破損事故: 過去1年で○件

業者B（宅配・小口）:
  ・月間利用料: ○万円（サイズ別単価精算）
  ・課題: 大型荷物の破損・紛失が散発している

業者C（混載・長距離）:
  ・月間利用料: ○万円
  ・リードタイム: 2〜3日（翌日着不可）

課題: 運送費用全体が年○百万円で、見直す余地があるか判断したい

出力形式:
・「運送業者評価シート（コスト・信頼性・サービス・対応力）」
・「輸送コスト削減のアプローチ」（複数業者の相見積もり・共同配送・
  梱包の軽量化・出荷頻度の見直し 等）
・「運送業者との交渉資料の骨子（ボリュームディスカウント・
  長期契約によるコストダウン交渉）」
・「物流KPIの設定（遅延率・破損率・配送コスト/売上比）」
・「運送業者との秘密保持・情報管理に関する確認事項」
```

## 輸送事故・貨物破損時の対応書類を作るプロンプト

輸送中の製品破損・誤配・遅延への対応書類を整理します。

```
以下の状況で、輸送事故・貨物破損への対応書類を
作成してください。

【発生した輸送事故の概要（例）】
状況: 顧客Aへの出荷品（製品○個・梱包重量○kg）が配送中に破損。
  顧客から「製品の一部が凹んでいる・梱包も壊れていた」と連絡が来た。
  配送業者（業者A）が荷物を落下させた可能性が高い。

影響:
  ・顧客の生産ラインが一時停止（2時間）
  ・代替品を緊急手配（特急対応・○万円の費用が発生）
  ・顧客への謝罪訪問が必要

出力形式:
・「顧客への第一報メール（謝罪・対応内容・代替品手配の連絡）」
・「運送業者への事故報告・損害賠償請求の手順」
・「事故品の写真撮影・証拠保全のチェックリスト」
・「輸送事故報告書（社内管理用: 事故内容・原因・対応・再発防止）」
・「梱包方法・輸送方法の見直し提案フォーマット」
  （同様の事故を防ぐための改善策）
```
