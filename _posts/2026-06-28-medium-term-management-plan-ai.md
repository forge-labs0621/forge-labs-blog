中小製造業でも、金融機関からの融資審査・ものづくり補助金の申請・社内の方向性統一のために、3カ年の中期経営計画書（事業計画書）を作成する機会があります。数字の根拠づくりと計画の文章化をAIで補助できます。

---

## 中期経営計画の骨子を作るプロンプト

3カ年中期経営計画の骨子（目次と概要）を作ります。

```
以下の情報をもとに、製造業の3カ年中期経営計画書の
骨子（章立てと各章のポイント）を作成してください。

【自社の概要・現状】
業種・規模: （例：受託プレス加工業 / 従業員65名 / 売上高 ¥8億円）
主要取引先: （例：自動車部品メーカー3社が売上の75%を占める）
現在の課題:
  （例：
  ① 取引先の集中リスク（3社依存）→ 新規開拓が課題
  ② ベテラン技術者5名が3年以内に退職予定 → 技能伝承が急務
  ③ プレス設備の老朽化 → 計画的な更新が必要
  ④ 価格競争が激化 → 高付加価値品への移行が課題
  ）

【3カ年の大方針（あれば）】
（例：「自動車部品依存からの脱却と、医療機器・電機業界への横展開。
  技能のデジタル化（マニュアル化・自動化）で生産性を2割向上させる。」）

出力形式:
・中期経営計画書の章立て（目次案）
・各章で書くべき内容のポイント（箇条書き）
・「経営ビジョン・ミッション」「現状分析（SWOT）」「3年間の数値目標」
  「重点施策」「投資計画・資金計画」の構成を含める
・3年間の売上・利益・投資のロードマップ（数値欄はプレースホルダー）
```

## SWOT分析を整理するプロンプト

自社の強み・弱み・機会・脅威を整理し、経営計画に活かします。

```
以下の情報をもとに、製造業のSWOT分析を整理し、
中期経営計画への示唆をまとめてください。

【自社の情報】
強み（Strengths）:
  （例：精密加工技術の蓄積（公差±0.01mm対応可）/
  ISO 9001認証取得 / 自動車部品での実績35年 /
  顧客密着型の対応力（短納期・小ロット対応））

弱み（Weaknesses）:
  （例：若手人材の育成が追いついていない /
  受注の自動車依存（75%） /
  老朽化した設備が多く、維持コストが増加 /
  デジタル化が遅れており、データ活用ができていない）

機会（Opportunities）:
  （例：医療機器・航空部品分野の国内回帰トレンド /
  EV化対応で新たな部品ニーズが生まれている /
  DX補助金・省エネ補助金を活用した設備更新が可能 /
  競合の廃業・縮小により受注拡大の余地がある）

脅威（Threats）:
  （例：材料費・エネルギーコストの高止まり /
  自動車の電動化（EV化）で既存部品の需要が長期的に減少 /
  海外製品との価格競争 /
  求人難で必要な人材確保が困難）

出力形式:
・SWOT分析の整理（表形式）
・「強みを活かして機会を取る」戦略（SO戦略）
・「弱みを克服して機会を取る」戦略（WO戦略）
・「強みで脅威を回避する」戦略（ST戦略）
・「最優先で対処すべき課題」のトップ3と理由
```

## 金融機関向け事業計画書の要約を作るプロンプト

銀行・信金への融資申請時に提出する事業計画書の要約を作ります。

```
以下の情報をもとに、金融機関（銀行・信用金庫）への
融資申請用の事業計画書（概要版）を作成してください。

【融資の目的・背景】
（例：老朽化した主力プレス機（25年稼働）の更新。
 新型設備の導入により加工精度が向上し、医療機器分野への
 参入を実現できる。投資総額 ¥5,000万円。
 うち補助金 ¥2,000万円申請中 / 自己資金 ¥1,000万円 /
 借入申請 ¥2,000万円（返済期間 7年）。）

【事業の現状と将来見通し】
売上推移: （例：2024年度 ¥7.8億 → 2025年度 ¥8.0億 → 2026年度 ¥8.2億（見込み））
主な取引先と安定性:
  （例：○○自動車工業（30年取引・基本発注量保証）/
  ○○精機（近年取引開始・成長中））
新規事業（設備更新後の計画）:
  （例：2027年度から医療機器部品の試作受注開始。
  2028年度に ¥5,000万円/年の医療部品売上を目指す。）
返済の原資:
  （例：現状の安定売上から年間 ¥3,000万円以上のキャッシュフロー。
  新規事業が加わることで返済余力はさらに向上する見込み。）

出力形式:
・1〜2ページ程度の事業計画書概要（金融機関の審査担当者が読む想定）
・「現状の事業の安定性」と「新規投資による成長性」をバランスよく記載
・キャッシュフロー計算・返済シミュレーションのプレースホルダー
```
