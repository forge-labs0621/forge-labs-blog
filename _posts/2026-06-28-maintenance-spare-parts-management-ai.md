工場設備のトラブル時に必要な保全部品（スペアパーツ）が手元にないと、生産停止が長引きます。AIを使って保全部品の在庫管理・発注ルール・TPM（全員参加の生産保全）の書類整備を効率化できます。

---

## 保全部品（スペアパーツ）台帳を整理するプロンプト

設備ごとの必要部品を体系的に管理します。

```
以下の情報をもとに、製造設備の保全部品管理台帳の
フォーマットと在庫管理ルールを整理してください。

【管理対象の設備（例）】
設備①: CNC旋盤 型番: OKUMA LB-3000
設備②: 油圧プレス 型番: 型番○○

【設備①のよく交換する消耗品・保全部品】
  ・チャックジョー（3爪） → 年2回交換 / 単価 ¥15,000/セット / 調達リードタイム 1週間
  ・主軸ベアリング（前側） → 3年ごと交換 / 単価 ¥45,000 / 調達LT 2週間
  ・油圧ユニットフィルター → 3ヶ月ごと交換 / 単価 ¥3,500 / 調達LT 3日
  ・切削油冷却ポンプ（Oリング） → 年1回予防交換 / 単価 ¥800 / 調達LT 1日

【現在の課題】
  ・ベアリングが急に壊れた際に在庫がなく、3週間生産停止になった経験がある
  ・消耗品の在庫がどこにいくつあるか把握できていない
  ・担当者が変わると部品の場所・型番がわからなくなる

出力形式:
・「保全部品管理台帳のフォーマット（設備名・部品名・型番・
  メーカー・在庫数・発注点・標準在庫数・保管場所・
  単価・調達先・調達LT）」
・「発注点の設定方法（使用頻度・調達LT・リスク評価から算出）」
・「重要度分類（A: 欠品で即停止 / B: 数日で停止 / C: 停止しない）
  による在庫方針の差別化」
・「保全部品の受払記録（入出庫管理）のフォーマット」
・「担当者が変わっても維持できる引き継ぎ書類の整備方針」
```

## 設備保全コスト管理・予算計画を整理するプロンプト

設備維持にかかるコストを把握し、予算管理します。

```
以下の情報をもとに、製造設備の保全コスト管理と
年次予算計画を整理してください。

【対象設備の一覧と保全コスト実績（昨年度）】
設備①CNC旋盤:
  ・定期点検（外注）: ¥120,000/回 × 2回 = ¥240,000
  ・消耗品交換: ¥180,000（実績）
  ・突発修理: ¥350,000（油圧ポンプ故障）

設備②油圧プレス:
  ・消耗品（金型・Oリング等）: ¥250,000
  ・突発修理: ゼロ

フォークリフト（2台）:
  ・法定点検（年2回）: ¥28,000/台 × 2台 × 2回 = ¥112,000
  ・タイヤ交換: ¥60,000

合計保全コスト実績: ¥1,192,000/年

【今年度の重点課題】
  ・旋盤の突発修理を減らす（予防保全の強化）
  ・旋盤主軸ベアリングの交換が今年度必要（¥80,000）

出力形式:
・「設備保全コスト管理表（設備別・種別（予防保全/事後保全/
  法定点検）・月次金額・年間合計）」
・「今年度の保全予算計画（実績ベースの積み上げ+今期追加事項）」
・「設備別の保全費用対効果分析（投資した保全費用 vs 
  停止損失の削減額）」
・「突発修理を減らすための予防保全強化計画」
・「設備の老朽化評価と更新タイミングの判断フレームワーク」
```

## TPM活動書類を整理するプロンプト

全員参加の生産保全（TPM）の活動記録を整理します。

```
以下の情報をもとに、TPM（全員参加の生産保全）の
活動書類を整理してください。

【TPMの取り組み状況】
活動レベル: 自主保全ステップ1〜2（清掃・給油・増し締めを
           オペレーターが自分で行う）

【自主保全チェックリストに含める主な活動】
清掃活動:
  ・設備外部の清掃（油汚れ・切粉の除去）: 毎日
  ・設備内部（カバー内）の清掃: 週1回
  ・フィルター清掃: 月1回

点検・確認:
  ・油圧ユニットのオイルレベル確認: 毎日
  ・冷却水量確認: 毎日
  ・ベルトの張り確認: 週1回
  ・各部ボルトの増し締め確認: 月1回

給油:
  ・主軸ベアリングへのグリスアップ: 月1回
  ・各スライド部への給油: 週1回

【現在の問題】
  ・清掃や点検の記録が紙でバラバラ
  ・異常を発見しても誰に報告すればいいかわからない
  ・「清掃するだけ」になって、設備の状態変化に気づけていない

出力形式:
・「TPM自主保全チェックリスト（日次/週次/月次に分けた
  設備別チェック表）」
・「チェック時の異常発見・報告フロー（誰が・いつ・
  どう報告するか）」
・「「清掃は点検なり」の考え方を現場に周知するための
  1枚説明資料の骨子」
・「TPM活動の見える化（活動記録の集計と改善効果の共有方法）」
・「TPM活動の評価方法（MTBF・MTTR等の保全指標の説明と記録方法）」
```
