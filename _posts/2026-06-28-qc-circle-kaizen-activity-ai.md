製造業の小集団改善活動（QCサークル）では、テーマ選定・現状把握・要因分析・対策実施・効果確認のサイクルを回し、発表会で成果を報告します。AIでこの流れの文書化と発表資料の整理を効率化できます。

---

## QCサークルのテーマ選定と問題の絞り込みプロンプト

複数の問題候補からQC活動テーマを選ぶ際の整理を行います。

```
以下の情報をもとに、QCサークルの活動テーマを選定し、
問題の重要度・優先順位を整理してください。

【対象のサークル・部門】
（例：製造部 プレスライン 班長○○リーダー / サークル名「○○サークル」
  メンバー5名）

【改善候補として挙がっている問題】
（例：
A. プレス品の外観不良（バリ・打痕）が多い: 月平均250個 / 不良率 0.5%
B. 段取り交換時間が長い: 平均65分 / 目標40分
C. 材料の端材廃棄量が多い: 月間○kg / 廃棄コスト ¥○万円/月
D. 残業が多い: 月平均25時間/人 / 残業代 ¥○万円/月
）

【テーマ選定の基準】
・重要度（安全・品質・コスト・納期への影響）
・取り組みやすさ（3か月程度で効果が出るか）
・データが取れるか（定量的な効果測定ができるか）

出力形式:
・各問題候補の「評価マトリクス」（重要度・取り組みやすさ・データ取得しやすさ）
・推奨テーマとその理由
・選定されたテーマの「テーマ設定理由書（発表会で使える文章）」
・「活動目標（数値目標）」の設定例
```

## 特性要因図（魚の骨図）の整理プロンプト

問題の原因を5M1E（人・機械・材料・方法・測定・環境）で整理します。

```
以下の問題について、特性要因図（魚の骨図）の
内容を整理してください。

【問題（結果）】
（例：プレス品の外観不良（バリ）が月250個発生している）

【現場でのヒアリング結果（気づいている原因の候補）】
人（Man）:
  ・初期流動確認の実施が不十分（担当者によってバラつき）
  ・検査員が変わると判断基準が違う

機械（Machine）:
  ・金型の摩耗状態の確認が不定期
  ・プレス機の打ち込み圧力の点検が月1回のみ

材料（Material）:
  ・材料の板厚ばらつきが一部ロットで大きかった
  ・同じSPCCでもメーカーによって硬さが違う気がする

方法（Method）:
  ・金型交換の際の芯出し手順が標準化されていない
  ・バリ取りの品質基準（どこまで許容するか）があいまい

測定（Measurement）:
  ・バリの高さを数値で測る方法がなく、目視判断のみ

環境（Environment）:
  ・夏は暑くて集中力が落ちる

出力形式:
・特性要因図の整理（5M1E別に要因を箇条書き）
・「根本原因として最も可能性が高い要因」の推定と根拠
・「要因ごとの検証方法（データで確認するか・現場観察か）」の提案
・発表会向けの「要因分析ページ」の文章（読み上げ原稿）
```

## QCサークル発表報告書を作るプロンプト

活動の成果をまとめた発表報告書の文章を整理します。

```
以下のQCサークル活動のデータをもとに、
発表報告書の各章の文章を作成してください。

【活動の概要】
テーマ: （例：プレス品のバリ不良を半減させよう）
活動期間: （例：2026年4月〜2026年9月（6か月））
サークル名・メンバー: （例：○○サークル（5名））

【現状把握】
活動前の状況:
  ・月間不良数: 250個 / 不良率: 0.5%
  ・不良の内訳: バリ60%・打痕30%・その他10%
  ・パレート図からバリが主問題と特定

【原因分析の結果】
主原因: 金型摩耗の管理基準がなく、摩耗が進んでいても
  使い続けていた（打数管理なし）

【対策の内容】
1. 金型の「打数管理台帳」を新設（管理担当を品質係に指定）
2. 金型ごとに「交換目安打数」を実績データから設定
3. 初物確認の基準を明確化（バリ高さの限度見本を作成）

【効果確認】
対策後6か月の結果:
  ・月間不良数: 110個（▲56%削減） / 不良率: 0.22%
  ・目標: 125個（▲50%）以下 → 目標達成

出力形式:
・QCサークル発表報告書の各章の文章（発表スライドの読み上げ原稿）
・「活動のまとめ」（一番伝えたいこと・苦労したこと・次のテーマ）
・「数値成果の要約文」（一文で効果を表現）
・「発表会でのQ&A想定質問と模範回答」3〜5問
```
