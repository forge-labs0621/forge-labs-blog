製造業では生産変動に対応するため、派遣労働者・業務請負会社の活用が多いです。労働者派遣法・偽装請負の問題は法令違反リスクが高く、管理書類の整備が重要です。AIを使って書類の整備と点検を効率化できます。

---

## 派遣先管理台帳のフォーマットを作るプロンプト

労働者派遣法で義務付けられた「派遣先管理台帳」を整理します。

```
以下の情報をもとに、派遣先管理台帳と
派遣受入れ管理書類を作成してください。

【派遣受入れの状況】
派遣元: （例: ○○㈱（一般労働者派遣事業許可: ○○号））
派遣期間: （例: 2026年○月○日〜2026年○月○日（3か月更新））
派遣人数: ○名
配属先: 製造部 プレスライン
業務内容: プレス加工品の検査・梱包作業
派遣先責任者: 製造部長 ○○
指揮命令者: 班長 ○○

【労働者派遣法の主な義務（派遣先）】
① 派遣先管理台帳の作成・保存（3年間）
② 派遣元への通知（就業状況・苦情処理等）
③ 派遣受入期間の制限（原則3年・事業所単位・個人単位）
④ 派遣先責任者・苦情処理担当者の選任

出力形式:
・「派遣先管理台帳のフォーマット」（法令上の記載項目を全て含む）
・「派遣受入期間の計算・管理方法」（3年ルールの確認シート）
・「派遣社員の就業開始時・終了時の手続きチェックリスト」
・「派遣元への定期報告フォーマット（就業日数・苦情・安全衛生等）」
・「派遣元との間で締結する派遣契約書で確認すべき項目一覧」
```

## 偽装請負リスクの確認書類を作るプロンプト

業務請負と派遣の境界線を確認し、偽装請負リスクを排除します。

```
以下の情報をもとに、業務請負における
偽装請負リスクの確認と是正書類を作成してください。

【自社の業務請負の実態（確認したい点）】
請負業者との契約内容:
  ・業務範囲: （例: プレス加工後の製品検査・梱包・出荷準備）
  ・作業場所: 自社工場内（自社設備を使用）
  ・指揮命令: 「請負」として契約しているが、自社の班長が
    実態として作業者に直接指示を出している
  ・報告・連絡: 作業者が自社担当者に直接報告している

偽装請負のリスク評価:
  ・上記の状況は「実態上の派遣」とみなされる可能性がある
  ・「偽装請負」と判断された場合: 労働者派遣法違反・直接雇用申し込み
    義務が発生するリスク

出力形式:
・「請負 vs 派遣の区分基準チェックリスト」
  （真正請負の条件: 指揮命令・設備・報告経路 の各確認項目）
・「現状の問題点の指摘と、リスクがある行為の具体例」
・「是正の選択肢」（契約を派遣に変更する / 真正請負に切り替える
  / 直接雇用に切り替える）の比較
・「真正請負に切り替えるための実務的な変更事項」
・「労働局・社労士に相談する際の準備書類一覧」
```

## 生産変動に伴う派遣・直接雇用の人員計画書類を作るプロンプト

繁忙期・閑散期に合わせた人員構成計画を整理します。

```
以下の情報をもとに、生産変動に対応した
人員計画（正社員・派遣の最適構成）を整理してください。

【自社の生産変動パターン】
年間の生産変動:
  ・繁忙期: 7〜9月（自動車向け部品の需要増 / 月産目標○個）
  ・閑散期: 12〜1月（年末年始休暇・取引先減産期）
  ・通常期: その他の月

現在の人員構成:
  ・正社員: ○名（製造部）
  ・派遣社員: 通常時○名 → 繁忙期○名（追加○名）
  ・パートタイム: ○名

課題:
  ・繁忙期に急に派遣が増えると、OJT・安全教育に時間を取られる
  ・閑散期に派遣を全員終了すると、次の繁忙期に再教育が必要
  ・正社員に対する繁忙期の残業が増えすぎる

出力形式:
・「月次人員計画表（正社員・派遣・パートの月次必要人員）」
・「派遣社員の受入れ・終了のタイミング最適化の考え方」
・「繁忙期の安全教育・OJTを効率化するための「派遣社員向け入社時
  教育パッケージ」の骨子」
・「生産変動に対応する人員コストのシミュレーション（残業 vs
  派遣追加のどちらがコスト低いか）」
```
