電子帳簿保存法は2024年1月から電子取引データの保存が完全義務化されました。製造業でもメールで届く請求書・注文書・発注書などを紙に印刷して保管する方法は認められなくなっています。社内ルール整備・従業員周知・保存方法の確認をAIで整理できます。

> **注意**: 税務・法律に関わる判断は必ず税理士・専門家に確認してください。このページはAIプロンプトの活用例であり、法律アドバイスではありません。

---

## 電子取引データ保存のルールを社内説明するプロンプト

従業員向けに電子帳簿保存法の対応ルールを説明する文書を作ります。

```
以下の情報をもとに、電子帳簿保存法の対応について
社内（事務・購買・営業担当者）向けの説明文書を作成してください。

【自社の対応方針（例）】
・メールで届いた請求書・注文書・見積書などは、印刷禁止（印刷した場合も
  電子データが原本）
・電子データはサーバーの「電子帳簿保存_電子取引」フォルダに保存する
・保存ファイル名のルール: 「取引日付_取引先名_書類種別_金額（例外なし）」
  例: 2026-06-01_○○製作所_請求書_108000
・保存期間: 7年間（消費税の保存義務期間）
・検索しやすいように、年月・取引先でフォルダを分ける

出力形式:
・A4 1枚に収まる社内説明文（箇条書き＋具体例で分かりやすく）
・「よくある質問（Q&A）」を3〜5件追加（よくある疑問: 紙で届いた場合は？/
  PDF以外のフォーマットは？/ 過去分はどうする？）
・末尾に「判断に迷った場合は経理部門（内線○○）に確認してください」
```

## 電子取引データの保存手順書を作るプロンプト

電子データの保存手順を担当者が実行できる手順書として整理します。

```
以下の情報をもとに、電子取引データの保存手順書を作成してください。

【自社の環境・ルール】
使用ツール: （例：Windowsのファイルサーバー / Googleドライブ / Box等）
対象書類: （例：メールで受取る請求書・注文書・見積書 /
  自社がメールで送付した請求書の控え）
ファイル名ルール: （例：YYYYMMDD_取引先名_書類名_金額）
保存先フォルダ: （例：\\server\経理\電子帳簿保存\電子取引\[年度]\[月])
保存担当者: （例：書類を受け取った本人が当日中に保存する）

【手順として説明してほしいこと】
(1) メールで請求書PDFが届いた場合の保存手順
(2) PDFではなくExcel・Word形式で届いた場合の扱い
(3) ファイル名の付け方（具体例で説明）
(4) 誤ってファイルを削除してしまった場合の対処
(5) 月次確認（経理担当者が月末にフォルダ内を確認する手順）

出力形式:
・担当者が1人でも実行できるSTEP形式の手順書
・スクリーンショットのプレースホルダー（「ここにスクリーンショットを貼る」）
・「よくある失敗例」のコラム（ファイル名ルール違反 / 保存忘れ等）
```

## 電子保存の社内ルール（規程）の骨子を作るプロンプト

電子帳簿保存法対応のための社内規程の骨子を作ります。

```
以下の情報をもとに、電子帳簿保存法に対応する
「電子取引データ保存規程」の骨子（条文案の目次と概要）を
作成してください。

【自社の規模・状況】
従業員規模: （例：50名）
取引の種類: （例：製造業（受発注・仕入・外注）、
  取引書類はほぼメール/FAXのスキャンで受け取り）
保存方法: （例：社内ファイルサーバーに保存）
会計ソフト: （例：マネーフォワード クラウド会計）

【規程に含めたい内容】
・目的・適用範囲
・電子取引データの定義（どの書類が対象か）
・保存責任者・担当者
・保存方法・ファイル名ルール・保存場所
・保存期間（最低7年）
・検索要件への対応
・定期点検・監査の手順
・違反した場合の対応

出力形式:
・規程の目次（第1条〜最終条までの章立て）
・各条の「この条で書くこと」の概要説明
・「この規程で定めるべき重要ポイント3つ」をコラムで
・※法令に適合した最終的な規程文書は税理士・社労士に確認を要する旨を記載
```

## スキャン書類（紙→電子化）の保存方法を確認するプロンプト

紙で届いた書類のスキャン保存ルールを整理します。

```
製造業での以下の紙書類について、
電子帳簿保存法のスキャナ保存の要件に合った保存方法を
確認・整理してください。

【スキャンしたい書類と現状】
(1) 取引先から郵送で届く請求書（A4 / 月20〜30件）
    → 現在は紙のまま7年間ファイル保存
(2) 仕入品の納品書（A4）
    → 現在は納品のたびに紙に押印して保存
(3) 現場で手書きした作業日報（自社書類）
    → これはスキャナ保存の対象外？

確認してほしいこと:
・各書類がスキャナ保存の対象かどうか
・スキャン保存する場合の解像度・ファイル形式の目安
・タイムスタンプ付与の要件（簡易的に代替できる方法があるか）
・このまま紙保存を続けてよい書類はどれか

出力形式:
・書類ごとに「スキャナ保存対象 / 紙保存継続可 / 要確認」を整理
・スキャン保存の基本手順（解像度・カラー/グレースケール・保存形式）
・「この判断は税理士に確認してから実行すること」の注記付き
```
