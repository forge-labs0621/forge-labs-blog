製造業では仕入れた材料や外注加工品の品質が自社製品の品質を左右します。外注先・仕入先への品質要求の伝え方と、品質問題が起きた際の対応書類をAIで整理できます。

---

## 外注先への品質保証要求書を作るプロンプト

外注先に品質管理の要求事項を伝える文書を作ります。

```
以下の情報をもとに、外注先への品質保証要求書を
作成してください。

【外注の内容】
外注先: （例：○○加工所 / 旋盤・研削加工）
発注内容: （例：当社からの粗材を受け取り、仕上げ加工を行って返納する）
発注量規模: （例：月間 500〜1,000個）

【品質上の要求事項】
寸法・精度:
  （例：当社図面に従った加工。公差外れは出荷前に全数確認すること。
  報告なしの不良納入は取引停止の対象とする）
外観基準:
  （例：傷・打痕・サビは禁止。限度見本は当社にて確認のこと）
トレーサビリティ:
  （例：加工ロット番号を現品票に記載して納品すること。
  加工実績記録を6か月間保存すること）
不良発生時の対応:
  （例：不良が発生した場合は24時間以内に当社に連絡。
  48時間以内に暫定対策を報告すること）

出力形式:
・外注先への品質保証要求書（正式な書面として使える形式）
・「要求事項を守った場合の確認方法（当社による受入検査）」の記載
・「要求事項違反の場合の取り扱い（是正請求・取引中断等）」の明記
・外注先の「承認サイン欄」（合意確認）
```

---

## 外注先の立入り監査（定期評価）のチェックリストを作るプロンプト

主要な外注先の工場を訪問して品質管理を確認する監査チェックリストを作ります。

```
以下の情報をもとに、外注先の立入り監査チェックリストを
作成してください。

【監査の概要】
対象外注先: （例：○○加工所 / 旋盤・研削加工専門 / 従業員8名の小規模業者）
監査実施頻度: （例：年1回（重要外注先））
監査目的: （例：当社の品質要求への対応状況確認と、品質トラブルの事前防止）

【確認したい項目】
品質管理:
  ・当社図面・仕様書の最新版を保有しているか
  ・検査基準書があり、担当者が内容を理解しているか
  ・測定器の校正管理ができているか
製造管理:
  ・作業標準書（手順書）が整備されているか
  ・4M変更時に当社への連絡が行われているか
  ・不良品の識別・隔離の仕組みがあるか
トレーサビリティ:
  ・加工ロット記録が残されているか
  ・使用する材料の仕入先を特定できるか
環境・安全:
  ・整理整頓（5S）の状態
  ・法令遵守（廃油・切り粉の処分方法等）

出力形式:
・立入り監査チェックリスト（確認済 / 未対応 / 要改善 / N/A の4択）
・各項目に「確認のポイント（何を見るか・何を聞くか）」を付記
・「監査後のフィードバックレポート（外注先に渡す改善依頼書）」のフォーマット
・「外注先評価スコアリング」のシンプルな計算方法
```

## 外注品の不良受入時の対応メールを作るプロンプト

外注先から受け取った品物に不良が見つかった場合の対応を整理します。

```
以下の情報をもとに、外注先への不良品発見の連絡メールと
返品・是正要求の文書を作成してください。

【不良の内容】
外注先: （例：○○加工所）
発見日: （例：2026年6月28日 受入検査時に発見）
不良内容: （例：当社発注の研削加工品（品番B-5678）50個のうち、
  15個で指定寸法（φ20h7）が外れていた（実測値: φ20.05〜20.12mm。
  許容 φ19.979〜20.000mm のため、全数不合格）)

【対応に必要な内容】
・不良品の返品（送料は外注先負担）
・良品への交換納入（最短でいつ可能か確認）
・原因調査と是正処置の報告（1週間以内）
・再発防止策の確認

出力形式:
・外注先への不良通知メール（事実を明確に、しかし関係を壊さない丁寧なトーン）
・「不良品の証拠写真・測定データを添付します」の文言
・「○日までに良品を再納入してください」の期限明示
・「是正処置報告書を△日までにご提出ください」の依頼
・「再発した場合の取り扱い」の注記（取引条件への影響）
```
