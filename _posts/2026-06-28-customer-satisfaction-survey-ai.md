ISO 9001では顧客満足度を継続的に把握・評価することが求められています（条項9.1.2）。また、主要取引先の満足度を定期的に確認することは、長期的な受注維持にもつながります。アンケート作成・集計・報告書の作成をAIで効率化できます。

---

## 顧客満足度アンケートを作るプロンプト

取引先に送付する満足度アンケートの質問票を作ります。

```
以下の情報をもとに、主要取引先向けの顧客満足度アンケートを
作成してください。

【自社の概要・アンケートの目的】
業種: （例：受託プレス加工業）
アンケートの目的: （例：ISO 9001の要求事項（9.1.2）への対応 /
  主要取引先の満足度把握と改善点の特定）
対象: （例：主要取引先10社の購買担当者・品質担当者）
実施時期: （例：年1回（○月）/ メールで送付）

【聞きたい項目】
品質面:
  ・納入品の品質（寸法・外観・不良率）は満足か
  ・品質問題発生時の対応（速さ・誠実さ）はどうか
納期・対応面:
  ・納期遵守率・緊急対応への対応力はどうか
  ・問い合わせへの返答の速さ・丁寧さはどうか
コスト・提案力:
  ・価格水準（価格と品質のバランス）はどうか
  ・コスト削減・改善提案の積極性はどうか
総合:
  ・総合満足度（100点満点または5段階）
  ・継続発注の意向はあるか
  ・今後期待すること（自由記述）

出力形式:
・メール添付できるアンケート（回答しやすい5段階評価 + 自由記述形式）
・「3分以内に回答できる」分量（15項目以内）
・「ご記入いただいた内容は改善活動に活かします」の文言
・送付用カバーレター（メール本文）
```

## 顧客満足度アンケートの集計・分析コメントを作るプロンプト

アンケートの回答結果を整理し、分析コメントを付けます。

```
以下のアンケート回答結果をもとに、
顧客満足度の集計レポートと分析コメントを作成してください。

【アンケート実施概要】
実施時期: （例：2026年6月）
回答数: （例：8社 / 送付数10社 / 回収率80%）

【各項目の平均スコア（5点満点）】
品質（寸法・外観・不良率）: （例：4.2点）
品質問題発生時の対応: （例：4.5点）
納期遵守率: （例：3.8点）
緊急対応力: （例：4.0点）
問い合わせ対応の速さ: （例：4.3点）
価格水準: （例：3.5点）
改善提案の積極性: （例：3.2点）
総合満足度: （例：3.9点）

【自由記述（主なコメント）】
ポジティブな意見:
  （例：「品質トラブルが発生したときの対応が迅速で助かる」
  「担当者の対応が丁寧」）
改善要望:
  （例：「納期回答がもう少し早いと助かる」
  「コスト削減の提案をもっと積極的に持ってきてほしい」
  「価格は競合他社と比べてやや高い」）

出力形式:
・顧客満足度調査結果レポート（経営会議・ISO審査向け）
・スコアのレーダーチャート（テキスト表形式でのグラフプレースホルダー）
・「強み（高スコア項目）」と「改善が必要な項目（低スコア）」の分析
・「来年度の改善施策（コメントから導き出した具体案）」の提案
・ISO 9001 9.1.2条「顧客の認識の監視」に対する対応記録として使えること
```

## 顧客への年次お礼・関係強化メールを作るプロンプト

アンケート実施後、回答してくれた取引先へのお礼と改善報告メールを作ります。

```
以下の情報をもとに、顧客満足度調査へのご協力のお礼メールと
改善報告メールを作成してください。

【送付の目的と内容】
送付先: （例：アンケートに回答してくれた主要取引先各社）
お礼の内容: （例：アンケートの回答に対する感謝。
  主な改善施策（来年度から納期回答を2営業日以内に改善・
  コスト削減提案を四半期1回定期実施）を伝える）
送付時期: （例：アンケート集計後1か月以内）

出力形式:
・丁寧なビジネスメール（感謝→アンケート結果の振り返り→来年度の改善約束の流れ）
・「ご意見が改善に活かされる」という具体的なメッセージ
・「今後ともよろしくお願いいたします」の締め
・テンプレート（各社宛名を差し込むだけで送れる形式）
```
