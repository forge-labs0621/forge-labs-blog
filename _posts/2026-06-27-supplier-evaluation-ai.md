製造業では、使用する材料・部品の品質管理のために、仕入先・外注先（協力会社）を定期的に評価・ランク付けする仕組みが重要です。ISO 9001でも「外部提供者の評価・選定・再評価・監視」が要求されています。AIを使うと、日頃の取引データから評価シートの記入・コメント作成・総合評価を効率化できます。

## 仕入先・外注先評価でAIが役立つ場面

- **評価シートのたたき台作成**: 評価項目・採点基準をゼロから考えるのが手間な場合
- **定量データからコメント生成**: 「不良率2%、納期遵守率95%」などの数値から評価コメントを作る
- **評価結果の報告書化**: 全社入先のランク一覧・上位/下位のまとめを整形する
- **改善要求書の下書き**: 評価が低い仕入先・外注先への改善要求書を丁寧な表現で作る

## 仕入先・外注先評価シートのたたき台を作るプロンプト

---

```
以下の条件で、製造業向けの仕入先（または外注先）定期評価シートを作成してください。

【評価対象】
□ 仕入先（材料・部品の調達先）
□ 外注先・協力会社（加工・組立・検査等の委託先）

【評価頻度】（例：年2回（6月・12月）、年1回（3月）など）

【主な評価観点（重要なものを選んでください）】
□ 品質（不良品・返品率、品質管理体制）
□ 納期（納期遵守率、急ぎ対応力）
□ 価格（競争力、見積もりの速さ）
□ 対応力（問い合わせへの応答速度、クレーム対応）
□ コンプライアンス（法令遵守、労働環境、情報管理）
□ 技術力・提案力（改善提案、新工法への対応）

【最終的に判定したいランク】
（例：S/A/B/C（Cは改善要求or取引見直し対象）など）

出力形式:
・評価項目ごとに「評価基準（点数）」と「評価欄」のある表形式
・各評価項目に点数の基準（例: 不良率0%→5点、1%未満→4点 等）を記載
・合計点からランク判定する基準も含める
・評価者・確認者・評価日欄
・評価対象（仕入先名・品目等）記入欄（プレースホルダー）
```

## 評価データからコメントを生成するプロンプト

各仕入先の数値データから評価コメントを作る際に使えます。

```
以下の仕入先データをもとに、評価シートの「総合コメント」と
「次期に向けた改善要望」を作成してください。

【仕入先名・取引品目】
（例：○○金属株式会社 / 機械加工部品（旋盤加工品））

【評価期間】（例：2026年1月〜6月）

【主な実績データ】
・発注件数: 24件
・納期遵守率: 95.8%（24件中23件を期日通り納品。遅延1件は素材調達遅れが原因で、
               事前連絡あり）
・品質不良率: 0.5%（1件、外観傷あり。速やかな代品対応済み）
・価格変動: 前期比±0%（据え置き）
・問い合わせ応答: 平均0.5営業日以内で返答

【特記事項】
（例：急な追加発注に2回対応してもらえた。
       担当者の引き継ぎがあったが品質への影響は特になかった。）

総合評価: A（85点）

出力形式:
・「総合コメント」（3〜5文で取引全体の評価）
・「良かった点」（箇条書き2〜3件）
・「改善を期待する点」（あれば箇条書き1〜2件、なければ「特になし」）
・「次期の取引継続/条件変更の方針」（1文）
```

## 低評価仕入先への改善要求書を作るプロンプト

評価が一定基準を下回った仕入先に対して、改善要求書を送る際に使えます。

```
以下の内容をもとに、仕入先への「品質・納期改善要求書」を作成してください。

【仕入先名】（プレースホルダー可）
【要求書の日付】
【担当者名・会社名】（プレースホルダー可）

【課題の内容】
（例：
・2026年上半期の納期遵守率: 78%（6件中2件遅延）
・品質不良率: 3.2%（月平均）
・不良内容: 寸法外れ（公差±0.1に対して±0.2〜0.3の不良が繰り返し発生）
）

【こちらが期待する改善目標】
（例：
・2026年下半期に納期遵守率95%以上を達成してほしい
・不良率を1%以下に改善してほしい
）

【改善計画書の提出を求めるかどうか】
□ 求める（提出期限:  ）
□ 求めない

文体: 取引先への正式な文書として丁寧に、ただし改善の必要性を明確に伝える。
```

## 評価結果一覧（サプライヤーマップ）を整理するプロンプト

複数の仕入先・外注先を一覧化して、ランクごとに整理する際に使えます。

```
以下の仕入先評価結果を整理し、ランク別サプライヤーリストを作成してください。

【各仕入先の評価結果（コピペ・入力）】
（例：
A社 / 旋盤加工 / 93点 / Sランク
B社 / 板金加工 / 82点 / Aランク
C社 / 材料（鉄鋼） / 61点 / Bランク
D社 / 溶接加工 / 45点 / Cランク（改善要求中）
）

出力形式:
・ランク別にグループ化した一覧表
・各仕入先の「取引品目」「点数」「特記事項」の列
・Cランク仕入先には「要対応: 改善要求書送付済み/未送付」の欄
・評価日・次回評価予定日
```

---

## AIを使う際の注意

- 仕入先・外注先の評価は**事実の数値**に基づいて行ってください。AIはデータを整形しますが、評価の判定は担当者が行ってください
- 改善要求書は送付前に上長・購買部門の確認を必ず取ってください
- 評価結果は仕入先との取引判断に影響するため、根拠となる記録（発注書・納品書・不良記録）を保管しておくことが重要です
