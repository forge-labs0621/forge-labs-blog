製造業・工場の採用活動では、「面接評価シート」「面接質問リスト」「内定通知書」などの書類作成が発生します。採用人数が少ない中小製造業では専任の採用担当がいないことも多く、現場リーダーや総務担当が兼務で対応するケースが多いです。AIを使って採用書類の効率化ができます。

---

## 製造現場向け面接評価シートを作るプロンプト

技能職・現場作業者の採用面接で使える評価シートを作ります。

```
以下の情報をもとに、製造業・工場の技能職採用面接用の評価シートを
作成してください。

【採用ポジション】
職種: （例：機械加工オペレーター（旋盤・マシニング操作）、正社員）
求める人物像: （例：
・基本的な旋盤加工経験（未経験でも可、要意欲）
・チームワークを大切にできる
・安全ルールを守れる
・残業・土曜出勤が月数回ある（事前確認必要）
）

【評価したい観点】
（例：技術スキル / 安全意識 / コミュニケーション / 責任感 / 定着意欲）

出力形式:
・評価項目を5〜7項目
・各項目を「5: 非常に良い / 4: 良い / 3: 普通 / 2: やや不足 / 1: 不足」で評価
・各項目に「面接官のメモ欄」
・「総合評価・採否意見」欄（合格 / 再考 / 不合格）
・面接官サイン欄・面接日時記入欄
```

## 製造現場向け面接質問リストを作るプロンプト

技能職・ライン作業者の面接でよく使う質問を整理します。

```
以下のポジション向けに、面接で聞くべき質問リストを作成してください。

【採用ポジション】
（例：製造ライン作業者（食品工場 / パート・アルバイト希望者））

【確認したい項目】
- 現場経験・スキル（あれば）
- 体力・勤務条件への適性（立ち仕事・シフト勤務）
- 安全意識・ルール遵守の姿勢
- チームワーク・コミュニケーション
- 長期就業の意欲・定着性
- 動機・志望理由

出力形式:
・質問を「必ず聞く（必須）」と「時間があれば聞く（任意）」に分類
・各質問に「この質問で何を確認したいか（面接官向けメモ）」を付ける
・NG質問（性別・出身地・家族構成など差別につながる質問）も最後に注意書きとして記載
・全体で20問以内（必須10問、任意10問程度）
```

## 内定通知書の文章を作るプロンプト

採用決定後に送る内定通知書（書面・メール）を作ります。

```
以下の情報をもとに、内定通知書の文章を作成してください。

【採用情報】
候補者氏名: （例：山田太郎様）
採用ポジション: （例：製造部 機械加工オペレーター（正社員））
入社予定日: （例：2026年9月1日）
勤務条件: （例：勤務時間 8:00〜17:00（休憩60分）、月給○○円、
  試用期間3か月）

【内定通知の形式】
（書面（郵送用）または メール どちらかを選択。書面の場合はA4 1枚）

出力形式:
・「内定通知書」の正式な書面形式
・内定のお祝いと入社への期待を伝える一文
・入社日・勤務条件・承諾期限（○月○日まで）の記載
・「内定承諾書を返送してください」の案内
・問い合わせ先（担当者名・連絡先のプレースホルダー）
```

## 不採用通知の文章を作るプロンプト

不採用時に送る通知文を作ります。

```
以下の情報をもとに、不採用通知の文章を作成してください。

【ポイント】
・丁寧な断り文を作る（応募者を傷つけない言葉遣い）
・不採用の具体的な理由は記載しない（トラブル回避のため）
・応募に感謝する気持ちを伝える

【送付方法】
（メール通知 / 書面通知 どちらかを選択）

出力形式:
・「誠に恐れながら」から始まる丁寧な不採用通知
・応募への感謝
・「慎重な選考の結果、今回はご縁がなかったとの判断」という表現で
・今後のご活躍を願うメッセージで締める
・署名欄（会社名・部署・担当者名・連絡先のプレースホルダー）
```

## 面接後の社内共有メモを作るプロンプト

面接終了後、採否検討に使う社内メモを整理します。

```
以下の面接メモをもとに、社内共有用の面接サマリーを作成してください。

【面接候補者】
（例：山田太郎 / 35歳 / 前職: ○○製作所 旋盤オペレーター 8年）

【面接でわかったこと（面接官のメモ）】
・旋盤経験8年、NC旋盤・汎用旋盤ともに対応可能
・前職は金属加工専門。離職理由は会社都合リストラ
・質問への回答が明確。安全意識の質問にも具体的なエピソードあり
・残業・土曜出勤についてはOKと回答。家族の理解も得ているとのこと
・志望動機は「ものづくりに集中できる環境を求めている」と話していた
・印象: 落ち着いており、経験豊富な印象。現場リーダーへの評価も聞きたい

出力形式:
・「候補者概要 / 経験・スキル / 採用ポジションへの適性 / 懸念点・確認事項 /
  面接官の感触（5段階）」のフォーマット
・社内で採否を検討する際に使える簡潔なまとめ（A4半ページ以内）
```
