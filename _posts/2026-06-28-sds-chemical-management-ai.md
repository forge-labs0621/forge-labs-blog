2023年の労働安全衛生法改正により、一定の化学物質については「リスクアセスメント」の実施と記録が義務化されました。工場・製造現場で使用する溶剤・洗浄剤・切削油・接着剤などの化学物質について、SDSと管理書類の文書化をAIで効率化できます。

> **注意**: 化学物質の危険性・有害性の情報は、必ずメーカーが発行する正式なSDS（安全データシート）を確認してください。AIはSDSの内容の整理・要約には使えますが、安全性の判断はSDS・専門家の知見に基づいてください。

## SDS管理台帳の記載内容を整理するプロンプト

工場内で使用する化学物質のSDS台帳エントリを整理します。

```
以下の情報をもとに、SDS管理台帳の1品目分の記載内容を整理してください。

【SDSから確認した情報】
製品名: （例：○○洗浄剤 EC-200）
製品番号: （例：EC-200-4L）
メーカー: （例：○○化学工業（株））
SDS発行日: （例：2024年3月）
主成分: （例：イソプロパノール（IPA）70%、水30%）

【GHS危険有害性区分（SDSから）】
引火性: 引火性液体 区分3（引火点12℃）
健康有害性: 急性毒性（経口）区分5 / 眼刺激性 区分2A
環境有害性: 該当なし

【使用場所・使用目的】
使用場所: （例：仕上げ工程・機械周辺の清掃）
保管場所: （例：可燃物保管庫A-2）

出力形式:
・台帳の1レコード（「製品名 / メーカー / 主成分 / 主な危険有害性 / 
  保管場所 / SDS発行日 / 次回確認日 / 担当」の欄）
・「主な注意事項」（3行以内、現場担当者向け）
・SDS更新確認期限（例：毎年4月に最新版を確認）
```

## 化学物質リスクアセスメント（簡易版）を整理するプロンプト

労安法改正対応の簡易リスクアセスメント記録を整理します。

```
以下の情報をもとに、化学物質リスクアセスメント記録（簡易版）を
作成してください。

【対象化学物質】
製品名: （例：切削油 MX-100）
使用量: （例：1週間で約2L使用、刷毛塗り）
SDS危険有害性: （例：皮膚刺激性 区分2、生殖毒性 区分2（注意要）、
  引火点: 65℃（非引火性））

【作業環境】
使用場所: （例：旋盤工程・室内・換気扇1台あり（換気回数：毎時6回程度））
使用時の保護具: （例：耐油ゴム手袋・保護メガネ）
暴露時間: （例：1日2〜3時間、加工時に飛沫が手に触れる可能性あり）

【現在のコントロール措置】
（例：局所排気なし。手袋着用。定期的に手洗い。）

出力形式:
・リスクアセスメント記録フォーマット（作業内容・危険有害性・ばく露評価・
  リスクレベル・コントロール措置・残留リスク・評価日）
・リスクレベルは「高・中・低」で（判断根拠も記載）
・追加の保護措置の提案（1〜2点、改善提案として）
・評価実施者サイン欄・次回評価予定
```

## 化学物質の取り扱い注意書き（現場掲示用）を作るプロンプト

SDSの内容をもとに、現場スタッフが読める取り扱い注意書きを作ります。

```
以下のSDS情報をもとに、現場に掲示する化学物質取り扱い注意書きを
作成してください。

【化学物質の情報】
製品名: （例：脱脂洗浄剤 クリーン-SP）
主な危険有害性:
  - 引火性液体（引火点: 18℃）→ 火気厳禁
  - 皮膚刺激性あり → 素手での扱い禁止
  - 蒸気の吸引注意 → 換気必要
保管条件: 日光・熱源から離れた冷暗所

【使用場所・対象者】
（例：仕上げ工程の作業者全員が使用）

出力形式:
・A4 1枚以内の現場掲示用注意書き
・「この溶剤の取り扱い上の注意」というタイトルで
・絶対禁止事項を大きく・目立つように（火気厳禁など）
・保護具（手袋・保護メガネ等）の着用ルール
・漏洩・皮膚付着時の応急処置
・「詳細はSDS（安全データシート）を確認してください」の注記
・文字は読みやすく大きめ（現場作業者が立ったまま読める）
```

## 化学物質管理の年次点検チェックリストを作るプロンプト

年1回の化学物質管理状況の総点検に使うチェックリストを作ります。

```
以下の条件で、化学物質管理の年次点検チェックリストを作成してください。

【対象】
（例：製造工場（プレス・溶接・仕上げ工程）で化学物質を使用している
 製造業SME。従業員30名。労安法のリスクアセスメント義務対象製品を
 3品目使用中）

【点検したい内容】
- SDS台帳の最新版確認
- リスクアセスメントの実施・記録確認
- 保管場所・保管状況の確認
- 保護具の準備・劣化確認
- 廃棄・処分記録の確認
- 現場作業者への周知・教育状況

出力形式:
・「確認項目 / 確認方法 / 担当部署 / 結果（記入欄） / 備考」の表形式
・「法令義務事項」と「推奨事項」に分類
・点検結果のサマリーを記入する欄
・次年度の改善事項記入欄
```
