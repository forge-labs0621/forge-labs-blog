製造業では、省エネ法（エネルギーの使用の合理化に関する法律）に基づくエネルギー管理や、取引先からのカーボンニュートラル対応要請への対応が増えています。月次の省エネ活動記録や、エネルギー使用量報告書の文章化をAIで効率化できます。

---

## 月次省エネ活動報告を作るプロンプト

月次の省エネ取り組みと成果を報告する文書を整理します。

```
以下の情報をもとに、月次省エネ活動報告書を作成してください。

【対象月】（例：2026年7月度）

【エネルギー使用量実績】
電力使用量: ○○ kWh（前月比 ○%・前年同月比 ○%）
ガス使用量: ○○ m³（前月比 ○%・前年同月比 ○%）
電力コスト: ¥○○万円

【今月実施した省エネ活動】
（例：
・昼休み中の空調・照明の停止を徹底（全工程リーダーへ周知）
・プレス工程の電源タイマー設定を変更（稼働前30分→15分に短縮）
・照明の間引き点灯（作業に影響しない箇所の照明を1/2に）
）

【エネルギー削減の実績】
（例：前月比 ○kWh削減（○%削減）。電力コスト削減効果: 約¥○○万）

【来月の重点省エネ活動予定】
（例：コンプレッサーの稼働管理強化。圧力設定の見直し。）

出力形式:
・省エネ活動報告書フォーマット
・活動内容・削減効果・翌月計画の3段構成
・「CO2削減換算」（あれば: ○kWh × 0.433kg-CO2/kWh = ○kg-CO2削減）
・承認者欄・作成日
```

## 省エネ目標設定書を作るプロンプト

年間の省エネ目標と取り組み計画を文書化します。

```
以下の情報をもとに、年間省エネ目標設定書を作成してください。

【前年実績・基準値】
電力使用量（前年実績）: ○○ MWh
ガス使用量（前年実績）: ○○ m³
製品1個あたりエネルギー原単位: ○○ kWh/個

【今年の省エネ目標】
電力使用量削減目標: 前年比 ○%削減（→ ○○ MWh以下）
エネルギー原単位改善目標: 前年比 ○%改善
削減方策の費用対効果目標: ¥○○万の削減効果

【主な省エネ施策（計画）】
（例：
・照明のLED化（第2工場 予算¥○○万 / ○月〜）
・インバーター導入（冷却ポンプ ¥○○万 / ○月〜）
・エアコン管理の強化（冷暖房設定温度の標準化）
）

出力形式:
・省エネ目標管理表のフォーマット
・目標値・施策・期待効果・担当の表
・「省エネ法の定期報告に使える様式」との整合性を意識した構成
・管理責任者サイン欄
```

## 取引先向けCO2排出量開示・報告文を作るプロンプト

サプライチェーン排出量（Scope 3）の把握に対応する報告文書を作ります。

```
以下の情報をもとに、取引先からのCO2排出量開示依頼への回答文を
作成してください。

【取引先からの依頼内容】
（例：○○自動車工業より、サプライヤーのScope 1/2排出量データの
 提供依頼が届いた。回答期限: ○月○日）

【自社のエネルギー・CO2データ（参考値）】
Scope 1（自社の直接排出）:
  ガス燃焼: ○ t-CO2（2025年度実績）
Scope 2（電力等の間接排出）:
  電力: ○○ MWh × ○ kg-CO2/kWh = ○ t-CO2
Scope 1+2合計: ○ t-CO2

【対応方針】
（例：今回は電力・ガスのデータを報告する。詳細な算定基準の整備は
 今後の課題とする）

出力形式:
・取引先への正式回答書
・データ開示内容（表形式：排出区分/排出量/算定方法）
・算定の前提・留意事項の注記（「自社計算のため精度は目安値」）
・「今後も情報開示に協力します」の趣旨を伝える文言
・担当者名・回答日・連絡先（プレースホルダー）
```

## 設備の省エネ改修効果レポートを作るプロンプト

照明LED化や設備のインバーター導入後の効果を報告します。

```
以下の情報をもとに、省エネ設備導入効果レポートを作成してください。

【導入設備】
設備名: （例：第2工場 照明 LED化工事）
導入費用: （例：¥○○万円）
完了日: （例：2026年5月）

【導入前後の比較】
導入前: （例：蛍光灯 40W × 50本 → 月間消費電力 ○○ kWh）
導入後: （例：LED 20W × 50本 → 月間消費電力 ○○ kWh）
月間削減量: （例：○○ kWh削減 / 月）
月間削減金額: （例：¥○○万円 / 月）

【投資回収の見込み】
単純回収年数: （例：¥○○万 ÷ ¥○○万/年 = ○年）

出力形式:
・省エネ設備導入効果レポートのフォーマット
・投資回収計算の表
・「環境効果（CO2削減換算）」の欄
・今後の追加投資候補の提案欄（任意）
・承認者サイン欄
```
