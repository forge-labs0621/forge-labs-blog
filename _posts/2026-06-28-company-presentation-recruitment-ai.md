人手不足が深刻な製造業では、若手採用に力を入れることが経営課題になっています。工業高校・専門学校への会社説明・合同説明会の資料・採用サイトの文章作成をAIで効率化できます。

---

## 高校生向け会社説明会の資料構成を作るプロンプト

工業高校の学生向けに自社の魅力を伝える説明会資料を作ります。

```
以下の情報をもとに、工業高校の学生向け会社説明会の
資料（スライド構成と各スライドの内容案）を作成してください。

【自社の情報】
会社名・業種: （例：○○製作所 / 精密プレス加工・溶接組立）
製品・取引先: （例：自動車ボディ部品を年間○百万個製造。
  トヨタ・ホンダ系1次サプライヤーへ納入）
規模・設立: （例：従業員75名 / 設立1985年 / 愛知県○○市）

【高校生に伝えたい自社の魅力】
・具体的な仕事内容: （例：最先端のプレス機・NC工作機械を操作する。
  図面を読んで精密な部品を作る技術が身につく）
・給与・待遇: （例：初任給 ○○万円 / 各種手当 / 昇給・賞与あり）
・資格取得支援: （例：溶接技能者・クレーン運転・フォークリフトなど
  取得費用を全額会社負担）
・社内の雰囲気: （例：20代・30代が多い / 先輩が丁寧に教える / 
  外国人社員も活躍中）
・将来のキャリア: （例：班長→職長→部門リーダーと成長できる。
  技術系・管理系の2つのキャリアパス）

出力形式:
・スライド構成案（1.会社紹介 / 2.仕事内容 / 3.働く環境 /
  4.キャリアパス / 5.採用情報・選考フロー）
・各スライドの「見出し + 2〜3のポイント（箇条書き）」
・高校生の目線で「ここが響くポイント」のコメント付き
・「製造業に対して高校生が持つイメージを払拭するための工夫」のアドバイス
```

## 採用サイト・求人ページのキャッチコピーを作るプロンプト

自社の採用ページに使えるキャッチコピーと紹介文を作ります。

```
以下の情報をもとに、製造業の採用ページのキャッチコピーと
会社紹介文を作成してください。

【自社の特徴（強み・アピールポイント）】
（例：
・自動車の「見えない部分」を支える精密加工の技術
・入社3年でNCを操れる。5年でリーダー候補に
・完全週休2日制・年間休日120日
・未経験者歓迎・工業高校卒でも技術者として活躍中
・社員の平均年齢35歳・若い職場
・愛知県○○市の地元企業で通勤も便利
）

【ターゲット（どんな人に応募してほしいか）】
（例：
・モノづくりが好きな人
・手に職をつけたい人
・チームで働くのが好きな人
・転勤なしで地元で長く働きたい人
）

出力形式:
・採用ページのメインキャッチコピー（3案・20字以内）
・サブキャッチコピー（各案に対応する補足文・50字以内）
・採用ページのリード文（200〜300字）
・「求める人材像（こんな人に来てほしい）」の文章（100字程度）
```

## 転職者向けのスカウトメール文を作るプロンプト

中途採用での転職サイト・ダイレクトリクルーティングのスカウトメールを作ります。

```
以下の情報をもとに、転職サイトでのスカウトメールを
作成してください。

【候補者の想定スペック】
（例：現在の仕事は製造業（プレス加工）の現場作業者。
  経験5年。班長経験あり。
  転職理由は「より大きい会社でキャリアアップしたい」と
  プロフィールにある。30代前半。）

【自社がアピールできること】
（例：
・前職と同業種（プレス加工）なので即戦力として活躍できる
・班長経験を活かして入社後すぐにリーダー候補として処遇する
・年収は前職比○〜○%アップを提示できる
・完全週休2日・有給消化率○○%
）

出力形式:
・個人宛のスカウトメール（自然な日本語で、堅すぎず親しみやすいトーン）
・「あなたのプロフィールを見て」という導入
・「なぜこの人にメッセージを送ったか」の具体的な理由
・「一度お話を聞かせてください」のカジュアルな打診
・カジュアル面談への案内（日程調整の依頼）
```

## 採用面接後の「不採用通知」「内定通知」に加えて「採用後のフォロー」を作るプロンプト

採用後の内定者フォローメールを作ります。

```
以下の情報をもとに、内定者への入社前フォローメールを
作成してください。

【内定者の状況】
内定者: （例：工業高校3年生 / 入社日: 2027年4月1日）
現在の時期: （例：内定出し直後（2026年9月）/ 入社まで7か月）

【フォローメールで伝えたい内容】
（例:
・内定おめでとうのメッセージ
・入社前に準備しておくこと（工具・作業服の準備 / 普通免許の取得推奨）
・工場見学への招待（秋に開催予定）
・社員との交流会の案内（12月に懇親会予定）
・不安なことがあれば相談窓口の案内
）

出力形式:
・内定者への温かいフォローメール
・「入社が楽しみになる」ポジティブなトーン
・「困ったことがあればいつでも連絡ください」の安心感を与える締め
・定期的なフォロー（月1回程度のメール）の案も提案
```
