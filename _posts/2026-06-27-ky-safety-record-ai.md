製造現場の安全活動として多くの工場で実施されているKY活動（危険予知活動・危険予知訓練）。「毎回同じようなことしか書けない」「KYシートを書くのが形式的になっている」という担当者向けに、AIを使ってKY活動記録を充実させる方法を紹介します。

## KY活動とは

KY活動（危険予知活動）とは、作業前に「この作業でどんな危険が潜んでいるか」を事前に予測・確認し、対策を決めてから作業に入る安全活動です。

一般的な流れ：

1. **現場の状況・作業内容を確認**（今日の作業内容・環境）
2. **危険の洗い出し**（どんな危険が考えられるか、全員で出し合う）
3. **重要な危険に絞り込む**（特に注意が必要なものを選ぶ）
4. **対策の決定と指差し確認**（「〇〇に注意、ヨシ！」）
5. **記録として残す**

AIはこの「危険の洗い出し」と「記録の文章化」を助けてくれます。

## KYシートをAIで作るプロンプト

### ステップ1: 危険項目の洗い出し

---

```
以下の作業条件で、製造現場のKY活動（危険予知）に使う危険項目の候補を出してください。

【作業内容】
（例：プレス機を使った金属部品のプレス加工）

【作業環境・特記事項】
（例：今日は雨で床が滑りやすい / 新しい品番の段取り替えが必要 /
      新人が1名参加 / 連休明けで集中力が落ちやすい　など）

【使用する設備・工具・材料】
（例：プレス機・金型・200mm角の金属板材）

出力内容:
・考えられる危険（5〜8項目）を箇条書きで
・各危険の「発生場面」と「起こりうる怪我・被害」のセットで
・現場の作業者が「あるある」と思える具体的な内容で
```

---

### ステップ2: KY活動記録の文章化

---

```
以下のKY活動の実施メモをもとに、KY活動記録シートを作成してください。

【日付・作業場所・担当班】
【今日の作業内容】
【洗い出した危険（上の出力または話し合いで出たもの）】
【今日の重点実施項目（1〜2つに絞った対策）】
【確認した安全ポイント（指差し確認の言葉）】
【参加者】

出力形式:
・KY活動記録シートとして一枚にまとまる形式
・「危険項目」「起こりうる被害」「対策」「重点実施項目」の列を持つ表
・「本日の安全目標」と「指差し確認ポイント」の欄
・日付・参加者・確認者のサイン欄
```

---

## 実際の出力例

プロンプト1に「プレス機でのプレス加工、雨で床が滑りやすい、新人1名参加」と入力すると：

---

**考えられる危険の候補**

1. **プレス機の稼働中に金型内に手を入れてしまう** → 指・手の挟まれによる切断・骨折
2. **段取り替え中に金型が落下する** → 足への落下・打撲・骨折
3. **雨で濡れた床で資材運搬中にすべる** → 転倒・腰・膝への打撲
4. **新人が作業手順を誤り、誤動作させる** → 誤プレス・設備損傷・怪我
5. **切削くずや材料の端材で手を切る** → 手指の切り傷
6. **連続作業で集中力が低下し、確認漏れが起きる** → 誤ったワークのプレス・品質不良

---

このリストから「今日特に注意が必要なもの」を1〜2項目に絞り、「〇〇に注意してヨシ！」の指差し確認を決めます。

## TBM（ツールボックスミーティング）の進め方メモをKY記録に変換

朝のTBMで話した内容をメモしておき、AIで記録に変換することもできます。

```
以下の朝のTBMのメモをもとに、KY活動記録を作成してください。

【TBMメモ（箇条書き）】
（例：
・今日は機械加工エリアで旋盤加工
・床の油汚れに注意（昨日清掃したが完全ではない）
・山田くん（新人・3ヶ月目）も参加
・重点: 旋盤の材料セット時に指を挟まないよう注意
・確認: 「材料セット時は必ず主軸停止を確認してからヨシ！」
・参加: 田中・山田・佐藤
）

出力形式: KY活動記録シートの形式で
```

## 繰り返し作業のKY記録を効率化する

毎日同じ作業をしていると、KY記録が形式的になりがちです。「今日の特記事項」だけを変えて使い回せる形にするプロンプトです。

```
以下の「基本のKY記録」をベースに、今日の特記事項を反映した更新版を作成してください。

【基本のKY記録】
（例：旋盤加工の標準KYシート。通常の危険項目と対策が記載されている）

【今日の特記事項】
（例：連休明けで集中力が低下しやすい / 新品の工具に交換 / 生産数が多い日）

変更点: 今日の特記事項に関する危険項目・対策を1〜2点追加・強調してください。
```

## ヒヤリハット発生後のKY見直しに活用する

ヒヤリハット（怪我には至らなかったが危なかった出来事）が発生した場合、その内容をKY活動にフィードバックできます。

```
以下のヒヤリハット事例をもとに、来週のKY活動に追加すべき
危険項目と対策を提案してください。

【ヒヤリハットの内容】
（例：旋盤加工中、工具交換の際に主軸の停止を確認せずに手を近づけた。
幸い停止中だったが、停止確認が抜けていた）

提案内容:
・KY記録に追加すべき危険項目（1〜2点）
・再発防止のための対策・指差し確認ポイント
```

→ ヒヤリハット報告書の書き方はこちら: [ヒヤリハットが報告されずに消えていく問題、AIで報告書化して防ぐ](/forge-labs-blog/2026/06/21/near-miss-ai.html)

## 注意事項

- KY活動はAIで「漏れなく危険を出す」補助として使い、**現場のメンバー全員で話し合うプロセスは省略しない**でください
- 実際のKY活動の内容は現場の状況・経験者の知識が最も重要です。AIの出力はあくまで参考・たたき台です
- 安全に関わる記録の保管・管理は社内の安全管理規程・ISO規格に従って行ってください
