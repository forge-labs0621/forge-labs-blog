製造業の排水処理は水質汚濁防止法・下水道法による規制を受けます。AIを使って排水処理の点検記録・水質分析結果の管理・異常時の対応手順を体系的に整備できます。

---

## 排水処理設備の日常点検記録を整理するプロンプト

設備管理と記録を標準化します。

```
以下の情報をもとに、排水処理設備の
日常点検記録書類を整理してください。

【当社の排水処理設備の概要】
業種: 金属加工（切削・研磨・めっき前処理）
排水の種類:
  ①切削油廃液（エマルション系）: 150L/日
  ②研磨廃液（SS・金属粉含有）: 50L/日
  ③酸性洗浄廃液（pH 2〜4程度）: 30L/日
  ④一般排水（冷却水・床洗浄水）: 500L/日

排水処理設備:
  ①中和処理槽（酸/アルカリ中和）: pH調整後 5.8〜8.6の範囲に
  ②凝集沈殿槽（SS除去・フロック形成）
  ③フィルタープレス（汚泥脱水）
  ④最終pH調整槽

放流先: 公共下水道（条例排水基準あり: pH 5〜9、SS 600mg/L以下）

出力形式:
・「排水処理設備の日常点検チェックシート
  （各設備の点検項目 / 判定基準 / 異常時の対処）」
・「排水のpH日常測定記録シート
  （測定時刻 / 測定場所 / 測定値 / 基準値との比較 / 処置記録）」
・「設備トラブル時の緊急対応フロー
  （処理能力低下 / pH異常 / 設備停止 の場合の手順）」
・「汚泥発生量の月次記録と産業廃棄物としての管理方法
  （汚泥の性状 / 発生量 / マニフェスト管理との連携）」
・「定期水質分析（外部機関委託）の頻度・検査項目・記録の管理方法」
```

## 水質汚濁防止法に基づく報告書類を整理するプロンプト

行政への届出・報告を漏れなく管理します。

```
以下の情報をもとに、水質汚濁防止法に
基づく届出・報告書類を整理してください。

【当社の法的な位置づけ】
特定施設: めっき前処理設備（酸洗い設備）を保有
法的分類: 水質汚濁防止法 特定施設（金属等を含む排水を排出する施設）

届出・報告の義務:
  ①特定施設の設置届出（施設設置時 → 60日前までに届出済み）
  ②排水基準の遵守（特定排水基準: pH / 浮遊物質量 / 重金属等）
  ③事故時の届出（排水基準超過・油の流出等の際は即時届出）
  ④定期水質測定の実施と記録の3年保管

昨年の問題:
  ①4月に pH基準を1度超過（pH 9.1）→ 設備調整後 翌日には基準内
  ②超過発生時の行政への報告手順が担当者に周知されていなかった
  ③測定記録が担当者個人のExcelにのみ保存（引き継ぎ体制なし）

出力形式:
・「水質汚濁防止法に基づく届出・報告の種類と手順の一覧表」
・「排水基準超過時の緊急対応・行政届出手順書
  （発見 → 排水停止 → 採水保管 → 行政連絡 → 原因調査 → 是正報告）」
・「排水基準超過時の行政（都道府県）への連絡文の文例」
・「定期水質測定記録の組織的な管理方法
  （クラウド保存 / 引き継ぎ手順 / 3年保管の確認方法）」
・「行政の立入検査に備えた記録の整理方法と点検チェックリスト」
```

## 廃液の適正処理・廃棄物業者管理を整理するプロンプト

廃液の産業廃棄物として適切に管理します。

```
以下の情報をもとに、切削油廃液・洗浄廃液の
適正処理と廃棄物業者管理書類を整理してください。

【廃液の発生状況と委託処理の実態】
発生する廃液の種類と量:
  ①使用済み切削油（廃油）: 200L/月（産廃区分: 廃油）
  ②切削廃液（水溶性廃液 / エマルション排液）: 500L/月（廃酸・廃アルカリ）
  ③脱脂洗浄廃液: 100L/月（廃アルカリ）

委託処理業者: 〇〇産業廃棄物処理（許可番号 保有）
収集運搬: △△運送（産廃収集運搬許可 保有）
処理方法: 焼却処分（廃油） / 中和処理後 廃水処理（廃液）

過去の管理上の問題:
  ①一部の廃液を「廃水処理の余剰」として下水へ放流していた
    （実際は産廃として処理が必要だった可能性あり）
  ②マニフェスト（E票）の返送を受け取っていない案件が3件あった
  ③処理業者の許可証の有効期限の確認を行っていなかった

出力形式:
・「廃液の産業廃棄物区分の整理
  （廃油・廃酸・廃アルカリの違いと適正な処理方法）」
・「電子マニフェストの活用方法と書面マニフェストの管理手順
  （E票の受取確認・5年保管の義務）」
・「廃棄物処理業者の許可証確認チェックリスト
  （許可の種類 / 有効期限 / 処理可能物の確認）」
・「廃液発生から処理完了までの管理フロー
  （発生 → 一時保管 → 収集運搬 → 処理 → マニフェスト確認）」
・「廃液管理台帳のフォーマット
  （発生日 / 種類 / 量 / 業者名 / マニフェスト番号 / 処理完了確認日）」
```
