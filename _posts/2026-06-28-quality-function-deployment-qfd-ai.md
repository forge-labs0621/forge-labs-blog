顧客の「声」を製品の設計品質に落とし込む品質機能展開（QFD）。AIを使ってVoC（顧客の声）の整理・品質表の構造設計・技術特性への展開を効率化することで、「作ったけど使いにくい」製品ミスを防止できます。

---

## 顧客の声（VoC）を整理するプロンプト

顧客要求を品質展開の入力として整理します。

```
以下の情報をもとに、製品開発に向けた
顧客の声（VoC）の整理と要求品質への展開を整理してください。

【収集した顧客の声（生の声）】
製品: 工場内搬送用アルミ製架台（新製品開発中）

顧客ヒアリング結果（3社×担当者2名へのインタビュー）:
①生産ライン担当者:
  「今使っている架台は重くて移動が大変。
   一人でも動かせる重量にしてほしい」
  「角が鋭くて作業者が手を切ることがある。
   安全面を改善してほしい」
  「高さを変えられると便利。ラインの作業者の
   身長に合わせたい」

②購買・調達担当者:
  「現在の架台は5年で錆びてしまう。
   もっと耐久性があるものが欲しい」
  「コストを下げてほしい（現在単価¥38,000 → ¥32,000以下）」

③品質・安全管理担当者:
  「転倒防止の機能がほしい。重い部品を乗せたとき
   倒れかけることがある」
  「清掃しやすい構造にしてほしい」

出力形式:
・「生の声を顧客要求（一次品質要求）に整理した一覧
  （重複の統合・曖昧表現の明確化）」
・「顧客要求を機能・使いやすさ・安全性・耐久性・コストの
  視点でグルーピング」
・「各要求の重要度の評価（顧客の言い方・頻度・競合比較から）」
・「品質表（QFD第1段階）の行（顧客要求）と重要度の整理表」
・「最も優先すべき顧客要求TOP5の選定根拠」
```

## 品質表（ハウス・オブ・クオリティ）の構造設計プロンプト

顧客要求と技術特性の関連付けを整理します。

```
以下の情報をもとに、品質表（QFD第1マトリクス）の
主要項目を整理してください。

【顧客要求（整理済み）】
  ①一人で移動できる重量（目標30kg以下）
  ②角が安全（エッジ処理・面取り）
  ③高さが調節できる
  ④耐久性が高い（屋内10年使用）
  ⑤転倒しにくい
  ⑥清掃しやすい

【技術特性（工学特性）の候補】
  a. 架台本体の重量（kg）
  b. エッジのR寸法（mm） / 面取りの角度
  c. 高さ調整機構の段数と調整範囲（mm）
  d. 表面処理の種類（アルマイト・粉体塗装等）とサビ試験基準
  e. 転倒モーメント / 安定性係数
  f. 洗浄可能かどうか（防水等級・分解のしやすさ）

競合製品の技術データ（参考）:
  競合A: 重量35kg・エッジR2mm・高さ3段調整・亜鉛メッキ処理
  競合B: 重量28kg・エッジR5mm・固定高さ・アルマイト処理

出力形式:
・「品質表マトリクス（顧客要求×技術特性の関係表）」
  （◎強い関係・○中程度・△弱い関係・空白＝無関係）
・「技術特性間の相関（プラス/マイナス相関の整理）」
  （例: 重量を下げる←→強度を保つ はトレードオフ）
・「各技術特性の設計目標値（競合比較と顧客要求から設定）」
・「設計目標の達成難易度と優先度の評価」
・「品質表を使った開発方針の要約
  （何に集中して開発するか 1〜2段落）」
```

## 工程展開（QFD第3段階）を整理するプロンプト

設計品質を製造工程に展開します。

```
以下の情報をもとに、品質特性から
製造工程への展開（QFD第3段階）を整理してください。

【製品の主要品質特性（設計で決定）】
  a. 重量: 26kg以下（アルミ構造材の使用）
  b. エッジ処理: R3mm以上の面取り（全面）
  c. 高さ調整: 680/780/880mm の3段階（ピンロック式）
  d. 表面処理: アルマイト（膜厚15μm以上）
  e. 安定性: 前傾30°でも転倒しない構造

【製造工程（概要）】
  1→ アルミ材料の切断（レーザー切断）
  2→ 曲げ加工（プレスブレーキ）
  3→ 面取り・バリ取り（手仕上げ）
  4→ 溶接・組立
  5→ アルマイト処理（外注）
  6→ 検査・梱包

出力形式:
・「各品質特性に影響する工程の特定
  （品質特性×工程の関係マトリクス）」
・「各工程での管理特性（何を測定・確認するか）」
  （例: 工程3 → 面取りR寸法をノギスで確認・全数）
・「工程管理の管理点一覧（工程名・管理特性・規格・
  測定方法・頻度・管理者）」
・「QC工程表（コントロールプラン）の骨子」
・「製造現場に展開するための教育資料（設計品質の背景を
  作業者に伝えるポイント）」
```
