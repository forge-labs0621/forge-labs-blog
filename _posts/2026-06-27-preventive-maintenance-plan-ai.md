「設備が壊れてから直す（事後保全）」ではなく、「壊れる前に手を打つ（予防保全）」に取り組む製造業が増えています。AIを使うと、保有設備の保全項目を整理したり、年間の保全スケジュールの骨子を作ったりする作業を効率化できます。

## 予防保全計画書で整理すべき内容

| 項目 | 内容 |
|---|---|
| 保全対象設備の一覧 | 対象設備・管理番号・担当者 |
| 点検・保全の種類と周期 | 日常点検・月次点検・年次点検・オーバーホール |
| 各保全項目の作業内容 | 清掃・給油・交換部品・点検方法 |
| 年間スケジュール | いつ・誰が・何をするか |
| 費用見込み | 消耗品・外注費用の概算 |

## 設備ごとの保全項目リストを整理するプロンプト

バラバラのメモや口頭情報を、設備の保全計画表として整理します。

```
以下の設備情報をもとに、予防保全の点検・作業項目リストを
作成してください。

【設備名・型式】
（例：NC旋盤 Model-B（△△機械工業製、2018年導入））

【メーカー推奨の保全項目（わかる範囲で）】
（例：
・日常点検: 起動前に主軸・チャック・刃物台の状態確認
・月次保全: 各摺動面への給油、フィルター清掃
・年次保全: 主軸精度確認、ベルト張り具合確認、全軸精度チェック
・3年に1回: オーバーホール（外注推奨）
）

【これまでの故障・修繕履歴（わかる範囲で）】
（例：
・2022年: 主軸ベアリング交換（突発停止後の修繕）
・2024年: 刃物台の位置ずれ → 調整（早期発見で軽微な修繕で済んだ）
）

出力形式:
・「○○ 保全計画表」として
・点検周期別（日常/月次/四半期/年次/3年）に保全項目を整理
・各項目に「作業内容 / 担当（社内/外注） / 必要な部品・消耗品」を付ける
・過去の故障履歴から「特に注意すべきポイント」を1〜2点コメントとして追記
```

## 複数設備の年間保全スケジュール骨子を作るプロンプト

複数の設備を一覧化して、年間のメンテナンスカレンダーの骨子を作ります。

```
以下の設備リストと保全周期をもとに、年間保全スケジュール（カレンダー形式）
の骨子を作成してください。

【保有設備と主な保全周期（わかる範囲で記入）】
（例：
・NC旋盤3台: 月次点検（自社）、年次精度チェック（外注）
・フライス盤2台: 月次点検（自社）、年次（自社）
・プレス機1台: 月次（自社）、年次点検（外注メーカー）、
  2年に1回プレス定期自主検査（法定）
・コンプレッサー1台: 月次フィルター清掃（自社）、年次安全弁検査（外注）
・天井クレーン1台: 月次（自社）、年次定期自主検査（有資格者）
・フォークリフト2台: 月次（自社）、年次特定自主検査（有資格者）
）

【スケジュール上の制約】
（例：
・8月は夏季休業（8/11〜8/15）のため、大型保全作業は8/11〜15に集中させたい
・年末は12/28〜1/5が休業
・法定点検の実施時期が決まっているものはその時期に合わせる
）

出力形式:
・「年間保全スケジュール（○○年）」として
・1〜12月の横軸、設備名の縦軸の表形式
・法定点検（★マーク）と自主保全を区別して記載
・各月の保全作業の件数・概算工数を下に集計
・「詳細な法定点検の要件は所轄官庁・専門業者に確認すること」の注記
```

## 外注保全業者への作業依頼文を作るプロンプト

専門業者に年次点検・オーバーホールを依頼する連絡文の下書きです。

```
以下の情報をもとに、設備保全業者への作業依頼メールを作成してください。

【依頼先】
（例：△△機電サービス株式会社 保全サービス部 鈴木様）

【依頼する作業内容】
（例：NC旋盤（Model-B、管理番号: M-2018-003）の年次点検・精度チェック）

【希望する作業日】
（例：2026年8月12日（水）または8月13日（木）のいずれか
（夏季休業期間中を希望））

【確認してほしいこと・特記事項】
（例：
・今年はスピンドルの振動が少し増えた感覚がある。精度チェック時に
  要確認とお伝えしたい
・前回（2025年8月）の実施報告書を参照いただきたい
・費用見積もりを先にいただいてから正式発注したい
）

出力形式:
・件名: 「年次点検作業のご依頼・日程確認のお願い」
・依頼内容（設備名・希望日・特記事項）を箇条書きで明確に
・見積もりを先にいただきたい旨
・回答の期限（例：7月末まで）
```

## 保全費用の年間予算見積もりコメントを作るプロンプト

年度予算の編成時に、設備保全費用の見積もりコメントを整理します。

```
以下のデータをもとに、設備保全費用の年間予算（来年度）のコメントを
作成してください。

【今年度の保全実績（参考）】
（例：
・外注保全費: ¥850,000（年次点検5設備、緊急修繕1件含む）
・消耗品・部品費: ¥320,000（オイル・フィルター・ベルト等）
・合計: ¥1,170,000
）

【来年度の増減要因】
（例：
・旋盤1台（2012年導入）のオーバーホールが来年度必要（費用見積もり約¥600,000）
・一方、昨年発生した緊急修繕（¥180,000）は再発しない見込み
・新規導入設備1台が追加（年次点検費用 約¥80,000増）
）

出力形式:
・「○年度 設備保全費用 予算案」として
・今年度実績と来年度予算の比較表
・増減の主な理由（箇条書き）
・「オーバーホールは外注見積もりを取得した上で正式予算化する」等の注記
```

## AIを使う際の注意

- 設備の法定点検（クレーン・プレス・フォークリフト等）の実施頻度・方法・記録保管は法令で定められています。法定要件の確認は官公庁・専門業者に依頼してください
- 消耗品の交換周期はメーカー推奨値を参考にしつつ、実際の使用状況・稼働時間・故障履歴に合わせて調整する必要があります
- 保全計画は作成して終わりではなく、実施後の記録（何を・いつ・誰が実施したか）を残すことが重要です
