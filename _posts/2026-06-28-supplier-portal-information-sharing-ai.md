製造業では仕入先・外注先との情報連携の質が品質と納期に直結します。AIを使って仕入先向け規格書・情報共有ルール・購買条件書・品質要求仕様書を効率的に整備できます。

---

## 仕入先向け品質要求仕様書を整理するプロンプト

仕入先に伝えるべき品質基準を明文化します。

```
以下の情報をもとに、仕入先向け品質要求仕様書
（サプライヤー品質マニュアル）を整理してください。

【当社の仕入先への品質要件】
当社業種: 精密板金加工（受託製造）
主要仕入品: 鋼板・アルミ板・ステンレス板（材料）、ネジ・ボルト（購入部品）

現在の課題:
  ①材料の受入検査で寸法不良・表面欠陥が月2〜3件発生
  ②納品時に成績書（材料証明書）が添付されていないことがある
  ③不適合材料の返品・交換が属人的で処理が遅い

当社が仕入先に求める主な要件:
  ・各納品ロットへの検査成績書の添付
  ・JIS規格への適合確認（鋼板: JIS G 3141等）
  ・サンプル品の事前承認（初回納品前）
  ・4M変更（材料種・製造方法・製造元・仕様変更）の事前報告義務
  ・不適合品発生時の速報（24時間以内）と8D報告提出

出力形式:
・「仕入先向け品質要求仕様書の構成と各章の記載内容
  （品質要求 / 納品規則 / 不適合品処置 / 変更管理）」
・「受入検査項目一覧と仕入先への提出書類一覧
  （品番ごとの検査項目 / 添付必要書類）」
・「4M変更報告書のフォーマット（仕入先が記入する様式）」
・「不適合品発生時の連絡フォーマット（速報 + 是正報告の2段階）」
・「新規仕入先評価（サプライヤー認定）フローの骨子」
```

## 仕入先との情報共有・コミュニケーション体制を整理するプロンプト

情報共有の抜け漏れをなくします。

```
以下の情報をもとに、仕入先との情報共有
体制と連絡ルールを整理してください。

【現在の仕入先との情報共有の問題】
仕入先数: 38社（主要5社・準主要10社・その他23社）

現在の連絡方法:
  ・発注: FAX（主流）/ メール（一部）
  ・品質連絡: 電話 + メール（担当者個人のアドレス）
  ・図面共有: 紙の図面を郵送 or メールでPDF添付
  ・仕様変更通知: 担当者が個別電話

問題点:
  ①FAXが届いているか確認できない（見落としリスク）
  ②担当者が不在の場合、情報が伝わらない
  ③図面の改訂版が仕入先に正しく届いているか管理できていない
  ④発注残・納期状況をリアルタイムで把握できない

出力形式:
・「仕入先ランク別のコミュニケーション方針
  （主要5社 / 準主要10社 / その他23社 でルールを分ける）」
・「情報共有ツール選定の評価基準
  （Excel共有 / クラウドストレージ / ポータルサービス の比較）」
・「図面・仕様書の改訂管理ルール
  （版管理の仕組みと仕入先への通知方法）」
・「発注・納期管理のExcelテンプレート骨子
  （発注番号 / 品番 / 数量 / 要求納期 / 回答納期 / 状況フラグ）」
・「仕入先との月次定例ミーティングのアジェンダ雛形（主要5社向け）」
```

## 仕入先コスト交渉・購買条件書を整理するプロンプト

購買条件を文書で明確にします。

```
以下の情報をもとに、仕入先との購買条件書と
コスト交渉の準備書類を整理してください。

【購買条件の見直し背景】
状況: 円安・原材料高騰により、主要仕入先から値上げ要求を受けている
値上げ要求の内容:
  ・A社（鋼板）: 現行価格比 +15%（鉄スクラップ上昇理由）
  ・B社（アルミ板）: +12%（LMEアルミ価格連動）
  ・C社（ネジ類）: +8%（人件費・エネルギー費上昇）

当社の対応方針:
  ①値上げを一部受容するが、全額は受け入れない
  ②価格上昇分をコスト改善で相殺できないか確認
  ③長期契約（1年間価格固定）と引き換えに数量コミットを提案

出力形式:
・「仕入先への値上げ要求に対する回答書の雛形
  （受容可能範囲 / 交渉余地のある項目 / 条件提示の構成）」
・「購買条件書（基本取引条件）の骨子
  （支払い条件 / リードタイム / 最低発注量 / 価格変動条項）」
・「コスト削減要求（VR活動）の依頼書のフォーマット
  （どこを・どれだけ・いつまでに削減してほしいかの依頼文）」
・「原材料価格連動条項の設計方法
  （LME等の公示価格に連動した価格変動ルール）」
・「複数社見積り取得による競争原理の活用方法（相見積もり戦略）」
```
