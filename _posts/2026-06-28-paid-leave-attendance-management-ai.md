2019年の労働基準法改正から「年10日以上の有給休暇が付与された従業員には年5日の取得義務」が課されました。製造業では繁忙期との調整が難しく、AIで管理台帳・個別周知・取得計画を効率化できます。

---

## 有給休暇の付与・残日数管理を整理するプロンプト

従業員ごとの有給付与と残日数を一括管理します。

```
以下の情報をもとに、従業員の年次有給休暇の
付与・残日数・取得義務の管理表を整理してください。

【会社の有給付与ルール】
付与日: 入社日の6ヶ月後に初回付与、以降1年ごとに更新
繰越: 2年間（当年付与分 + 前年繰越分が上限）
付与日数: 労働基準法に準拠（下記の法定日数）
  入社6ヶ月: 10日 / 1年6ヶ月: 11日 / 2年6ヶ月: 12日
  3年6ヶ月: 14日 / 4年6ヶ月: 16日 / 5年6ヶ月: 18日
  6年6ヶ月以上: 20日

【従業員の現状データ】（一部抜粋）
  従業員A: 入社2019年4月1日 / 直近付与日2025年10月1日（6年6ヶ月） /
           付与日数20日 / 繰越残10日 / 今年度使用済み3日
  従業員B: 入社2023年10月1日 / 直近付与日2025年4月1日（1年6ヶ月） /
           付与日数11日 / 繰越残0日 / 今年度使用済み1日
  従業員C: 入社2024年4月1日 / 直近付与日2025年10月1日（1年6ヶ月） /
           付与日数11日 / 繰越残0日 / 今年度使用済み0日

出力形式:
・「従業員別有給休暇管理台帳（付与日・付与数・繰越・
  使用数・残日数・取得義務残日数・期限）の一覧表」
・「5日取得義務の残日数計算（各従業員の義務達成状況）」
・「取得義務の期限が近い（3ヶ月以内）従業員へのアラートリスト」
・「有給取得義務を果たすための個別指示文書テンプレート」
・「出勤簿と連動させる月次更新フローの説明」
```

## 有給取得計画表（年間計画）の整理プロンプト

繁忙期を避けながら年5日を消化させる計画を立てます。

```
以下の情報をもとに、製造部門の
年間有給休暇取得計画を整理してください。

【生産カレンダーの繁忙パターン】
繁閑のパターン（前年実績）:
  閑散期: 1〜2月・6〜7月（受注量が平均比-20〜-30%）
  標準期: 3〜5月・8〜9月
  繁忙期: 10〜12月（自動車部品の年度末対応・受注量平均比+40%）

有給が取りにくい時期（製造部の実情）:
  ・11〜12月（繁忙期のため全員出勤が必要）
  ・毎月月末（締め日の出荷集中）

【現在の有給取得状況（昨年度）】
  ・従業員35名中、年5日取得義務を果たせなかった: 8名（23%）
  ・原因: 「言い出しにくい」「繁忙で断られる」

出力形式:
・「推奨有給取得月（閑散期集中）の設定と案内文」
・「計画的付与（会社が取得日を指定する）の活用方法と
  就業規則への記載例」
・「班・シフト別の取得分散計画（誰かが必ず出勤している状態）」
・「部門長向けの有給管理チェックリスト（月1回確認）」
・「年5日未取得リスクの早期アラート設計
  （7月時点・10月時点のチェックポイント）」
```

## 出勤簿・タイムカード管理を整理するプロンプト

法定帳票としての出勤記録を管理します。

```
以下の情報をもとに、出勤簿の
管理方法と法的要件を整理してください。

【当社の出勤管理の現状】
記録方法: タイムカード（打刻式）
保管状況: 1〜2年分は事務所に保管、それ以降は廃棄していた
問題点: 労基署の調査で「3年間の保管義務があるのに2年分しかない」
       と指摘を受けた（是正勧告ではなく口頭注意）

法定要件（確認が必要な事項）:
  ・出勤簿の保存期間: 労働基準法109条（5年間・当分の間は3年）
  ・記録すべき内容: 出勤日・退勤日・労働時間（残業含む）
  ・有給取得日の記録方法（有給か欠勤かの区別）

出力形式:
・「出勤簿の法的記録要件の整理（記録項目・保存期間・保管方法）」
・「タイムカードのデジタル化・スキャン保存の適法性の確認ポイント」
・「有給休暇の取得理由を記載する必要がないことの確認と
  取得記録の残し方」
・「出勤簿と残業時間記録の突合確認フロー（月次）」
・「労基署調査に備えた書類整理の優先順位
  （出勤簿・36協定・給与台帳の3点セット）」
```
