製造業では保有する設備・機械を一覧で管理する「設備台帳」と、減価償却・税務申告に使う「固定資産台帳」の2種類の台帳管理が必要です。AIを使うと、台帳に記載するコメント・特記事項の文章化や、設備情報の整理・棚卸しを効率化できます。

## 設備台帳と固定資産台帳の違い

| | 設備台帳（現場管理用） | 固定資産台帳（会計・税務用） |
|---|---|---|
| 主な目的 | 設備の場所・状態・点検履歴を管理 | 減価償却計算・固定資産税申告 |
| 主な管理者 | 製造部門・総務 | 経理・会計担当 |
| 主な記載内容 | 設備名・設置場所・点検日・担当者 | 取得価額・耐用年数・減価償却累計額 |
| AIが特に役立つ部分 | 特記事項・メモの文章化・台帳整理 | 項目の整理・コメント生成（計算は除く） |

## 設備台帳の記載内容を整理するプロンプト

---

```
以下の情報をもとに、製造現場の設備台帳の1行分（1設備の記録）を
整理してください。

【設備の基本情報】
設備名・型番: （例：NC旋盤 ○○型 Model-B）
製造メーカー: （例：△△機械工業）
取得年月: （例：2018年3月）
設置場所: （例：第1製造棟 旋盤エリア No.3）
管理番号: （例：M-2018-003）

【使用状況・状態のメモ】
（例：
・主力の中型部品加工に使用。週40〜45時間稼働
・2024年にスピンドルのベアリングを交換済み
・現在、若干の振動があり要観察中
・予防保全の点検: 年2回（6月・12月）実施
）

出力形式:
・設備台帳の「特記事項」欄に記入できる形で整理
  （A4半ページ以内・箇条書き可）
・「保全履歴の要点」「現状の課題・観察事項」「次回点検予定」の3ブロック
```

## 複数設備の情報を台帳用に一覧整理するプロンプト

バラバラのメモや口頭情報を台帳形式に整えます。

```
以下のメモをもとに、設備台帳の一覧表に使える情報を整理してください。

【設備情報のメモ（バラバラで構いません）】
（例：
・旋盤3台（2台は2015年導入、1台は2020年に更新。旧型2台は老朽化が進んでいる）
・フライス盤2台（どちらも2010年代導入。1台はNCでもう1台は汎用）
・プレス機1台（2019年導入の油圧プレス。能力50t）
・コンプレッサー1台（エアーツール用。年1回の法定点検実施）
・クレーン1台（天井クレーン。2tクレーン法適用）
）

出力形式:
・「No. / 設備名 / 型式（わかる範囲） / 取得年（推定） / 台数 / 
   メモ・特記事項」の6列で表にまとめる
・法定点検が必要な設備には「★法定点検あり」と明記する
・今後確認が必要な情報（不明な値）は「要確認」と記載する
```

## 設備更新・廃棄の記録を整理するプロンプト

古い設備を廃棄・更新する際の記録作成に使えます。

```
以下の情報をもとに、設備の廃棄・更新の記録（申請書または報告書）を
作成してください。

【廃棄・更新する設備の情報】
設備名・管理番号: （例：NC旋盤 M-2015-001）
廃棄・更新の区分: （廃棄のみ / 更新設備と入れ替え）
廃棄予定日: （例：2026年8月末）

【廃棄・更新の理由】
（例：
・取得から10年以上経過し、スピンドルの精度が劣化
・修繕費用が更新費用を上回ると判断
・後継機（型番○○）を2026年7月に導入予定
）

【廃棄方法】
（例：産業廃棄物として専門業者に依頼 / スクラップ業者に売却）

出力形式:
・「設備廃棄（更新）記録書」として
・廃棄理由・設備情報
・廃棄方法・廃棄業者（プレースホルダー）
・固定資産台帳からの除却申請先（経理部門）への連絡事項
・申請者・承認者・廃棄完了日欄
```

## 法定点検が必要な設備のチェックリストを作るプロンプト

製造業では法令で点検が義務付けられている設備があります。

```
以下の設備リストから、法定点検が必要な設備を抽出し、
点検管理チェックリストを作成してください。

【保有設備のリスト】
（例：NC旋盤3台、フライス盤2台、天井クレーン1台（2t）、
       プレスブレーキ1台、コンプレッサー1台（空気圧縮機）、
       フォークリフト2台、ボイラー1台（小型温水ボイラー）、
       消火器12本（5F建て）、防火シャッター3基）

各設備について:
・法定点検の有無（労働安全衛生法・消防法・フロン排出抑制法等）
・点検頻度の目安（毎日・月次・年次・2年に1回等）
・点検主体（有資格者が必要か・社内点検でよいか）
を整理してください。

注意: 詳細な法的要件は必ず官公庁の情報・専門業者に確認してください。
AIの回答は参考情報です。
```

## AIを使う際の注意

- 設備台帳の「取得価額・減価償却」に関わる計算はAIに任せず、会計ソフトまたは税理士が担当してください
- 法定点検の要否・頻度は法令改正で変わることがあります。最終判断は官公庁の情報または専門家に確認してください
- 設備台帳のデータ（設備情報・点検記録）はAIに入力する際、外部サービスへの送信になることを意識してください
