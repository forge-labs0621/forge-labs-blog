製造業でも人材育成・定着のために人事評価制度の整備が重要になっています。評価シートの作成・MBO（目標管理）の目標設定・評価面談の記録をAIで効率化できます。

---

## 職種別の人事評価シートを作るプロンプト

製造現場・技術・事務の職種ごとに人事評価シートを作ります。

```
以下の情報をもとに、製造業の○○職向けの人事評価シートを
作成してください。

【評価対象の職種】
（以下から選択して記載してください）
・製造現場（オペレーター・加工担当）
・班長・リーダー（現場監督職）
・品質保証担当
・営業・受注担当
・設計・技術担当
・管理部門（総務・経理）

例として「製造現場（オペレーター）」を選択する場合の情報:

【評価の対象者・制度の概要】
評価対象: 製造現場の一般作業者（班長以外）
評価時期: 年2回（6月・12月）
評価者: 直属の班長（一次評価）/ 製造部長（二次評価）

【評価したい要素】
業績評価（成果）:
  ・生産数量・品質（不良率）の実績
  ・目標達成度
能力評価（スキル・技能）:
  ・機械操作の熟練度（1〜4段階評価を想定）
  ・品質意識（不良の検出・報告）
  ・安全遵守（安全作業・5S）
情意評価（態度・姿勢）:
  ・勤怠（出勤率・時間厳守）
  ・チームワーク・協力性
  ・改善提案への積極性

出力形式:
・A4 1枚に収まる評価シートのフォーマット
・各項目を「5段階評価（1〜5）+ コメント欄」の形式
・「評価基準の目安（3点の基準）」を各項目に付記
・総合評価・コメント欄・評価者サイン欄を末尾に
```

## MBO目標設定シートを作るプロンプト

個人ごとの目標管理（MBO）の目標設定書類を作ります。

```
以下の情報をもとに、MBO（目標による管理）の
目標設定シートと目標案を作成してください。

【対象者・期間】
対象者: （例：製造部 班長 ○○（勤続8年・現場リーダー歴3年））
目標設定期間: （例：2026年10月〜2027年3月（下期））
上司との面談日: （例：2026年9月25日）

【会社・部門の方針（上位目標）】
（例：
・製造部門目標: 不良率0.3%以下 / 生産性前年比5%向上
・品質: ISO 9001の内部監査不適合ゼロ
）

【個人の役割・期待値】
（例：
・工程管理: 担当ラインの生産計画の達成（遅延なし）
・品質管理: 自ラインの不良率を0.5%→0.3%に改善
・後輩育成: OJT担当として若手1名の技能評価を"一人前"にする
・改善活動: 月1件以上の改善提案を提出する
）

出力形式:
・MBO目標設定シートのフォーマット
・各目標について「目標内容 / 指標（KPI） / 達成期限 / 測定方法」を記載
・SMART原則（Specific / Measurable / Achievable / Relevant / Time-bound）で
  整理した目標の文例
・「目標達成に必要な支援（上司に依頼したいこと）」の欄を含める
```

## 評価面談のフィードバックメモを整理するプロンプト

評価面談の結果をフィードバックレポートとして整理します。

```
以下のメモをもとに、評価面談のフィードバックレポートを
整理してください。

【面談の概要】
対象者: （例：製造部 オペレーター ○○（入社3年目））
面談者: （例：班長 ○○）
面談日: （例：2026年6月30日）

【面談で出た内容（メモ）】
評価結果の伝達:
  （例：「今期の総合評価は3（標準）。品質面は良くなっている。
  遅刻が2回あったのが残念だった」）
本人の振り返り:
  （例：「遅刻は気をつける。仕事はやりがいを感じている。
  もっと難しい加工に挑戦したい」）
来期の目標:
  （例：「難削材（SUS）の加工を習得したい。班長代理を経験したい」）
上司として伝えたかった点:
  （例：「勤怠を改善すれば来期には班長候補として考えたい。
  SUSの習得は来期OJT計画に組み込む」）

出力形式:
・評価面談レポート（担当者の記録として残す形式）
・「本人の強み・改善点」「来期への期待・目標」を整理
・「上司からの具体的な支援約束（OJT計画に組み込む等）」の記録
・本人サイン欄・日付
```
