QC工程表（品質管理工程表）・コントロールプラン（管理計画書）は、製造業の品質管理における中核書類です。各工程で「何を」「どのように」管理するかを一覧化したもので、ISO 9001・IATF 16949での提出・維持が求められます。AIでの整理・更新作業を効率化できます。

---

## QC工程表の骨子を作るプロンプト

製品の全工程を網羅したQC工程表（品質管理工程表）を作ります。

```
以下の情報をもとに、QC工程表（品質管理工程表）を作成してください。

【製品情報】
品番・品名: （例：A-1234 プレスブラケット（亜鉛めっき仕様））
材料: （例：SPCC t1.6mm）
適用規格・取引先要求: （例：取引先仕様書 No.○○ / ISO 9001準拠）

【製造工程の流れ（順番に記載）】
工程1: 材料受入・受入検査
  ・管理項目: 材料規格（規格証明書確認）/ 外観（さび・傷）/ 寸法（厚み）
  ・管理方法: 規格証明書の確認 + 目視検査 + ノギス計測
  ・記録: 受入検査記録簿
  
工程2: プレス加工（ブランク抜き）
  ・管理項目: 外形寸法 / 穴位置 / バリの大きさ
  ・管理方法: 初物確認（毎ロット冒頭5個の寸法測定）/ 以降サンプリング
  ・記録: 初物確認書 / 工程内検査記録
  
工程3: プレス加工（曲げ加工）
  ・管理項目: 曲げ角度 / 曲げ後の外形寸法
  ・管理方法: 専用ゲージ確認 / 定期サンプリング（2時間おき）
  ・記録: 工程内検査記録

工程4: 脱脂・洗浄
  ・管理項目: 洗浄液の濃度・温度 / 処理時間
  ・管理方法: 洗浄液を毎日測定・記録 / 処理時間のタイマー管理
  ・記録: 洗浄液管理記録

工程5: 外注（亜鉛めっき）
  ・管理項目: めっき厚み（最低5μm以上） / 外観（はがれ・まだら）
  ・管理方法: 外注仕様書で管理（仕入先管理） / 受入後にサンプリング確認
  ・記録: 外注業者の検査成績書

工程6: 最終検査・出荷
  ・管理項目: 全寸法（抜取）/ 外観全数 / 特性記録（CoC発行）
  ・管理方法: 抜取検査（AQL適用）/ 全数外観確認
  ・記録: 出荷検査記録 / 品質成績書（CoC）

出力形式:
・QC工程表の表形式（工程番号 / 工程名 / 管理項目 / 管理方法 /
  管理基準 / 検査頻度 / 使用設備・ゲージ / 記録書類）
・「管理基準欄」に規格値のプレースホルダーを設ける
・工程内不良が発生した場合の「処置欄」も含める
```

## コントロールプラン（管理計画書）の骨子を作るプロンプト

IATF 16949などで求められるコントロールプランをQC工程表から展開します。

```
以下のQC工程表の情報をもとに、コントロールプラン
（管理計画書・Control Plan）のフォーマットを作成してください。

【製品情報】
品番: （例：A-1234）
品名: （例：プレスブラケット）
取引先: （例：○○自動車工業）
コントロールプラン番号: （例：CP-A1234-001）
作成日: （例：2026年6月28日）

【主要工程の管理情報（QC工程表から転記）】
工程No.2 プレス加工:
  ・管理特性（特殊特性か否か）: 外形寸法（一般特性）/ 穴位置（特殊特性 ★）
  ・規格値: （例：外形±0.5mm / 穴位置±0.1mm）
  ・測定システム（使用計測器）: ノギス / 穴位置ゲージ
  ・サンプルサイズ・頻度: 5個 / 2時間ごと
  ・管理方法: X-bar R管理図 / 専用ゲージ
  ・反応計画（規格外の場合）: ラインを止めて班長に報告 / 原因追求

出力形式:
・AIAG形式に準拠したコントロールプランのフォーマット
  （部品番号 / 工程番号 / 工程名 / 管理特性 / 特殊特性の識別 /
  仕様・公差 / 評価・測定方法 / サンプルサイズ・頻度 /
  管理方法 / 反応計画）
・「特殊特性（★印など）」の識別方法の説明
・「反応計画（不適合発生時の対応手順）」の記載例
```

## QC工程表の変更・改訂プロンプト

4M変更（人・機械・材料・方法の変更）に伴うQC工程表の更新箇所を整理します。

```
以下の変更内容をもとに、QC工程表の改訂が必要な箇所を
特定し、改訂案を作成してください。

【変更の内容（4M変更）】
変更の種類: （例：設備変更（M: Machine））
変更内容: 
  （例：プレス工程で使用していたNC-1000プレス機（200ton）を、
  NC-1500プレス機（300ton）に更新した。
  加工速度が15%向上、段取り時間が20分→12分に短縮。）
変更実施日: （例：2026年7月1日）
変更の影響:
  ・加工精度への影響: なし（取引先の検証試作で確認済み）
  ・管理方法への影響: 初物確認の頻度を毎ロット→最初の3ロットで確認して安定を確認後、
    通常通りに戻す（暫定措置）
  ・リスク: 立上げ初期は不具合の可能性があるため、初期流動管理を実施

出力形式:
・QC工程表の「改訂が必要な工程・項目」の特定
・改訂前後の対比（変更前の記載 → 変更後の記載）
・「初期流動管理計画（立上げ初期のモニタリング強化計画）」の骨子
・4M変更管理票との連動確認チェックリスト
```
