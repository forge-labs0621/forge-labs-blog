品質コスト（COQ: Cost of Quality）分析は、品質に関するコストを「予防コスト・評価コスト・失敗コスト（内部・外部）」に分類し、改善投資の優先度を決める手法です。AIを使って品質コストの分類・集計・経営報告書類の整備を効率化できます。

---

## 品質コストの分類・集計フォーマットを作るプロンプト

自社の品質関連コストをCOQ4分類で整理します。

```
以下の情報をもとに、品質コスト（COQ）分析の
集計フォーマットと月次レポートを作成してください。

【品質コストの4分類と自社での具体例】
1. 予防コスト（Prevention Cost）: 不良を出さないための費用
   ・品質担当者の人件費（QC・品質保証部門）
   ・仕入先品質管理・外注管理の費用
   ・品質教育・訓練の費用
   ・QC活動・改善活動の費用
   ・設備点検・予防保全の費用

2. 評価コスト（Appraisal Cost）: 品質を確認するための費用
   ・受入検査・工程内検査・出荷検査の人件費
   ・測定器・試験設備の維持費・校正費
   ・外部試験機関への依頼費用

3. 内部失敗コスト（Internal Failure Cost）: 出荷前に発見した不良
   ・工程内不良品の廃棄コスト（材料費・加工費の損失）
   ・手直し・再加工費用
   ・スクラップ廃棄費用

4. 外部失敗コスト（External Failure Cost）: 出荷後に発覚した不良
   ・クレーム対応・返品処理費用
   ・無償補修・交換費用
   ・客先訪問・陳謝費用
   ・ペナルティ・損害賠償費用

【昨年度の実績データ（例）】
（概算値で構いません。月次または年次で入力してください）

出力形式:
・「品質コスト月次集計表（4分類別・勘定科目別）」のフォーマット
・「品質コスト比率の計算（売上高対比・製造原価対比）」
・「品質コストの内訳グラフ（円グラフ・棒グラフの説明文）」
・「業界標準値との比較・目標値の設定」（参考値の説明）
・「品質コスト削減のROI計算（予防投資を増やすと失敗コストがどれだけ減るか）」
```

## 品質コスト改善計画・優先施策の整理プロンプト

品質コスト分析の結果から改善の優先順位を決めます。

```
以下の品質コスト分析の結果をもとに、
改善施策の優先順位と計画を整理してください。

【品質コスト分析の結果（例）】
年間品質コスト総額: ○百万円（売上高○億円の○%）
  ・予防コスト: ○百万円（○%）
  ・評価コスト: ○百万円（○%）
  ・内部失敗コスト: ○百万円（○%）← 最大
  ・外部失敗コスト: ○百万円（○%）

内部失敗コストの内訳:
  ・不良廃棄・スクラップ費用: ○百万円
    主な発生工程: プレス工程（全体の65%）/ 塗装工程（25%）
  ・手直し・再加工費用: ○百万円

外部失敗コストの内訳:
  ・クレーム対応（返品・補修・訪問）: ○百万円
    主な顧客: 顧客A（○件）/ 顧客B（○件）
  ・ペナルティ: ○百万円

【改善の方向性（現場の認識）】
・プレス工程の不良発生原因: 金型の摩耗管理が不十分
・塗装工程の不良: 下地処理の条件が不安定

出力形式:
・「品質コスト改善施策の優先順位（コスト削減効果 × 実現可能性）」
・「各施策の「予防コスト投資対効果（予防に○万円かけると失敗が
  何万円減るか）」の試算枠組み
・「品質コスト削減目標（1年後・2年後）の設定」
・「経営会議向けの品質コスト現状報告と改善計画の1枚サマリー」
・「品質コスト削減プロジェクトのKPI（進捗確認指標）」
```

## 不良コストを見える化するプロンプト

「不良品を1個出したらいくら損するか」を計算して現場に意識付けます。

```
以下の情報をもとに、製造業の「不良1個当たりコスト」の
計算と現場への見える化資料を作成してください。

【対象製品・工程】
製品: （例: 自動車用ブラケット / 完成品単価: ○円）
製造工程: 原材料 → プレス → バリ取り → 洗浄 → 検査 → 梱包

【工程別の付加価値（コスト）】
原材料費: ○円/個
プレス加工: ○円/個（機械時間 + 作業者費 + 間接費）
バリ取り: ○円/個
洗浄: ○円/個
検査: ○円/個
梱包: ○円/個

【不良発生工程と損失の内訳】
プレス後の不良の場合:
  ・廃棄損失: 原材料費 + プレス加工費 = ○円
  ・手直し対応: 追加作業費 + 材料費 = ○円

出荷後のクレーム（外部失敗）の場合:
  ・返品処理: 物流費 + 受入再検査費 = ○円
  ・顧客での対応（生産停止・特急対応等）: ○円（変動大）
  ・担当者の移動・説明費用: ○円

出力形式:
・「不良1個あたりの損失コスト計算表（工程別・不良種類別）」
・「現場掲示用の不良コスト見える化ポスターの文章」
  （「プレスで1個不良を出すと、何個分を余計に売らないと取り戻せないか」）
・「品質意識向上のための朝礼・TBMでの伝え方のスクリプト」
・「不良率が○%改善すると年間いくら利益が改善するかの試算フォーマット」
```
