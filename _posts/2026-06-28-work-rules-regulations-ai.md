従業員数10人以上になると就業規則の届出義務が生じ、テレワーク・副業・育児休業など法改正への対応も頻繁に求められます。就業規則の確認・見直し・社内規程の整備をAIで補助できます。

> **重要**: 就業規則は労働基準法・労働契約法に基づく法的文書です。作成・変更には必ず社会保険労務士・労働基準監督署に確認してください。AIの出力はあくまでも下書き補助用です。

---

## 就業規則の「抜けている項目」を確認するプロンプト

現在の就業規則で不足している記載項目をチェックします。

```
以下の情報をもとに、就業規則の記載内容を確認し、
法定必要記載事項や最近の法改正への対応状況を確認してください。

【現在の就業規則の状況】
最後に更新した時期: （例：2019年）
主な記載内容（章立て）:
  （例：1.総則 / 2.採用・異動 / 3.勤務時間・休日 / 4.給与 /
  5.退職・解雇 / 6.懲戒）

【自社の状況・変化】
（例：テレワークを2022年から導入したがテレワーク規程が未整備。
  育児休業を取る男性社員が出てきた。副業を認める方向で検討中。）

確認してほしいこと:
・法定必要記載事項（絶対的記載事項）が揃っているか
・最近の法改正（2022年育児介護休業法改正 / 2023年60時間超割増賃金引上げ /
  2024年時間外労働上限規制等）への対応が必要か
・テレワーク・副業・男性育休について規程を追加すべきか

出力形式:
・「あり / なし / 要確認」のチェックリスト形式
・各不足項目について「なぜ必要か / 大まかにどんな内容を記載すべきか」
・「最終的な確認・届出は社会保険労務士・労基署へ」の注記
```

## テレワーク規程の骨子を作るプロンプト

製造業でテレワークを行う事務職・設計職向けのテレワーク規程の骨子を作ります。

```
以下の情報をもとに、テレワーク規程の骨子（条文案の目次と概要）を
作成してください。

【自社の状況】
業種: （例：金属加工業（製造業））
テレワーク対象者: （例：事務職・設計職のみ。製造現場は対象外）
テレワークの形態: （例：在宅勤務のみ。サテライトオフィスは利用しない）
実施頻度の想定: （例：週1〜2回まで。原則として上司の許可制）
使用機器: （例：会社貸与のノートPC。私用PCは不可）

【規程に含めたい主な内容】
・テレワークの対象者・申請手順
・勤務時間・休憩の扱い（コアタイム設定等）
・通信費・光熱費の会社負担の有無（自社方針を記入）
・情報セキュリティ（作業環境・データの取り扱い）
・業務の記録・報告方法

出力形式:
・規程の目次と各条の概要（実際の条文は社労士に作成を依頼する前提）
・「製造業でテレワーク規程を作る際の注意点」のコラム
・※法的効力のある規程には社労士・労基署への届出が必要な旨を明記
```

## 育児休業・介護休業制度の社内周知文を作るプロンプト

法改正に対応した育休・介護休業制度を従業員に周知する文書を作ります。

```
以下の情報をもとに、育児・介護休業制度について
従業員全員への周知文書を作成してください。

【自社の状況】
従業員数: （例：65名）
最近の変化: （例：男性社員からパタニティ休暇・育休の相談が増えてきた）
制度の現状: （例：育児休業は就業規則に記載あり。男性育休の実績はこれまでなし）

【周知したい主な制度内容（2022年改正後）】
・産後パパ育休（出生時育児休業）: 子の出生後8週以内に4週間まで取得可
・育児休業: 子が1歳まで（条件次第で最大2歳まで）
・育休中の就業: 一定の条件で可能（労使協定が必要）
・2025年4月〜: 従業員1000人超では育休取得率の公表義務
  （自社は対象外だが、積極的に取り組む方針）

出力形式:
・全従業員向けの周知文書（読みやすい平易な文章）
・「男性もとれます！」という積極的なメッセージを含める
・相談先（人事担当者の部署名・内線のプレースホルダー）
・「詳細は人事部門にお気軽に相談ください」の締め
```

## 副業・兼業規程の考え方を整理するプロンプト

副業を認める場合のルール整備について社内で検討する資料を作ります。

```
以下の情報をもとに、副業・兼業を認める場合の
社内ルール（就業規則の改定案）の骨子を整理してください。

【自社の状況】
業種: （例：精密部品製造業）
従業員数: （例：45名）
副業検討の背景: （例：優秀な若手社員が副業を希望しており、
  禁止したままでは離職リスクがある。厚労省のモデル就業規則でも
  副業を原則認める方向に変わっている）

【懸念していること】
・競合他社での副業をどう防ぐか
・副業で過労・健康問題が起きた場合の責任
・機密情報・ノウハウの漏洩リスク
・労働時間の通算管理（労基法での取り扱い）

出力形式:
・「副業を認める場合の主なルール項目」とその概要
・「許可制 vs 届出制」の比較（製造業ではどちらが多いか）
・「禁止にしたほうがよい副業の種類（例：競合他社・情報漏洩リスクのあるもの）」
・「副業を認める際に整備が必要な事項チェックリスト」
・※最終的な規程改定は社労士に相談すること
```
