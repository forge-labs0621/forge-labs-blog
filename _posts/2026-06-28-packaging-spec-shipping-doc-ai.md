製造業の出荷業務では、製品に合わせた「梱包仕様書」や客先向けの「出荷明細書」「荷姿指示書」を作成する必要があります。取引先ごとに梱包条件が異なることも多く、それぞれの書類作成に手間がかかります。AIを使って文章化・整理の効率化が図れます。

---

## 梱包仕様書の骨子を作るプロンプト

製品の梱包方法を社内・外注先に伝える梱包仕様書を作ります。

```
以下の情報をもとに、製品の梱包仕様書を作成してください。

【製品情報】
品番・品名: （例：精密プレス部品 品番A-1234、材質SPCC、外径φ50mm×高さ30mm）
重量（1個）: （例：約200g）
取引先: （例：○○自動車工業、品番確認票に梱包指定あり）

【梱包条件（取引先指定または社内基準）】
個別包装: （例：防錆ビニール袋（個別）に入れる）
箱詰め単位: （例：1箱50個。平積み、縦横2×5段）
内装材: （例：気泡緩衝材（プチプチ）でセパレーター挿入）
外装箱: （例：ダンボールB式（310×220×200mm）、取引先指定の自社ラベル貼付）
ラベル内容: （例：品番 / 品名 / 数量 / ロット番号 / 製造日 / 自社名）
輸送条件特記: （例：精密品につき積み上げ禁止。「天地無用」「精密機器」表記）

出力形式:
・梱包仕様書フォーマット（品番・品名・梱包手順・ラベル仕様・注意事項）
・工程ごとの手順を番号付きで（作業者が見て迷わないレベルで）
・完成品の荷姿図（テキストで説明）
・作成日・作成者・承認者欄（プレースホルダー）
```

## 出荷明細書（パッキングリスト）を作るプロンプト

出荷時に添付する出荷明細書（パッキングリスト）の文章を整理します。

```
以下の情報をもとに、出荷明細書（パッキングリスト）を作成してください。

【出荷情報】
出荷日: （例：2026年8月5日）
送付先: （例：○○自動車工業 株式会社 岐阜工場 受入担当 ○○様）
自社名: （例：○○製作所 出荷担当 ○○）

【出荷明細（品番・品名・数量）】
- 品番A-1234 / プレス部品 / 200個 / ダンボール4箱（各50個）
- 品番B-5678 / 旋盤加工品 / 50個 / ダンボール1箱
- 品番C-0099 / 検査成績書（書類） / 1式 / A4封筒

【輸送条件】
（例：ヤマト運輸 コレクト便。指定時間: 午前中指定。）

出力形式:
・「出荷明細書（Packing List）」のフォーマット
・明細を表形式（品番 / 品名 / 数量 / 箱数 / ロット番号 / 備考）
・合計箱数・合計重量（概算）の欄
・輸送方法・特記事項の欄
・日英バイリンガル（海外向けに使う場合の追加指示: 英文も作ってください）
```

## 荷姿見本（梱包見本）の説明文を作るプロンプト

取引先への初回納品時や、新梱包方法の承認申請に使う「荷姿見本の説明文」を整理します。

```
以下の情報をもとに、取引先への荷姿見本（梱包見本）説明文を作成してください。

【提出の背景】
（例：新規取引開始にあたり、客先の品質部門へ梱包方法の事前確認申請を行う）

【梱包の特徴・こだわり点】
（例：
・防錆対策: 個別防錆袋（VCI袋）を使用。素手では触れず専用手袋で作業
・衝撃対策: 各層に気泡緩衝材（厚さ10mm）でセパレーター挿入
・トレーサビリティ: ロット番号シール（品番/ロット/日付）を全個に貼付
・外箱: JIS規格A式ダンボール強化仕様。積み上げ段数は最大3段まで
）

【客先への依頼事項】
（例：この梱包方法でよいか承認をいただけますか？変更点があればご指示ください）

出力形式:
・取引先へ提出する正式ビジネス文書
・写真説明文（「①〜③の写真のとおり」という形式。写真はご担当者が添付）
・梱包仕様のポイントを箇条書き
・「ご確認・ご承認をお願いいたします」の締め
```

## 返品・引き取り品の梱包指示を作るプロンプト

不良品返品や試作品引き取りの際に、相手先に伝える梱包依頼文を作ります。

```
以下の情報をもとに、取引先からの不良品返品または試作品引き取りの
梱包依頼文を作成してください。

【返品の内容】
品番・品名: （例：A-1234 プレス部品）
数量: （例：返品品 15個）
理由: （例：受入検査で外観不良と判断された品）

【梱包のお願い】
（例：
・不良箇所を保護するため、個別にビニール袋で包んでください
・段ボール箱に緩衝材を入れた状態で発送ください
・不良の状態がわかるよう、各個に「不良内容メモ」（手書き可）を添付いただけると助かります
）

【集荷方法】
（例：こちらから集荷便を手配します。○月○日に集荷させていただく予定です）

出力形式:
・取引先へのビジネスメール形式
・お礼と謝意を最初に
・梱包依頼内容を番号付きで簡潔に
・「ご不明な点はご連絡ください」の締め
```
