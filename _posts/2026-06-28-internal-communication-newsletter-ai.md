製造業の現場では「伝える」ことがモチベーション管理の重要な手段です。AIを使って社内報・朝礼スピーチ・表彰状・記念行事の案内を効率的に作成できます。

---

## 社内報（工場ニュースレター）を整理するプロンプト

月1〜2回発行する社内報の骨格を作ります。

```
以下の情報をもとに、製造現場向けの
社内報（月次版・A4 2ページ）を整理してください。

【社内報に掲載したい内容（今月）】
会社・経営情報:
  ・今月の受注状況（通常月比 +20%・繁忙期突入）
  ・新規顧客（D自動車部品様）との取引開始

現場トピック:
  ・品質改善達成: 今月不良率0.8%（目標1.0%以下・達成）
  ・改善提案表彰: 梱包工程の動線改善（製造2課 田中さん）
  ・ヒヤリハット件数: 3件（先月比 -2件）

人事・社員情報:
  ・新入社員紹介（7月入社 佐藤さん・高卒）
  ・定年退職お知らせ（経理部 山田さん・勤続32年）

その他:
  ・夏季休暇のお知らせ（8月13〜16日）
  ・熱中症対策の呼びかけ

出力形式:
・「社内報の構成案（見出し + 各コーナーの分量配分）」
・「各コーナーの本文（読みやすく・現場感を大切にした文章）」
  （全体で A4 2ページ相当・文字数 1,800字程度）
・「写真・図表を入れる場合の推奨レイアウト案（テキスト説明）」
・「次号の予告コーナーのテキスト案」
・「社内報を効果的に活用するためのコツ
  （配布タイミング・掲示場所・読んでもらうための工夫）」
```

## 朝礼スピーチ原稿を整理するプロンプト

工場長・班長が使える朝礼スピーチ素材を作ります。

```
以下の条件で、製造現場の朝礼スピーチ原稿を
3パターン作成してください。

【スピーチの用途・条件】
対象: 製造1課（15名）の朝礼（毎朝8時・約5分）
話者: 課長・山本（40代・ベテラン管理職）
時期: 夏（7月・熱中症シーズン / 繁忙期突入）

スピーチパターン:
  パターン1: 安全スピーチ（熱中症予防）
    ・今月すでに1件、軽度の熱中症疑い事象があった
    ・対策の徹底を呼びかけたい
    ・具体的な行動（水分補給・声かけ）を促す

  パターン2: 品質スピーチ（繁忙期の品質管理）
    ・受注が20%増で現場が忙しくなっている
    ・忙しい時ほどミスが増えることを伝えたい
    ・ヒヤリハット報告の促進も含める

  パターン3: モチベーション系（新規顧客獲得の感謝）
    ・新規顧客D社との取引が始まったことへの感謝
    ・現場の仕事の質が新規受注につながっていることを伝えたい

出力形式:
・「各パターンのスピーチ原稿（1パターン400〜500字・読んで3〜4分）」
  （口語調・聞いている人が続きを聞きたくなる書き出し）
・「朝礼スピーチを続けるためのネタ収集メモ
  （日常業務から見つけるヒントとなるポイント10選）」
```

## 表彰制度の設計と表彰状を整理するプロンプト

現場のモチベーションを高める表彰体制を作ります。

```
以下の情報をもとに、製造現場向けの
社内表彰制度を設計してください。

【表彰制度の背景と目的】
現状の問題:
  ・頑張っている人が報われていないと感じている社員がいる
  ・改善提案は形骸化（出すけど実行されない）
  ・若手のモチベーション低下が課題

表彰制度の方向性:
  ・月次表彰（毎月・全社員が対象）
  ・年次表彰（年1回・年度末）
  ・予算感: 1件あたり商品券¥3,000〜¥5,000以内

表彰したいテーマ:
  ①品質改善（不良率改善・クレームゼロ達成）
  ②安全（ヒヤリハット件数削減・安全ゼロ達成）
  ③改善提案（実行された改善提案・コスト削減実績）
  ④勤怠皆勤（ノーミス出勤・無遅刻無欠勤）
  ⑤多能工（新しい業務を習得した人）

出力形式:
・「表彰制度の概要と運用ルール
  （表彰の種類 / 評価基準 / 推薦方法 / 授与式の進め方）」
・「月次表彰の推薦・選考フォーマット（誰が推薦し・誰が決めるか）」
・「表彰状の文例（各テーマ別・3パターン）」
  （品質改善賞 / 安全推進賞 / 改善提案賞 の文面）
・「表彰式を盛り上げるためのアイデア（短時間・低コストで効果的な方法）」
・「表彰制度がうまくいかなくなる典型的な失敗パターンと予防策」
```
