## 「また日報の時間か…」をなくしたい

中小製造業の現場では、作業日報・見積書・点検記録・ヒヤリハット報告など、
「毎日同じような書類」を書く時間が地味に積み重なっています。1件あたりは
数分でも、現場と事務を兼任している人にとっては馬鹿にならない時間です。

最近はChatGPTやClaudeのようなAIチャットツールを使えば、こうした書類の
下書きを数十秒で作ってもらえます。ただ、実際に試してみると「何を入力
すればいいか分からない」「毎回プロンプトを考えるのが面倒」という壁に
ぶつかる方も多いはずです。

今回は、実際に「作業日報」を例に、どんな入力でどんな下書きが出てくるかを
紹介します。

## 試してみた例

入力したメモはこれだけです。

```
日付: 2026-06-21
作業者: 田中
担当工程・作業内容: NC旋盤でシャフト加工、午後から検査も対応
使用設備: NC旋盤2号機
生産数量・進捗: 予定120本中100本完了
トラブル・気づいた点: 14時頃に切削油が少なくなったので補充した
```

これをAIに「社内向けの作業日報として整形して」という指示と一緒に渡すと、
次のような下書きが返ってきます。

```
1. 日付・作業者
2026-06-21 / 田中

2. 本日の作業内容
NC旋盤2号機を用いたシャフト加工を実施した。午後からは加工品の検査作業も
併せて対応した。

3. 使用設備
NC旋盤2号機

4. 生産数量・進捗
予定120本のうち100本を完了した（進捗率 約83%）。

5. トラブル・気づいた点
14時頃、切削油の残量が少なくなったため補充を行った。
```

メモ書き程度の入力から、読みやすい報告文がすぐに出てきます。もちろん
最終チェックは人がする前提ですが、「ゼロから書く」と「下書きを直す」の
差は大きいです。

## ポイントは「指示の型」を持っておくこと

この例で実際にAIへ渡しているのは、メモだけではなく「どう整形してほしいか」
という指示（プロンプト）です。このプロンプトの「型」を一度作っておけば、
毎回同じ型を使い回すだけで、誰が入力しても同じクオリティの下書きが
出るようになります。

逆に言うと、プロンプトの型を毎回考え直すのが一番の手間です。
