フォークリフトとクレーンは製造業の現場に欠かせない荷役機械ですが、死亡事故の多い設備でもあります。日常点検・資格管理・安全作業手順書をAIで整備することで、重大事故を防止できます。

---

## フォークリフトの日常点検記録を整理するプロンプト

法定の始業前点検を確実に記録します。

```
以下の情報をもとに、フォークリフトの
日常点検チェックシートと管理方法を整理してください。

【当社のフォークリフトの概要】
台数: 3台
  FLT-1: トヨタ製 3t カウンターバランス型（1F倉庫）
  FLT-2: 日産製 1.5t カウンターバランス型（2F製品倉庫）
  FLT-3: ニチユ製 1t リーチ式（工場内通路）

フォークリフト運転資格保有者: 8名（構内限定免許3名含む）

法的根拠:
  ・労働安全衛生規則第151条の21：毎日の作業前に点検（始業前点検）
  ・点検内容: 制動装置・操縦装置・荷役装置・警報装置・前照灯・後照灯
  ・特定自主検査: 年1回（資格を持った事業内検査者または業者に依頼）

出力形式:
・「フォークリフト始業前点検チェックリスト（台数×点検項目）」
  （制動テスト・フォーク・タイヤ・バッテリー・警報・照明の確認項目）
・「点検記録表のフォーマット（日付・機番・点検者・各項目の合否・
  異常処置・承認）」
・「異常発見時の対応フロー
  （使用停止→班長報告→整備業者連絡→修理完了→復帰の手順）」
・「資格者管理台帳（氏名・資格証番号・取得日・有効期限・担当機番）」
・「特定自主検査の年次計画（検査時期・依頼業者・費用・記録保管）」
```

## 天井クレーン・ホイストの点検記録プロンプト

クレーンの法定点検を管理します。

```
以下の情報をもとに、天井クレーンの
点検計画と点検記録書類を整理してください。

【当社のクレーンの概要】
機種: 天井走行クレーン（電動ホイスト付き）
吊上荷重: 3t
設置場所: 第1工場（スパン25m・10年前設置）
主な用途: プレス金型の交換・大型部品の搬送

法定点検の種類（クレーン等安全規則）:
  ①月例自主点検（月1回 / 自社で実施）
  ②年次定期自主検査（年1回 / 記録保存3年）
  ③作業開始前点検（毎日の使用前 / 法定義務）

クレーン運転の資格:
  ・5t未満: クレーン運転特別教育（吊上荷重3tのため必要）
  ・現在の有資格者: 4名

出力形式:
・「毎日の作業開始前点検チェックリスト
  （ホイスト・ワイヤーロープ・フック・ブレーキ・走行・横行の確認）」
・「月例自主点検チェックリスト（電気系統・構造部材・安全装置等）」
・「年次定期自主検査の実施計画（検査項目・実施者・記録保管方法）」
・「ワイヤーロープの定期交換基準（素線切れ・摩耗の判断基準）」
・「クレーン使用前のKY（危険予知）活動のフォーマット
  （作業内容・危険ポイント・対策・確認者）」
```

## 荷役作業安全手順書を整理するプロンプト

フォークリフト・クレーン使用時の安全手順を標準化します。

```
以下の情報をもとに、工場内荷役作業の
安全作業手順書を整理してください。

【標準化する作業の種類】
作業①: フォークリフトによるパレット積み替え・移動
作業②: 天井クレーンによる金型の交換（プレス機への金型吊り込み）
作業③: フォークリフトとクレーンの協調作業（大型部品の位置決め）

【過去に発生したヒヤリハット・事故（参考）】
  ・フォークリフト走行中に歩行者が交差点で急に現れた
    → 接触寸前（速度制限5km/h・ミラーで確認するルールを徹底）
  ・クレーンで吊り上げ中にワイヤーが滑りかけた（掛け方が不適切）
    → 玉掛け者の資格確認・2点掛けの徹底

作業者への教育方法:
  ・OJTで教えてきたが、文書化されていない
  ・外国人技能実習生がいるため多言語対応が今後必要

出力形式:
・「フォークリフト作業の安全手順書（使用前確認→積み込み→
  走行→荷下ろし→使用後確認の手順と禁止事項）」
・「クレーン金型交換の安全手順書（玉掛け方法・合図の確認・
  吊り上げ・移動・降ろしの手順）」
・「工場内の交通ルール（速度制限・歩行者優先区域・
  フォークリフト専用通路の設定）の規定案」
・「安全作業手順書のポイントを絵・図で示すためのイラスト指示案」
・「多言語版（日本語+英語）の注意事項カード（主要禁止事項5項目）」
```
