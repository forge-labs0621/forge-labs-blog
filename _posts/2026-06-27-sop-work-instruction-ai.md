製造現場で「作業標準書を作りたいけど、どこから手をつければいいかわからない」「ベテランの手順を文書化したいが時間がない」という悩みは多いです。AIを使うことで、作業手順書の作成・改訂・わかりやすい表現への書き換えを大幅に効率化できます。

## 作業標準書とは

作業標準書（Work Instruction / SOP: Standard Operating Procedure）は、製造現場で**誰がやっても同じ品質で作業できる**ように、手順・方法・条件を文書化したものです。

ISO 9001では「業務を管理するための文書化された情報」として求められており、新人育成・多能工化・品質管理の基盤になります。

## AIが役立つ場面

- **ゼロから手順書を作る**: 口頭ノウハウ・メモをもとに整形
- **既存手順書を改訂する**: わかりにくい表現を整える、抜けを補う
- **作業手順の翻訳・平易化**: 外国人スタッフ向け、新人向けに書き換える
- **複数工程の手順書を一括で作る**: 類似工程を効率よく文書化

## 作業標準書をゼロから作るプロンプト

---

```
以下の情報をもとに、製造現場向けの作業標準書を作成してください。

【作業名】
（例：旋盤による丸棒切削加工）

【対象機械・設備】
（例：NC旋盤 ○○型）

【作業者の前提知識・資格】
（例：旋盤操作経験1年以上、社内資格「旋盤Bクラス」取得者）

【作業の目的・概要】
（例：指示書に従い、丸棒から所定の形状に切削加工する）

【主な作業手順（メモ・箇条書きで構いません）】
（例：
1. 材料の品番・寸法を確認する
2. チャックに材料をセットし、芯出しを行う
3. 加工プログラムを選択する
4. 試切りを行い、寸法を確認する
5. 本加工を行う
6. 加工後、全数寸法確認する
7. 合格品を次工程へ送る
）

【品質確認ポイント】
（例：切削後の外径・長さ、面粗度、バリの有無）

【安全上の注意事項】
（例：保護メガネ着用必須、切粉処理は素手禁止）

出力形式:
・「作業標準書」として整形
・目的・適用範囲・作業条件
・作業手順（番号付き・各ステップに判断基準や注意点を含む）
・品質確認方法・基準
・安全注意事項
・改訂欄（文書番号・版数・作成日・承認者プレースホルダー）
```

## 既存手順書を改訂・わかりやすくするプロンプト

ベテランが書いた手順書が「意味はわかるが伝わらない」「表現が曖昧」という場合に使えます。

```
以下の作業手順書の文章を、新入社員でも理解できるように改訂してください。

【現在の手順書の内容】
（現在の文章をそのまま貼り付けてください）

改訂の方針:
・専門用語は残しつつ、括弧内に簡単な説明を追加する
・「〜すること」という指示形式に統一する
・曖昧な表現（「適切に」「適量」「しっかりと」）は、
  具体的な数値・条件・判断基準に書き換える
・安全に関わる手順は【安全】と明記して目立たせる
```

---

## 多能工・外国人スタッフ向けに平易化するプロンプト

```
以下の作業手順書を、日本語能力初級の外国人スタッフでも理解できるように
書き換えてください。

【手順書の内容】
（現在の文章）

書き換えの条件:
・1文を短くする（20字以内を目安）
・カタカナよりひらがな・漢字を使う
・専門用語には読み仮名をつける（例：旋盤（せんばん））
・写真・イラストを入れる場所を【写真】と明記する
・数値・NG条件は太字または別枠で強調する
```

## 手順書の構成チェックリスト

AIで作った手順書を確認する際のチェック項目です。

```
以下の作業手順書が「良い手順書」の要件を満たしているか確認し、
不足している項目を指摘してください。

【確認する手順書の内容】
（手順書の内容を貼り付け）

確認観点:
□ 対象者・前提知識・資格が明記されているか
□ 使用する設備・工具・材料が列挙されているか
□ 手順が順番通りで抜け・飛びがないか
□ 各手順に「判断基準」（OKの条件・NGの条件）があるか
□ 品質確認方法・測定タイミングが明確か
□ 安全上の注意事項・禁止事項が記載されているか
□ 異常時（機械停止・品質NG）の対処方法があるか
□ 文書番号・版数・承認欄があるか
```

---

## AIを使う際の注意

- 生成された手順書は**ベテランが実作業と照らし合わせて確認・修正**することが必須
- 機密の工法・配合・特殊条件はAIに入力しない（社内専用手順書の場合はオンプレAIを検討）
- 定期見直し（年1回以上）を忘れずに実施する
