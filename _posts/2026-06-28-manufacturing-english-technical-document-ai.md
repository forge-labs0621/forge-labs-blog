製造業でのグローバル取引や外国人労働者対応に、技術文書の英語化が求められています。AIを使えば、専門用語を含む製造業の英語文書を効率的に作成・翻訳できます。

---

## 英語版作業標準書（SOP）を整理するプロンプト

日本語の作業手順書を英語化します。

```
以下の日本語の作業標準書を、外国人作業者
（英語話者）向けに英語版へ翻訳・最適化してください。

【翻訳対象の日本語作業標準書（抜粋）】
工程名: NC旋盤加工（ワーク取り付け・加工・取り外し）

手順:
  ①加工前に刃具の摩耗確認を実施すること。
    使用限度を超えた刃具は直ちに交換する。
  ②ワーク（φ50 × L200 炭素鋼）をチャックに取り付ける。
    把握力 850N 以上であること。
  ③プログラム番号「NC-2024-A」を選択し、
    試加工（1個目）を実施して寸法確認を行う。
  ④試加工品の寸法測定：外径φ40（公差±0.05）、長さ180mm（公差±0.1）
  ⑤測定値が公差内であれば量産加工を開始する。
    公差外の場合は班長に報告し加工を中止する。
  ⑥量産加工中は、20個ごとに抜き取り検査を実施すること。

安全注意:
  ・加工中は必ず保護眼鏡を着用する。
  ・チャックが完全に固定されたことを確認してから主軸を回転させる。

出力形式:
・「英語版作業標準書（上記の日本語手順を英語に翻訳）」
  （製造現場で使われる平易な英語・箇条書き・能動態で記述）
・「製造現場でよく使われる英語技術用語対訳集
  （日本語 → 英語 / この作業で使われる専門用語）」
・「外国人作業者向けの安全注意事項（英語 + 視覚的な伝達のヒント）」
・「英語版 + 日本語版のバイリンガル書式の設計案
  （日英対訳レイアウトの例）」
```

## 海外顧客への英文技術メール・提案書を整理するプロンプト

海外ビジネスでの技術的なやり取りを英語化します。

```
以下の状況について、海外顧客への英文メールと
技術提案書の本文を整理してください。

【状況】
送信者: 当社営業技術部・山本（英語は初級〜中級レベル）
受信者: 米国のXYZ Automotive Parts社 調達マネージャー・John Smith
案件: サンプル部品の試験結果報告と量産移行の提案

報告内容:
  ・試作品10個を納品し、全数検査合格を確認
  ・寸法測定結果: 全て図面公差内（測定報告書は添付）
  ・表面粗さ: Ra 1.6μm以下（要求値 Ra 3.2μm以下）を達成
  ・提案: 2026年9月から月産500個での量産開始を提案
  ・見積単価: $45/個（500個/月以上・FOB名古屋港）

先方からの懸念（事前の電話で聞いた）:
  ・材料の成分証明書（Mill Test Report）の提出を求められた
  ・第一回目の量産前にサプライヤー工場監査を実施したいとのこと

出力形式:
・「英文メール本文（プロフェッショナルな書き方・件名・本文・結び）」
・「技術提案書（Executive Summary 1ページ分の英文）」
  （試作結果のまとめ / 量産提案 / 価格 / スケジュール）
・「Mill Test Report（材料成績書）の英語での提出方法と説明文」
・「工場監査への対応準備で英語で伝えるべき情報の整理
  （ISO認証有無 / 品質管理体制 / 設備リスト）」
・「ビジネス英語での技術的な数値表現の書き方ルール
  （単位 / 公差表記 / 統計的な記述）」
```

## 英語版品質マニュアル・顧客向け品質規格書を整理するプロンプト

グローバル取引で必要な英語品質文書を作ります。

```
以下の情報をもとに、海外顧客向けの
英語版品質規格書（Quality Specification）を整理してください。

【品質規格書の対象と背景】
対象製品: 精密切削部品（炭素鋼・S45C相当材）
用途: 建設機械部品（海外向け）
顧客: 欧州の建設機械メーカー（ドイツ系）

顧客からの要求:
  ①ISO 9001の認証取得（またはそれに準じた品質管理体制の証明）
  ②製品の寸法・外観・材料の検査基準の英文書面提出
  ③検査成績書（Certificate of Conformance）の各ロット添付
  ④不適合品発生時の8D報告（英文）提出

当社の現状:
  ・ISO 9001は未取得（ISO認証を取得する計画はない）
  ・日本語の品質マニュアルはある
  ・検査成績書は日本語版のみ

出力形式:
・「英語版品質保証ステートメント（ISO未取得でも提出できる品質体制説明書）」
  （品質方針 / 検査体制 / 記録の保管方法を英語で1〜2ページにまとめる）
・「英語版検査成績書（Certificate of Conformance）のフォーマット」
  （ロット番号 / 数量 / 検査日 / 検査項目 / 測定値 / 合否判定）
・「英語版8D報告書のフォーマット」
  （D1〜D8の各ステップを英語で記入する様式）
・「製造業でよく使われる英語品質用語の対訳集
  （日本語 → 英語 50語）」
```
