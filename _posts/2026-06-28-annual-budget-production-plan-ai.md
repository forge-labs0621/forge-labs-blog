製造業では毎年度初めに年度予算・生産計画を策定し、月次で実績と比較しながら管理します。予算書の骨子・生産能力の計算・予算実績レポートの文章化をAIで効率化できます。

---

## 年度予算書の骨子を作るプロンプト

年度の売上・製造原価・利益計画の骨子を整理します。

```
以下の情報をもとに、製造業の年度予算書の骨子を作成してください。

【予算策定の前提条件】
対象年度: （例：2027年4月〜2028年3月）
前年度実績:
  ・売上高: （例：¥8.2億円）
  ・製造原価: （例：¥6.4億円 / 原価率78%）
  ・粗利益: （例：¥1.8億円 / 粗利率22%）
  ・販管費: （例：¥1.2億円）
  ・営業利益: （例：¥0.6億円 / 利益率7.3%）

【来年度の前提・目標】
売上目標: （例：¥9.0億円（+9.8%） / 主力取引先の発注増を見込む）
原材料費の変動: （例：鉄鋼材が5〜10%値上がり見込み）
新規採用: （例：正社員2名採用予定 / 人件費増加 ¥8,000千円/年）
設備投資: （例：プレス機更新 ¥30,000千円予定 / 減価償却費増加）
目標利益率: （例：営業利益率8%以上を目指す）

出力形式:
・年度予算書のフォーマット（売上 / 製造原価 / 粗利益 / 販管費 / 営業利益 の構成）
・「前年比増減の根拠」を各項目に記述する欄
・「月別売上計画（月次展開の考え方）」の説明
・製造業特有の「原価明細（材料費・労務費・製造経費）」の内訳表
```

## 生産能力計画表を作るプロンプト

自社の設備・人員の生産能力を計算し、受注計画と比較します。

```
以下の情報をもとに、生産能力計画表を作成してください。

【設備情報】
設備名・台数:
  （例：プレス機 10台 / CNC旋盤 5台 / 溶接ライン 3ライン）

設備稼働条件:
  ・稼働日数: （例：252日/年（年間稼働日数））
  ・1日の稼働時間: （例：8時間（残業・休日除く基準）/ 繁忙期は10時間）
  ・設備稼働率（目標）: （例：85%（段取り・メンテ等で100%にはならない）)

【製品ごとの生産サイクル（代表品番で試算）】
品番A-1234（ブラケット）:
  ・プレス工程: 1個あたり 1分30秒（プレス機1台使用）
  ・溶接工程: 1個あたり 3分（溶接ライン1ライン使用）
  ・月間受注予定: 3,000個/月

品番B-5678（カバー類）:
  ・プレス工程: 1個あたり 45秒（プレス機1台使用）
  ・月間受注予定: 8,000個/月

出力形式:
・設備ごとの「理論生産能力（時間）」の計算表
・品番ごとに必要な設備時間を積算する表
・「現在の受注量 vs 生産能力」のギャップ分析
・「能力不足が発生した場合の対策案」（残業・外注・設備増設）
```

## 予算実績管理レポートを作るプロンプト

月次の予算と実績の差異を報告する資料を整理します。

```
以下の実績データをもとに、月次予算実績管理レポートを
作成してください。

【対象月・基本情報】
対象月: （例：2026年7月）
レポート作成日: （例：2026年8月10日）

【売上の予実】
予算: ¥70,000千円 / 実績: ¥67,500千円 / 差異: ▲¥2,500千円（▲3.6%）
差異の主な原因:
  （例：主要取引先の月末出荷が翌月にズレ込んだ（約¥3,000千円分）。
  緊急受注が¥500千円上乗せとなり、一部相殺）

【製造原価の予実】
材料費 予算: ¥40,000千円 / 実績: ¥41,500千円 / 差異: ¥1,500千円超過
  原因: （例：鉄鋼材の単価が予算比+7%）
労務費 予算: ¥12,000千円 / 実績: ¥11,800千円 / 差異: ▲¥200千円良化
  原因: （例：残業が予算より少なかった）
製造経費 予算: ¥5,000千円 / 実績: ¥4,800千円 / 差異: ▲¥200千円良化

【7月末時点の年度累計】
累計売上予算達成率: （例：年間目標¥840,000千円に対して7月累計¥487,000千円 / 進捗率58%）

出力形式:
・月次予算実績レポート（経営会議提出用・A4 1枚目標）
・差異の「良化 / 悪化」と「一時的要因 / 構造的要因」の区分
・「8月以降の見通し・留意点」の記述
・「年間目標達成に向けた対策」のアクションプラン欄
```
