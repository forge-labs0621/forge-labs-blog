棚卸しで「帳簿在庫と実棚が合わない」問題は多くの製造業に共通する課題です。AIを使って棚卸し計画書・実施手順・差異分析報告書・在庫精度改善計画を体系化できます。

---

## 棚卸し計画書・実施手順書を整理するプロンプト

```
以下の情報をもとに、工場の棚卸し計画書と実施手順書を
整理してください。

【棚卸しの概要】
工場規模: 従業員60名・倉庫3棟（原材料倉庫・仕掛品置場・製品倉庫）
品目数: 原材料500品目・仕掛品250品目・製品180品目（計930品目）
在庫管理システム: 手作業でExcel管理（リアルタイムではない）
棚卸し頻度: 年次（期末）＋半期（中間）
前回の問題点:
  ・棚卸し準備が不十分で、当日に差異が多発
  ・実棚票の記入ミスが多い（品番と数量の転記ミス）
  ・棚卸し完了まで2日かかった（目標: 1日で終わらせたい）
  ・差異の多かったエリア（仕掛品置場）の原因追及が不十分

目標: 棚卸し精度98%以上・1日完了・差異の根本原因を突き止める

出力形式:
・「棚卸し実施計画書（目的 / スケジュール / 担当配置 / 禁止事項）」
・「棚卸し1週間前から当日のチェックリスト（事前準備の徹底）」
・「棚卸し実施手順書（エリア別・チーム割り当て・数え方の統一ルール）」
・「実棚票フォーマット（品番・品名・単位・実棚数量・2重チェック欄）」
・「棚卸し当日タイムスケジュール（開始宣言→実棚作業→集計→差異確認→終了宣言）」
```

## 実棚差異分析と在庫精度改善計画を整理するプロンプト

```
棚卸し後の実棚差異分析レポートと
在庫精度改善計画書を整理してください。

【棚卸し結果（2026年6月末）】
総品目数: 930品目
差異ゼロ（帳簿一致）: 783品目（84.2%）
数量差異あり: 117品目（12.6%）
品番違い・不明品: 30品目（3.2%）

主な差異内容:
  原材料倉庫: 差異58品目（差異合計額: 約42万円）
    → 帳簿より多い（プラス差異）: 31品目（受入登録漏れが主因と推測）
    → 帳簿より少ない（マイナス差異）: 27品目（払出登録漏れが主因と推測）
  仕掛品置場: 差異39品目（差異合計額: 約67万円）
    → 工程間移動時の記録漏れ・仕掛品の行先不明が多い
  製品倉庫: 差異20品目（差異合計額: 約18万円）
    → 出荷後の在庫更新タイムラグ

在庫精度（帳簿一致率）: 84.2%
目標: 95%以上

出力形式:
・「実棚差異分析報告書（差異の全体サマリー・エリア別分析・重大差異の詳細）」
・「差異原因の分類と仮説（受入漏れ / 払出漏れ / 工程間記録漏れ / 実棚ミスの切り分け）」
・「在庫精度改善計画書（問題エリア別: 原因 / 対策 / 担当 / 完了期限 / 効果測定方法）」
・「日常の在庫管理改善ルール（入出庫のダブルチェック・週次サイクルカウントの導入）」
・「次回棚卸しに向けた在庫精度目標と月次確認ルーティン」
```
