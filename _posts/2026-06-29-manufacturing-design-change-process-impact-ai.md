設計変更・工程変更の管理不備は品質トラブルと顧客クレームの大きな原因です。AIを使って変更影響評価・承認フロー設計・変更後の品質確認計画を体系化できます。

---

## 設計変更の影響評価・承認フロー書類を整理するプロンプト

```
以下の情報をもとに、設計変更の影響評価書と
社内・顧客承認フローの書類を整理してください。

【変更内容の概要】
変更の種類: 設計変更（材料変更）
変更内容: 部品A（樹脂成型品）の使用樹脂材料を変更
  変更前: グレードXX（国内メーカー製）
  変更後: グレードYY（コスト削減のため代替品採用）
変更理由: 材料コスト20%削減（年間効果: 約85万円）
対象品目: 42品目（すべて自社製品）
出荷先: 3顧客（A社・B社・C社）

潜在リスク:
  ・材料の流動性が若干異なるため、成型条件の微調整が必要な可能性
  ・耐熱性が若干低下する（高温環境での使用製品に注意）
  ・顧客の製品認定（承認図面）に材料グレード記載あり→顧客承認必要

出力形式:
・「設計変更影響評価書（影響範囲：品目リスト / 工程 / 品質特性 / 顧客への影響）」
・「顧客別の変更通知・承認依頼書（A社・B社・C社それぞれの対応方針）」
・「変更検証計画書（試作評価項目 / 評価基準 / サンプル数 / 承認基準）」
・「変更後の初期生産管理計画（初期ロット全数検査・条件記録強化・トラブル対応フロー）」
・「4M変更記録（変更の記録フォーマット・社内承認・顧客承認の履歴管理）」
```

## 工程変更の管理書類を整理するプロンプト

```
製造工程で設備更新・加工条件変更が発生した際の
変更管理書類を整理してください。

【変更内容の概要】
変更の種類: 工程変更（加工設備の更新）
変更内容: NC旋盤の更新（老朽化による更新）
  変更前: A社製NC旋盤（15年使用）
  変更後: B社製NC旋盤（最新型）
変更理由: 老朽化による精度低下・スペアパーツ入手困難
対象工程: シャフト加工工程（月産約2,000個）
顧客: 主要顧客2社（工程変更届提出義務あり）

変更に伴うリスク:
  ・新旧設備で加工精度の差異が出る可能性
  ・NCプログラムの移植確認が必要
  ・刃物経路の最適化が旧機と異なる場合がある

出力形式:
・「工程変更届（顧客提出用）のフォーマット
  （変更理由 / 変更内容 / 変更前後比較 / 品質への影響 / 初期確認計画）」
・「設備切替えの検証計画書（新設備での初期加工 / 精度確認 / 比較測定 / 承認基準）」
・「NCプログラム移植確認記録（旧プログラム→新機での動作確認・寸法比較）」
・「設備切替え前後のサンプル測定記録フォーマット（新旧比較データ記録）」
・「顧客への工程変更事前通知文（変更内容・スケジュール・サンプル提供の案内）」
```
