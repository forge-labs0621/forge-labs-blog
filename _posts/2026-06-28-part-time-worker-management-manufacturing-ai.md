製造業では繁忙期のパートタイム・アルバイト活用が欠かせません。AIを使って雇用契約書・シフト管理・教育記録・同一労働同一賃金対応書類を効率的に整備できます。

---

## パートタイム雇用契約書を整理するプロンプト

法令要件を満たした雇用契約書を作成します。

```
以下の情報をもとに、パートタイム社員の
雇用契約書（兼労働条件通知書）を整理してください。

【採用予定のパートタイム社員の情報】
採用目的: 繁忙期（6〜9月）の梱包・検査補助作業
勤務形態:
  ・勤務日: 週4日（月・火・水・金）
  ・勤務時間: 8:00〜16:30（休憩1時間 / 実労働7.5時間）
  ・時給: ¥1,050
  ・交通費: 実費支給（上限 月¥15,000）
  ・時間外労働: 月10時間以内で発生する場合あり

契約期間: 2026年7月1日〜2026年9月30日（更新あり）
試用期間: 最初の2週間（条件は同じ）
雇用保険: 加入対象（週20時間以上のため）
社会保険: 不加入（勤務時間が週30時間未満）

出力形式:
・「雇用契約書（兼労働条件通知書）の全文
  （パートタイム・有期雇用労働法の法定記載事項を含む）」
・「同一労働同一賃金の観点から確認すべき事項
  （正社員との待遇差の説明義務 / 基本給・賞与の考え方）」
・「更新・不更新の判断基準の明示方法
  （雇い止めトラブルを防ぐための「期待を持たせない」書き方）」
・「試用期間中の能力確認チェックリスト（本採用判断のための評価項目）」
・「入社時に渡すべき書類一覧（本人確認 / マイナンバー / 口座情報等）」
```

## パートタイム社員のシフト管理・業務割り当てを整理するプロンプト

週次・月次のシフトを効率的に管理します。

```
以下の情報をもとに、パートタイム社員の
シフト管理と業務割り当て計画を整理してください。

【現在のパートタイム社員体制】
パート社員数: 12名（女性8名・男性4名）
担当業務: 梱包（6名）・外観検査（4名）・仕分け・ラベル貼り（2名）
各自の制約（シフト希望）:
  ・Aさん（梱包）: 月曜定休・子供の送迎で15時上がり
  ・Bさん（検査）: 週4日勤務希望
  ・Cさん（梱包）: 月1回病院（水曜午前）
  ・Dさん（検査）: フルタイム勤務可能
  ・他8名: 特別制約なし・週3〜5日

生産計画からの人員需要:
  7月1週: 梱包6名・検査3名・仕分け2名（通常需要）
  7月2週: 梱包8名・検査4名・仕分け2名（増産）
  7月3〜4週: 梱包6名・検査4名（維持）

出力形式:
・「月次シフト表の作成手順（制約条件を考慮した割り当て方法）」
・「シフト希望収集用紙のフォーマット
  （希望出勤日・都合の悪い日・その他要望）」
・「繁忙期の人員不足が予測された場合の対応フロー
  （追加採用 vs 既存パートへの増勤依頼 vs 正社員残業）」
・「多能工化（梱包 ↔ 検査）の育成計画
  （複数業務を担当できるパートを増やすOJT計画）」
・「急な欠勤・遅刻時の連絡・対応フロー（誰が代替するか）」
```

## 同一労働同一賃金対応の待遇確認書類を整理するプロンプト

パートタイム労働法・短時間・有期雇用労働法への対応を確認します。

```
以下の情報をもとに、同一労働同一賃金への
対応状況確認書類を整理してください。

【当社の正社員とパートタイムの待遇比較】
対象: 梱包担当パートタイム vs 梱包担当正社員

基本給:
  ・正社員梱包担当: 月給¥220,000（時給換算 約¥1,320）
  ・パートタイム: 時給¥1,050

待遇の違い:
  ①賞与: 正社員あり（年2回）/ パートなし
  ②通勤手当: 正社員あり（全額実費） / パート（上限¥15,000）
  ③食事補助: 正社員あり（社食補助¥300/日） / パートなし
  ④退職金: 正社員あり / パートなし
  ⑤有給休暇: 両者あり（法定通り）
  ⑥教育訓練: 正社員向けOJT・研修あり / パート向けは入社時のみ

義務:
  ・待遇差の説明義務（パート本人から求められた場合）
  ・不合理な待遇差の禁止（同じ仕事なのに大幅差はNG）

出力形式:
・「待遇比較一覧表（正社員 vs パートタイム / 均衡・均等の観点で整理）」
・「各待遇差の合理的な理由の説明文（説明義務への対応用）」
  （例: 賞与なしの理由 → 「業績連動・正社員責任加算分が含まれるため」）
・「待遇差が不合理と判断されるリスクのある項目の指摘と改善案」
・「パートタイム社員から待遇差の説明を求められた場合の対応手順書」
・「正社員化・無期転換ルールへの対応状況確認チェックリスト
  （5年超の有期雇用者の無期転換申込権の発生状況）」
```
