製造業の生産管理では、設備の稼働率・生産能力・OEE（設備総合効率）を定期的に分析し、ボトルネックの特定と改善活動に活かします。AIを使って分析レポートと経営会議向け報告書を効率化できます。

---

## 稼働率・OEE（設備総合効率）計算と分析レポートを作るプロンプト

月次の設備稼働状況と生産効率の分析報告書を整理します。

```
以下の情報をもとに、製造ラインの稼働率・OEE分析レポートを
作成してください。

【対象設備・集計期間】
対象: プレスラインA（プレス機2台・搬送設備）
集計期間: （例: 2026年5月 / 稼働予定: 20日間 × 2直 × 8時間 = 320時間）

【実績データ】
① 負荷時間（予定稼働時間）: 320時間
② 停止時間（ダウンタイム）の内訳:
   ・計画停止（昼休憩・定時停止・整備）: 40時間
   ・故障停止: 12時間（主な故障: 潤滑系トラブル2回）
   ・段取り替え時間: 25時間
   ・材料待ち・段取り前後の空き: 8時間
③ 稼働時間 = 負荷時間 - (故障停止 + 段取り): ___時間

④ 生産量と速度に関するデータ:
   ・設計サイクルタイム（標準サイクルタイム）: 10秒/個
   ・実績サイクルタイム（平均）: 11.5秒/個
   ・稼働時間での生産可能数（設計速度）: ___個
   ・実際の生産数: 82,000個

⑤ 品質データ:
   ・総生産数: 82,000個
   ・不良品数: 820個（不良率1.0%）
   ・良品数: 81,180個

出力形式:
・OEE計算結果（時間稼働率・性能稼働率・良品率・OEE総合値）
・「OEEのウォーターフォールチャート（どのロスが大きいか）」の表形式
・「業界平均OEEとの比較・目標値の設定案」
・「改善優先順位（最も効果が大きいロスの特定）」
・経営会議向けの「生産効率月次報告書の1ページ要約フォーマット」
```

## ボトルネック工程の特定と改善計画を作るプロンプト

複数工程の中で制約となっているボトルネックを特定し改善計画を立てます。

```
以下の情報をもとに、製造ラインのボトルネック分析と
改善計画を整理してください。

【製造工程の流れと能力】
製品: （例: プレス加工品 → 月産目標: 100,000個）

工程別の生産能力:
  工程1（プレス加工）: 理論能力 120,000個/月 / 実績 105,000個/月
  工程2（バリ取り）: 理論能力 90,000個/月 / 実績 85,000個/月 ← 課題
  工程3（洗浄）: 理論能力 130,000個/月 / 実績 125,000個/月
  工程4（外観検査）: 理論能力 110,000個/月 / 実績 108,000個/月
  工程5（梱包・出荷）: 理論能力 100,000個/月 / 実績 98,000個/月

工程間の在庫（WIP）:
  工程1→2の間: 平均在庫 5,000個（約2.5日分）← 滞留
  工程2→3の間: 平均在庫 500個（0.25日分）
  工程3→4の間: 平均在庫 800個（0.4日分）
  工程4→5の間: 平均在庫 400個（0.2日分）

出力形式:
・「ボトルネック工程の特定（TOC理論に基づく）と根拠の説明」
・「バリ取り工程の能力向上のための改善アイデア3案」
  （設備増強案・外注化案・工程改善案）
・「各案のコスト・効果・実施期間の比較表」
・「改善実施の優先順位と推奨案」（理由付き）
・経営会議向けの「生産能力改善計画書の骨子」（1枚に収まる形式）
```

## 需要予測と生産計画の整合性分析を作るプロンプト

顧客の受注変動と自社の生産能力のギャップを分析します。

```
以下の情報をもとに、需要予測と生産能力のギャップ分析と
対応計画を整理してください。

【需要予測データ（受注内示）】
7月: 110,000個（内示）/ 8月: 125,000個（内示）/ 9月: 140,000個（予測）
10月: 130,000個（予測）/ 11月: 120,000個（予測）

【自社の生産能力】
現在の月産能力（通常稼働）: 100,000個/月
残業対応時の最大生産能力: 115,000個/月
外注活用時の追加上限: さらに+20,000個/月（外注B社・リードタイム3週間）

課題:
  8月〜9月に能力超過。8月: 10,000個不足、9月: 40,000個不足
  （顧客への対応が急務）

出力形式:
・「需要と生産能力のギャップ表（月別・対応後の充足状況）」
・「不足分への対応シナリオ比較」（残業・外注・増産設備・顧客と納期交渉）
・「顧客への不足見込み連絡メールの骨子」（早期の情報共有が重要）
・「繁忙期に向けた生産計画の前倒しと材料先行手配の提案」
・経営・製造・営業が共有すべき「生産能力管理会議のアジェンダと
  モニタリング指標」
```
