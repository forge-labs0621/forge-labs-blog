「ものづくり補助金に申請したいが、事業計画書の書き方がわからない」「書類作成に時間がかかりすぎて申請を諦めた経験がある」という製造業の経営者・管理者は多いです。AIは補助金申請書類の「整形・構成・表現の改善」を補助できますが、全自動ではないことも理解しておく必要があります。

## AIで補助できること・できないこと

| できること | できないこと |
|---|---|
| 現状と課題の文章化・整理 | 最新の補助金制度・公募要領の確認 |
| 「ですます調」→「体言止め」などの表現統一 | 採択されやすい「戦略」の判断 |
| 箇条書きを論文調に変換 | 公認の数値・実績の作成 |
| 構成（目次）の案出し | 審査員の加点ポイントの見極め |
| 類似表現の重複チェック | 公募要領の最新変更への対応 |

**重要**: 生成AIが作った申請書をそのまま提出しても採択率は低いです。AIは「たたき台と表現の整形」に使い、中身（自社の強み・課題・解決策）は経営者・担当者が考えることが必要です。

## ものづくり補助金の「事業計画書」を整理するプロンプト

ものづくり補助金では「革新的な製品・サービス開発」または「生産プロセスの革新」の計画を記載します。

---

```
以下のメモをもとに、ものづくり補助金の「事業計画書」の下書き（骨子）を
作成してください。

【自社の現状と課題（自由に書いてください）】
（例：
・現在、金属部品の加工を受託しているが、熟練工の退職が相次ぎ
  技術継承が課題になっている
・生産能力が不足しており、受注を断ることが月に3〜4件ある
・NC機械が旧型で精度が出せない品番が増えている
）

【導入を検討している設備・システム】
（例：5軸加工機 / 協働ロボット / 生産管理システム / AIを使った品質検査装置 など）

【設備導入で実現したいこと（定量的な目標があれば）】
（例：
・加工精度を現在の±0.1mmから±0.02mmに向上させる
・1人でこなせる工程数を2工程から4工程に増やす
・生産リードタイムを現在の10日から6日に短縮する
）

【自社の強み（競合に比べて優れているところ）】
（例：○○メーカーとの30年の取引実績、航空部品の加工資格保有、
       熟練工が社内に3名在籍 など）

出力形式:
・「事業計画書（骨子）」として
・現状と課題（3〜5段落）
・導入計画（設備・時期・実施方法）
・期待される効果（定量・定性の両面）
・自社の強みと実現可能性

注意: この出力はあくまで下書きです。実際に申請する際は
公募要領の最新版を確認し、専門家（認定支援機関）の確認を受けてください。
```

## 小規模事業者持続化補助金の「経営計画書」を整理するプロンプト

持続化補助金では「自社の経営状況の分析」と「補助事業で何をするか」の2点が重要です。

```
以下の情報をもとに、小規模事業者持続化補助金の「経営計画書」の
下書きを作成してください。

【事業の概要】
（例：従業員12名の金属プレス加工業。主要顧客は自動車部品メーカー3社。
売上の80%が受注生産。年商約1.5億円）

【現在の強み・特徴】
（例：
・設立40年の信頼と取引実績
・特殊形状のプレス加工技術（他社では対応が難しい品番を受注できる）
・ISO 9001取得済み
）

【経営課題】
（例：
・主要顧客への依存度が高く、売上が集中している
・新規顧客の開拓をしてこなかった
・若手採用が難しく、技術継承が心配
）

【補助事業でやりたいこと】
（例：自社のホームページを作成して、新規顧客からの問い合わせを
受けられるようにしたい。補助金でWeb制作費と写真撮影費を使いたい）

出力形式:
・「経営計画書」として
・企業概要・事業の強み
・経営課題の整理（箇条書き + 2〜3文の説明）
・補助事業の内容（何をするか・期待する効果）
・文体は補助金申請書として（ですます調・簡潔に）
```

## 補助金申請書の表現を整えるプロンプト

下書きを書いたあと、表現を引き締める際に使えます。

```
以下の補助金申請書の文章を、審査員に伝わりやすい表現に改訂してください。

【現在の文章】
（現在書いている文章をそのまま貼り付けてください）

改訂の方針:
・「〜したいと思います」→「〜します」と断定形に変える
・同じ内容の繰り返しや冗長な表現を削る
・抽象的な表現（「大幅に」「さらに」）を具体的な数値・事実に差し替える提案をする
・「補助金をもらいたい」という書き方ではなく、「自社の事業展開に必要な投資」として伝わるよう調整する

注意: 事実と異なる内容・実績の誇張は行わないでください。
```

---

## AIを使った補助金申請書作成の流れ

### ステップ1: 骨子（構成）を作る
AIに現状・課題・やりたいことをメモ書きで渡して、論文の骨子を出してもらう。

### ステップ2: 中身を経営者が確認・修正する
AIが出した骨子の「事実」「数値」「強み」が本当に正しいか経営者・担当者が確認する。誇張・間違いを修正する。

### ステップ3: 認定支援機関に相談する
ものづくり補助金は認定支援機関（商工会・中小企業診断士・税理士等）の確認が必要な補助金もあります。

### ステップ4: 表現の整形にAIを使う
最終的な文章のトーン・表現統一・読みやすさの改善にAIを使う。

## 製造業向けの主な補助金（参考）

| 補助金名 | 対象 | 補助上限の目安 |
|---|---|---|
| ものづくり補助金 | 設備投資・システム導入 | 750万〜数千万円 |
| 小規模事業者持続化補助金 | 販路開拓・業務効率化 | 50〜200万円 |
| IT導入補助金 | ITシステム・ソフトウェア導入 | 5〜450万円 |
| 省エネ設備補助（各省庁） | 省エネ設備の更新 | 案件により異なる |

**必ず最新の公募要領を確認してください。** 補助金制度は頻繁に変更されるため、AIの回答より公式サイト（中小企業庁・各省庁）の情報を優先してください。

---

## AIを使う際の注意

- **最新の公募要領を必ず確認**: AIは訓練データの締め切り時点の情報しか持っていません。補助金制度は毎年・毎公募で変わります
- **事実と異なる内容は書かない**: 採択後に実績報告があり、事実と違う内容を書くと後のトラブルになります
- **認定支援機関・専門家への相談を推奨**: 特にものづくり補助金は認定支援機関の確認が実質必要です
