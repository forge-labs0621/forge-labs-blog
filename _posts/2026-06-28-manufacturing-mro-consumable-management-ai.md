製造業では材料・製品だけでなく、切削工具・研磨材・消耗品（MRO品）の管理も重要なコスト管理テーマです。AIを使ってMRO品の在庫・発注・コスト分析書類を効率的に整備できます。

---

## MRO品（切削工具・消耗品）の在庫管理を整理するプロンプト

工具・消耗品の在庫を適切に管理します。

```
以下の情報をもとに、切削工具・消耗品（MRO品）の
在庫管理書類を整理してください。

【当社のMRO品の実態】
主な管理対象品目:
  切削工具:
    ドリル（各サイズ）/ エンドミル / バイト / タップ
    月平均使用額: ¥180,000/月
  研磨・バリ取り用品:
    研磨布 / 砥石 / ヤスリ / リューター
    月平均使用額: ¥50,000/月
  消耗品:
    切削油 / 冷却水（クーラント）/ グリス / オイル
    月平均使用額: ¥70,000/月

現在の問題:
  ①工具が「なくなってから発注」して加工ラインが止まる事態が月2〜3回
  ②在庫の把握が現場任せで、過剰在庫（使わないまま古くなった工具）が発生
  ③工具費のコストが増加しているが、どの品目・工程が原因か不明
  ④ドリル等の研磨再利用をしているが、使用限度の判断基準がない

出力形式:
・「MRO品在庫管理台帳のフォーマット
  （品目 / 規格 / 発注点 / 発注量 / 現在庫 / 次回発注日）」
・「発注点・発注量の設定方法
  （月次使用量 × 調達リードタイム + 安全在庫 の計算式）」
・「工具の使用限度判断基準書（摩耗・欠け・寸法精度の観点）」
・「工具費の工程別・品目別コスト配賦方法
  （どの機械・工程に何円かかったかを記録する方法）」
・「工具の棚管理（5S的な整理整頓）改善のアイデア
  （定位置 / 数量表示 / 色別管理 / カンバン発注の設計）」
```

## 切削油・クーラントの管理書類を整理するプロンプト

機械稼働に不可欠な液体管理を効率化します。

```
以下の情報をもとに、切削油・クーラント液の
管理書類を整理してください。

【切削油・クーラントの現状】
使用機械: NC旋盤×3台 / マシニングセンター×2台
クーラント種類: 水溶性エマルション（希釈濃度 8〜10%）
クーラントタンク容量: 各タンク 200L

現在の管理問題:
  ①濃度管理が不定期（毎日計測していない）
    → 低濃度のまま使用して工具寿命が短くなっている懸念
  ②クーラントの腐敗（臭い発生）が2ヶ月ごとに起きている
    → 交換するが費用も手間もかかる
  ③クーラント廃液の処理を外部業者に委託しているが、量の記録がない
    （産業廃棄物管理の観点からも記録が必要）

日常管理で実施すべき項目（現在不十分なもの）:
  ・濃度測定（リフラクトメーター使用）: 毎日実施が理想 → 週1回程度
  ・pH測定: 週1回（腐敗判定に有効）
  ・外観・臭い確認: 毎日（腐敗の早期発見）
  ・浮き油・切粉の除去: 週1回

出力形式:
・「クーラント日常管理チェックシート
  （測定項目 / 測定頻度 / 判定基準 / 対処方法）」
・「クーラント交換判断基準書
  （pH / 濃度 / 外観 / 臭い / 細菌数 の判定基準）」
・「クーラントの長寿命化のための管理ポイント解説
  （浮き油除去 / 切粉除去 / 補充方法 / 殺菌剤の使用）」
・「クーラント廃液の産業廃棄物記録フォーマット
  （交換日 / 機械番号 / 廃液量 / 処理業者 / マニフェスト番号）」
・「クーラントコストの月次計算方法
  （補充量×単価 + 廃液処理費用 の集計）」
```

## 副資材の一括購買・コスト削減を整理するプロンプト

消耗品の調達を一元管理してコスト削減します。

```
以下の情報をもとに、副資材・消耗品の
一括購買によるコスト削減計画を整理してください。

【副資材・消耗品の購買の現状】
現在の購買方法:
  ・各現場の班長が「必要になったら発注」（個別発注）
  ・発注先が担当者個人のコネクションで決まっており、相見積もりなし
  ・事務用品は月1回まとめて発注しているが、現場消耗品はバラバラ

主な消耗品と月次購入額:
  手袋・使い捨て軍手: ¥35,000/月（担当者Aが1社から発注）
  清掃用品（モップ・ウエス等）: ¥20,000/月（担当者Bが別の業者から）
  梱包テープ・段ボール: ¥40,000/月（購買部が発注）
  安全保護具（耳栓・保護メガネ）: ¥15,000/月（安全担当が発注）
  切削油・グリス（少量補充分）: ¥30,000/月（各課が個別に発注）

問題点:
  ・同じ商品を複数部門が別々の業者から別の価格で買っている
  ・年間で¥1,700,000以上の副資材購入があるが一括交渉できていない

出力形式:
・「副資材の品目分類と購買の一元管理方針の整理
  （品目別に一括購買すべきもの vs 現場個別の方が効率的なものを分類）」
・「一括購買に向けた業者選定・見積もり取得の手順書
  （現状の品目・数量・仕様をまとめてRFQを作る方法）」
・「副資材の年間コスト削減目標設定と施策一覧
  （一括発注 / 代替品への切り替え / 数量の適正化）」
・「消耗品費の部門別・品目別の月次コスト管理シートのフォーマット」
・「社内への一括購買移行の説明・根回し文書の骨子
  （現場が反発しにくい伝え方）」
```
