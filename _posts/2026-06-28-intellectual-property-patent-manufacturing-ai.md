製造業では、技術開発の成果を特許・実用新案で守ることや、他社特許への抵触リスクを確認することが重要です。AIを使って知財管理書類・先行技術調査・職務発明管理の整備を効率化できます（出願・権利化の実務は弁理士が必要）。

---

## 社内の知的財産管理台帳を作るプロンプト

自社が保有・出願中の特許等の一覧管理を整理します。

```
以下の情報をもとに、社内の知的財産（IP）管理台帳の
フォーマットを作成してください。

【自社の知財保有状況（例）】
特許:
  ・特許第○○号: 「○○加工方法」 / 出願日: ○年○月 / 登録日: ○年○月
    権利存続期限: 出願から20年（○年○月）/ 年金納付: 毎年○月
  ・特願○○: 「○○装置」（審査請求済み・審査中）

実用新案:
  ・実用新案登録第○号: 「○○治具」/ 存続期限: ○年○月

意匠・商標:
  ・なし（今後取得を検討）

秘密管理（特許出願しないノウハウ）:
  ・○○の加工条件（書類として社内で管理中: `assets/confidential/...`）

出力形式:
・IP管理台帳のフォーマット（権利種別・権利番号・発明名・
  出願日・登録日・存続期限・年金納付期限・担当者・メモ欄）
・「年金納付期限のアラート管理方法」（失効リスクの防止）
・「社内で保有するノウハウ（秘密情報）の管理台帳フォーマット」
・「知財管理担当者の引き継ぎ書類チェックリスト」
```

## 新技術の先行技術調査メモを作るプロンプト

新製品開発・設備改良の前に行う先行技術調査の記録を整理します。

```
以下の情報をもとに、先行技術調査の記録メモと
他社特許への抵触リスク確認書類を作成してください。

【調査対象の技術・開発テーマ】
（例: プレス加工における「金型摩耗を自動検知する画像認識システム」
  の開発を検討している）

【調査の目的】
① この分野に他社の強い特許があるかを確認する（Freedom to Operate: FTO確認）
② 自社技術を特許出願できる新規性・進歩性があるかを確認する
③ 類似する先行技術の参考情報を収集する

【J-PlatPatで調査する際の検索キーワード案】
・「金型」×「摩耗」×「画像認識」
・「プレス」×「工具状態監視」×「カメラ」
・IPC分類: B21D（金属プレス加工）× G06T（画像処理）

出力形式:
・「先行技術調査の記録フォーマット」（調査日・調査者・検索式・
  ヒット件数・重要文献リスト・リスク評価・コメント）
・「他社特許の要旨メモのフォーマット（請求項のポイント整理）」
・「J-PlatPatの調査結果を弁理士に相談する際の提示資料の骨子」
・「特許出願を検討する場合の発明の要点整理フォーマット
  （発明の名称・解決する課題・従来技術との違い・発明の詳細）」
```

## 職務発明規程・社内発明届出書のフォーマットを作るプロンプト

従業員の発明を適切に会社の権利にするための社内ルールを整理します。

```
以下の情報をもとに、職務発明規程の骨子と
社内発明届出書のフォーマットを作成してください。

【自社の状況】
従業員数: ○名（うち技術・開発担当: ○名）
現状: 職務発明規程がなく、従業員が発明した際の権利帰属・
  報奨金の扱いが不明確
過去の経験: 退職した技術者が自社で生み出した発明について、
  外部で特許出願するリスクが生じた

職務発明規程で明文化したいこと:
  ① 発明の権利が会社に帰属する条件
  ② 発明者への相当の利益（報奨金等）の内容と基準
  ③ 発明届出・権利評価・出願判断のフロー
  ④ 退職後の義務（秘密保持・競業避止）との関係

出力形式:
・「職務発明規程の骨子（条文の見出しと要点）」
・「社内発明届出書のフォーマット（発明者・発明の内容・
  創作の経緯・関連業務との関係・出願希望の有無）」
・「発明を評価・出願判断するための社内会議（IP会議）の
  アジェンダフォーマット」
・「注意事項: 職務発明規程は就業規則の付属規程として
  労働基準監督署への届出・従業員への周知が必要」のコメント
```
