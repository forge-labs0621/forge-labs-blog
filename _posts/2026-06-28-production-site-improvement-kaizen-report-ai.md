「なんとなく改善した」ではなく、数字で成果を見せる改善活動が組織の信頼を高めます。AIを使って改善テーマの設定・活動記録・成果の定量化・社内発表資料を効率化できます。

---

## 改善テーマの選定と活動計画プロンプト

どの問題から取り組むかを体系的に決めます。

```
以下の情報をもとに、製造現場の
改善テーマ選定と3ヶ月間の活動計画を整理してください。

【当社の現場課題（班長からのヒアリング結果）】
  ① バリ取り工程の残業が多い（月30〜40時間）
  ② 部品の探し時間が長い（治具・工具が所定位置にない）
  ③ 段取り時間が長い（型交換に平均45分かかっている）
  ④ 不良品が多い工程がある（検査工程で月不良率2.3%）
  ⑤ 作業者の腰痛が多い（重量物の手運搬）

課題の優先順位付けに使う評価軸:
  インパクト（改善できた時の効果の大きさ）
  実現可能性（3ヶ月以内に改善できる可能性）
  緊急性（放置した場合のリスク）

出力形式:
・「課題の優先順位評価表（5課題×3軸・各5段階評価）」
・「最優先テーマとその選定根拠」
・「3ヶ月間の改善活動スケジュール
  （テーマ決定→現状分析→対策立案→実施→効果確認の月別計画）」
・「改善チームの構成と役割（誰が・何を担当するか）」
・「改善活動の開始宣言（チームへの最初のメッセージ）」
```

## 改善活動の記録と成果定量化プロンプト

「何が変わったか」を数字で見せます。

```
以下の情報をもとに、改善活動の
成果記録と定量化レポートを整理してください。

【改善テーマ】
テーマ: 「段取り時間の短縮（型交換45分→20分以下）」
対象設備: プレス機200t（1号機）
活動期間: 4〜6月（3ヶ月間）

【改善前の現状（4月計測値）】
段取り時間（型交換）の内訳:
  ・現在の型を外す作業: 8分
  ・次の型を持ってくる作業: 12分（倉庫から台車で運搬）
  ・型をセットして固定する作業: 15分
  ・高さ調整・位置決め: 7分
  ・試し打ち・調整: 3分
  合計: 45分（測定12回の平均）

【実施した改善策と完了日】
  ①型置き場を機械の横（2m以内）に移設（4月末完了）
    → 「持ってくる作業」が倉庫運搬不要になった
  ②型の固定ボルトを4本→2本に変更（標準化・5月中旬完了）
  ③位置決めのガイドピン設置（毎回位置決め不要に・6月初旬完了）

【改善後の計測値（6月末）】
  ・現在の型を外す作業: 8分（変化なし）
  ・次の型を取りに行く: 2分（12→2分 ▲10分削減）
  ・型セット・固定: 8分（15→8分 ▲7分削減）
  ・高さ調整・位置決め: 1分（7→1分 ▲6分削減）
  ・試し打ち・調整: 2分（3→2分 ▲1分削減）
  合計: 21分（45→21分 ▲24分削減 53%短縮）

出力形式:
・「改善成果サマリー（Before/After・削減時間・削減率）」
・「改善効果の金銭換算（1日の型交換回数×削減分×稼働日数×
  設備・人件費/時間）」
・「改善の要因分析（何が最も効果的だったか）」
・「横展開の対象設備・工程の特定（他のプレス機にも適用できるか）」
・「改善成果発表のプレゼン資料の骨子（5分・社内発表用）」
```

## 改善活動の社内発表資料プロンプト

改善成果を全社に伝える発表を準備します。

```
以下の情報をもとに、全社改善発表会での
プレゼン資料（10分間）を整理してください。

【発表の概要】
発表者: 製造部第1班 班長・山田
発表会: 年2回の社内改善発表会（参加者: 全社員55名）
発表テーマ: 「型交換段取り短縮プロジェクト」
発表時間: 10分（質疑応答5分含む）

【アピールしたいポイント】
  ①定量的な成果（段取り45分→21分・53%短縮）
  ②チームの取り組み（班員3名で協力した過程）
  ③お金への貢献（年間削減効果¥1,200,000試算）
  ④横展開（2号機・3号機にも適用予定）
  ⑤次のテーマ（バリ取り工程の残業削減に着手予定）

聴衆の特徴:
  ・社長・工場長: 数字と横展開への関心が高い
  ・他の班の作業者: 自分たちの改善ヒントになるかを見ている
  ・品質・営業部門: 自部門への影響・連携に関心がある

出力形式:
・「プレゼンスライドの構成（8枚以内・各スライドのメッセージ）」
  スライド1: タイトル・チームメンバー紹介
  スライド2: 問題提起（なぜこのテーマを選んだか）
  スライド3〜4: 現状と原因分析（ビジュアル重視）
  スライド5〜6: 改善策と実施プロセス
  スライド7: 成果（Before/After・金額効果）
  スライド8: 今後の展開と次のテーマ
・「各スライドの発言内容（台本・口語体）」
・「想定される質問と回答（3問）」
・「発表後に全員に配る「改善事例サマリー1枚資料」のフォーマット」
```
