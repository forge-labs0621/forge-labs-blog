製造業が直面する法令は労働基準法・製造物責任法・下請法・環境法・独占禁止法など多岐にわたります。AIを使って法令遵守の確認チェックリスト・研修教材・内部通報制度の整備を効率化できます。

---

## 製造業の法令遵守チェックリストを整理するプロンプト

年次で主要法令の遵守状況を確認します。

```
以下の情報をもとに、製造業の年次
コンプライアンスチェックリストを整理してください。

【当社の業種・規模】
業種: 金属プレス加工業（受託製造）
従業員: 45名 / 年商: ¥350M / 主要顧客: 自動車部品Tier1

【確認が必要な主要法令（担当部署ごとに整理）】
労務・人事関連:
  ・労働基準法（36協定・就業規則・賃金・休暇）
  ・労働安全衛生法（安全管理・健康診断・ストレスチェック）
  ・最低賃金法（都道府県別最低賃金との比較）

取引・商取引関連:
  ・下請代金支払遅延等防止法（下請法）
  ・独占禁止法（価格カルテル・不公正取引の禁止）

製品・環境関連:
  ・製造物責任法（PL法）
  ・廃棄物処理法（産業廃棄物の適正処理）
  ・化学物質規制（RoHS・REACH・PRTR法）

情報・個人情報関連:
  ・個人情報保護法（従業員・顧客情報の管理）
  ・不正競争防止法（営業秘密・図面の管理）

出力形式:
・「法令コンプライアンスチェックリスト（確認項目・
  確認方法・担当者・確認頻度の一覧表）」
・「製造業で特に違反リスクが高い項目TOP5の解説」
・「年1回の法令遵守確認の実施スケジュール」
・「コンプライアンス違反を発見した場合の対応フロー
  （発見→報告→対策→再発防止の手順）」
・「全社員向けコンプライアンス誓約書のフォーマット」
```

## 製造物責任（PL法）対策書類を整理するプロンプト

製品が原因で損害が発生した場合に備えます。

```
以下の情報をもとに、製造物責任法（PL法）への
対応書類と手順を整理してください。

【当社の製品と責任範囲】
製品: アルミダイカスト部品（自動車用ブラケット等）
最終製品への組み込み: 顧客（自動車メーカーTier1）が組み立て
当社の立場: 部品製造業者（最終製品の製造者ではない）

【PL法の基本確認】
PL法（製造物責任法）では、製品の欠陥によって消費者に
損害が発生した場合、製造業者は過失の有無に関わらず賠償責任を負う

当社が問われる欠陥の種類:
  ①製造上の欠陥（製造プロセスの問題）
  ②設計上の欠陥（設計自体に問題・客先設計の場合は責任分担が論点）
  ③指示・警告上の欠陥（使用上の注意が不十分）

現在の保険状況:
  ・PL保険（生産物賠償責任保険）: 加入済み（補償上限¥100M）

出力形式:
・「PL法対応の社内体制整備チェックリスト
  （製造記録・品質記録・ロット管理・
  製品保証書・クレーム対応窓口の整備状況確認）」
・「「設計は客先支給図面」の場合の責任分担の考え方
  （欠陥の発生箇所による責任分担の整理）」
・「PL事故が発生した場合の初動対応フロー
  （事故報告受理→原因調査→回収判断→監督官庁への報告→賠償交渉）」
・「PL保険の補償範囲と保険金請求に必要な書類の確認」
・「製造業者として取っておくべき品質記録の保存期間と管理方法」
```

## 内部通報制度の整備書類を整理するプロンプト

コンプライアンス違反を早期発見する仕組みを作ります。

```
以下の情報をもとに、製造業の内部通報制度を
整備するための書類と手順を整理してください。

【当社の現状】
従業員数: 45名
現在の内部通報制度: なし
問題意識:
  ・ある部門で少量の材料の持ち出し（横領）の疑いがあった
  ・上司に言いにくいことが報告されず、問題が長期化する傾向
  ・2022年の公益通報者保護法改正で300人以下でも窓口整備が努力義務

【整備したいこと】
  ①社内通報窓口の設置（気軽に相談できる窓口）
  ②通報者の秘密保護と不利益取扱いの禁止
  ③通報内容の調査・対応プロセス

通報窓口の候補:
  社内窓口: 総務部長（現場から独立）
  社外窓口: 顧問弁護士（中立性の担保）

出力形式:
・「内部通報制度の設計（窓口・対象案件・
  通報方法・秘密保護・不利益取扱禁止の各規定）」
・「内部通報規程の骨子（目的・定義・通報窓口・
  調査手続き・通報者保護・処分の各章）」
・「全社員への制度説明文（なぜ必要か・どう使うか・
  プライバシーは守られるか）」
・「通報を受けた場合の調査プロセス（受理→確認→
  調査→対処→結果通知の流れ）」
・「公益通報者保護法の要点と当社への適用（45名規模の対応義務範囲）」
```
