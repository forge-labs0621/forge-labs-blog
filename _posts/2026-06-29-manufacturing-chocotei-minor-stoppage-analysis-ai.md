チョコ停（ちょこ停・短時間停止）は1回あたりの損失が小さくても積み重なるとOEEの大きな足枷になります。AIを使って発生記録の集計・パレート分析・対策書類を体系化できます。

---

## チョコ停発生記録の集計・分析を整理するプロンプト

```
以下の情報をもとに、チョコ停（短時間停止）の発生記録と
原因分析・対策書類を整理してください。

【チョコ停の発生状況（過去1ヶ月の記録）】
対象設備: 自動組立機（1号機）
集計期間: 2026年6月（稼働日数22日）

発生記録（抜粋）:
  ① 部品詰まり（シュート部）: 47回 / 計2時間18分
  ② センサー誤検知（光電センサー）: 31回 / 計1時間05分
  ③ 部品供給切れ（パーツフィーダー）: 18回 / 計45分
  ④ ビジョン判定エラー（画像検査部）: 12回 / 計32分
  ⑤ エアー圧低下（シリンダー作動不良）: 8回 / 計24分

目標OEE: 85%
現状OEE: 71%（チョコ停による速度・停止ロスが主因）

改善の方向性:
  ・発生頻度トップのシュート詰まり対策を最優先
  ・センサー誤検知の根本原因調査を実施する
  ・部品供給方法の見直しを検討

出力形式:
・「チョコ停パレート分析（発生件数・停止時間・累積比率の表）」
・「上位2件の原因究明書類（発生状況詳細 / 一次原因 / 真の原因 /
  改善の方向性）を5W1Hで整理」
・「シュート部品詰まり対策の改善計画書（対策案 / 効果予測 /
  担当 / 完了期限 / 効果確認方法）」
・「チョコ停ゼロ化に向けた目標設定と週次モニタリング項目」
・「来月の目標OEEと改善後の予測値計算（停止ロス削減効果の試算）」
```

## チョコ停日次記録フォーマットを整理するプロンプト

```
製造現場でオペレーターが記入しやすいチョコ停日次記録シートの
フォーマットを作成してください。

【記録シートの要件】
記入者: ライン担当オペレーター（スマートフォン or 紙）
対象設備: 自動組立機、ロボット溶接機、検査装置（3設備）
記録粒度: 1回の停止ごとに記録
記録項目: 最低限に絞る（記入負担を減らしたい）

必須項目:
  ①発生時刻 ②停止設備 ③停止場所（部位）
  ④停止分類（詰まり・センサー・供給・その他）
  ⑤復旧時刻 ⑥オペレーター対応内容 ⑦再発防止メモ（任意）

集計: 日次でリーダーがExcelに転記→月次集計

出力形式:
・「現場記入用チョコ停記録シート（A4 1枚・横レイアウト）」
・「停止分類コード一覧（現場が迷わない分類定義）」
・「週次集計フォーマット（設備別・原因別の件数・時間マトリクス）」
・「チョコ停記録の定着を促す朝礼でのアナウンス文案」
```
