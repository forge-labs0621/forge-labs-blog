製造業では設備投資・運転資金のために銀行融資・政策金融公庫の借入れが必要になります。AIを使って融資申請に必要な事業計画書・資金繰り表・決算説明資料を整理し、金融機関との折衝を有利に進めましょう。

---

## 銀行融資用の事業計画書を整理するプロンプト

設備投資融資の申請に向けた事業計画書を作ります。

```
以下の情報をもとに、銀行融資の申込みに向けた
事業計画書の骨子を整理してください。

【借入の目的と金額】
借入目的: 設備投資（CNCマシニングセンタ 新規導入）
必要金額: ¥12,000,000
返済期間の希望: 7年（設備の耐用年数10年を考慮）

【設備投資の目的・効果】
現状の課題:
  ・現在は旋盤加工と手仕上げで対応しているが、外注に依存している
  ・自動車部品の受注増加（年間受注量が現在比120%を見込む）
  ・外注費のコスト増加（月50万円程度かかっている）

投資効果（試算）:
  ・内製化による外注費削減: 年間 ¥3,600,000
  ・加工時間短縮による生産能力向上: 現在比 140%
  ・投資回収期間: 約3.5年（投資額¥12,000,000 ÷ 年間効果¥3,600,000）

【会社の概況】
設立: ○年 / 従業員: ○名 / 主要顧客: 自動車部品メーカー
直近3期の売上推移: ○期前 ¥180M → ○期前 ¥195M → 直近期 ¥210M

出力形式:
・「銀行融資の事業計画書の構成（目次と各章の骨子）」
・「事業計画書で銀行が重視するポイント（返済能力・担保・
  事業の将来性の視点）」
・「投資効果の説明文（数値を使った説得力のある表現）」
・「借入返済計画（元金+利息の試算: 金利○%で○年返済の場合）」
・「銀行担当者への説明資料の構成案（PowerPoint想定）」
```

## 資金繰り表を整理するプロンプト

月次の資金繰り見通しを作成して金融機関に提出します。

```
以下の情報をもとに、12ヶ月の資金繰り予定表を
整理してください。

【資金繰りの現状】
現在の月末手元現金: ¥8,500,000

毎月の収入（おおよそ）:
  ・売上入金（翌々月払いが主体）: 月約 ¥15,000,000
  ・設備売却収入: 4月に¥500,000予定

毎月の支出（おおよそ）:
  ・仕入・外注費支払い（翌月末払い）: 月約 ¥6,000,000
  ・人件費（給与・賞与）:
    通常月 ¥3,500,000 / 賞与月（6月・12月）¥5,500,000
  ・設備ローン返済: 月 ¥180,000
  ・地代家賃: 月 ¥350,000
  ・その他経費: 月 ¥500,000

特殊な支出:
  ・新規設備購入（4月）: ¥12,000,000（融資前払い想定）
  ・納税（法人税・消費税）（5月）: ¥2,800,000（前期実績）

出力形式:
・「12ヶ月の資金繰り予定表フォーマット（月別: 収入合計・
  支出合計・収支差引・月末残高）」
・「資金不足が発生する月の特定と必要借入額の試算」
・「資金繰り改善のための提案（入金サイト短縮・支払サイト延長等）」
・「銀行への資金繰り説明のポイント（月末残高の最低水準の確保）」
```

## 決算書の説明・銀行との折衝資料を整理するプロンプト

決算結果を銀行担当者に説明するための資料を作ります。

```
以下の決算情報をもとに、銀行担当者向けの
決算説明資料と業績説明文を整理してください。

【直近決算の結果（前期比較）】
売上高: 前期 ¥195,000,000 → 当期 ¥210,000,000（+7.7%）
売上総利益率: 前期 28.5% → 当期 27.1%（-1.4ポイント）
営業利益: 前期 ¥6,200,000 → 当期 ¥4,900,000（-21.0%）

【業績変動の主な原因】
プラス要因:
  ・主要顧客（自動車部品メーカー）からの増産要請で売上増加
  ・新規顧客を2社開拓

マイナス要因:
  ・材料費（鋼材）の高騰により原価率が悪化（材料費+¥8M）
  ・設備の老朽化に伴う修理費増加（+¥2M）
  ・新規顧客開拓のための営業費用増加（+¥1.5M）

来期の見通し:
  ・受注残あり（¥45M）
  ・材料費の高騰分は顧客への価格転嫁を交渉中（一部承認済み）

出力形式:
・「決算説明資料の構成（業績サマリー・主要因分析・来期見通し）」
・「利益率低下の理由を銀行に説明するための表現（前向きな要因分析）」
・「財務指標の自己評価（自己資本比率・流動比率・借入金依存度）」
・「銀行担当者が懸念するポイントへの先手対応コメント集」
・「来期黒字回復の蓋然性を示す根拠の整理」
```
