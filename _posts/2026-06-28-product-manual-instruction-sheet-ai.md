製造業では製品とともに提供する取扱説明書（取説）・製品仕様書の整備がPL法対応や顧客満足度に直結します。AIを使えば、安全上の注意から操作手順まで体系的に整備できます。

---

## 取扱説明書の構成と内容を整理するプロンプト

読みやすく・法令対応した取説を作ります。

```
以下の情報をもとに、製品取扱説明書（取説）の
構成と各章の内容を整理してください。

【製品情報】
製品名: 小型電動コンベヤ（工場内部品搬送用）
型式: CV-300-AC
主な仕様: 搬送速度0.1〜1.0m/s可変 / 最大積載5kg / AC200V / 重量38kg
対象使用者: 工場内の製造担当者（機械に不慣れな新人も含む）
販売先: 国内製造業（中小工場向け）

取説に含める必要がある情報:
  ・安全上の注意（PL法対応の警告・注意ラベル含む）
  ・各部名称と機能
  ・設置・配線の手順
  ・操作方法（起動・停止・速度調整）
  ・日常点検・定期メンテナンスの手順
  ・故障かな？と思ったときのチェック（トラブルシュート）
  ・仕様表
  ・保証規定

出力形式:
・「取扱説明書の章構成と各章の記載内容・分量配分」
・「安全上の注意（第1章）の全文案」
  （警告・注意・禁止事項の分類 / PL法に対応した表現）
・「操作方法の手順書本文（起動〜停止の標準手順）」
  （番号付き手順・写真説明箇所のキャプション案）
・「トラブルシューティング表の構成」
  （症状 / 考えられる原因 / 対処方法 の3列形式）
・「PL法対応の警告表示（ラベル）文言の例
  （感電・挟まれ・落下等のリスク別に警告シールの文言）」
```

## PL法（製造物責任法）対応の安全警告文書を整理するプロンプト

製品事故を防ぐための警告・注意文書を体系化します。

```
以下の情報をもとに、PL法（製造物責任法）に
対応した製品安全文書を整理してください。

【製品のリスクと現状の課題】
製品: 産業用ガス噴射クリーナー（PCB・部品洗浄用）
使用するガス: 高圧アルゴン（可燃性なし、不活性ガス）
リスクの種類:
  ①高圧ガスによる噴射圧力（目への飛散・皮膚への当て傷）
  ②密閉空間での使用時の酸欠リスク（アルゴンで酸素が希釈される）
  ③ノズル詰まり時の圧力上昇（逆流・破裂のリスク）

現在の警告表示の問題:
  ・日本語のみ（外国人作業者への対応不十分）
  ・「使用上の注意」レベルの表記のみ（PL法上の警告文言として弱い）
  ・絵記号（ピクトグラム）がない

出力形式:
・「PL法で求められる製品安全文書の要件の整理」
  （警告 / 注意 / 禁止事項の3段階表示の基準）
・「改善後の警告ラベル文言（日本語版 + 英語版 + ピクトグラムの説明）」
  各リスク（目傷 / 酸欠 / 圧力）別の警告文
・「取扱説明書の『安全上の重要事項』ページの全文案」
  （PL法に対応した強度の表現を使用したバージョン）
・「製品事故発生時の対応フロー（製造者としての義務）
  （PSE法・消費生活用製品安全法・重大製品事故報告制度の基礎）」
・「社内の製品安全確認プロセス（設計段階から出荷までのリスク確認手順）」
```

## 多言語版製品説明書（日英対応）を整理するプロンプト

海外販売や外国人労働者対応のため多言語化します。

```
以下の日本語製品仕様書・簡易操作手順を
英語版に翻訳・最適化してください。

【翻訳対象】

【製品仕様書（抜粋）】
製品名: 精密NC旋盤（型式: PL-450）
最大旋削径: φ450mm
最大旋削長: 1,000mm
主軸回転数: 50〜4,000rpm（無段変速）
主軸電動機出力: 11kW
制御方式: CNC（G/Mコード対応）
外形寸法: 2,600(L) × 1,400(W) × 1,600(H) mm
本体質量: 3,800kg
電源: AC 200V 3相 / 50/60Hz

【操作手順（抜粋）】
1. 電源スイッチをONにし、起動音を確認する。
2. 安全扉が閉まっていることを確認し、主軸を低速（100rpm）で試運転する。
3. 加工プログラムをCNC操作盤から呼び出す。
4. 試加工前にドライラン（切削なしの空運転）でプログラムを確認する。
5. 加工中は安全扉内に手を入れないこと。

出力形式:
・「英語版製品仕様表（SI単位系 + インペリアル単位系の両方を記載）」
・「英語版操作手順（上記5ステップの英語化）」
  （命令形・能動態・シンプルな英語で記述）
・「技術仕様書でよく使われる英語略語の対訳集
  （rpm / kW / CNC / G-code / M-code / DRY RUN 等）」
・「製造機械の安全操作に関する英語警告文の定型表現集（10例）」
  （例: "Do not reach inside the machine while it is in operation."）
・「日英バイリンガル仕様書のレイアウト設計案
  （左日本語/右英語 vs 上日本語/下英語 の比較）」
```
