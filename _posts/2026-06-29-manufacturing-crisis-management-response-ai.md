製造業では製品事故・工場災害・取引先倒産など多様な危機が突発します。AIを使って危機対応マニュアル・初動対応フロー・ステークホルダー対応書類を素早く整理できます。

---

## 重大品質問題・製品事故の初動対応書類を整理するプロンプト

```
以下の情報をもとに、重大品質問題発生時の
初動対応書類を整理してください。

【緊急事態の概要】
発生日時: 2026-06-29 午前10時
問題内容: 納入済み製品（自動車内装部品）で市場での割れ不良が発覚
  顧客（自動車メーカー）からの連絡: 「組立ライン停止中・原因究明と
  代替品の緊急供給を要請」
影響範囲:
  ・当該ロット: 2026年6月1〜15日製造品（出荷済み3,200個）
  ・顧客の生産ラインが停止（影響: 約1,500台/日の組立が停止）
  ・社内在庫: 同ロット材料で製造中・未出荷品 約800個
  ・当該材料の残在庫: 1,200kg（まだ未加工）

自社の初期状況:
  ・不良の原因はまだ不明（材料ロット不良? 成型条件? 梱包?）
  ・プレス工程・射出成型・梱包の各担当者が原因調査中
  ・副社長が顧客工場へ向かっている

対応チームの緊急招集完了: 社長・品質部長・製造部長・営業部長

出力形式:
・「緊急事態初動対応フロー（発生〜4時間以内のアクションリスト）」
・「緊急対策会議（クライシスミーティング）のアジェンダ
  （現状確認 / 影響範囲特定 / 暫定対策決定 / 役割分担 / 顧客への第一報）」
・「顧客への第一報（電話後のフォローメール）の文案
  （謝罪 / 現状報告 / 暫定措置 / 次の連絡タイミング）」
・「社内出荷停止・在庫隔離の指示書（発令文・対象範囲・担当者）」
・「24時間以内の原因調査スケジュール（各担当の調査項目と期限）」
```

## 取引先倒産・供給停止時の緊急対応書類を整理するプロンプト

```
主要仕入先が突然倒産した場合の
緊急対応書類を整理してください。

【緊急事態の概要】
発生: 主要部品サプライヤー（〇〇部品工業）が民事再生申請
影響:
  ・当社売上の約30%に使用する部品Xを〇〇社から仕入れていた
  ・現在の手持在庫: 部品X → 3週間分のみ
  ・代替サプライヤー: 現時点で候補なし（〇〇社が業界シェア70%の特殊部品）
  ・顧客への出荷に支障が出るまでの猶予: 約3週間

対応の方向性:
  ・〇〇社の管財人と在庫・製造再開交渉
  ・代替品・代替仕入先の緊急探索
  ・顧客へのサプライチェーン問題の早期通知

出力形式:
・「サプライヤー倒産時の緊急対応チェックリスト（最初の72時間）」
・「代替仕入先の緊急探索書類（要件整理・問い合わせ先リスト・RFQ文案）」
・「顧客への供給リスク通知書（現状・影響範囲・対応状況・今後のスケジュール）」
・「管財人・金融機関向けの在庫・受注確認交渉書類」
・「経営層への緊急報告書（損失試算・対応方針・必要な意思決定事項）」
```
