製造業における労働災害は企業として最も重大なリスクの一つです。万が一の発生時に冷静に対応し、正確な事故調査・原因分析・再発防止策の立案・監督署への報告書作成をAIで効率化できます。

---

## 事故調査報告書を整理するプロンプト

労働災害発生後の詳細な原因調査を文書化します。

```
以下の労働災害の情報をもとに、
事故調査報告書を整理してください。

【労働災害の概要】
発生日時: 2026年6月15日（月）午前10:25頃
発生場所: 第2工場 プレス加工ライン（工程3）
被災者: 製造部 加工係 田中○○（35歳・男性・入社10年）
被害状況: 右手薬指の切傷（長さ2cm・深さ5mm） 
         全治2週間の見込み（休業4日間の予定）
災害の種類: 切れ・こすれ（機械作業中）

【事故発生の状況（被災者・目撃者からのヒアリング）】
被災者の証言:
  「作業開始前に、プレス金型の端材が引っかかっているのに気づいた。
   機械を止めずに素手で端材を取り除こうとした時に
   金型のエッジで指を切傷した」

目撃者（同僚B氏）の証言:
  「田中さんが機械を止めずに金型付近に手を入れているのを
   見て声をかけようとしたが、間に合わなかった」

作業環境の確認:
  ・その時間帯は班長不在（休憩で離席中）
  ・端材処理の手順は「機械停止→ロック→端材除去」が正規手順
  ・当日は生産目標達成のプレッシャーがあった（週次目標の最終日）

出力形式:
・「事故調査報告書の構成と各項目の記載内容」
・「直接原因・間接原因・根本原因の分析（なぜなぜ分析の展開）」
・「4M分析（Man・Machine・Method・Material）で見た原因の整理」
・「再発防止策の立案（設備・教育・管理・ルールの各面から）」
・「労働安全衛生法に基づく監督署への報告手続き
  （休業4日 → 労災様式第8号の提出時期・提出先）」
```

## 労働基準監督署への報告書プロンプト

労働者死傷病報告書（様式第23号/8号）の記載を支援します。

```
以下の情報をもとに、労働基準監督署への
労働者死傷病報告書（労基署報告）の記載内容を整理してください。

【法的報告義務の確認】
休業日数: 4日間（休業4日未満は翌四半期に提出、4日以上は遅滞なく）
→ 今回は休業4日 = 遅滞なく提出が必要（労働者死傷病報告 様式第8号）

【報告書に記載する情報（確認済み項目）】
事業場名: ○○製作所 第2工場
所在地: ○○県○○市○○町1-2-3
業種: 金属プレス加工業（製造業）
労働保険番号: 12-345678-000（仮）
被災者氏名: 田中○○
性別: 男性
年齢: 35歳
入社年月日: 2016年4月1日（入社10年）
雇用形態: 正社員
経験年数（現在の業務）: 8年
被災年月日時: 2026年6月15日（月）10時25分
発生場所: 第2工場プレス加工ライン
休業日数（予定）: 4日間

出力形式:
・「様式第8号（労働者死傷病報告書）の各記載欄の内容説明と
  今回のケースでの記載例」
・「事故の原因（記入欄）の適切な書き方
  （簡潔・事実ベース・法的問題になりにくい表現）」
・「提出先（管轄の労働基準監督署）の調べ方」
・「報告書提出後の監督署の対応（調査に来る可能性と準備事項）」
・「労災保険の申請書類（様式第5号・療養補償給付申請）との連携」
```

## 再発防止策と安全管理の改善計画プロンプト

事故後の組織的な安全管理改善を整理します。

```
以下の情報をもとに、今回の労働災害を
受けた安全管理の改善計画を整理してください。

【今回の再発防止策として決定した事項】
①即時対策（当日・翌日）:
  ・全工程の「インターロック機能（機械が動いていると人が接触できない
    物理的機構）」の作動確認（点検実施済み）
  ・端材除去の手順の再教育（全加工作業者15名への緊急KY実施）

②短期対策（1ヶ月以内）:
  ・端材が引っかかりにくい金型のエッジ改良（設計部と協議中）
  ・作業手順書への「機械停止確認→ロック→除去」の明記と掲示
  ・班長不在時の代行者ルールの明確化

③中期対策（3〜6ヶ月以内）:
  ・安全衛生委員会での審議と対策フォローアップ
  ・労働安全衛生マネジメントシステム（ISMS）の見直し

出力形式:
・「再発防止対策の進捗管理表（対策名・担当者・期限・完了確認）」
・「類似作業・類似工程での水平展開チェックリスト
  （全加工ラインで同じリスクがないか確認する）」
・「安全衛生委員会への報告・審議資料の骨子」
・「全社員への水平展開教育（朝礼・KY活動での共有のやり方）」
・「今後の安全パトロール強化計画（頻度・確認ポイント・記録方法）」
```
