金属部品の防錆管理は、製造業において重大なクレームを防ぐ重要な課題です。AIを使って防錆仕様書・梱包基準書・長期保管管理記録を体系的に整備できます。

---

## 防錆処理仕様書を整理するプロンプト

品種ごとの防錆処理基準を文書化します。

```
以下の情報をもとに、防錆処理仕様書を整理してください。

【当社の製品と防錆の実態】
主要製品: 精密機械部品（炭素鋼・合金鋼・ステンレス混在）
顧客先: 自動車部品メーカー3社、農業機械メーカー1社

防錆に関する過去のクレーム実績（過去2年）:
  ・炭素鋼部品の赤錆発生（輸送中に発錆）: 2件
  ・長期在庫品のさび・変色（半年以上保管）: 3件
  ・海外向け輸出品の輸送中発錆（湿度変化）: 1件

現在の防錆処理:
  ・炭素鋼部品: 防錆油塗布（ユシローケン防錆油 #102）
  ・ステンレス部品: 防錆油なし（耐錆性あり）
  ・塗装品: 防錆前処理なし（塗装のみ）

出力形式:
・「防錆処理仕様書（素材別・用途別の防錆処理基準一覧表）」
  （例: 炭素鋼・保管3ヶ月以内 → 防錆油塗布/3ヶ月超 → 防錆フィルム追加）
・「輸送環境別の防錆対策基準
  （国内陸送 vs 輸出海上輸送 vs 長距離保管の違い）」
・「防錆処理の工程チェックリスト
  （処理前清浄度確認 → 処理方法 → 梱包 → 保管条件）」
・「防錆油の種類と適用基準の説明（水置換型 vs 溶剤系 vs 長期型）」
・「発錆発生時の原因分析フロー（何が起きたか → 原因 → 再発防止策）」
```

## 梱包仕様書・出荷基準を整理するプロンプト

品種ごとの梱包基準を標準化します。

```
以下の情報をもとに、梱包仕様書を整理してください。

【梱包・出荷の現状と課題】
主な梱包形態:
  ・小物部品（〜50mm角）: プラスチックケース + 防錆紙包み
  ・中物部品（50〜200mm）: 段ボール箱 + 発泡スチロール緩衝材
  ・長尺物（500mm超）: 木枠 + 防錆フィルム巻き

過去の輸送クレーム:
  ①段ボール箱の角つぶれによる部品へのキズ（2件）
  ②プラケース蓋の開口（輸送中振動で外れた）（1件）
  ③発泡材不足で部品が箱内で動き、傷が発生（3件）

顧客の特別要件:
  ・A社: 樹脂袋への個包装 + バーコードラベル添付が必須
  ・B社: 木枠梱包禁止（虫害リスク）→ スチールフレーム指定
  ・C社: 梱包明細書（ロット番号・数量・製品番号）の同梱必須

出力形式:
・「梱包仕様書の構成（部品種別 × 輸送先 × 保管期間 の梱包基準表）」
・「梱包材選定の判断基準
  （重量・形状・素材・輸送距離・保管期間ごとの推奨梱包材）」
・「梱包作業手順書（梱包前確認 → 防錆処理 → 緩衝材 → 封函 → 表示ラベル）」
・「顧客別梱包特記事項の一覧表
  （A社・B社・C社の要件を一覧で管理）」
・「梱包後の出荷前チェックリスト（外観確認・ラベル確認・数量確認）」
```

## 長期保管品の品質維持管理記録を整理するプロンプト

倉庫に眠る在庫の品質管理を体系化します。

```
以下の情報をもとに、長期保管品の
品質維持管理手順と記録書類を整理してください。

【長期保管品の実態】
現在の長期保管品の状況:
  ・保管品: 完成品在庫 + 内示品（受注確定前の先行製造品）
  ・長期保管の定義: 3ヶ月以上倉庫保管の品
  ・現在の長期保管品数量: 40品番・合計1,200個

品質問題の懸念:
  ①防錆処理の有効期限切れ（油が揮発）
  ②ゴムパーツの硬化・変質（ゴム部品を含む製品）
  ③Oリング等シール部品の経年劣化
  ④長期保管中の包装材劣化（段ボール箱の湿気変形）

現在の管理状況:
  ・入庫時の保管開始日記録: あり（受入票に手書き）
  ・定期点検: 実施していない
  ・保管期限の設定: 品種によって異なるが基準がない

出力形式:
・「長期保管品管理台帳のフォーマット
  （品番 / 数量 / 入庫日 / 品質有効期限 / 最終点検日 / 次回点検日）」
・「品種別の保管期限基準（素材・部品種類別の推奨保管限度）」
  （例: 防錆油塗布の炭素鋼 → 6ヶ月 / ゴムパーツ含む → 1年 等）
・「定期巡視（月次）チェックシート
  （防錆状態 / 梱包状態 / 保管環境 / 問題品の対処記録）」
・「期限切れ在庫の処置フロー
  （再処理可能 vs 廃棄判定の基準と担当者決裁）」
・「在庫管理システムへの期限アラート設定の提案（Excelでの管理方法）」
```
