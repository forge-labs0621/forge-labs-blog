食品製造業では2021年から食品衛生法に基づくHACCP対応が義務化されました（全食品事業者対象）。AIを使ってHACCP計画書・衛生管理記録の文書化・更新を効率化できます。

---

## HACCP一般衛生管理プログラム（PP/PRP）の整理プロンプト

HACCP構築の前提となる一般衛生管理（前提条件プログラム）を整理します。

```
以下の情報をもとに、食品製造業の一般衛生管理プログラム
（前提条件プログラム: PP/PRP）の文書を整理してください。

【自社施設・製品の概要】
施設: （例: 食肉加工工場 / 総床面積○○m2 / 従業員○名）
製造品目: （例: ハム・ソーセージ類 / 日間生産量: ○t）
アレルゲン: （例: 小麦・乳・大豆を使用する製品あり）

【現在の主な衛生管理の取り組み】
・手洗い消毒: 入室時・作業前・トイレ後に実施（手洗い手順書あり）
・施設清掃: 日次・週次・月次の清掃スケジュールあり
・温度管理: 冷蔵保管（4℃以下）・冷凍保管（-18℃以下）
・異物混入対策: 金属探知機（感度Fe2mm・SUS2.5mm）・目視確認
・害虫管理: 月1回専門業者による点検・捕獲記録

整理したい書類:
  ① 施設・設備の衛生管理手順書（清掃・消毒方法）
  ② 従業員の衛生管理規則（健康管理・服装・行動規範）
  ③ 害虫管理記録フォーマット
  ④ 温度管理記録フォーマット

出力形式:
・上記4つの書類の骨子・フォーマット（記録欄含む）
・「法令（食品衛生法・HACCP義務化）の根拠と要求事項のまとめ」
・「第三者検査・保健所立入調査で指摘されやすいポイント」3点
・「一般衛生管理の教育訓練記録のフォーマット」
```

## HACCP計画書（危害分析・CCP設定）を整理するプロンプト

製品ごとのHACCP計画書（フローダイアグラム・危害分析・CCP）を作ります。

```
以下の情報をもとに、HACCP計画書の骨子を作成してください。

【対象製品・プロセス】
製品名: （例: 加熱食肉製品「ロースハム」）
製品特性: （例: 亜硝酸塩含有・pH5.8・Aw0.95・要冷蔵（10℃以下））
対象者: 一般消費者（アレルギー患者への注意書き記載済み）
賞味期限: 製造後21日間

主な製造工程（フローダイアグラム）:
① 原料受入（豚ロース肉・冷蔵4℃以下）
② 下処理（筋取り・カット）→ ③ 塩漬け・ピックル注入（4℃ / 48時間）
④ 充填（ケーシング充填）→ ⑤ スモーク・加熱（中心温度63℃以上 / 30分以上）
⑥ 冷却（1時間以内に10℃以下）→ ⑦ スライス・包装 → ⑧ 金属探知 → ⑨ 冷蔵保管・出荷

【想定される危害要因（事前情報）】
生物学的: リステリア菌（原料由来）/ サルモネラ菌 / 腸管出血性大腸菌
化学的: 亜硝酸塩の過剰使用 / アレルゲン交差汚染
物理的: 骨片 / 金属片 / 包装材の破片

出力形式:
・危害分析表（工程別・危害要因・発生可能性・重篤度・CCP判定）
・CCP一覧（CCP No. / 管理手段 / 管理基準 / モニタリング方法・頻度）
・「CCPモニタリング記録のフォーマット」（加熱工程・金属探知）
・「逸脱（管理基準超過）が発生した場合の是正処置フロー」
・「HACCP計画の妥当性確認（バリデーション）の方法と記録」の骨子
```

## アレルゲン管理書類を作るプロンプト

食品表示法のアレルゲン（8品目+20品目）管理書類を整理します。

```
以下の情報をもとに、アレルゲン管理書類を整理してください。

【自社製品とアレルゲン使用状況】
使用アレルゲン（特定原材料8品目のうち):
  ・小麦（つなぎ使用: 一部製品）
  ・乳（生クリーム・バター: デザート製品）
  ・卵（全卵: 惣菜製品）
  ・大豆（植物油: 全製品の油脂）

製造ライン:
  ・ラインA: 卵・小麦含有品専用
  ・ラインB: アレルゲンフリー品（乳・卵・小麦不使用）
  ・共用設備: 充填機・包装機（切り替え時に洗浄）

管理課題:
  ・ラインBでのコンタミネーション（交差汚染）防止
  ・切り替え時の洗浄効果の確認方法
  ・原料受入時のアレルゲン確認

出力形式:
・アレルゲン管理書類の一覧（必要な書類の洗い出し）
・「原料別アレルゲン一覧表」のフォーマット（仕入先・原料名・アレルゲン有無）
・「ライン切り替え時の洗浄確認手順書・記録フォーマット」
・「食品表示チェックリスト（アレルゲン表示の確認）」
・「アレルゲンコンタミネーション事故発生時の対応フロー」の骨子
```
