年末調整は毎年11〜12月に行われる所得税の精算作業で、経理・総務担当者にとって大きな負担です。AIを使って申告書のチェックポイント整理・控除内容の確認・従業員への案内文作成などを効率化できます。

---

## 年末調整の対象者確認と手続きの流れを整理するプロンプト

年末調整の全体フローと対象者を整理します。

```
以下の情報をもとに、年末調整の実施手順と
対象者の整理を行ってください。

【当社の従業員構成（例）】
正社員: 15名（全員・居住者）
パートタイム: 8名（うち扶養控除等申告書を提出している者: 6名）
外国人従業員: 2名（居住者・特定技能）
中途入社（11月）: 1名（前職あり）

【年末調整のスケジュール（予定）】
  11月中旬: 申告書類の配布・回収開始
  11月末:   書類回収〆切
  12月中旬: 年末調整計算完了・12月給与で過不足精算
  翌年1月末: 法定調書（源泉徴収票・給与支払報告書）提出期限

年末調整の対象外となるケース:
  ・給与年収が2,000万円超
  ・副業があり確定申告が必要
  ・年の途中で退職（退職月の翌月以降に再就職なし → 本人が確定申告）
  ・非居住者

出力形式:
・「当社従業員の年末調整対象者・対象外者の整理（各従業員の対応）」
・「年末調整業務の全体フロー（11月〜翌年1月のタイムライン）」
・「配布する書類一覧（扶養控除等申告書・保険料控除申告書・
  配偶者控除等申告書・住宅借入金控除申告書）と回収期限」
・「従業員への案内文（申告書の提出依頼メール・書き方のポイント）」
・「よくある質問と回答（Q&A: 保険料控除の証明書が来ていない場合等）」
```

## 各種控除の確認チェックリストを作るプロンプト

申告書を受け取ったあとの確認ポイントを整理します。

```
以下の控除項目について、年末調整での
確認チェックリストを作成してください。

【確認すべき主な控除項目】

① 配偶者控除・配偶者特別控除:
  ・確認ポイント: 配偶者の合計所得金額の見込み
    （パート年収の場合: 103万円・150万円・201万円の壁を確認）
  ・申告者の給与収入が1,095万円（合計所得900万円）以下か

② 扶養控除:
  ・確認ポイント: 扶養親族の年齢・所得・続柄
  ・16歳未満（年少扶養）は控除額なし（確認のみ）
  ・特定扶養親族（19〜22歳）・老人扶養親族（70歳以上）は
    控除額が異なる

③ 生命保険料控除:
  ・控除証明書（保険会社から10月ごろ送付される）との照合
  ・一般生命保険料・介護医療保険料・個人年金保険料の区分
  ・新契約（2012年1月以降）と旧契約の区別

④ 地震保険料控除:
  ・証明書との照合
  ・旧長期損害保険契約との区別

⑤ 住宅借入金等特別控除（住宅ローン控除）:
  ・税務署発行の「年末調整のための証明書」の受領確認
  ・適用2年目以降は税務署・市区町村からの通知で申告

出力形式:
・「控除確認チェックリスト（各控除項目・確認内容・
  必要書類・OK/NG判定欄）」
・「よくある記入ミス・もれやすいケースの注意喚起リスト」
・「従業員へのリマインド文（〆切前の催促メール）」
・「証明書が未入手の場合の対応方法（後日提出の手続き）」
```

## 源泉徴収票の内容確認と従業員への説明を整理するプロンプト

発行した源泉徴収票の見方を従業員に説明します。

```
以下の情報をもとに、源泉徴収票の確認方法と
従業員への配布・説明文を整理してください。

【源泉徴収票の主要記載項目（説明が必要なもの）】
  支払金額: 1年間に支払った総給与額（残業手当・賞与を含む）
  給与所得控除後の金額: 支払金額から給与所得控除を差し引いた金額
  所得控除の額の合計額: 扶養控除・生命保険料控除等の合計
  源泉徴収税額: 1年間で実際に天引きされた所得税額
  （過不足精算後の最終税額）

【よくある従業員からの質問】
  Q: 「去年より源泉徴収税額が増えた理由は？」
  → 給与増加・配偶者の収入増（配偶者控除の減額）・
    扶養家族の人数変化

  Q: 「確定申告は必要ですか？」
  → 医療費控除・ふるさと納税（ワンストップ特例以外）・
    副業所得がある場合は本人が確定申告必要

  Q: 「住宅ローン控除の初年度だが手続きは？」
  → 初年度は確定申告が必要（2年目以降は年末調整で対応可）

出力形式:
・「源泉徴収票の記載項目の見方説明文（従業員向け・わかりやすい解説）」
・「確定申告が必要な場合の判定チェックリスト」
・「源泉徴収票の再発行依頼が来た場合の対応手順」
・「翌年1月末の法定調書提出（e-Tax・市区町村への給与支払報告書）
  の作業フロー」
```
