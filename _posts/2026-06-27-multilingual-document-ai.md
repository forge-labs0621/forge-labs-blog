日本の製造業では技能実習生・特定技能外国人の受け入れが増えており、安全指導資料・作業手順書・社内ルール等を多言語で用意することへのニーズが高まっています。AIを使うと、日本語で作成した文書を複数言語に翻訳・平易化する作業を大幅に効率化できます。

## AIが役立つ多言語対応の場面

- **安全教育資料の翻訳**: ヒヤリハット・KY活動・緊急時対応の手順を多言語化
- **作業手順書の平易化・翻訳**: 難しい専門用語を含む手順書を読みやすくする
- **社内ルール・就業規則の要点の翻訳**: 服務規程・禁止事項・申告手順など
- **採用時・入職時の説明資料**: 初日の説明・施設案内・緊急連絡先

## 日本語の文書を英語に翻訳するプロンプト

---

```
以下の製造現場向けの日本語文書を英語に翻訳してください。

【翻訳する文書】
（日本語の文章をそのまま貼り付けてください）

翻訳の方針:
・製造現場で働く外国人スタッフ（日常英語レベル）が読んで理解できる表現にする
・専門用語は英語でも括弧内に簡単な説明を添える（例: lathe (旋盤 / 金属を削る機械)）
・禁止事項・安全注意事項は明確に目立つ書き方にする（MUST NOT / DANGER などを使う）
・1文は短く（20〜30語以内を目安）
```

## 日本語→ベトナム語（Tiếng Việt）に翻訳するプロンプト

製造業では最も多い外国人労働者の出身国の一つです。

```
以下の製造現場向けの日本語文書をベトナム語（Tiếng Việt）に翻訳してください。

【翻訳する文書】
（日本語の文章）

翻訳の方針:
・製造工場で働くベトナム人スタッフが読んで理解できるわかりやすい表現にする
・安全上の禁止事項は「CẤM（禁止）」「NGUY HIỂM（危険）」など
  明確な警告語を使う
・重要な数値・条件は太字または別枠で目立たせる（出力はMarkdown可）
・翻訳後に「難しい専門用語のリスト（日本語/ベトナム語/英語対照）」を
  5〜10個まとめてください

注意: AIの翻訳は参考品質です。ネイティブスピーカー・
ベトナム語を話せる同僚・翻訳サービスによる最終確認を推奨します。
```

## 日本語→インドネシア語（Bahasa Indonesia）に翻訳するプロンプト

```
以下の製造現場向けの日本語文書をインドネシア語（Bahasa Indonesia）に
翻訳してください。

【翻訳する文書】
（日本語の文章）

翻訳の方針:
・製造工場で働くインドネシア人スタッフが読んでわかる表現にする
・禁止事項には「DILARANG（禁止）」「BERBAHAYA（危険）」を使う
・翻訳後に「製造業でよく使う用語（日本語/インドネシア語）」を
  10個リストアップしてください

注意: 最終的にはネイティブスピーカーによる確認を推奨します。
```

## 安全教育資料を多言語化するプロンプト

外国人スタッフ向けの安全教育資料（日本語原文から作成）です。

```
以下の安全規則・注意事項を、外国人スタッフ向けの多言語安全カードとして
整理してください。

【工場の主な安全規則（日本語）】
（例：
1. 作業エリアでは必ず保護メガネを着用すること
2. 機械の動作中は安全カバーを外さないこと
3. 指定された通路以外は歩かないこと
4. 体調が悪い場合は無理をせず、すぐに上司に報告すること
5. 緊急時は赤いボタン（非常停止）を押すこと
）

【用意してほしい言語】
□ 英語（English）
□ ベトナム語（Tiếng Việt）
□ インドネシア語（Bahasa Indonesia）
□ タガログ語（Filipino）
□ その他

出力形式:
・各言語を縦に並べたカード形式
・各項目に番号をつけ、視覚的に対応しやすくする
・「危険マーク：⚠️」「禁止マーク：🚫」などの絵文字を適切に使用
```

## 作業手順書を「やさしい日本語」に変換するプロンプト

翻訳の前に、まず「やさしい日本語」に変換すると翻訳の質が上がります。

```
以下の作業手順書を「やさしい日本語」に変換してください。

【変換する手順書の内容】
（現在の手順書の文章）

「やさしい日本語」の基準:
・1文を短くする（20字以内が目安）
・外来語・専門用語には読み仮名または簡単な説明をつける
  （例：旋盤（せんばん）= 金属をけずる機械）
・「〜すること」という指示形式に統一する
・否定的な表現（〜しないでください）は
  肯定的に言い換える（〜します、または、これはしません）
・「難しい漢字」は仮名に変える
```

## 多言語文書を活用する際の注意

**AIの翻訳の限界について**
- ChatGPTやClaudeの翻訳は高品質ですが、完全ではありません
- 安全に関わる文書・法的な文書は、ネイティブスピーカーまたは翻訳業者による確認を必ず行ってください
- 特に「禁止事項」「緊急時の対応」「怪我・事故時の報告先」は正確さが最重要です

**実用的な活用方法**
- AIで翻訳した文書を、自社の外国人スタッフに見てもらい「意味が伝わるか」を確認する
- 定期的に内容を更新し、古い文書が現場に残らないようにする
- 翻訳文書は「見やすい場所に掲示」することで実際に活用される
