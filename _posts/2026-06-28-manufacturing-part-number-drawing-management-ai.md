品番体系の整備は部品管理・図面管理・BOMの基盤です。AIを使って品番採番ルール・図面番号規則・改訂管理手順を体系化できます。

---

## 品番体系と図面管理規則を整理するプロンプト

```
以下の情報をもとに、製造業の品番採番ルールと
図面管理規則を整備してください。

【品番・図面管理の現状と課題】
現状:
  品番: 各担当者が自分のルールで番号を付けており統一されていない
  図面: Excelと紙図面が混在・最新版がどれか分からないケースがある
  改訂: 図面変更しても旧版が現場に残っていて誤使用が発生したことがある

品番に含めたい情報（ニーズ）:
  ・製品カテゴリ（機械部品 / 電気部品 / 購入品 / 外注品）
  ・材質区分（鉄 / アルミ / SUS / 樹脂等）
  ・連番

図面管理ツール: 現状はWindowsフォルダ共有
将来: PDM（製品データ管理）ツール導入を検討中

出力形式:
・「品番採番ルールの提案
  （例: XX-YYYY-ZZZ で XX=カテゴリ / YYYY=材質・形状 / ZZZ=連番）
  記号の選び方と桁数の根拠まで含めて提案」
・「図面番号規則の設計例（品番との対応関係の付け方）」
・「図面管理台帳のフォーマット
  （図面番号 / 図面名称 / 版数 / 発行日 / 改訂内容 / 保管場所）」
・「図面改訂時の手順書（旧版の回収 / 現場への周知 / 記録の残し方）」
・「フォルダ管理で図面バージョン管理をするためのフォルダ構成の提案」
```
