原材料価格の変動は製造業の収益を直撃します。鋼材・アルミ・樹脂・銅の価格上昇局面でのコスト影響試算・顧客への価格改定申請・リスクヘッジ計画をAIで効率化できます。

---

## 原材料価格変動の収益影響を分析するプロンプト

材料費上昇が利益に与えるインパクトを試算します。

```
以下の情報をもとに、原材料価格上昇による
収益インパクト分析と対策を整理してください。

【主要原材料の価格変動】
材料名 / 使用量（月） / 前年単価 / 現在単価 / 上昇率:
  ① 冷延鋼板（SPCC）: 8,000kg / ¥120/kg → ¥158/kg / +31.7%
  ② アルミ板（A5052）: 2,000kg / ¥380/kg → ¥520/kg / +36.8%
  ③ SUS304: 500kg / ¥680/kg → ¥890/kg / +30.9%
  ④ 銅板（C1100）: 300kg / ¥1,050/kg → ¥1,380/kg / +31.4%

【当社の収益構造（月次平均）】
  月次売上高: ¥40,000,000
  原材料費（上記4品目）: ¥16,000,000（売上比40%）
  加工費・人件費・その他: ¥18,400,000（売上比46%）
  経常利益（前年実績）: ¥5,600,000（利益率14%）

出力形式:
・「材料費上昇額の計算（前年 vs 現在）」
・「利益圧縮額と現在の推定利益率」
・「売上ベースでの価格転嫁必要額
  （材料費上昇分を全て顧客に転嫁する場合の値上げ率）」
・「顧客別の価格改定申請の優先順位
  （売上シェアが大きい顧客から）」
・「材料費変動の月次モニタリング表のフォーマット」
```

## 顧客への原材料価格改定申請書プロンプト

材料費上昇を顧客に説明して値上げ交渉を行います。

```
以下の情報をもとに、主要顧客向けの
原材料価格改定（値上げ）申請書を整理してください。

【申請先顧客の情報】
顧客名: ○○産業機械株式会社 調達部 御中
取引品目: プレス部品（型番: PA-2205, PA-3301他 計15品番）
現在の売上（月次）: 約¥12,000,000
現在の価格設定: ¥2,050/個（PA-2205 の場合）

【価格改定の根拠】
  ・主要材料（冷延鋼板SPCC）の市場価格が前年比+31.7%上昇
  ・社内努力（コスト削減・作業改善）での吸収限界: 10%
  ・今回申請する価格改定率: +13%（材料費上昇分の一部転嫁）
  ・申請内容: ¥2,050 → ¥2,317/個（+267円/個）
  ・適用開始希望: 2026年9月1日

根拠データ（添付資料として作成）:
  ・日本鉄鋼連盟の鋼材価格推移グラフ
  ・当社の原価構成表（材料費比率を明示）

出力形式:
・「価格改定申請書の文案
  （宛先・件名・価格改定の理由・根拠・申請内容・
  適用希望日・今後の方針）」
・「価格改定の根拠データ添付資料の構成案」
・「顧客が「現状維持を希望する」と言ってきた場合の
  次の交渉ステップ」
・「価格改定が一部認められた場合の計算（¥267 → ¥150円/個の場合）」
・「価格改定交渉記録のフォーマット（交渉日・出席者・結果・次回日程）」
```

## 原材料調達リスクヘッジ計画を整理するプロンプト

価格変動リスクを事前に抑える調達戦略を立てます。

```
以下の情報をもとに、原材料の
調達リスクヘッジ計画を整理してください。

【当社の調達リスクの現状】
現在の調達方針:
  ・主要材料（鋼板）: 特定の鋼材商社1社から全量調達（依存度100%）
  ・在庫保有: 通常1ヶ月分を保有

懸念事項:
  ・国際情勢の変化による価格の更なる上昇リスク
  ・特定仕入先への依存（災害・倒産時の供給リスク）
  ・為替変動による輸入原材料の価格変動

ヘッジ手段として検討中のもの:
  ①長期契約（3〜6ヶ月の価格固定）
  ②複数仕入先への分散（デュアルソース化）
  ③在庫増量（2〜3ヶ月分の戦略在庫）
  ④アルミ→鋼材への材料代替の可能性検討

出力形式:
・「各ヘッジ手段のメリット・デメリット・コスト・効果の比較表」
・「長期契約交渉時の仕入先への申し入れ文の骨子」
・「複数仕入先開拓のための新規サプライヤー評価基準」
・「在庫増量時の資金コスト計算
  （在庫¥3M追加 × 金利3% = 年¥90,000の資金コスト）」
・「調達リスク管理の月次報告フォーマット
  （価格動向・在庫水準・仕入先状況の定期チェック）」
```
