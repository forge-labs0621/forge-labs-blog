製造業の間接部門（総務・経理・人事・購買・品質管理）は、書類作成・集計・メール対応・会議準備に多くの時間を費やしています。AIを活用して間接業務を大幅に効率化できます。

---

## 間接業務の時間コストを整理するプロンプト

「どの業務に時間がかかっているか」を可視化します。

```
以下の情報をもとに、間接部門の業務時間分析と
効率化優先テーマを整理してください。

【当社間接部門の業務時間の実態調査結果（1週間）】
対象部門: 総務・経理・人事兼任（2名体制）

業務時間の内訳（2名合計 / 週）:
  ・書類作成（報告書・稟議書・申請書等）: 12時間/週
  ・Excel入力・集計（売上・在庫・勤怠）: 15時間/週
  ・メール対応（仕入先・社内・役所等）: 8時間/週
  ・電話対応・問い合わせ対応: 6時間/週
  ・会議・打ち合わせ（含む準備）: 5時間/週
  ・請求書・証憑の整理・ファイリング: 4時間/週
  ・その他（雑務・来客対応等）: 10時間/週
  合計: 60時間/週（2名 × 30時間 = 週60時間）

担当者からの声:
  「Excelで同じような表を何度も作っている」
  「報告書の文章を作るのに30分以上かかる」
  「メールの返信文を考える時間が意外とかかる」
  「会議の議事録が後回しになって思い出せなくなる」

出力形式:
・「AI活用で削減効果が最も高い業務TOP5の選定と根拠
  （削減可能時間 × 費用換算で優先順位付け）」
・「業務別のAI活用可能度の評価
  （書類作成 / Excel集計 / メール / 会議議事録 それぞれの削減率試算）」
・「1週間で実行できる「AIで時短」のロードマップ
  （Day1: 〇〇を試す → Day3: △△を整備 → Day7: ルーティン化）」
・「間接部門のAI導入で想定される懸念点と対処法
  （機密情報の取り扱い / 出力確認の必要性 / 社内ルール整備）」
・「削減した時間の「使い道」提案（コア業務への集中・価値創造への転換）」
```

## 経理事務のAI活用プロンプト集

月次・年次の経理業務をAIでサポートします。

```
製造業の経理担当者が月次・年次で行う
以下の業務について、AIを活用できる場面と
使えるプロンプトを整理してください。

【経理業務のAI活用ニーズ】
月次業務:
  ①請求書・領収書のデータ整理
    （仕訳補助・勘定科目の振り分け案の生成）
  ②月次報告書の文章作成
    （数字を伝えて「経営報告書」の文章化）
  ③未払い・未収の管理リスト更新
    （未払い催促メールの文章作成）

年次業務:
  ④決算準備の書類一覧と進捗管理
    （税理士への確認事項・資料収集リスト）
  ⑤新入社員の社会保険・雇用保険の届出書類一覧
    （手続き漏れを防ぐチェックリスト）

特に困っている点:
  ・税理士に渡す書類の「説明文」を書くのに時間がかかる
  ・「〇〇について確認してください」という問い合わせメールの文章作成
  ・前年比・予算比の解説コメントを数字から作成する作業

出力形式（各業務に対して）:
・「ChatGPTに使えるプロンプト（コピーして使える形式）」
・「入力すべき情報と出力される内容のサンプル」
・「活用時の注意点（機密情報の範囲・確認必須の箇所）」
```

## ペーパーレス化推進計画を整理するプロンプト

書類の電子化を段階的に進めます。

```
以下の情報をもとに、工場・間接部門の
ペーパーレス化推進計画を整理してください。

【当社の紙書類の実態】
1日に使用する紙: 推定500枚/日
ファイルの棚: 事務所に5段×1.8mキャビネット 12台
紙書類の種類（主なもの）:
  ・製造指示書・作業日報（現場に配布・収集）: 100枚/日
  ・仕入先からのFAX: 50枚/日
  ・受発注書類のコピー: 30枚/日
  ・社内回覧物・連絡書: 20枚/日
  ・その他（領収書・検査記録等）: 300枚/日

社内の意識・環境の現状:
  ・現場作業者（40〜60代）はスマートフォン操作に不慣れな人が多い
  ・FAXは廃止すると仕入先対応が難しい（一部の仕入先がFAXのみ）
  ・電子帳簿保存法（2024年1月〜義務化）への対応が未完

ペーパーレス化の目標:
  ・まず半年以内に紙を30%削減（→ 350枚/日に）
  ・将来的にはFAX廃止・現場への端末（タブレット）導入を視野に

出力形式:
・「ペーパーレス化の優先順位（削減効果 × 実施しやすさのマトリクス）」
  （今すぐできる / 3ヶ月以内 / 半年〜1年 の3段階に分類）
・「現場への端末（タブレット）導入のメリット・デメリット・必要条件」
・「電子帳簿保存法の対応チェックリスト
  （スキャナ保存要件 / タイムスタンプ / 検索機能の整備方法）」
・「ペーパーレス化移行計画書の骨子（6ヶ月ロードマップ）」
・「作業者向けの「なぜペーパーレスにするか」の説明文（理解促進用）」
```
