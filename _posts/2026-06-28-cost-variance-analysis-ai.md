製造業の原価管理では、「標準原価と実際原価の差異（原価差異）」を月次で分析し、報告・改善することが求められます。数値の集計は会計システムで行っても、「なぜ差異が生じたか」「改善策は何か」という文章の説明を書くのに時間がかかることがあります。この記事では、AIを使って原価管理レポートの文章部分を効率化する方法を解説します。

> **注意**: 原価差異の数値データは会計・ERPシステムから取得したものを使用してください。AIに数値を「推測」させることは避けてください。

---

## 原価差異分析コメントを作るプロンプト

月次の差異分析データをもとに、上司・経営層向けの説明コメントを作ります。

```
以下の原価差異データをもとに、月次製造原価差異分析の
「差異の説明コメント」を作成してください。

【対象期間】
（例：2026年7月度 製造2課）

【差異データ（計画値 vs 実績値）】
材料費:
  計画: ¥3,200,000 / 実績: ¥3,450,000 / 差異: +¥250,000（不利差異）
  主な要因: （例：材料歩留まり悪化。不良率が計画2.0%に対し実績3.5%）

労務費:
  計画: ¥1,800,000 / 実績: ¥1,950,000 / 差異: +¥150,000（不利差異）
  主な要因: （例：設備ダウンタイムにより残業が増加。設備故障2回発生）

経費（製造間接費）:
  計画: ¥600,000 / 実績: ¥620,000 / 差異: +¥20,000（不利差異）
  主な要因: （例：設備修繕費が計画外で発生）

【今月の特殊事情】
（例：月中旬に主力設備が2日間停止したため稼働率が低下した）

出力形式:
・差異の原因を項目別（材料費・労務費・経費）に3〜5行で説明
・月次会議で上司に報告できる文章のトーン（箇条書き＋要約）
・「主な課題」「改善アクション」を最後に1〜2行でまとめる
```

## 改善アクション提案をまとめるプロンプト

差異の原因が判明した後、次月に向けた改善案の文章を整理します。

```
以下の状況をもとに、製造コスト改善アクション計画（次月実施分）を
まとめた文章を作成してください。

【課題の内容】
（例：
・材料費: 不良率が計画の2%に対して実績3.5%が続いている
・主な不良原因: プレス工程のダイス摩耗による寸法外れ
・労務費: 不良品の手直し工数が月平均40時間発生している
）

【既に決定している対策】
（例：
・ダイス交換サイクルを3か月→2か月に短縮（今月から実施）
・始業前の型確認チェックリストを追加（来週から）
）

【未定・検討中の対策】
（例：
・生産ラインの作業標準書を改訂するかどうか
・外注先から材料仕入れ先を変更するかどうか
）

出力形式:
・「今月確定アクション」「来月以降の検討事項」に分けて箇条書き
・各アクションに「担当者（○○課）」「期限（○月末まで）」の欄
・改善の効果見込み（不良率△%低減、材料費約△万円削減見込み）を最後に追加
```

## 月次コスト管理報告書のサマリーを作るプロンプト

経営会議向けの簡潔なコスト管理サマリーを作ります。

```
以下のデータをもとに、経営会議向けの製造コスト管理月次サマリーを
作成してください。

【月次データ】
売上高: ¥○○○万円
製造原価合計: ¥○○○万円（計画比：○%増・減）
原価率: ○%（前月比：+○%）

【主な差異項目（差異が大きいもの上位3件）】
1. ○○費 ¥○○万円超過 / 原因: ○○
2. ○○費 ¥○○万円超過 / 原因: ○○
3. ○○費 ¥○○万円削減（改善項目） / 原因: ○○

【前月比トレンド】
（例：材料費は先月比▲3%改善。労務費は+2%増加継続中）

出力形式:
・A4半ページ以内の経営会議用サマリー
・数字+原因+対策のセットで、読んですぐわかる構成
・「課題事項」「次月の注目ポイント」を最後に
・数値はそのまま使用。文章補完のみAIに任せる
```

## 予算実績対比表の「コメント欄」を整理するプロンプト

月次予実対比表に添付するコメント欄の文章を整理します。

```
以下の情報をもとに、予算実績対比表に添付するコメント文を作成してください。

【科目と差異】
製造部門費用（7月度）:
- 消耗品費: 計画¥120,000 / 実績¥165,000 / 差異+¥45,000
  理由: 刃工具の交換頻度増加（品種切り替えが増えたため）
- 水道光熱費: 計画¥80,000 / 実績¥73,000 / 差異-¥7,000
  理由: 節電活動の効果。昼休みの空調停止を徹底
- 外注加工費: 計画¥500,000 / 実績¥680,000 / 差異+¥180,000
  理由: 特急品の外注が3件発生（計画外受注に対応）

出力形式:
・各科目のコメントを2〜3行（数値、理由、今後の見通し）
・「来月は○○を実施して解消予定」という改善見通しを含める
・経理・財務部門に提出できるフォーマルなトーン
```

## 工程別・品番別のロス分析コメントを作るプロンプト

現場で不良ロス・廃棄ロスを集計した後の分析コメントを作ります。

```
以下のロスデータをもとに、工程別ロス分析コメントを作成してください。

【対象月】（例：2026年7月度）

【工程別不良・廃棄ロスデータ】
- プレス工程: 不良品15個 / 廃棄重量8.5kg / ロス金額¥21,000
  主な不良内容: 寸法外れ（バリ発生）
- 旋盤工程: 不良品3個 / ロス金額¥9,000
  主な不良内容: 面粗さ不良（切削条件変更後に集中発生）
- 仕上げ・検査工程: 手直し12件 / 手直し工数4.5時間 / 工数ロス¥13,500

【合計ロス】
金額: ¥43,500（計画ロス¥28,000に対し+¥15,500超過）

出力形式:
・工程ごとの要因サマリー（各2〜3行）
・「最も影響の大きい工程」を明確に
・改善優先項目（1位・2位）をまとめて最後に提示
```
