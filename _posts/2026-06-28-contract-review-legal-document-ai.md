製造業では顧客・仕入先との取引基本契約・製造委託契約・秘密保持契約など多様な契約書を管理します。AIを使って契約書の確認ポイント整理・交渉すべき条項の洗い出し・契約台帳の管理を効率化できます。

> **免責事項**: 本記事はAIによる契約書チェックの補助的な活用方法を紹介するものです。最終的な法的判断は必ず弁護士にご相談ください。

## 取引基本契約書の確認ポイントを整理するプロンプト

顧客から送られた取引基本契約書のリスクを確認します。

```
以下の取引基本契約書の条項について、
製造業（受託加工業）の立場からリスクの高い条項と
交渉すべきポイントを整理してください。

【確認したい主な条項（例）】
① 品質保証・瑕疵担保責任:
  「製品引渡し後3年以内に発見された瑕疵については
  無償で補修または代品を提供しなければならない」
  → 長すぎないか？一般的な期間は？

② 損害賠償の範囲:
  「本契約の違反により損害が生じた場合、
  甲（顧客）は乙（当社）に対し、直接損失のみならず
  逸失利益についても賠償を請求できる」
  → 「逸失利益」まで含むのはリスクが高いのでは？

③ 知的財産権:
  「乙が本製品の製造に際し創作した発明・考案は
  甲の単独所有とする」
  → 製造ノウハウも含まれるのか？範囲が広すぎないか？

④ 代金支払い条件:
  「納品月の翌々月末払い」
  → 下請法の観点で問題ないか？

出力形式:
・「各条項のリスク評価（高/中/低）と懸念点の説明」
・「業界の一般的な条件との比較（何が標準的か）」
・「交渉で変更を求めるべき条項のリスト（優先度順）」
・「代替案の提案文（「このように変更してほしい」という
  具体的な文言の例）」
・「法律専門家（弁護士）に確認を依頼すべき条項の特定」
```

## 製造委託契約書の主要条項を確認するプロンプト

製造を外注する際の委託契約書のポイントを整理します。

```
以下の情報をもとに、製造委託契約書に
記載すべき主要条項を整理してください。

【委託の概要】
委託者（甲）: 当社（組立製品の受注メーカー）
受託者（乙）: A加工（株）（旋削加工専門の外注先）
委託内容: 精密旋削部品の加工（月産1,000個）

【契約書に盛り込みたい主要事項】
① 品質管理: 図面・仕様書の遵守・不良品発生時の対応・
  検査記録の保管義務（10年間）
② 納期・数量: 月次発注・1ヶ月前の内示・確定発注のルール
③ 機密保持: 当社図面・設計情報・顧客情報の第三者への開示禁止
④ 知的財産: 当社が提供する金型・治具の帰属・管理
⑤ 外注の禁止: 無断での再委託禁止（当社承認なし）
⑥ 損害賠償: 品質不良による損害賠償の範囲（下限・上限の設定）

出力形式:
・「製造委託契約書の条項構成（目次）」
・「各条項の主要な記載内容（箇条書きで要点）」
・「製造業でよくある委託契約のトラブルと予防策
  （品質不良時の費用負担・金型管理の問題等）」
・「下請法との関係（支払条件・書面交付義務の確認）」
・「契約書の作成を弁護士に依頼する場合の費用目安と判断基準」
```

## 契約台帳の管理を整理するプロンプト

社内で締結している契約書を一元管理します。

```
以下の情報をもとに、契約台帳の
フォーマットと管理方法を整理してください。

【当社の契約の概要】
取引基本契約: 顧客10社（毎年更新・5年ごとに内容見直し）
製造委託契約: 外注先15社（常時取引 8社・スポット取引 7社）
NDA（秘密保持契約）: 50件超（新規引合いのたびに締結）
設備リース・割賦契約: 5件

【管理上の課題】
  ・古い契約書がキャビネットに紙で保管されていて
    検索しにくい
  ・契約の有効期限・更新時期の把握ができていない
  ・NDAは件数が多く、誰が何を送ったか不明

出力形式:
・「契約台帳のフォーマット（契約ID・契約種別・相手先・
  締結日・有効期限・自動更新条件・担当者・保管場所・
  重要条項メモ）」
・「契約の期限アラート管理（何ヶ月前に誰に通知するか）」
・「NDA管理の効率化方法（一般的な管理実務）」
・「電子契約サービスの活用検討ポイント（コスト・法的有効性・
  取引先の対応状況）」
・「契約書の電子保存の要件（電子帳簿保存法との関係）」
```
