製造業では定期健康診断・ストレスチェック・産業医との連携など、従業員の健康管理に関する法的義務があります。AIを使って健診結果の管理・産業医面談記録・健康管理計画書の整理を効率化できます。

---

## 定期健康診断の実施管理を整理するプロンプト

年1回の定期健診の実施から結果管理までを整理します。

```
以下の情報をもとに、定期健康診断の実施計画と
結果管理書類を整理してください。

【当社の従業員構成】
正社員: 15名（全員対象）
パートタイム（週30時間以上）: 6名（対象）
パートタイム（週30時間未満）: 3名（任意）

【健診実施の概要】
実施時期: 毎年○月（会社として集団健診を実施）
健診委託先: ○○健診センター
健診項目（法定項目）:
  ・問診（既往歴・自覚症状）
  ・身長・体重・BMI・腹囲
  ・視力・聴力
  ・血圧・心電図
  ・血液検査（血糖・脂質・肝機能・貧血）
  ・胸部X線
  ・尿検査

特殊健診（有機溶剤を扱う工程あり）:
  ・有機溶剤健診（6ヶ月に1回）

出力形式:
・「定期健康診断の実施スケジュール（通知→当日→結果管理の流れ）」
・「従業員への健診案内文（受診日程・持参物・受診義務の説明）」
・「健診結果の管理フォーマット（要精密検査者・要治療者の把握）」
・「法的義務の確認（健診結果の5年間保存・産業医への提供の義務）」
・「未受診者への対応手順（受診勧奨・記録の保管）」
```

## ストレスチェックの実施管理書類を整理するプロンプト

50人以上の事業場で義務化されているストレスチェックの管理を整理します。

```
以下の情報をもとに、ストレスチェック制度の
実施計画と結果管理書類を整理してください。

【当社の状況】
従業員数: 21名（法定対象: 50名未満だが義務はなし、任意実施を検討）
→ 取引先から「ストレスチェックの実施状況を確認したい」との要請あり

【ストレスチェックの実施計画】
実施時期: 毎年12月
実施方法: 外部業者（EAP会社）のWebシステムを活用
調査票: 職業性ストレス簡易調査票（57問版）
産業医: 嘱託産業医 ○○先生（月1回訪問）

【実施手順の概要】
1. 実施者（産業医）・実施事務従事者の選定
2. 従業員への事前説明・同意取得
3. 回答（Web入力・30分程度）
4. 結果通知（個人の結果は本人のみ）
5. 高ストレス者の産業医面談（希望者）
6. 集団分析結果を職場改善に活用

出力形式:
・「ストレスチェック実施計画書のフォーマット」
・「従業員への事前説明文（目的・手順・プライバシー保護の説明）」
・「高ストレス者への産業医面談案内文」
・「集団分析結果の読み方と職場改善計画への活用方法」
・「50名未満の事業場が任意でストレスチェックを行う場合の注意点」
```

## 産業医との連携・長時間労働対応書類を整理するプロンプト

産業医面談・長時間労働者への対応記録を整理します。

```
以下の情報をもとに、産業医との連携と
長時間労働対応に必要な書類を整理してください。

【長時間労働の実態】
対象者: 製造部 ○○さん（36歳男性）
直近2ヶ月の残業実績:
  ・先月: 88時間（面接指導の対象: 月80時間超）
  ・今月: 76時間（累計 164時間）

健康状態の訴え:
  ・本人から「最近疲れやすい・頭痛が続いている」との申し出

法的義務:
  ・月80時間超の時間外労働者は産業医による面接指導の実施が義務（労安衛法）
  ・本人の申し出により産業医面談を実施する

産業医の意見:
  ・面談後、産業医から「就業上の措置が必要」との意見書を受領
  ・「2ヶ月間は残業を月45時間以内に制限すること」を推奨

出力形式:
・「長時間労働者への面接指導実施記録のフォーマット
  （対象者特定→本人連絡→面談→産業医意見→会社措置の流れ）」
・「産業医意見書を受けた会社側の就業措置計画書の骨子」
・「本人・所属上長・産業医・人事への情報共有の手順
  （プライバシー保護の観点）」
・「健康管理台帳のフォーマット（要管理者の状況を一元管理）」
・「長時間労働を繰り返さないための残業管理の仕組みの提案」
```
