TPM（Total Productive Maintenance）は「設備のロスをゼロにして生産性を最大化する」取り組みです。AIを使ってTPM活動の計画・OEE分析・自主保全ステップの手順書・改善報告書を効率化できます。

---

## OEE（総合設備効率）の計算と改善テーマ設定プロンプト

設備のロス構造を分析して改善の方向を決めます。

```
以下の情報をもとに、設備のOEE（総合設備効率）の
計算とロス構造の分析を整理してください。

【対象設備】
設備名: プレス加工ライン（主要設備：プレス機200t）
計画稼働時間: 480分/日（8時間×1交代）
稼働日: 週6日（月〜土）

【今月（6月）の実績データ】
計画停止（段取り）: 45分/日（型交換・材料補充）
故障停止: 合計25分/月 → 1.2分/日
チョコ停（小停止）: 合計80分/月 → 3.8分/日
速度低下: 理論サイクルタイム8秒/個に対し実サイクル10秒/個
不良品数: 18個/日（総生産数 280個/日中）
修正・手直し品: 上記18個の内5個は手直しで良品化

出力形式:
・「OEEの3指標（時間稼働率・性能稼働率・品質率）の計算」
  時間稼働率: (480-45-1.2-3.8) / 480
  性能稼働率: 8秒 / 10秒
  品質率: (280-18+5) / 280
  OEE = 3指標の積
・「6大ロスの分析（故障ロス・段取りロス・チョコ停ロス・
  速度低下ロス・不良ロス・立ち上がりロス）」
・「OEE改善による生産増加量の試算（OEE 85%達成時）」
・「最優先で攻める改善テーマの選定（ロスの大きい項目から）」
・「月次OEE管理グラフのフォーマット（今月比較・目標比較）」
```

## 自主保全活動のステップ管理プロンプト

作業者が設備を自ら守る自主保全の推進書類を整理します。

```
以下の情報をもとに、TPM自主保全活動の
ステップ1〜3の推進計画と記録書類を整理してください。

【自主保全の概要（TPM7ステップのうち第1〜3ステップ）】
ステップ1: 初期清掃（設備をきれいにして不具合を発見）
ステップ2: 発生源・困難箇所対策（汚れの発生源をなくす）
ステップ3: 清掃・点検基準書の作成（守れる基準を作る）

【対象設備と実施チーム】
設備: プレス加工ライン（プレス機・コンベア・バリ取り機）
チーム: 第1班 5名（班長: 山田）
実施スケジュール: 今月（6月）からステップ1開始

【ステップ1の初期清掃で発見した不具合（第1回実施後）】
  ・ボルト緩み: 3箇所（プレスフレーム・コンベアレール）
  ・油漏れ: 1箇所（油圧シリンダーのOリング摩耗）
  ・金型の欠け: 1箇所（目視で発見）
  ・コンベアベルトの摩耗: 交換時期に近い

出力形式:
・「ステップ1：初期清掃の実施記録書（清掃箇所・
  発見不具合・処置内容・担当者）」
・「不具合復元計画（専門保全への依頼票のフォーマット）」
・「ステップ2：発生源対策シート（どこで汚れが出るか・
  どう対策するか）」
・「ステップ3：清掃・給油・点検基準書
  （何を・どこで・どの方法で・どのくらいの頻度で）」
・「自主保全進捗管理表（ステップ別チェックと活動時間記録）」
```

## TPM個別改善（ロスゼロ）の改善提案書プロンプト

テーマを絞って設備ロスをゼロに近づけます。

```
以下の情報をもとに、TPM個別改善活動の
改善提案書と成果報告書を整理してください。

【改善テーマ】
テーマ: 「プレス機のチョコ停（小停止）をゼロにする」
改善チーム: プレス班・山田リーダー（3名）
活動期間: 3ヶ月（4月〜6月）

【現状（4月時点・改善前）】
チョコ停の内訳（1ヶ月分の観察記録）:
  ①材料の詰まり（送り量設定ミス）: 12回 / 月
  ②金型端材の引っかかり: 8回 / 月
  ③コンベアの位置ずれ（センサー誤動作）: 5回 / 月
  合計: 25回 / 月（1回平均3.2分のロス = 80分 / 月）

改善策（実施した内容）:
  ①送り量の自動調整設定の見直し + 作業手順書改定
  ②端材処理ガイドの追加設置（金型側面に板を設置）
  ③コンベアセンサーの調整・固定方法の改善

【改善後の結果（6月末）】
チョコ停: 3回 / 月（88%削減）
改善後のロス時間: 10分 / 月（70分の削減）

出力形式:
・「個別改善活動報告書（テーマ・現状・目標・
  改善策・結果・効果の標準フォーマット）」
・「ビフォー・アフターの効果比較（チョコ停回数・
  ロス時間・OEEへの影響試算）」
・「金銭的効果の計算（ロス70分×設備時間コスト×稼働日数）」
・「横展開計画（他の設備・他の班での水平展開）」
・「次のテーマ選定（改善後の新たなロスNo.1の攻め方）」
```
