製造業では、製品の設計変更が発生するたびに「設計変更通知書（ECN: Engineering Change Notice / DCN: Design Change Notice）」を関係者に発行し、影響部署・仕入先・外注先に周知する必要があります。変更内容の整理・連絡文の作成・影響範囲の確認チェックリストをAIで効率化できます。

> **注意**: 設計変更の内容（寸法・材質・仕様）は設計部門が正式に確定した変更内容のみを使用してください。AIが変更内容を推測・生成することは避けてください。

---

## 設計変更通知書の骨子を整理するプロンプト

設計変更の内容をまとめた社内向けECNを作ります。

```
以下の情報をもとに、設計変更通知書（ECN）の文章を作成してください。

【変更の基本情報】
ECN番号: （例：ECN-2026-0720-001）
対象製品・品番: （例：組立品 ASM-1234、サブアセンブリ SA-456も影響あり）
変更理由: （例：客先○○社からのデザインレビューで強度不足の指摘があったため）
変更区分: （例：設計変更（材質変更））
有効日: （例：2026年8月1日出荷分から適用）

【変更内容（変更前 → 変更後）】
- 変更箇所1: ブラケット材質
  変更前: SPCC t1.6mm
  変更後: SECC（電気亜鉛めっき鋼板）t2.0mm
- 変更箇所2: 固定ボルト本数
  変更前: 4本（M5×20）
  変更後: 6本（M5×20）
- 図面改訂: Rev.B → Rev.C（図面番号DWG-ASM-1234-C）

【影響を受ける工程・部門】
- 製造部（ブラケット成形工程）
- 購買部（SECC材の発注先変更）
- 品質保証部（検査基準書の更新）
- 外注先: ○○プレス工業（ブラケットの試作・量産移行）

出力形式:
・ECN正式フォーマット（変更概要・変更内容比較・影響部署・有効日・承認欄）
・変更前後を比較した表形式を含める
・承認者欄・確認者欄（プレースホルダー）
・「この変更により既存在庫・仕掛品の扱いを確認すること」の注記
```

## 仕入先・外注先への設計変更連絡文を作るプロンプト

変更内容を仕入先・外注先に正式に伝える連絡文を作ります。

```
以下の情報をもとに、仕入先（外注先）への設計変更連絡文を作成してください。

【連絡先】
送付先: （例：○○プレス工業 株式会社 製造部 ○○様）
自社担当: （例：○○製作所 購買部 ○○）

【変更内容の概要】
品番: （例：SA-456 ブラケット（プレス加工品））
変更内容: （例：材質: SPCC→SECC、板厚: 1.6mm→2.0mm、固定穴数: 4→6）
図面改訂: （例：Rev.B → Rev.C）
有効日: （例：2026年8月1日出荷分から）

【仕入先への依頼事項】
- （例：(1) 改訂図面（Rev.C）を確認し、変更可否・量産移行コスト・
  リードタイムへの影響をご回答ください
  (2) 新材質（SECC）の調達可否を確認ください
  (3) 変更後の試作品（10個）を○月○日までに提出ください）

出力形式:
・取引先へのビジネスメール形式
・変更内容の比較表（簡略版）を本文に含める
・依頼事項を番号付きで明確に
・回答期限（○月○日まで）を明記
・添付資料一覧: 「改訂図面DWG-ASM-1234-C（別途送付）」
```

## 設計変更の影響確認チェックリストを作るプロンプト

変更発生時に確認すべき影響範囲を洗い出すチェックリストを作ります。

```
以下の変更内容をもとに、設計変更時の社内確認チェックリストを作成してください。

【変更の種類】
（該当するものを選択: 材質変更 / 形状・寸法変更 / 部品追加・削除 / 
  外注先変更 / 工程変更）
→ 今回: 材質変更 + 寸法変更（板厚・穴数）

【影響確認が必要な領域】
- 設計・図面管理
- 製造工程（加工方法の変更有無）
- 検査基準・検査方法
- 購買・調達（材料・部品の調達先変更）
- 在庫・仕掛品（変更前の在庫の取り扱い）
- 外注先（加工条件の変更通知）
- 顧客（変更承認が必要か）
- ISO 9001 文書（4M変更管理記録との連動）

出力形式:
・「確認項目 / 担当部門 / 確認期限 / 確認結果（記入欄） / 承認（記入欄）」
  の表形式
・「重要確認」事項（顧客承認が必要な変更か、試作・評価が必要か）を
  最初に大きく記載
・完了サイン欄（変更管理責任者）
```

## 変更後の作業標準書・手順書の改訂ポイントを整理するプロンプト

設計変更に伴い、作業標準書・検査基準書の改訂が必要な箇所を整理します。

```
以下の設計変更内容をもとに、作業標準書・検査基準書の改訂が必要な
箇所をリストアップしてください。

【変更内容】
（例：
・材質変更: SPCC t1.6mm → SECC t2.0mm
・穴数変更: 4穴 → 6穴
・重量変化: 約150g → 約200g（推定）
）

【現在存在する関連文書（わかる範囲で）】
（例：
・ブラケット加工作業標準書（SOP-BRKT-001）
・プレス工程検査基準書（QI-002）
・組立工程作業標準書（SOP-ASM-003）
）

出力形式:
・文書ごとに「改訂が必要な箇所・変更前・変更後の内容」を箇条書き
・「改訂不要（変更の影響なし）」の文書も明記
・改訂優先度（高・中・低）を付ける
```
