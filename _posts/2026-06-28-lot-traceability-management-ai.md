製造業では品質問題が発生した場合に「どのロットの製品が影響を受けるか」を特定できることが必須です。特に自動車部品・食品・医療機器では取引先からトレーサビリティの確保を義務付けられることがほとんどです。製造記録・ロット管理のルール整備をAIで補助できます。

---

## ロット管理ルール・ロット番号体系を整理するプロンプト

自社のロット番号の付け方と管理ルールを整理します。

```
以下の情報をもとに、ロット管理ルール（ロット番号体系と
トレーサビリティのルール）を整理してください。

【自社の生産形態・製品】
業種・製品: （例：受託プレス加工業 / 自動車部品）
生産ロット: （例：受注単位（注文書1件＝1ロット）または日付単位）
材料の種類: （例：鉄板材料を仕入先から購入 → 加工 → 出荷）
現在のロット管理の状況:
  （例：ロット番号はあるが付け方がバラバラで担当者によって異なる。
  市場クレームが起きたときに追跡できるか不安がある）

【設計したいロット番号の体系（案）】
（例：「YYYYMMDD-品番-シーケンス番号」形式
  例: 20261001-A1234-001
  → 2026年10月1日に加工した品番A-1234の1番目のロット）

【トレーサビリティで追跡したいこと】
・使用した材料の仕入先・ロット（材料成績書に記載のロット番号）
・加工した設備の番号・担当者
・品質検査の結果（合否・不良数）
・出荷日・出荷先（どの取引先の何の注文に使ったか）

出力形式:
・ロット番号体系の設計案（記号の意味の説明付き）
・「製造ロット記録票」のフォーマット（1ロットごとに記録する内容）
・「材料ロット → 製品ロット → 出荷先」の追跡フローの説明
・「ロット番号を現品票・製品に貼付する方法」の提案
```

## 製造記録・現品票のフォーマットを作るプロンプト

ロットごとに製造実績を記録する「製造記録」と「現品票」を作ります。

```
以下の情報をもとに、製造ロット記録票（製造記録）と
現品票（実物に貼る識別ラベル）のフォーマットを作成してください。

【記録する内容（製造記録票）】
・ロット番号
・品番・品名・数量
・製造日
・使用材料（仕入先ロット番号）
・使用設備番号
・担当作業者
・工程内品質検査の結果（測定値・合否）
・最終検査の結果
・出荷先・出荷日
・検査員の承認サイン

【現品票（製品またはケースに貼るラベル）に含める情報】
・品番・品名
・ロット番号
・数量
・製造日（または出荷日）
・合否（検査済みスタンプまたは記載）

出力形式:
・製造ロット記録票のフォーマット（A4 1枚、記入欄付き）
・現品票（ラベル）のフォーマット案（名刺サイズ程度を想定）
・「記録票の保管期間」の提案（製品寿命・取引先要求を考慮）
・「電子化への移行ステップ」のアドバイス（紙記録 → Excel → システム）
```

## 市場クレーム・品質問題のトレーサビリティ調査プロンプト

問題が発生した際に影響を受けるロットを特定する調査手順を整理します。

```
以下の情報をもとに、市場クレームへのトレーサビリティ調査の
手順と調査報告書のフォーマットを作成してください。

【問題の概要】
（例: 取引先から「出荷品A-1234に外観不良（打痕）がある。
  ロット番号20261001-A1234-003の一部に発生。
  現在在庫中の同ロット品と、直近の3ロット（001〜003）の確認を依頼された」）

【調査すべきこと】
(1) 問題ロットの製造記録の確認
  ・使用した材料のロット番号（仕入先まで遡る）
  ・加工した設備・担当者
  ・加工当日の品質記録（初物確認・中間検査）
  
(2) 影響範囲の特定
  ・同じ材料ロットを使用した他のロット品がないか
  ・同じ設備を使用した前後のロットに同様の不良がないか
  ・出荷済み品と在庫品の数量・行き先

(3) 回収・対応判断
  ・在庫品の全数確認 → 不良品の隔離
  ・出荷済み品の返品依頼が必要かどうかの判断基準

出力形式:
・トレーサビリティ調査の手順（STEP形式）
・調査報告書のフォーマット（問題概要・調査内容・影響範囲・処置）
・「調査完了後に客先に報告する内容のテンプレート」
・「今回の問題から導かれるトレーサビリティ改善の提案」
```
