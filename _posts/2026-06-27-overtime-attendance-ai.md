製造業の現場では、繁忙期の残業申請・シフト変更・振替休日の管理が頻繁に発生します。AIを使うと、これらの申請書・通達文・連絡文の下書きを素早く作ることができます。

## AIが役立つ勤怠管理の場面

- **残業申請書**: 理由・対象者・予定時刻を整理して文書化する
- **振替休日申請書**: 出勤した休日と代替の休日を記録する
- **シフト変更の社内連絡**: 班全体へのシフト変更通知を作る
- **休日出勤の連絡文**: 急な休日出勤を従業員に連絡する
- **勤怠の月次集計コメント**: 月次の残業時間・欠勤状況を報告する文章

## 残業申請書を作るプロンプト

工場の班長・ライン長が上司に出す残業申請書の下書きです。

```
以下の情報をもとに、残業申請書を作成してください。

【基本情報】
申請者: （例：製造2課 係長 田中○○）
申請日: （例：2026年7月15日）
残業を行う日: （例：2026年7月17日（木曜日））
残業予定時刻: （例：定時17:30終了 → 20:00まで）

【残業する理由】
（例：
・受注品 B-500型プレス部品300個の出荷期限が7月18日の午前中に設定されている
・現在の加工進捗は250個完了。本日の定時までに完了する見込みが立たなくなった
・納期遅延を避けるため、明日の残業で残り50個の加工・梱包を完了する必要がある
）

【残業予定者】
（例：
・山田○○（旋盤担当）
・佐藤○○（梱包担当）
の2名）

出力形式:
・「時間外労働申請書」として
・申請日・申請者・所属・残業日時・残業理由・残業予定者
・承認者欄（所属長・部門長のプレースホルダー）
・提出先・承認方法（自社の規程に合わせて確認してください）を注記
```

## 振替休日申請書を作るプロンプト

休日に出勤した際に、別の日を振替休日として取得する申請書です。

```
以下の情報をもとに、振替休日申請書を作成してください。

【基本情報】
申請者氏名: （例：製造1課 鈴木○○）
申請日: （例：2026年7月20日）

【休日出勤した日】
- 出勤した日（本来の休日）: （例：2026年7月13日（日曜日））
- 出勤した理由: （例：緊急の設備修理対応のため）
- 出勤時刻〜退勤時刻: （例：9:00〜16:00）

【振替で取得する休日】
- 振替休日取得日: （例：2026年7月25日（金曜日））

出力形式:
・「振替休日申請書」として
・出勤した日・理由・時刻、振替取得予定日
・申請者氏名・所属・署名欄
・承認者（直属上司・人事担当）欄
```

## シフト変更を班全体に連絡するプロンプト

急なシフト変更を班全体に周知する社内連絡文を作ります。

```
以下の情報をもとに、班全体へのシフト変更連絡文（掲示板・チャット用）を
作成してください。

【変更内容】
- 対象: （例：製造2課 第1班 全員）
- 変更後のシフト: （例：来週7月21日（月）〜25日（金）は
  早番（7:00〜16:00）に変更。通常の昼番（8:00〜17:00）から変更）
- 変更の理由: （例：来週、大型受注品の出荷期限が月初に集中するため、
  加工時間を確保するために早番に切り替える）
- 変更後の連絡先（質問がある場合）: （例：係長 田中まで）
- 伝達日（掲示日）: （例：2026年7月18日）

出力形式:
・掲示板（A4）またはチャットグループ向けに
・要点が一目でわかる形式（変更内容・期間・理由・問い合わせ先）
・丁寧だが簡潔に。読み飛ばされないよう変更日時を目立たせる
```

## 急な休日出勤の連絡文を作るプロンプト

設備トラブルや納期急変で急きょ休日出勤が必要になった場合の連絡文です。

```
以下の状況で、急な休日出勤の連絡文（SMSまたはグループチャット用）を
作成してください。

【状況】
- 連絡する相手: （例：製造1課 第2班 4名）
- 出勤要請する日: （例：2026年7月20日（日曜日））
- 出勤時刻: （例：8:00〜12:00（午前のみ））
- 急に出勤をお願いする理由: （例：前日の夜に設備の一部が故障し、
  月曜の稼働再開に向けて日曜午前中の点検・復旧作業が必要になった）
- 対応できる場合の連絡期限: （例：今夜（土曜日）22時まで返信をお願いしたい）

出力形式:
・短くて読みやすいチャット/SMS向けの文体
・「誰が・いつ・何時から何時まで・何のために」を明確に
・来られない場合の連絡も一言添える
```

## 残業時間の月次集計コメントを作るプロンプト

毎月の残業実績を管理職・人事に報告する際のコメント文です。

```
以下の数値をもとに、月次残業実績の報告コメントを作成してください。

【今月の残業実績（○月分）】
- 対象部署・人数: （例：製造1課 12名）
- 残業時間の合計: （例：312時間（前月比 +68時間）)
- 最大残業者: （例：鈴木○○ 45時間（36協定上の上限に近い）)
- 残業が増加した主な原因: （例：7月中旬に大口受注が入り、
  月末に向けて出荷が集中したため）
- 来月の見通し: （例：受注は落ち着く見込み。8月は残業を大幅に
  削減できる予定）

出力形式:
・「○月 残業実績報告」として
・今月の実績と前月比較
・増減の主な原因
・36協定上限への接近状況（注意が必要な場合）
・来月の対応方針
```

## AIを使う際の注意

- 残業申請書・振替休日申請書の書式は、各社の就業規則・労使協定（36協定）で定められた形式に従う必要があります。AIが作成した文書は下書きとして使用し、**必ず自社の書式・規程に合わせて修正してください**
- 36協定（時間外・休日労働に関する労使協定）では、残業の上限時間が定められています。残業申請書の内容が36協定の範囲内に収まっているかは、人事・総務担当者が確認してください
- 従業員の勤怠情報（氏名・残業時間・出勤日）は個人情報です。外部AIサービスへの入力時は取り扱いに注意してください
