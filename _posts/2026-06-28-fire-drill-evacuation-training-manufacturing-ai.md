製造業では年2回以上の避難訓練が消防法で義務付けられています。AIを使って訓練計画書・実施記録・消防計画の更新を効率的に整備できます。

---

## 避難訓練実施計画書を整理するプロンプト

訓練前の計画書を素早く作成します。

```
以下の情報をもとに、工場の避難訓練
実施計画書を整理してください。

【当社工場の概要と訓練の背景】
工場規模: 建物2棟（工場棟1,500m² + 事務棟500m²）
在籍人数: 従業員48名（正社員28名 + パート・派遣20名）
前回訓練: 半年前（昨年12月）/ 消防法上 年2回義務 → 次回6月実施必要

過去の訓練の問題点:
  ①パート社員への事前周知が徹底されず、知らなかった人が数名いた
  ②工場棟の煙充満時を想定した「低い姿勢での避難」が形式的だった
  ③集合場所で全員の人数確認に8分かかった（目標3分以内）
  ④外国人スタッフ（3名）への多言語案内がなかった

今回の訓練テーマ: 「夜間火災を想定した初動対応と連絡体制確認」
（昼間のみではなく、夜間・早朝の少人数対応を意識した訓練）

出力形式:
・「避難訓練実施計画書（A4 2枚程度）の全文構成」
  （実施日時 / 想定火災 / 訓練の目的 / 役割分担 / タイムライン / 評価基準）
・「役割別の行動シナリオ
  （火災発見者 / 消火担当 / 連絡・通報担当 / 避難誘導担当 / 安全確認担当）」
・「外国人スタッフ向けの簡易英語版訓練案内
  （避難ルート・集合場所・役割の英語説明文）」
・「訓練時間を短縮するための事前準備チェックリスト
  （名簿作成 / 役割カード配布 / 放送内容の事前確認）」
・「消防署への訓練実施通知文（提出が推奨される場合の例文）」
```

## 避難訓練実施後の記録書を整理するプロンプト

訓練後の反省・是正を文書化します。

```
以下の訓練実施後の観察データをもとに、
避難訓練の実施記録・評価報告書を整理してください。

【訓練実施後のデータ（観察・測定値）】
実施日: 2026年6月28日（土）10:00〜
想定: 工場棟2階東側から出火（電気系統の火災想定）
参加者数: 45名（3名欠席・病欠）

訓練の経過（タイムライン）:
  10:00 訓練開始・火災発見者（製造2課・田中）が発報
  10:01 場内放送「火災発生・避難開始」
  10:04 消火担当が消火器で初期消火訓練実施（消火成功と判定）
  10:05 全員が指定の避難経路で外の集合場所へ
  10:08 人数確認完了（所要時間 8分）→ 目標3分を超過

訓練中に気づいた問題点（観察者メモ）:
  ・非常口ドアの内側に台車が置かれており、開けるのに手間取った
  ・2名が携帯電話を取りに戻ろうとして注意が必要だった
  ・集合場所での人数確認が班ごとではなく「全員まとめて数える」になっていた
  ・消火器を実際に噴射した経験のある社員が5名しかいなかった

訓練後のアンケート（自由回答より）:
  「避難経路を知らなかった（入社3ヶ月のパート社員）」
  「煙が出たらどうすればいいか分からなかった」

出力形式:
・「避難訓練実施記録書（法定記録として保管する様式）」
  （実施日時 / 参加人数 / 訓練内容 / タイムライン / 評価まとめ）
・「問題点の整理と是正措置の一覧表
  （問題 / 原因 / 是正措置 / 担当者 / 期限 の4列）」
・「次回訓練での改善ポイントTOP3の選定と具体的な改善案」
・「消防計画の更新が必要な箇所の指摘
  （非常口前の物品放置禁止規定の追記等）」
・「全社員向けの訓練後フィードバック通知文（一文一文を短く・具体的に）」
```

## 消防計画書の作成・更新を整理するプロンプト

法令で義務付けられた消防計画を整備します。

```
以下の情報をもとに、防火管理者が作成・届出する
消防計画書の構成と内容を整理してください。

【当社の概要と消防計画の現状】
施設区分: 工場・倉庫（一般工場）
面積: 工場1,500m²・事務所500m²（合計2,000m²）
防火対象物の用途: 準防火対象物（4号）
防火管理者: 製造部長・山本（甲種防火管理者講習修了）
現在の消防計画: 3年前に作成（大幅な変更なし・最新の人員構成に未更新）

更新が必要な背景:
  ①従業員が32名→48名に増加（外国人スタッフ3名の追加含む）
  ②工場内レイアウト変更（加工エリアが東から西に移動）
  ③パート社員の増加（夜間・早朝シフト対応）

消防計画に含める必要がある主な内容:
  ①防火管理体制（防火管理者・火元責任者の氏名・役割）
  ②出火防止の措置（工場内の火気管理ルール）
  ③消防設備の点検・維持管理
  ④避難訓練の計画・実施記録
  ⑤自衛消防組織の編成

出力形式:
・「消防計画書の標準的な構成（章立て）と各章の記載内容」
  （消防法施行規則第3条に基づく作成内容の解説も含む）
・「防火管理体制図（役職者の名前・役割を入れた組織図の記載方法）」
・「工場内の火気管理ルール文書（禁煙場所・火気使用届・作業前後の点検）」
・「消防計画書を消防署に届け出る際の手続き手順
  （届出先 / 様式 / 記載例）」
・「消防計画の定期見直しのタイミングと更新チェックリスト
  （人員変更・レイアウト変更・設備追加時の更新基準）」
```
