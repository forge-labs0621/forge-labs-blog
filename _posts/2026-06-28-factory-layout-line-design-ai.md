工場レイアウトは生産性・安全性・動線効率に直接影響します。AIを使って現状分析・レイアウト変更の効果シミュレーション・設備配置計画書・工事計画書を効率的に整理できます。

---

## 現状の工場レイアウト課題を整理するプロンプト

工場の動線・設備配置の問題点を体系的に把握します。

```
以下の情報をもとに、工場レイアウトの課題と
改善方針を整理してください。

【工場の概要】
工場面積: 延床 1,200m²（1階建て）
主な製造工程（フロー順）:
  材料受入 → 切断 → プレス加工 → バリ取り → 検査 → 梱包 → 出荷

現在の設備配置（大まかな位置）:
  ・材料置き場: 入口付近（東側）
  ・切断機: 北東エリア
  ・プレス機: 中央〜南側
  ・バリ取り（手作業台）: 北西エリア（プレスから遠い！）
  ・検査台: 北側
  ・梱包・出荷: 西側（入口と反対端）

【現在の課題】
  ・バリ取りエリアとプレスが離れているため、運搬時間が長い
  ・材料と完成品の動線が交差している（フォークリフトと人の交差）
  ・出荷エリアが入口と逆サイドなので、トラックへの積み込みに時間がかかる
  ・検査エリアが暗く（照明不足）、見落としのリスクがある

出力形式:
・「工場レイアウト分析（動線・距離・安全性の観点から課題を整理）」
・「TPSの観点でのムダ（運搬のムダ・動作のムダ）の定量化
  （1日の移動距離・時間の試算）」
・「改善後のレイアウト案の骨子（どの設備をどこに移動するか）」
・「レイアウト変更による効果の試算（搬送時間削減・安全性向上）」
・「改善案の実施優先順位（すぐできる・設備移動が必要・工事が必要の分類）」
```

## 新ライン設計・タクトタイム計算をするプロンプト

新製品の生産ラインを設計します。

```
以下の情報をもとに、新製品向けの組立ラインの
設計とタクトタイム計算を整理してください。

【新製品の概要】
製品名: 新型電子機器用ブラケット組立品
月産計画: 5,000個/月
稼働日数: 20日/月
1日あたりの生産目標: 250個/日

【各組立工程の作業内容と標準時間（想定）】
工程①「部品セット」: 本体プレートに部品をセットする（2分/個）
工程②「ネジ締め」: 4本のネジを締める（3分/個）
工程③「ハーネス接続」: 電線の接続（4分/個）
工程④「外観確認」: 外観検査・NG品の取り除き（1.5分/個）
工程⑤「梱包」: 梱包材に入れ、ラベル貼付（1分/個）

作業者数の制約:
  ・投入できる作業者は最大4名
  ・1人が複数工程を担当する柔軟な配置を検討

出力形式:
・「タクトタイム計算（1日の稼働時間 ÷ 1日の生産目標 = タクト）」
・「各工程のサイクルタイムとタクトタイムの比較
  （ボトルネック工程の特定）」
・「作業者の最適配置案（4名でどう工程を分担するか）」
・「ラインバランス計算（最も効率的な工程分担の数値評価）」
・「生産ライン設計書（工程フロー・設備・作業者配置・必要面積）」
```

## レイアウト変更工事計画書を作るプロンプト

設備移設・レイアウト変更の工事計画を整理します。

```
以下の情報をもとに、工場レイアウト変更工事の
計画書を整理してください。

【レイアウト変更の概要】
変更内容:
  ① バリ取り作業台（手作業台×3台）をプレス機の隣に移設
  ② 材料保管ラック（2,400×900×1,800mm × 5連）を東壁沿いに整理
  ③ 検査エリアの照明設備を増設（LED投光器2台）

制約条件:
  ・工事中も生産は止めたくない（土日での作業が必要）
  ・バリ取り台の移設は電源配線工事が必要（200V・30A）
  ・照明工事は電気工事士が必要（外注依頼予定）

工事予算（概算）:
  ・作業台移設（設備業者+社内人員）: ¥150,000
  ・照明工事（電気工事業者外注）: ¥120,000
  ・合計: ¥270,000

スケジュール希望:
  ・3週間後の土曜日（作業台移設）→ 翌々週の土曜日（照明工事）

出力形式:
・「工事計画書（目的・変更内容・作業手順・スケジュール・
  安全注意事項）のフォーマット」
・「工事中の生産への影響と暫定対応策（一時的な代替手順）」
・「工事後の確認チェックリスト（設備の正常稼働・安全確認）」
・「作業者への事前説明文（工事の目的・当日の作業手順変更の案内）」
・「工事効果の測定計画（工事前後の搬送時間・不良率の比較）」
```
