VE（Value Engineering / バリューエンジニアリング）は「機能を維持しながらコストを削減する」体系的な手法です。AIを使ってVE提案書・コスト削減アイデアの整理・実施計画書・効果測定レポートを効率化できます。

---

## VE提案のアイデアを整理するプロンプト

機能とコストの観点から改善案を洗い出します。

```
以下の製品・部品情報をもとに、VE（バリューエンジニアリング）の
観点からコスト削減アイデアを整理してください。

【分析対象の部品・製品】
部品名: （例: 機械装置用カバー部品）
現在の仕様:
  ・材質: SPCC（冷延鋼板）t2.0mm
  ・加工方法: プレス成形 + 溶接 + 粉体塗装
  ・現在の製造コスト: ¥2,800/個
  ・生産数量: 月500個

【この部品に求められる機能】
  必要機能（Must機能）:
    ・内部機構を物理的衝撃から保護する
    ・防塵性の確保（IP54相当）
    ・設置・取り外しが工具なしで可能
  あると良い機能（Want機能）:
    ・外観が整っている（顧客が目にする面）
    ・軽量であること

【コスト削減目標】
  ・現行 ¥2,800 → 目標 ¥2,200以下（21%削減）

VEの検討視点:
  ① 機能の廃止・統合（不要な機能を削除）
  ② 材料の変更（代替材・薄肉化・軽量化）
  ③ 工法の変更（溶接→曲げ成形のみ、プレス→樹脂射出等）
  ④ 購入品への切り替え（内製→購入品/標準品活用）
  ⑤ 設計変更（形状簡略化・部品統合）

出力形式:
・「VEアイデア一覧（変更内容・期待コスト削減額・実現性の3列）」
・「優先度ランキング（削減額×実現性の2軸マトリクス）」
・「採用候補案の詳細検討（材料変更 or 工法変更のトップ2〜3案）」
・「VE実施に向けた確認事項（品質・強度・顧客承認の必要性）」
```

## コスト削減提案書を作るプロンプト

VEアイデアを社内・顧客に提案できる形にまとめます。

```
以下の情報をもとに、コスト削減提案書（VE提案書）を
作成してください。

【提案の概要】
提案タイトル: （例: カバー部品の材質変更による原価低減提案）
提案の背景:
  ・顧客からのコスト削減要求（年率2%）に対応するため

【提案内容】
現行仕様: SPCC t2.0mm + プレス成形 + 溶接 + 粉体塗装
提案仕様: ABS樹脂（射出成形品に変更）
  ・溶接・塗装工程の廃止
  ・部品点数の削減（溶接箇所4ヶ所を一体化）

【効果の試算】
現行コスト: ¥2,800/個（月500個 → 月間¥1,400,000）
提案後コスト: ¥1,950/個（試算）
削減額: ¥850/個（30.4%削減）
月間削減効果: ¥425,000

【懸念点と対応策】
・強度: 現行品と同等のIPX4を樹脂で達成できるか（試作評価必要）
・金型費用: 初期投資 ¥1,500,000が必要（回収期間: 約3.5ヶ月）
・顧客承認: 材料変更は顧客の変更承認が必要

出力形式:
・「コスト削減提案書の構成（背景・提案内容・効果試算・
  スケジュール・リスクと対策）」
・「費用対効果の分析（初期投資 ÷ 月次削減額 = 回収期間）」
・「提案書の説得力を高める表現（数値の見せ方・リスク説明）」
・「顧客への変更承認申請のポイント（3変更：機能/外観/材料）」
・「試作から量産までのスケジュール骨子」
```

## 原価低減活動の年次計画・進捗管理プロンプト

組織全体でのコスト削減活動を体系的に管理します。

```
以下の情報をもとに、製造業の原価低減活動の
年次計画書と進捗管理フォーマットを整理してください。

【今年の原価低減目標】
売上高に対する原価低減目標: 売上高×1.5%削減
（具体額: 年間売上¥200M × 1.5% = ¥3,000,000/年）

【原価低減のカテゴリ別目標額（例）】
材料費削減（仕入先見直し・VE）: ¥1,500,000
加工工程効率化（工数削減・自動化）: ¥800,000
外注費見直し（内製化・仕入先交渉）: ¥400,000
廃棄・手直しロス削減（不良率低下）: ¥300,000

【現在進行中のテーマ】
テーマ①: カバー部品の樹脂化（上述のVE案）
  担当: 技術部 ○○さん・購買部 ○○さん
  目標削減額: ¥5,100,000/年
  進捗: 試作品評価中（完了予定: 3月）

テーマ②: 旋盤加工の段取り時間短縮
  担当: 製造部 ○○さん
  目標削減額: ¥480,000/年
  進捗: 改善案策定中

出力形式:
・「原価低減活動年次計画書のフォーマット（目標額・テーマ一覧・
  担当・スケジュール）」
・「月次進捗管理フォーマット（各テーマの達成率・見込み・
  課題・次月アクション）」
・「原価低減効果の集計・見える化（月次・累計・目標達成率の
  グラフ化イメージ）」
・「経営報告用サマリー（月次報告への組み込み方）」
・「原価低減テーマを増やすためのアイデア出し会議の進め方」
```
