製造業の製品開発・技術戦略は企業の将来を左右します。AIを使って製品開発ロードマップ・技術戦略書・競合分析レポートを効率的に整備できます。

---

## 製品開発ロードマップを整理するプロンプト

3〜5年の製品開発計画を可視化します。

```
以下の情報をもとに、製品開発ロードマップを
整理してください。

【当社の製品開発の状況】
業種: 精密板金加工（受託製造）→ 自社製品への展開を検討中
現状の強み: 精密板金加工技術（公差±0.05mm）/ 少量多品種対応 / 溶接技術
市場の動き: 脱炭素・省エネ関連設備の需要増 / 農業IoT / 医療機器

検討中の自社製品方向性（3候補）:
  A案: 農業用スマート灌漑ユニット（アルミ板金 + 制御基板）
    → 農業IoT需要 / 初期投資 ¥3,000,000 / 開発期間 18ヶ月
  B案: 工場向け省エネ配電盤カバー（板金 + 断熱材）
    → 既存顧客への横展開 / 初期投資 ¥1,000,000 / 開発期間 6ヶ月
  C案: 医療機器外装部品（SUS316L精密板金）
    → 高付加価値 / 規制対応（QMS省令）/ 開発期間 24ヶ月

出力形式:
・「3候補の比較評価（市場規模 / 開発難易度 / リスク / 収益性 の4軸）」
・「推奨案の選定と理由（ROI・リスクのバランスで最適案を提示）」
・「選定案の製品開発ロードマップ（月次ガントチャート形式）」
  フェーズ1（市場調査・コンセプト）→ フェーズ2（試作）→ フェーズ3（量産）
・「開発に必要な外部リソース（技術連携先 / 補助金 / 人材）の整理」
・「CEO・役員向けの製品開発投資提案書骨子（1ページ）」
```
