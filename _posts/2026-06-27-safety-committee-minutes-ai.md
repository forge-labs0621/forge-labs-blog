製造業では、従業員50人以上の事業所で安全衛生委員会（または安全委員会・衛生委員会）の設置と月1回以上の開催が労働安全衛生法で義務付けられています。毎月の議事録作成が担当者の負担になりがちですが、AIを使うと会議のメモから整形された議事録を素早く作れます。

## 安全衛生委員会議事録のAI活用ポイント

安全衛生委員会の議事録に記載が必要な主な内容:

- 開催日時・場所・出席者
- 付議事項（調査審議した内容）
- 審議結果（決定事項・継続審議事項）
- 委員からの意見・提案
- 産業医の意見（衛生委員会がある場合）

これらを毎回ゼロから整形するのは手間ですが、会議中に取ったメモをAIに渡すと、議事録の形式に整えてくれます。

## 安全衛生委員会の議事録を作るプロンプト

---

```
以下のメモ・情報をもとに、安全衛生委員会の議事録を作成してください。

【開催情報】
開催日時: （例：2026年7月3日（木）16:00〜17:00）
開催場所: （例：第2会議室）
出席者: （例：委員長 ○○工場長、副委員長 ××製造課長、委員 △△安全衛生担当、
          産業医 □□先生、その他委員○名）
欠席者: （なし、または欠席者名）

【前回からの継続事項・進捗】
（例：
・フォークリフト通路の区画ライン引き直し → 完了（6/25実施）
・新規入場者教育の記録様式の見直し → 継続中、7月中に改訂予定
）

【今回の付議事項・審議内容のメモ】
（例：
1. 6月の労働災害・ヒヤリハット発生状況の報告
   → 労働災害ゼロ。ヒヤリハット3件（転倒2・はさまれ1）。全件再発防止策実施済み。
2. 熱中症対策の進捗確認
   → 休憩所の冷水機を増設（2台→4台）、WBGT計測を毎日記録開始。
   → 産業医より「午後の水分補給を15時にも1回入れるとよい」と意見あり。
3. 7月の安全活動計画の審議
   → 7月12日に安全パトロールを実施する（委員全員参加）。
   → 熱中症予防の朝礼教育を週1回実施することを決定。
）

【次回開催予定】
（例：2026年8月7日（木）16:00〜）

出力形式:
・「安全衛生委員会 議事録」としてフォーマット
・開催情報・出席者一覧
・前回確認事項の進捗
・付議事項ごとに「審議内容」「審議結果・決定事項」の2段で整理
・産業医意見があれば明記
・次回予定
・作成者・確認者欄（プレースホルダー）
```

## 安全パトロール記録をAIで整理するプロンプト

委員会でよく報告される安全パトロールの結果もAIで文書化できます。

```
以下の安全パトロールの記録を整理してください。

【パトロール実施日・場所】
（例：2026年7月12日、第1製造棟・第2製造棟・倉庫）

【パトロール中に気づいた問題（メモ・箇条書き）】
（例：
・第1製造棟 旋盤エリア: 通路に材料が置かれていて歩行スペースが狭い
・第2製造棟 出口付近: 消火器が棚の裏に隠れていて見つけにくい
・倉庫: 段ボール箱が高く積まれすぎている（目測3m超）
・全体的に問題なかった箇所: 溶接エリア（整理整頓良好）、休憩室
）

出力形式:
・「安全パトロール実施報告書」として
・日時・場所・参加者
・指摘事項を「場所」「指摘内容」「改善要求事項」「期限・担当者」の4列で一覧化
・良好だった箇所のコメント
・総評（1〜2文）
```

## 月次安全衛生報告書（経営層向け）を作るプロンプト

委員会の審議内容を経営層に報告するサマリーも作れます。

```
以下の安全衛生活動の実績をもとに、経営者向けの月次安全衛生報告書を作成してください。

【対象期間】（例：2026年6月）

【労働災害・ヒヤリハット実績】
（例：労働災害 0件、ヒヤリハット 3件（前月比-1件））

【今月実施した主な安全衛生活動】
（例：
・KY活動（毎朝実施）
・安全パトロール 1回実施、指摘6件中4件改善済み
・新入社員1名の安全教育実施
）

【課題・来月の対策】
（例：
・熱中症リスクが上昇するため、水分補給の周知を強化
・残り未完了の指摘2件を7月中に対応
）

出力形式:
・A4一枚に収まるサマリー形式
・数値（件数・比較）は目立たせる
・「今月のトピック」「数値実績」「課題と対策」の3構成
```

## AIを使う際の注意

- 議事録は**委員会の正式な記録**です。AIが整形した後、必ず出席した委員が内容を確認・修正してください
- 個人名（委員名・産業医名）の取り扱いは社内のプライバシーポリシーに従ってください
- 労働安全衛生法では議事録の**3年間保存**が義務付けられています
