製造業では生産会議・品質会議・月次定例会など多数の会議があります。AIを使って議事録・アジェンダ・アクションアイテム管理を効率化し、会議の質と速度を同時に改善できます。

## 生産会議のアジェンダと議事録を整理するプロンプト

週次生産会議の準備から記録まで効率化します。

```
以下の情報をもとに、製造部の週次生産会議の
アジェンダと議事録を整理してください。

【週次生産会議の概要】
開催: 毎週月曜 08:30〜09:00（30分）
出席者: 工場長・製造1課長・製造2課長・品質管理係長・生産管理担当
目的: 先週の生産実績確認 / 今週の生産計画共有 / 問題点の早期発見

今週（6月28日）の確認事項:
  先週の実績:
    - 生産数: 計画850個 → 実績810個（未達40個）
    - 未達原因: 設備トラブル（NC旋盤 6/24 2時間停止）
    - 不良率: 1.2%（目標1.0%以下 → 超過）
    - 残業: 製造1課 合計12時間（先週比 +4時間）

  今週の計画:
    - 計画生産数: 890個（先週の未達40個を上乗せ）
    - 懸念事項: 新製品（品番 C-2024）の初物加工が今週開始
    - 必要な準備: 初物確認書の準備・品質部立会い依頼

  持ち越しアクションアイテム（先週から）:
    - NC旋盤の予防保全スケジュールの見直し（設備担当・6/30期限）
    - C-2024の加工手順書の確認（品質部・6/28期限 → 本日確認）

出力形式:
・「今週の生産会議アジェンダ（時間配分付き 30分版）」
  （5分: 先週実績 / 10分: 今週計画 / 10分: 問題点協議 / 5分: アクション確認）
・「会議中に確認すべき質問リスト（議長が使う確認事項）」
・「議事録のフォーマット（記録者が埋めやすい形式）」
  （決定事項 / アクションアイテム / 次回確認事項 の3ブロック構成）
・「アクションアイテム管理表（担当・期限・進捗をトラッキングする表）」
・「週次生産会議を30分以内に終わらせるためのファシリテーションのコツ5つ」
```

## 月次品質会議の議事録・報告書を整理するプロンプト

品質トレンドを可視化した会議資料を作ります。

```
以下の情報をもとに、月次品質会議の
報告書と議事録を整理してください。

【月次品質会議の概要と今月のデータ】
開催: 毎月最終週・火曜（2時間）
出席者: 工場長・品質部長・製造部長・各課長

今月の品質データ（2026年6月）:
  ①不良率:
    製造1課: 0.8%（目標1.0% → 達成）
    製造2課: 1.5%（目標1.0% → 未達）
  ②クレーム受領: 1件（顧客B社から 寸法不良 × 3個）
  ③不良の主な原因分類:
    工程不良（加工精度）: 65%
    外注部品不良: 20%
    作業者ミス: 15%
  ④是正措置の進捗:
    先月の不適合是正（5件）: 完了3件 / 進行中2件
    顧客クレームB社の8D報告: 提出済み（6/20）
    B社からの承認: 未受領（確認中）

会議での決議事項（事前に想定される内容）:
  ①製造2課の不良率改善への対応（原因深掘り・対策立案）
  ②B社クレームへのフォロー（承認取得のタイムライン）
  ③来月の品質目標（全課 0.8%以下に引き下げを検討）

出力形式:
・「月次品質会議 報告書の構成（A4・3〜4枚）」
  （今月の品質KPI / 不良トレンド / 主な不良事例 / 是正措置進捗 / 来月計画）
・「不良率のトレンドグラフ説明文
  （数値を言葉で説明するコメント文）」
  （「製造2課は3ヶ月連続して目標を超過しており...」等の文章化）
・「品質会議の議事録（決定事項・アクション・担当・期限を明記した形式）」
・「製造2課の不良率改善に向けた問題解決フロー
  （なぜなぜ分析の実施 → 対策立案 → 効果確認のサイクル）」
```

## 経営会議向け製造部報告書を整理するプロンプト

経営幹部に伝わる製造報告書を作ります。

```
以下の情報をもとに、経営会議（月1回）での
製造部の報告書を整理してください。

【経営会議への報告で求められる情報】
報告者: 製造部長（工場長）
聴衆: 社長・営業部長・管理部長（製造の専門家ではない）
報告時間: 10〜15分

今月の製造部の実績と課題:
  生産面:
    ・生産数: 計画3,400個 → 実績3,280個（達成率 96.5%）
    ・主な未達原因: 設備トラブル（NC旋盤）と新製品の立ち上がり遅延
    ・来月の生産計画: 3,600個（新規受注追加）
  
  品質面:
    ・不良率: 平均1.1%（目標1.0%・わずかに超過）
    ・顧客クレーム: 1件（B社・是正中）
  
  コスト面:
    ・残業代: 今月¥450,000（先月¥380,000 / 予算¥400,000）
    ・設備修理費: ¥200,000（NC旋盤の部品交換）
  
  人材面:
    ・欠員: 1名（定年退職・来月採用予定）
    ・外国人スタッフの日本語レベルが課題

出力形式:
・「経営会議用の製造部月次報告書（A4 2枚・非専門家向け）」
  （数字だけでなく「意味」と「次のアクション」を必ず付ける構成）
・「経営幹部が聞きたい3つの質問への回答を準備するプロンプト」
  Q1「来月の生産は大丈夫か？」
  Q2「品質クレームは解決したか？」
  Q3「何かCEOに判断してほしいことはあるか？」
・「製造部の現状を1枚のサマリーにまとめるダッシュボード骨子
  （赤・黄・緑の信号機形式で即座に把握できる形）」
```
