製造業の設備投資・AI導入・工場増設等の意思決定では、投資対効果（ROI）の計算と経営層・金融機関向けの投資計画書が必要です。AIを使って計算の枠組みと報告書の文章を整理できます。

---

## 設備投資のROI・投資回収期間を計算するプロンプト

新設備導入の費用対効果を整理し経営判断書類にまとめます。

```
以下の情報をもとに、設備投資案件のROI計算と
投資評価レポートを作成してください。

【投資案件の概要】
設備名: （例: 5軸マシニングセンタ（精密加工用））
投資金額: ○百万円（設備費 + 搬入・据付費 + 初期設定費）
補助金・助成金: ものづくり補助金○百万円の採択見込み
実質負担額: ○百万円（ファイナンスリース、月額○万円 × 7年）

【投資によって得られる効果（数値）】
① コスト削減効果:
  ・現在: 外注加工費 ○万円/月 → 内製化後: 材料費+電力費 ○万円/月
  ・削減効果: ○万円/月 = ○万円/年

② 売上増加効果:
  ・内製化により月○工程分の追加受注が可能
  ・追加売上見込み: ○万円/月（粗利率○%で粗利○万円/月）

③ 品質改善効果:
  ・外注加工の不良返品コストが年間○万円 → 内製化で削減見込み

合計年間効果: ○百万円/年

出力形式:
・「投資回収期間（Payback Period）の計算」（単純回収法・割引回収法）
・「NPV（正味現在価値）の計算枠組み」（割引率の設定の考え方も含む）
・「ROI（投資対効果率）= 年間純利益 / 投資金額 の計算」
・「設備投資評価レポートの1枚サマリー（経営会議向け）」
・「投資リスクと感度分析（売上が予測比○%下振れした場合の回収期間）」
```

## AI・DX投資の費用対効果を整理するプロンプト

AIシステム・デジタル化投資の効果を経営層に示す書類を整理します。

```
以下の情報をもとに、AI/DX投資の費用対効果を
整理してください。

【AI/DX投資の内容】
投資対象: （例: 生産管理システム（MES）の導入 / クラウドERPへの移行）
初期費用: ○百万円（システム構築・ライセンス・初期設定）
ランニングコスト: ○万円/月（SaaS利用料・保守費）
導入期間: ○か月

【期待される効果（定量）】
① 事務工数の削減:
  ・日報入力・集計: 月○時間の削減（担当2名 × ○時間/月）
  ・在庫管理・発注作業: 月○時間の削減
  ・時間当たりコスト: ○円/時間 → 月○万円の削減

② 品質管理効率化:
  ・検査記録の自動集計により不良トレンドの発見が3日→即日に短縮
  ・クレーム対応コスト削減: 年間○万円

③ 意思決定の高速化:
  ・リアルタイムの生産状況把握により在庫削減（運転資金削減 ○百万円）

【定性的な効果】
・社員の業務満足度向上（単純作業からの解放）
・属人化の解消（担当者不在時のリスク低減）
・顧客への情報提供スピード向上（競争力強化）

出力形式:
・「AI/DX投資の費用対効果一覧（定量＋定性の両面）」
・「月次ROI推移の予測（導入月から投資回収月まで）」
・「役員・経営会議向けの1枚投資提案資料の構成」
・「金融機関（設備資金融資）向けの投資計画説明の骨子」
・「AI/DX投資の効果を正直に評価するためのモニタリング指標KPI」
```

## 製造業の財務分析・経営判断レポートを作るプロンプト

月次・年次の財務データを経営判断に使いやすい形に整理します。

```
以下の財務データをもとに、製造業経営者向けの
月次財務分析レポートを作成してください。

【月次財務データ（例）】
売上高: ○百万円（前月比+○%・前年同月比+○%）
製造原価: ○百万円（原価率○%）
  うち材料費: ○% / 労務費: ○% / 経費: ○%
売上総利益（粗利）: ○百万円（粗利率○%）
販管費: ○百万円
営業利益: ○百万円（営業利益率○%）

製造KPI:
・稼働率: ○%（前月比○ポイント）
・不良率: ○%（前月比○ポイント）
・在庫回転日数: ○日（前月比○日）

出力形式:
・「社長・役員向けの月次経営ダッシュボード（1枚）」
・「原価率悪化の場合の原因分析フレームワーク」
  （材料費変動 / 労務費 / 固定費比率 を分解する方法）
・「前月・前年比の増減要因コメントの書き方（例文）」
・「製造業の財務改善に使えるKPI指標一覧（10指標）と目標値の設定方法」
・「銀行・金融機関に月次報告する場合の説明資料フォーマット」
```
