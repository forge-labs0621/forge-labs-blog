50人以上の事業場では年1回のストレスチェックが義務化されています。AIを使って実施計画・結果の集団分析・職場環境改善計画・相談窓口の案内文を効率化できます。

---

## ストレスチェックの実施計画書を整理するプロンプト

年1回の法定ストレスチェックの準備を整えます。

```
以下の情報をもとに、ストレスチェックの
実施計画書と関係書類を整理してください。

【当社の基本情報】
従業員数: 65名（正社員45名・パートタイム15名・派遣5名）
→ 50人以上のため実施義務あり

前回実施: 2025年10月（昨年度）
今回実施予定: 2026年10月

【実施体制】
実施者: 産業医（田村医師・月2回来訪）
実施事務従事者: 衛生管理者・人事部・佐藤（個人情報保護の誓約書提出済み）
使用する調査票: 職業性ストレス簡易調査票（57項目・国が推奨）

【前回の課題】
  ・受検率が78%（目標90%以上）
  ・高ストレス者のうち、医師への面接指導の申し出があったのは0名
  ・「ストレスチェックをやっているが何も変わらない」という声

出力形式:
・「ストレスチェック実施計画書の骨子
  （実施日程・実施者・実施事務従事者・調査票・
  結果の通知・集団分析・改善への流れ）」
・「全従業員への実施案内文（目的・方法・
  プライバシー保護の説明・受検への呼びかけ）」
・「受検率向上のための工夫（前回78%→90%以上へ）」
・「高ストレス者への面接指導の案内文（申し出しやすい言い方）」
・「衛生管理者・産業医・人事部の役割分担と情報管理ルール」
```

## ストレスチェック結果の集団分析と職場改善プロンプト

結果を組織改善に活かします。

```
以下の情報をもとに、ストレスチェックの
集団分析結果と職場環境改善計画を整理してください。

【今回のストレスチェック集団分析の結果】
（部署別・10名以上の集団のみ分析・個人特定なし）

製造部（45名）:
  ・「仕事の量的負担」: 高得点者が43%（全社平均28%・要注意）
  ・「上司のサポート」: 低得点者が38%（全社平均22%・要注意）
  ・「仕事のコントロール」: 良好

品質部（12名）:
  ・全体的に中程度、特記事項なし

管理部（8名）:
  ・10名未満のため集団分析の対象外

高ストレス者の割合: 全体の12%（7.8名・目標10%以下）

出力形式:
・「集団分析結果の見方と製造部の課題の整理
  （量的負担の多さ・上司サポート不足の因果関係の考察）」
・「製造部の職場環境改善計画案（半年以内に実施する具体策）」
  例: 業務量の見直し・班長向け部下との面談スキル研修等
・「改善策の実施体制（誰が・何を・いつまでに）」
・「安全衛生委員会への報告資料の骨子」
・「6ヶ月後の効果確認方法（ストレスチェック以外のモニタリング手段）」
```

## 職場のメンタルヘルス相談窓口の整備プロンプト

悩みを持った従業員が相談しやすい環境を作ります。

```
以下の情報をもとに、職場のメンタルヘルス
相談窓口の設置と運用方法を整理してください。

【当社の現状】
  ・相談窓口: 現在は「人事部に相談してください」とだけ伝えている
  ・問題: 人事=会社側というイメージから、従業員が相談しにくい
  ・産業医: 月2回来訪（第1・第3金曜日・各2時間）

【整備したいこと】
  ・相談しやすい窓口の設置（社内・社外の両方）
  ・管理職が部下の変化に気づくための研修
  ・メンタル不調者の復職支援の手順書

利用可能なリソース:
  ・EAP（従業員支援プログラム）: 未導入（月額¥5,000〜/人が目安）
  ・産業医: 対応可能
  ・労働者健康サービスセンター（無料の地域窓口）: 近隣に有り

出力形式:
・「社内相談体制の設計（産業医相談・衛生管理者面談・
  第三者（EAP）利用の3層構造）」
・「従業員向けメンタルヘルス相談案内ポスター・チラシの文案
  （どんなときに相談できるか・秘密は守られるか・
  相談した結果どうなるか）」
・「管理職向けラインケア研修のアジェンダ（2時間版）
  （部下の変化のサイン・声のかけ方・専門家への繋ぎ方）」
・「休職・復職支援の手順書の骨子
  （休職申請→主治医診断書→休職→復職面談→
  試し出勤→本格復帰の各ステップ）」
・「EAP導入の費用対効果の考え方（ROIの試算方法）」
```
