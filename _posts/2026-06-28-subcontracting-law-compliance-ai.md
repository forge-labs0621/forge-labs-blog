下請代金支払遅延等防止法（下請法）は、親事業者（発注する側）と下請事業者（受注する側）の取引を公正にするための法律です。中小製造業の多くは下請事業者の立場で取引をしており、不当な代金引き下げや支払遅延に対する知識が重要です。また自社が外注先に発注する場合は親事業者として法律を遵守する必要があります。

> **注意**: 下請法の解釈・具体的な法的判断は公正取引委員会・弁護士に確認してください。このページはAI活用の例示であり、法的アドバイスではありません。

## 発注書（書面）の必要記載事項を確認するプロンプト

外注先への発注書に必要な記載事項（下請法対応）を確認します。

```
以下の情報をもとに、下請法に適合した発注書に
必要な記載事項を確認し、現在の発注書の不備を指摘してください。

【自社の立場】
（例: 自社が親事業者（発注する側）として、外注先（下請事業者）に
 機械加工を依頼している。年間取引額 ¥5,000万円程度）

【現在の発注書の記載内容】
（例:
・品番・品名: 記載あり
・数量: 記載あり
・単価・金額: 記載あり
・納期: 記載あり
・発注日: 記載あり
・支払期日: 記載なし（口頭で「月末締め翌月末払い」としている）
・検査完了期日: 記載なし
・返品条件: 記載なし
）

確認してほしいこと:
・下請法第3条が定める「書面の必要記載事項（11項目）」のうち何が不足しているか
・不足している項目を追加する際の記載例
・「口頭での取り決め→書面化」が必要な事項はどれか

出力形式:
・「下請法3条書面 必要記載事項チェックリスト（11項目）」
・各項目の「有 / 無 / 要確認」と「記載がない場合のリスク」
・発注書テンプレートの改善案（追加すべき項目の記載例）
・※最終確認は公正取引委員会のガイドラインまたは弁護士に
```

## 不当な代金引き下げ交渉への対処法を整理するプロンプト

取引先から一方的なコスト削減要請があった場合の対応を整理します。

```
以下の状況について、下請法の観点から対応方針を整理してください。

【状況】
（例: 取引先（親事業者）から「来月から単価を10%引き下げてほしい。
 理由の説明なし。断るなら取引を打ち切ると言われた」という
 電話・メールを受けた。既に合意した単価で2年間取引してきた。）

確認してほしいこと:
・この行為が下請法の「買いたたき（第4条1項5号）」に該当する可能性があるか
・下請法違反の場合、公正取引委員会への相談・申告の方法
・取引先への回答文の方向性（断り方・交渉の進め方）
・記録として残すべき書類・メール・メモの種類

出力形式:
・状況の法的整理（可能性のある下請法違反の種類と根拠）
・「こういう場合は相談できる窓口」のリスト（公取委 / 中小企業庁 / 弁護士等）
・取引先への返答メール案（「すぐに回答できない・書面での提示をお願いしたい」の旨）
・証拠保全のためのアクション（会話録音・メール保存・日付入りメモ等）
・※法的判断は弁護士・公正取引委員会に確認することを明記
```

## 支払遅延・下請代金の受取確認メール文を作るプロンプト

下請代金の支払いが遅延している場合の確認メールを作ります。

```
以下の状況で、取引先への支払確認メールを作成してください。

【状況】
（例: 2026年5月31日が支払期日だった請求書（金額 ¥850,000）の
 入金が2026年6月28日現在も確認できていない。
 取引先への問い合わせは今回が2回目。）

【自社の立場と関係】
（例: 自社は下請事業者。取引先は年間 ¥3,000万円以上の
 大口取引先で、関係を壊したくない。ただし資金繰りに影響が出ている。）

出力形式:
・ビジネスメール（丁寧だが事実を明確に伝えるトーン）
・「○月○日が支払期日の請求書（請求書番号○○）の入金が
  確認できていない」という事実の明示
・「ご確認いただけますでしょうか」の依頼
・「入金予定日をお知らせいただけると助かります」の文言
・「もし何か手続きが必要であればご連絡ください」の配慮
```

## 外注先への発注条件変更通知書を作るプロンプト

自社が親事業者として外注先に発注条件を変更する際の通知書を作ります。

```
以下の情報をもとに、外注先への発注条件変更通知書を作成してください。

【変更内容】
（例:
・支払サイト変更: 現在 月末締め翌々月末払い（60日）
  → 変更後 月末締め翌月末払い（30日）
  ※下請法では60日以内の支払が必要。以前の条件は下限ギリギリだったため
  取引先への配慮で改善する。
・適用時期: 2026年10月1日以降の発注分から
）

出力形式:
・取引先への正式な変更通知書
・変更の理由・適用時期・対象取引の明確な記載
・「ご不明点はご連絡ください」の担当者連絡先欄
・ポジティブな変更（支払サイト短縮）の場合は「取引先への配慮・感謝」を添える
```
