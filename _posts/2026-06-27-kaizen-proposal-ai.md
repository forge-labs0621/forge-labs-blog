「改善提案はいいアイデアがあるけど、書き方がわからない」「書いている時間がない」という現場担当者は多いです。AIを使うと、口頭や箇条書きのメモから、改善提案書の形式に整えた文書を素早く作れます。

## 改善提案書でAIが役立つ理由

改善提案書は「何が問題で」「どう改善して」「どれくらい効果があったか」を記録・共有するための文書です。アイデアを思いついた現場担当者が書くことが多いため、「文章が苦手で提出をためらう」という声が出やすい書類の一つです。

AIを使うと:
- アイデアのメモを整えた文書に変換できる
- 「問題の現状」「改善案」「期待効果」の構成に整理できる
- 改善前・改善後の比較表を自動で作れる

## 改善提案書を作るプロンプト

---

```
以下のメモ・状況をもとに、製造現場向けの改善提案書を作成してください。

【改善対象の工程・作業・設備】
（例：組立工程Aの部品ピッキング作業）

【現在の問題・困っていること】
（例：
・部品棚が乱雑で、必要な部品を探すのに毎回2〜3分かかる
・ピッキングミスが月に3〜4件発生している
・棚の配置がバラバラで新人が特に迷う
）

【改善のアイデア（思いついたこと）】
（例：
・部品棚を工程順に並べ替える
・棚にラベル（品番・写真）を貼る
・使用頻度の高い部品を手前・目線の高さに配置する
）

【改善の結果として期待すること】
（例：ピッキング時間が半減、ミスゼロ、新人でも1週間で習熟）

出力形式:
・「改善提案書」として整形
・改善テーマ（タイトル）
・現状の問題（事実・数値で記載）
・改善案の内容（実施方法・手順）
・改善前・改善後の比較（表形式）
・期待効果（定量・定性の両面）
・実施時期・担当者（プレースホルダー）
・費用の有無（費用が発生する場合は概算）
```

## 改善提案の「before/after」比較表を作るプロンプト

改善提案書の中でも「改善前後の比較」は、経営者・上長に効果を伝えるうえで特に重要です。

```
以下の改善について、改善前・改善後を比較する表を作成してください。

【改善の概要】
（例：部品ピッキング作業の棚レイアウト変更）

【改善前の状況（数値があれば数値で）】
・ピッキング時間: 1回平均3分
・ミス発生件数: 月3〜4件
・新人習熟期間: 約2週間
・棚の整理状態: 乱雑（担当者によってバラバラ）

【改善後の状況（数値があれば数値で、なければ見込み値）】
・ピッキング時間: 1回平均1分半（目標）
・ミス発生件数: 月0〜1件（目標）
・新人習熟期間: 約1週間（目標）
・棚の整理状態: ラベルで統一（誰でもわかる）

出力形式:
・項目/改善前/改善後/改善幅の4列の表
・表の下に改善ポイントの簡単なコメント
```

## 改善効果を報告する「効果確認書」を作るプロンプト

改善を実施してから一定期間後に「どのくらい効果があったか」を報告する際に使います。

```
以下の情報をもとに、改善活動の効果確認報告書を作成してください。

【改善の内容（実施したこと）】
（例：部品棚のレイアウト変更・ラベル整備）

【実施時期】
（例：2026年7月、3日間で完了）

【改善前の実績】
・ピッキング時間: 平均3分/回
・ミス件数: 月4件

【改善後の実績（計測期間を記載）】
・計測期間: 2026年8月〜9月（2ヶ月間）
・ピッキング時間: 平均1分40秒/回
・ミス件数: 月0〜1件

【費用がかかった場合】
（例：ラベルシール・棚板代 約2,000円）

【現場担当者のコメント】
（例：新人から「探しやすくなった」という声があった）

出力形式:
・「改善効果確認報告書」として
・改善テーマ・実施期間
・改善前後の数値比較（表形式）
・効果の評価（目標達成度）
・追加課題・残課題があれば
・報告者・確認日
```

## 改善テーマを発掘するヒント

「改善提案のネタが思い浮かばない」という場合、AIに現場の状況を説明すると改善テーマを出してもらえます。

```
製造現場での改善テーマを見つける手助けをしてください。

【現場の状況（困っていること・気になっていること）】
（例：
・残業が続いている（どの工程で詰まっているかわからない）
・材料の在庫がいつも多すぎるか少なすぎる
・新人がすぐ辞める
・不良品が繰り返し同じ工程から出る
）

改善テーマとして取り組みやすいものを5〜10個提案してください。
各テーマに「改善の方向性（何をすれば解決しそうか）」と
「効果の大きさ（大・中・小）」「取り組みやすさ（高・中・低）」を添えてください。
```

---

## 改善提案書を書く際の注意

- AIが生成した内容は**たたき台**です。現場の実情と合っているか必ず確認してください
- 費用が発生する改善案は、社内の承認ルールに従って申請してください
- 改善効果の数値（目標値）は、現実的な範囲で設定することが重要です
