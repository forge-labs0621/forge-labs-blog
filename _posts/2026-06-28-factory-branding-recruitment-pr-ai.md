中小製造業は技術力があっても「見せる力」が弱く、採用難・受注獲得の機会損失が生じます。AIを使って会社案内・採用PR・プレスリリースを効率的に制作することで、ブランド力を高められます。

---

## 会社案内・企業PRパンフレットの骨子プロンプト

自社の強みを伝える会社案内を整理します。

```
以下の情報をもとに、BtoB向け会社案内
パンフレットの骨子と文案を整理してください。

【当社の概要】
会社名: ○○製作所
業種: 精密板金加工（受託製造）
設立: 1978年（創業48年）
従業員: 42名
所在地: ○○県○○市
主要設備: レーザー切断機（最大加工サイズ3m×1.5m）・
         プレスブレーキ×5台・溶接設備・表面処理ライン
主要顧客: 産業機械メーカー・医療機器メーカー・半導体製造装置メーカー
年商: 約¥480M

【当社の強み（自社の認識）】
  ・精度: 公差±0.05mmの精密加工が可能
  ・品質: ISO 9001取得・不良率0.08%
  ・対応力: 試作1個から量産まで一貫対応・最短3日納期対応
  ・技術: 難材質（SUS316L・チタン・インコネル）の加工実績

【ターゲット読者】
  ・新規取引先の購買・調達担当者
  ・展示会での名刺交換後のフォローアップ

出力形式:
・「会社案内のページ構成案（全8ページの構成）」
・「各ページのキャッチコピー・見出し・本文の文案」
・「強みの伝え方（数値・実績・事例を使った説得力のある文章）」
・「会社案内に載せるべき写真・図の指示一覧」
・「デジタル版（PDF・Webサイト掲載）への展開の注意点」
```

## 採用ブランディング・求職者向けPR資料プロンプト

製造業の採用力を高めるコンテンツを整理します。

```
以下の情報をもとに、若手求職者向けの
採用ブランディングコンテンツを整理してください。

【採用の現状と課題】
  ・毎年3〜4名の採用を目標、昨年は2名採用（1名欠員）
  ・20代の応募が少ない（30代以上が中心）
  ・「製造業＝きつい・古い・将来性がない」のイメージが障壁

【求職者に伝えたいこと（採用側の想い）】
  ・最新設備（昨年導入：5軸CNCマシニングセンタ）があること
  ・週休2日・残業月平均20時間以下（実績）
  ・資格取得支援（技能検定受験料・CAD/CAM研修費用を会社負担）
  ・20代で年収360万〜400万円スタート・3年で昇給実績あり
  ・町工場だが顧客は大手上場メーカー（製品が社会を動かしている）

ターゲット: 高校・工業高校・専門学校・工業系大学の卒業予定者
媒体: 求人サイト（マイナビ・ハローワーク）・自社Webサイト・学校訪問資料

出力形式:
・「採用コンセプト・キャッチフレーズ案（3案）」
・「求人票の本文（仕事内容・求める人物像・
  待遇・職場の雰囲気の文案）」
・「自社Webサイトの採用ページの構成と文案」
・「学校訪問時のプレゼン資料の骨子（先生・学生への説明）」
・「応募者へのメール返信テンプレート（選考案内・合否通知）」
```

## プレスリリース・技術広報の作成プロンプト

新製品・技術・受賞などのニュースを発信します。

```
以下の情報をもとに、プレスリリースの
文案を整理してください。

【発信するニュースの内容】
種別: 新技術・新設備の導入
概要: 国内最大級の5軸CNCマシニングセンタ（ヤマザキマザック製）を
     今月導入。複合加工により従来3工程必要だった加工を1工程で完了。
     加工精度±0.01mm・加工時間40%短縮を実現。

背景: 医療機器・半導体製造装置向けの精密部品需要増加に対応するため。
投資額: 約¥180M（CEO承認済み）
初受注: 医療機器メーカーX社から試作品を受注、量産検討中。

【プレスリリースの配信先】
  ・業界紙（日刊工業新聞・機械情報）への投稿
  ・自社Webサイトのニュースページ
  ・展示会でのプレス資料

出力形式:
・「プレスリリース本文（見出し・リード・本文・
  会社概要・問い合わせ先の標準フォーマット）」
・「業界紙への投稿時の注意点（記者が使いたくなる切り口）」
・「写真撮影の指示（プレスリリースに添付する写真の撮り方）」
・「SNS（X/LinkedIn）への展開版（200文字以内の短縮版）」
・「顧客・取引先向けのお知らせメール（法人向け）の文案」
```
