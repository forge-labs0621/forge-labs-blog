製造業では、安全衛生管理の一環として「月次安全パトロール（安全巡視）」を実施し、指摘事項と是正措置を記録することが求められます。パトロール後の「文書化・是正報告書作成・経営層への報告」をAIで効率化できます。

---

## 安全パトロール報告書を整理するプロンプト

月次の安全パトロールで見つけた指摘事項をまとめます。

```
以下のパトロールメモをもとに、安全パトロール報告書を作成してください。

【基本情報】
実施日: （例：2026年7月10日）
パトロール実施者: （例：安全衛生管理者 ○○・部長 ○○）
対象エリア: （例：第一工場（プレス・溶接・仕上げ工程）、倉庫、廊下・階段）

【パトロールで発見した指摘事項（メモ）】
・プレス工程Aライン: 通路に材料箱が2個はみ出していた。黄色い通路ライン内に戻す必要あり
・溶接工程: 消火器1本の点検シールが2026年6月で期限切れ。更新が必要
・倉庫: 高積み（3m超）のパレットが1か所。転倒防止措置なし → 積み直しが必要
・仕上げ工程: 整理整頓は良好。特記なし
・廊下: 照明1個球切れ。薄暗い → 交換が必要

出力形式:
・安全パトロール報告書のフォーマット
・指摘事項を重要度（高・中・低）で分類
・各指摘に「是正担当者（部門）」「是正期限（目安）」の欄
・「総合所見」（全体的な安全状況の評価・前月との比較）を100字程度
・承認者欄・作成日
```

## 是正措置報告書を作るプロンプト

パトロールで指摘された事項への是正措置を報告する文書を作ります。

```
以下の情報をもとに、安全パトロール指摘事項の是正報告書を作成してください。

【元の指摘事項】
指摘番号: （例：2026-07-001）
指摘日: （例：2026年7月10日）
指摘内容: （例：倉庫内で高積みパレット（3m超）の転倒防止措置がなかった）
重要度: （例：高）

【是正措置の内容】
実施日: （例：2026年7月12日）
実施者: （例：倉庫担当 ○○）
実施した対応:
  - パレットの積み直しを実施（最大2段2.0m以下に統一）
  - 倉庫入口に「最大積み高さ2.0mまで」の表示を新設
  - 全倉庫作業者（3名）に口頭で周知

【再発防止策】
（例：今後は月次パトロール前に倉庫担当が自主点検を実施することとした。
 月初め第1月曜日を定例点検日とした。）

出力形式:
・是正措置報告書のフォーマット（指摘内容 / 是正内容 / 再発防止策）
・「写真添付欄」（是正前・是正後）のプレースホルダー
・是正完了確認者サイン欄
・次回パトロールでの確認予定日
```

## 経営層向け安全活動報告（月次）を作るプロンプト

社長・部長向けの安全活動月次サマリーを作ります。

```
以下のデータをもとに、経営層向けの安全活動月次サマリーを作成してください。

【対象月】（例：2026年7月）

【安全活動の実績】
- 安全パトロール: 実施1回（7月10日）/ 指摘事項5件（是正完了3件・対応中2件）
- ヒヤリハット報告: 4件（前月比+1件）
- 安全衛生委員会: 実施（7月15日）/ 出席率 8/10人
- 安全教育: 新入社員1名に入社時安全教育実施
- 労働災害: 0件（今年度累計0件）

【先月からの主な改善事項】
（例：倉庫の高積み是正完了。消火器の点検シール更新完了。）

【翌月の安全活動予定】
（例：消火訓練（8月20日）/ 安全パトロール（8月12日予定））

出力形式:
・A4 1枚以内の経営層向けサマリー
・「問題のない点」と「継続対応事項」を区別して記載
・翌月の重点活動を最後に明記
・災害ゼロ継続日数（あれば）を目立つ場所に記載
```

## KYT（危険予知トレーニング）シートを作るプロンプト

特定の作業に対するKYT（危険予知訓練）シートの文章を作ります。

```
以下の作業について、KYT（危険予知トレーニング）シートを作成してください。

【作業の概要】
（例：プレス機のダイス（型）交換作業）

【作業の手順（概略）】
1. 電源OFFの確認
2. スライドを下死点に降ろしてロック
3. ボルトを緩めてダイスを取り外す
4. 新ダイスをセットしてボルト締め
5. 試し打ち確認

【想定される危険・ヒヤリポイント】
（わかる範囲で）
・ダイス落下によるケガ（ダイスは重い）
・スライドが下降してくる危険
・ボルトのかじり・ネジ山破損

出力形式:
・KYTシートの4ラウンド法で（危険把握→重点設定→対策立案→目標設定）
・各ラウンドで3〜4点の記載
・「確認サイン（リーダー・担当者）」欄
・チームで話し合うための「ディスカッション用の問いかけ文」を1〜2個追加
```
