大手製造業・自動車メーカーを中心に、Scope3（サプライチェーン全体のCO2排出量）の開示が広がり、中小製造業にも取引先からのCO2排出量報告要求が増えています。自社のCO2排出量の可視化・削減計画・取引先への報告書類をAIで整理できます。

---

## Scope1・2のCO2排出量を計算・整理するプロンプト

自社の直接排出（Scope1）と間接排出（Scope2）を集計します。

```
以下の情報をもとに、自社のScope1・Scope2 CO2排出量を
整理し、報告書の形式にまとめてください。

【算定対象年度】
（例：2025年4月〜2026年3月（2025年度））

【Scope1（直接排出）: 自社で燃料を燃やして出るCO2】
都市ガス使用量: （例：12,500 m³/年）
LPGガス使用量: （例：0 kg/年）
重油使用量: （例：2,000 L/年（フォークリフト燃料））
軽油使用量: （例：500 L/年（社用車・トラック））

【Scope2（間接排出）: 購入した電力によるCO2】
電力使用量: （例：850,000 kWh/年）
購入先の電力会社: （例：中部電力ミライズ）
使用する排出係数: （例：0.453 kg-CO2/kWh（最新の電力会社公表値 / または
  資源エネルギー庁の調整後排出係数））

確認してほしいこと:
・各エネルギー源のCO2換算量（kg-CO2）の計算
・Scope1合計・Scope2合計・総排出量（tCO2）の集計
・「原単位（製品1個あたり or 売上高1円あたり）」の計算方法

出力形式:
・Scope1・2のCO2排出量一覧表（エネルギー種別ごとの内訳付き）
・前年比較の欄（前年データがあれば）
・「この数値をGHGプロトコルに基づく算定として使う際の注意点」
```

## CO2削減計画書を作るプロンプト

自社のCO2削減目標と具体的な施策を整理した削減計画書を作ります。

```
以下の情報をもとに、製造業のCO2削減計画書を作成してください。

【現状の排出量】
（例：2025年度 Scope1+2排出量: 450 tCO2/年）

【削減目標】
（例: 2030年度までに Scope1+2排出量を50%削減（225 tCO2以下）を目指す。
  ※政府目標（2030年46%削減）に合わせた設定）

【削減施策の候補】
（例：
① 太陽光発電設備の導入（屋根面積500㎡）
  → 年間発電量80,000 kWh相当の電力調達分を削減可能
  → 削減効果: 約36 tCO2/年 / 設置費用: 約¥1,500万円
  
② 照明のLED化（製造棟・倉庫）
  → 消費電力30%削減 / 年間9,000 kWh削減
  → 削減効果: 約4 tCO2/年 / 投資費用: 約¥200万円
  
③ 圧縮空気システムの見直し（エアー漏れ修繕・インバータ化）
  → 年間15,000 kWh削減
  → 削減効果: 約7 tCO2/年 / 投資費用: 約¥100万円）

出力形式:
・削減計画書のフォーマット（施策 / 削減効果（tCO2） / 投資費用 / 実施時期）
・「2030年目標への施策の積み上げ」の表（合計削減量と目標のギャップ確認）
・省エネ補助金・GX投資促進税制の活用可能性のコメント
・「現時点での達成見通し（現状施策で足りるか / 追加施策が必要か）」の評価
```

## 取引先へのCO2排出量報告書を作るプロンプト

大手取引先からCO2排出量の報告を求められた場合の回答書類を作ります。

```
以下の情報をもとに、取引先（大手メーカー）への
CO2排出量報告書を作成してください。

【報告先・目的】
報告先: （例：○○自動車工業 サステナビリティ推進部）
報告の背景: （例：取引先のScope3排出量算定のために、
  当社の排出量データ提供を依頼された）
報告対象期間: （例：2025年4月〜2026年3月（2025年度））

【報告内容】
Scope1排出量: （例：85 tCO2/年）
Scope2排出量: （例：385 tCO2/年）
排出量合計: （例：470 tCO2/年）
前年比: （例：▲8%（削減施策の効果））
主な削減施策: （例：照明LED化・省エネ設備更新）
今後の目標: （例：2030年に2023年比50%削減を目指す）
第三者検証の有無: （例：なし（自己計算）/ あり（○○機関が検証）））

出力形式:
・取引先への正式な報告書フォーマット
・「算定の根拠（エネルギー使用量・排出係数の出所）」を脚注で明記
・「今後の脱炭素への取り組み方針」を一段落で記載
・「本データは自社算定によるもので第三者検証を受けていない旨」の免責注記
```
