多能工化は欠員・繁閑に対応できる強い現場をつくる鍵です。AIを使ってスキルマトリックス・多能工化計画・OJTロードマップ・人員配置計画を体系化できます。

## スキルマトリックスと多能工化計画書を整理するプロンプト

```
以下の情報をもとに、製造現場の
スキルマトリックスと多能工化推進計画書を整理してください。

【現場の状況】
工程数: 6工程（NC旋盤 / マシニング / 研削 / 組立A / 組立B / 検査）
作業者: 正社員9名・パート4名

現在の問題:
  ①NC旋盤は山田さん（ベテラン）しかできないため
    山田さんが休むと生産が止まる（一人依存）
  ②繁忙期に研削・組立が手薄になるが、他工程の作業者が入れない
  ③どの作業者がどの工程をどのレベルでできるかが可視化されていない

目標（1年後）:
  ・全工程を3名以上が担当できる状態にする
  ・特に NC旋盤は山田さん以外に2名育成する

出力形式:
・「スキルマトリックスのフォーマット（縦：作業者名 / 横：工程名）
  各セルに習熟度レベル（0:未習 / 1:研修中 / 2:監督付き / 3:独立可 / 4:指導可）
  を記入するテーブル形式」
・「スキルマトリックスの現状をレーダーチャートで表現する方法の説明」
・「多能工化推進計画書のフォーマット（対象工程 / 育成対象者 / 目標レベル /
  育成期間 / OJT担当 / 確認時期）」
・「多能工化の優先順位付け基準（一人依存リスク / 繁閑差 / 難易度）」
・「多能工化手当・技能評価への反映方法の考え方」
```
