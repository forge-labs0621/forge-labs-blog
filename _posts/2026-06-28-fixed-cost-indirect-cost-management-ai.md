製造業では変動費だけでなく固定費・間接費の管理が利益率に大きく影響します。AIを使って固定費分析・削減計画・間接費の見える化書類を効率的に整備できます。

---

## 固定費分析・削減計画を整理するプロンプト

固定費の現状を可視化して削減テーマを特定します。

```
以下の情報をもとに、固定費の分析と
削減計画書を整理してください。

【当社の固定費（年間）の実態】
売上高: ¥380,000,000/年
変動費: ¥240,000,000/年（材料費・外注費・変動加工費）
固定費合計: ¥115,000,000/年（内訳は以下）

固定費の内訳:
  ・人件費（正社員25名・間接部門含む）: ¥70,000,000
  ・設備減価償却費: ¥12,000,000
  ・工場賃料: ¥9,600,000（¥800,000/月）
  ・リース料（フォークリフト等）: ¥2,400,000
  ・保険料: ¥1,500,000
  ・電気・ガス・水道: ¥8,000,000
  ・その他（通信・消耗品等）: ¥11,500,000

営業利益: ¥25,000,000（利益率 約6.6%）
業界平均利益率: 8%（目標: 10%）

出力形式:
・「固定費の分解・ABC分析（コスト大順での並び替えと改善余地の評価）」
・「固定費削減テーマTOP5の選定と削減可能額の試算
  （電気使用量削減 / リース見直し / 間接人員の配置見直し 等）」
・「損益分岐点分析（現状のBEP売上高と目標利益率10%達成に必要な固定費削減額）」
・「削減余地の大きい固定費項目の詳細調査指示
  （データ収集方法・調査すべき項目）」
・「固定費削減プロジェクトの推進体制案（誰が何を担当するか）」
```

## 間接費（製造間接費）の見える化プロンプト

現場以外のコストを部門別に整理します。

```
以下の情報をもとに、製造間接費の
見える化と管理書類を整理してください。

【当社の間接費の実態と問題点】
問題認識: 「間接部門のコストが高い」と感じているが、
どの部門・どの費目が問題か把握できていない

間接部門の人員:
  ・総務・経理（2名）: 給与・社会保険・経理・庶務全般
  ・購買・資材（2名）: 仕入先対応・発注・在庫管理
  ・品質管理（3名）: 受入検査・出荷検査・顧客クレーム対応
  ・生産管理（2名）: 生産計画・スケジュール管理・工程管理
  ・営業（1名）: 受注管理・見積対応

間接費の主な項目（月次・概算）:
  事務用品・消耗品: ¥150,000/月
  電話・通信費: ¥80,000/月
  接待交際費: ¥120,000/月
  新聞・図書: ¥30,000/月
  外注委託費（経理・IT等）: ¥200,000/月
  教育研修費: ¥50,000/月

出力形式:
・「間接費の部門別・費目別の可視化表（現状整理）」
・「間接業務の「なくす・減らす・まとめる」視点での削減アイデア
  （各費目の削減余地と具体的な方法）」
・「外注委託費の内容確認と内製化余地の検討（何をどこに委託しているか）」
・「間接業務時間の調査方法（誰が・何に・どれだけ時間をかけているか）」
・「間接費予算管理の仕組み（月次で超過を検知する管理フォーマット）」
```

## 電気代削減計画を整理するプロンプト

製造業で最も削減余地が大きいユーティリティコストです。

```
以下の情報をもとに、工場の電気代削減計画を
整理してください。

【当社の電気使用状況】
工場面積: 2,000m²
年間電気使用量: 480,000kWh/年
年間電気代: ¥8,000,000/年（平均単価 ¥16.7/kWh）

主な電気消費設備（年間消費量の推計）:
  ・生産設備（NC旋盤・プレス・溶接機等）: 35%（168,000kWh）
  ・コンプレッサー: 25%（120,000kWh）
  ・空調（工場・事務所）: 20%（96,000kWh）
  ・照明: 10%（48,000kWh）
  ・その他（事務機器等）: 10%（48,000kWh）

過去に検討した削減策:
  ①LED照明への切り替え（試算で ¥300,000削減/年、工事費¥800,000）
  ②コンプレッサーの適正圧力設定（担当者が「効果があると聞いた」）
  ③昼休みの設備停止（現状は昼休みも待機電力発生中）

出力形式:
・「電気代削減施策の一覧表（施策 / 削減効果試算 / 初期投資 / 回収年数）」
  LED照明・コンプレッサー・デマンド管理・契約見直し・設備停止等を含む
・「コンプレッサー適正運転の改善ポイント
  （圧力設定 / エア漏れ対策 / 不使用時停止の手順）」
・「電力デマンド管理の概念と警報アラート設定方法
  （契約電力をオーバーしないための管理方法）」
・「省エネ法対象（年間1,500kL以上の燃料等使用事業者）の該当確認と
  対象外の場合の代替的な省エネ取り組み記録のあり方」
・「電力会社との契約見直し（低圧 vs 高圧 / 時間帯別料金プラン）のチェックポイント」
```
