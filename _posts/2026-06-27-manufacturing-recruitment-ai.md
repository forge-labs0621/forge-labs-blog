製造業の採用では「求人票の文章が他の会社と似たようなものになってしまう」「自社の魅力を言語化できていない」「面接で何を聞けばよいかわからない」という悩みが多いです。AIを使うと、求人票の文章・魅力の言語化・面接質問の設計を効率化できます。

## 製造業の採用でAIが役立つ場面

- **求人票・募集要項の作成**: 職種・業務内容・条件を整理した文章に変換
- **自社の魅力の言語化**: 「うちはこんないいところがある」を採用候補者に伝わる文章に
- **面接質問の設計**: 職種別・経験レベル別の面接質問リスト
- **採用基準の言語化**: 「こういう人が欲しい」を具体的な採点基準に

## 製造オペレーター・技能職の求人票を作るプロンプト

---

```
以下の情報をもとに、製造オペレーター（技能職）の求人票を作成してください。

【会社・工場の基本情報】
業種: （例：金属機械加工、食品製造、電子部品組立など）
工場の場所: （例：愛知県○○市）
従業員規模: （例：50名）

【募集する職種・業務内容】
（例：
・NC旋盤・マシニングセンタによる金属部品の機械加工
・加工プログラムは既存のものを使用（プログラム作成は別担当）
・製品の検査・梱包作業も含む
）

【募集条件】
雇用形態: （例：正社員、最初は3ヶ月の試用期間）
経験・資格: （例：未経験歓迎、機械加工経験者優遇）
給与: （例：月給22〜28万円（経験・能力による）、残業代別途）
勤務時間: （例：8:00〜17:00（実働8時間）、残業月平均20時間程度）
休日: （例：土日祝日、年間休日120日）
待遇: （例：社会保険完備、通勤手当あり、制服支給）

【自社のアピールポイント（思いついたことを箇条書きで）】
（例：
・残業は少なく定時で帰れる日が多い
・先輩が丁寧に教えるので未経験でも安心
・長く安定して働ける（創業40年、主要取引先は大手自動車メーカー）
・資格取得支援制度あり（技能検定の受験料を会社が負担）
）

出力形式:
・求人票（ハローワーク・求人サイト掲載用）として
・職種名・仕事内容・応募資格・雇用条件・アピールポイントの構成
・「こんな方に向いています」のセクションも含める
・文体は採用候補者が読みやすい、親しみやすいトーン
```

## 製造業の魅力を言語化するプロンプト

「うちの工場の良さが伝わらない」という場合に使えます。

```
以下の情報から、製造業（工場）として採用候補者に伝えるべき魅力を
言語化してください。

【工場・会社の状況（思いつくまま書いてください）】
（例：
・創業35年で廃業の心配がない
・主要顧客が航空部品メーカーで技術力が必要な加工をしている
・社員の平均在籍年数が12年で離職率が低い
・昼食は工場内の食堂で600円で食べられる
・子育て中の女性社員も多い（時短勤務制度あり）
・社長が現場出身で現場の意見を聞いてくれる
）

「製造業は地味」「3K（きつい・汚い・危険）のイメージがある」という
偏見を覆すような魅力の伝え方をしてください。

出力形式:
・「当社で働く魅力」として、5〜7項目
・各項目にキャッチーな見出し（例：「技術力でキャリアを積める」）と
  2〜3文の説明文
・求人票の「アピールポイント」欄にそのまま使える形で
```

## 面接質問リストを作るプロンプト

職種・経験レベルに合わせた面接質問を設計できます。

```
以下の条件で、製造業の採用面接で使う質問リストを作成してください。

【職種】
（例：製造オペレーター（未経験）/ NC機械加工（経験者）/ 生産管理職 など）

【採用で重視したいこと】
（例：
・長く続けてくれるか（定着性）
・安全への意識が高いか
・チームワークを大切にできるか
・新しい技術を学ぶ意欲があるか
）

【応募者の経験レベル】
□ 未経験者（製造業初めて）
□ 経験者（他社の製造業出身）
□ キャリア採用（管理職候補）

出力形式:
・「必ず聞く質問」5〜7問（採用基準に直結する質問）
・「状況によって聞く追加質問」3〜5問
・各質問に「この質問で何を確認したいか」の意図を1行で添える
・「聞いてはいけない質問」（プライバシー・差別につながる禁止事項）も
  最後に3〜5項目まとめる
```

## 採用基準（評価シート）を作るプロンプト

面接後の合否判断を客観的にするための採点基準も作れます。

```
以下の職種の採用面接で使う評価シート（採点基準）を作成してください。

【職種】（例：製造オペレーター・新卒・未経験）

【重視する評価項目】
（例：コミュニケーション能力、安全意識、体力・継続力、会社への関心度、
       素直さ・成長意欲）

出力形式:
・評価項目ごとに「5段階評価の基準（5=非常に良い〜1=懸念あり）」
・各段階の判断基準を具体的な言動・発言例で記載
・総合評価欄（A採用推奨/B条件付き/C見送り）
・評価者・面接日・備考欄
```

## 内定通知書・不採用通知の下書きを作るプロンプト

採用・不採用の通知文も素早く作れます。

```
以下の内容で【内定通知書 / 不採用通知】を作成してください。

□ 内定通知書（採用の場合）
  入社予定日・職種・処遇（給与・勤務条件）の概要を含める
  内定承諾の期限と連絡方法も明記

□ 不採用通知（見送りの場合）
  丁寧な表現で、理由は書かない
  応募への感謝を忘れずに

文体: 書面（封書・メール）として丁寧に。
     会社名・担当者名はプレースホルダーで。
```

## AIを使う際の注意

- 求人票の条件（給与・残業・休日）は**正確な情報**を記載してください。誇張した表現は後トラブルの原因になります
- 面接質問は法律上禁止されている事項（本籍・家族構成・宗教・思想信条など）が含まれないよう、AIの出力をかならず確認してください
- 採用基準・選考プロセスは社内の人事担当・経営者が最終確認してください
