製造現場のシフト管理は「欠員が出たとき誰を入れるか」「生産計画に合わせた人数をどう確保するか」が常に課題です。AIを使って月次シフト作成・多能工スキルマトリクスの活用・欠員補充計画を効率化できます。

---

## 月次シフト表の作成を支援するプロンプト

制約条件を踏まえた工場シフトを設計します。

```
以下の情報をもとに、製造部門の月次勤務
シフト表を整理してください。

【工場の勤務体系】
勤務形態: 2交代制
  早番: 6:00〜15:00（休憩1時間・実働8時間）
  遅番: 14:00〜23:00（休憩1時間・実働8時間）
稼働日: 月〜土（日曜・祝日は休業）

【配置対象の作業者（製造部第1班）】
名前 / 雇用形態 / 担当工程 / 制約条件:
  山田: 正社員 / プレス・バリ取り両方可 / なし
  鈴木: 正社員 / プレスのみ / 月曜早番NG（通院）
  田中: 正社員 / バリ取り・検査両方可 / なし
  佐藤: パートタイム / 検査のみ / 水・金のみ勤務可
  伊藤: 派遣社員 / バリ取りのみ / 週4日まで

【7月の生産計画と必要人員】
  早番必要人員: プレス1名 + バリ取り1名 + 検査1名 = 最低3名
  遅番必要人員: プレス1名 + バリ取り1名 = 最低2名
  週3回（月・水・金）は検査工程の需要が高いため検査に+1名追加

【7月の休暇申請】
  山田: 7月19日（土）年次有給申請済み
  田中: 7月14日（月）〜16日（水）夏季休暇申請済み

出力形式:
・「7月の週次シフト割り当て案（日付×担当者×早/遅番の表）」
・「田中の夏季休暇中（7/14-16）の補完案（誰がカバーするか）」
・「シフト上の問題点（人員不足になる日）の指摘と対応案」
・「翌月以降の人員補充必要性の判断（現在の体制で対応可能かどうか）」
・「シフト表のExcel/Googleスプレッドシート向けフォーマット設計の説明」
```

## 多能工スキルマトリクスを活用した配置最適化プロンプト

誰がどの工程を担当できるかを可視化します。

```
以下の情報をもとに、製造ラインの多能工
スキルマトリクスと人員配置の最適化を整理してください。

【現在の作業者スキル状況（5段階評価）】
     / プレス / バリ取り / 検査 / 溶接 / 組立
山田 /   5   /    4    /  3  /  1  /  2
鈴木 /   5   /    2    /  1  /  0  /  0
田中 /   2   /    5    /  5  /  0  /  3
佐藤 /   0   /    2    /  5  /  0  /  4
伊藤 /   1   /    5    /  2  /  0  /  1
渡辺 /   0   /    1    /  3  /  5  /  5
（5=一人立ち可・4=ほぼ一人立ち・3=補助ありで可・2=練習中・1=未経験・0=不可）

【今後6ヶ月の多能工化目標】
  鈴木: 検査を3以上に（現在1）
  伊藤: プレスを3以上に（現在1）
  佐藤: バリ取りを3以上に（現在2）

【問題点】
  ・プレス工程は山田・鈴木の2名に依存しており、どちらかが欠勤すると停止
  ・溶接・組立は渡辺1名に依存（渡辺が欠勤した場合の代替なし）

出力形式:
・「多能工スキルマトリクス表（完成形・視覚的に弱点が分かる形）」
・「代替不可のボトルネック工程と緊急度の評価」
・「多能工化育成計画（6ヶ月間のOJT・教育スケジュール）」
・「各作業者の育成目標と達成確認の基準（何ができれば3以上か）」
・「育成進捗を管理する月次チェックシートのフォーマット」
```

## 急な欠員・生産量変化への対応計画プロンプト

突発的な欠員や受注急増への対応を整理します。

```
以下の状況をもとに、突発的な欠員と
急な生産増加への対応計画を整理してください。

【現在の状況（今朝）】
  ・第2班のプレス担当・加藤（正社員）が急病で当日欠勤連絡
  ・今日の生産計画: プレス600個・バリ取り600個
  ・プレスは加藤1名のみが一人立ちレベル（スキルマトリクス5）
  ・他にプレス経験者: 班長（山田、スキル4）・別班の安田（スキル3、
    現在バリ取り担当）

来週からの状況:
  ・顧客からの急受注: プレス部品を来週末までに800個追加納品
  ・通常の生産能力: 600個/週
  ・対応策として残業・休日出勤・応援要員を検討中

出力形式:
・「当日の緊急シフト変更案（誰がプレスに入るか・
  バリ取りはどうカバーするか）」
・「欠員が長引いた場合（加藤が1週間以上休む場合）の対応計画」
・「急受注800個に対応するための週次生産計画の組み換え案」
  （残業/休日出勤/外注活用/納期交渉の4択の検討）
・「緊急人員対応のエスカレーションルール
  （班長→工場長→人事部への報告基準と連絡フロー）」
・「今回の欠員対応の教訓を踏まえた体制改善提案（再発防止）」
```
