工場の温湿度は製品品質・測定精度・作業者の快適性に影響します。AIを使って温湿度記録・空調設備保全・環境管理基準書を体系的に整備できます。

---

## 測定室・精密組立室の温湿度管理書類を整理するプロンプト

```
以下の情報をもとに、精密測定室と精密組立エリアの
温湿度管理書類を整理してください。

【温湿度管理の要件】
管理エリア:
  ①精密測定室（3次元測定機設置）
    基準: 温度 20±1℃（JIS B 7153準拠）/ 湿度 45〜65%RH
  ②精密組立室
    基準: 温度 20±3℃ / 湿度 40〜70%RH

測定設備: 温湿度データロガー（1時間毎に自動記録）

現在の運用:
  ・データロガーのデータはPCに取り込んでいるが管理方法が不明確
  ・異常（温度外れ）が発生した時の製品への影響確認手順がない
  ・夏場に測定室の冷却能力不足で温度超過することがある

出力形式:
・「温湿度管理基準書のフォーマット
  （管理範囲 / 測定機器 / 測定頻度 / 記録方法 / 異常時対応手順）」
・「月次温湿度記録の集計・分析フォーマット（最高・最低・逸脱件数）」
・「温湿度異常発生時の製品品質への影響確認フロー
  （逸脱時間 / 逸脱幅 / 影響を受けた可能性のある製品の特定）」
・「空調設備（精密クーラー）の定期点検記録フォーマット
  （フィルター清掃 / 冷媒圧確認 / 温度制御精度確認）」
・「夏場の測定室温度対策（冷却能力補強 / 入室制限 / 朝一測定推奨）の
  社内通知文例」
```
