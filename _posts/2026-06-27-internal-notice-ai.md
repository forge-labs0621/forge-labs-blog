工場・製造現場では、作業ルールの変更、設備メンテナンスの予告、安全注意の呼びかけなど、現場スタッフ全員に伝えるべき情報を「社内通達」「掲示物」「回覧板」などで周知することが日常的に発生します。AIを使うと、伝えたい要点をまとめた連絡文を素早く作れます。

## AIが役立つ社内通達・連絡文の場面

| 種類 | 具体例 |
|---|---|
| ルール変更の通達 | 作業手順変更、服装規定の改定、入退場ルール変更 |
| 設備停止・工事の予告 | 定期メンテ、電気設備工事、断水工事 |
| 安全注意喚起 | ヒヤリハット事例の共有、季節の安全注意 |
| 表彰・おしらせ | 無事故達成、品質表彰、人事異動のご挨拶 |
| 行事・催しの案内 | 安全大会、健康診断、忘年会案内 |

## 作業ルール変更の通達文を作るプロンプト

```
以下の情報をもとに、作業ルール変更の社内通達文を作成してください。

【変更する内容】
（例：工場内でのスマートフォン使用ルールの変更）

【変更前のルール】
（例：作業中のスマートフォン使用は原則禁止。ただし休憩中はロッカーエリアに限り使用可）

【変更後のルール（変更日含む）】
（例：
2026年8月1日（月）より、以下のルールに変更します。
・作業中の使用禁止（変更なし）
・休憩中はロッカーエリアに加え、1F休憩室内でも使用可
・ただし、撮影・録画は引き続き全エリアで禁止
）

【変更の理由（任意）】
（例：休憩室の設置に伴い、より快適な休憩環境を提供するため）

【通達先】
（例：全従業員）

出力形式:
・「社内通達」として
・件名（通達のタイトル）
・変更内容を「変更前 / 変更後」で対比
・施行日を強調
・問い合わせ先（プレースホルダー）
・発行者（会社名・総務部等）・発行日
```

## 設備停止・工事予告の通達を作るプロンプト

計画停電・設備メンテナンスなどで工場の一部または全体が停止する際の事前連絡文です。

```
以下の情報をもとに、設備停止・工事予告の社内通達文を作成してください。

【停止・工事の種類】
（例：電気設備の法定点検（電気主任技術者による年次点検））

【停止の日時・範囲】
停止する日: （例：2026年8月3日（日））
停止時刻: （例：7:00〜17:00（予定））
停止する範囲: （例：第1・第2工場全域（電力・空調・照明すべて停止））
影響範囲: （例：上記時間帯は工場に入れません）

【従業員への影響・お願いこと】
（例：
・当日は全従業員お休みです（有給取得推奨日とします）
・前日（8/2土曜日）に仕掛品の整理・機器のシャットダウンを行います
・8/3夜17:00以降に電源復旧・動作確認を行い、翌8/4（月）より通常稼働予定
）

出力形式:
・「社内通達」として
・停止日時・範囲を太字や表形式で目立たせる
・従業員への連絡事項を箇条書きで
・不明点の問い合わせ先（プレースホルダー）
・発行者・発行日
```

## 安全注意喚起の通達（季節・ヒヤリハット共有）を作るプロンプト

季節の変わり目の熱中症対策や、ヒヤリハット事例を全員に共有する通達文です。

```
以下の情報をもとに、安全注意喚起の社内通達文（掲示物用）を
作成してください。

【注意喚起のテーマ】
（例：夏季の熱中症予防 / または ヒヤリハット事例の共有）

【内容（箇条書きで構いません）】
テーマ例A: 熱中症予防
（例：
・7月〜8月は工場内温度が35℃を超える日もある
・こまめな水分補給（30分おきを推奨）
・暑さ指数（WBGT）が28以上の時は作業強度を下げる
・頭痛・めまい・吐き気を感じたらすぐに申告
・塩分タブレット・経口補水液を各ラインに設置した
）

テーマ例B: ヒヤリハット事例共有
（例：
・先週、2階から1階へ部品を搬送中に、階段で足元の切粉を踏んで
  転倒しそうになった（けがなし）
・原因: 階段清掃が週1回だったが、加工量増加で切粉が早く溜まっていた
・対策: 階段清掃を毎日実施。切粉の落下が多い加工エリアの養生板を追加
・皆さんも類似状況を見かけたら「KY活動」で報告してください
）

出力形式:
・A4掲示物サイズで、現場スタッフが立ち読みしても内容が伝わる形式
・重要ポイントを箇条書きで
・「こんな時はすぐ報告（相談先）」を末尾に明記
・発行者・発行日
```

## 社内表彰・おしらせの連絡文を作るプロンプト

無事故達成・品質目標達成・人事異動などの周知文です。

```
以下の情報をもとに、社内表彰・おしらせの連絡文を作成してください。

【内容の種類（いずれかを選択）】
□ 無事故達成の表彰（例：製造2課が1,000日無災害を達成）
□ 品質目標達成のお知らせ（例：第1四半期 不良率 目標達成）
□ 人事異動の挨拶（例：部署異動や定年退職者のご挨拶）
□ 入賞・受賞報告（例：5Sコンテスト表彰・安全表彰）

【具体的な内容】
（例（無事故達成）：
製造2課は2026年7月1日をもって、1,000日無災害を達成しました。
これは2023年10月からの全員での取り組みの成果です。
安全第一を常に心がけてくれた製造2課の皆さんに感謝します。
引き続き全社で安全文化の維持・向上に取り組んでいきましょう。
）

出力形式:
・社内掲示・社内メール向けの温かみのある文体
・達成内容を冒頭に明確に
・お礼・感謝の言葉を添える
・継続への呼びかけで締める
・発行者・発行日
```

## 社外連絡（夏季休業・年末年始の案内）を作るプロンプト

取引先へ送る休業案内も、同じプロンプトで対応できます。

```
以下の情報をもとに、取引先向けの夏季休業案内メールを
作成してください。

【休業期間】
（例：2026年8月11日（月）〜8月15日（金）の5日間）

【休業前の最終営業日】
（例：2026年8月8日（金））

【休業明けの最初の営業日】
（例：2026年8月18日（月））

【休業中の緊急連絡先（ある場合のみ）】
（例：製品不具合・納期に関する緊急連絡は、携帯番号○○○-○○○-○○○○まで）

【自社名・担当者名】
（例：○○製作所株式会社 営業部 山田○○）

出力形式:
・取引先メール向けの丁寧なビジネスメール
・件名: 「夏季休業のご案内」
・休業期間を目立つ場所に
・緊急連絡先の扱いを明確に（ない場合は省く）
・日頃のお引き立てへの感謝を冒頭に
```

---

## AIを使う際の注意

- 社内通達は「誰が」「何を」「いつから」「どこで」を明確にすることが最重要です。AIの下書きで抜け漏れがないか確認してください
- 表彰・人事に関する連絡文は、本人の同意・事前確認が必要な場合があります
- 取引先向けの休業案内は、誤った日程が記載されていないか、送付前に必ず確認してください
