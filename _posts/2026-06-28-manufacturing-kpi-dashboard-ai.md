製造業ではOEE・不良率・納期遵守率など多くのKPIを管理します。AIを使ってKPIの設計・ダッシュボードの構成・月次経営指標の報告書を効率化できます。

---

## 製造業のKPI体系を設計するプロンプト

自社に合ったKPIを体系的に整理します。

```
以下の情報をもとに、製造業の経営・現場の
KPI体系を設計してください。

【当社の業種・規模】
業種: 金属プレス加工業（受託製造）
従業員: 25名 / 売上: 年¥200M / 主要顧客: 自動車部品Tier1 × 3社

【現在管理しているKPI（ばらばらで管理中）】
  ・月次売上高
  ・不良品率（工程別）
  ・残業時間（月次）
  ・設備稼働率（プレス機のみ）

【経営層が気にしていること】
  ・「なぜ利益率が改善しないのか」
  ・「どの工程でロスが大きいのか」
  ・「納期遵守率が低いと顧客からクレームがくる」

【現場が気にしていること】
  ・「今月の目標に対して進捗がどうなのかわからない」
  ・「不良が多いのはわかるが、どこで発生しているかデータがない」

【KPIの階層設計（バランススコアカード的なアプローチ）】
  財務の視点: 売上・利益・原価
  顧客の視点: 納期遵守率・品質クレーム・顧客満足
  内部プロセスの視点: OEE・不良率・歩留まり・リードタイム
  学習・成長の視点: 改善提案件数・資格取得数・多能工化率

出力形式:
・「製造業KPI体系（4視点×各2〜4指標・計12〜16個のKPI一覧）」
・「各KPIの定義（計算式・分子・分母・測定頻度）」
・「現状データで計算できるKPI vs 今後データ収集が必要なKPIの分類」
・「経営層向けKPI（3〜5個のエグゼクティブKPI）の選定推奨」
・「現場向けKPI（日次・週次で確認する指標）の選定推奨」
```

## 月次KPIダッシュボードの構成を整理するプロンプト

ワンページで全体が見えるダッシュボードを設計します。

```
以下の情報をもとに、製造業の月次KPI
ダッシュボードの構成を整理してください。

【ダッシュボードの目的】
  ・毎月の経営会議（月初の第1月曜日）で使用
  ・参加者: 社長・工場長・品質部長・営業部長
  ・A3用紙1枚 または PowerPoint1スライドに収める

【掲載したいKPI（月次データが揃っているもの）】
売上・財務:
  ・月次売上高（目標 vs 実績 / 前年同月比）
  ・粗利益率（目標 vs 実績）

生産・品質:
  ・総合設備効率（OEE）: 目標85%
  ・不良品率（工程別 + 合計）: 目標0.5%以下
  ・歩留まり率: 目標98.5%以上

納期・顧客:
  ・納期遵守率: 目標98%以上
  ・顧客クレーム件数: 目標0件

安全:
  ・労働災害件数（累計）

人材:
  ・残業時間（部門別）: 目標45時間/月以下
  ・改善提案件数

出力形式:
・「ダッシュボードのレイアウト案（信号機方式の色分けを含む）」
・「各指標の表示形式の推奨（棒グラフ/折れ線/信号機の使い分け）」
・「前月比・前年同月比・目標達成率の3軸の見せ方」
・「ダッシュボード作成のためのデータ収集テンプレート（誰が何を集めるか）」
・「会議でダッシュボードを使った議論の進め方（アジェンダ）」
```

## 部門別KPI目標の設定と管理を整理するプロンプト

年度計画のKPI目標を部門ごとに設定します。

```
以下の情報をもとに、製造部・品質部・購買部の
年度KPI目標を設定してください。

【会社全体の年度方針】
今年度のテーマ: 「品質と生産性の両立」
経営目標（今年度）:
  ・売上高: 前年比+5%（¥200M → ¥210M）
  ・営業利益率: 3.5%以上（前年3.1%）
  ・重大クレーム: ゼロ

【前年の実績（参考）】
  ・不良品率: 1.2%（目標0.8%で未達）
  ・OEE: 75%（目標85%で未達）
  ・納期遵守率: 96.5%（目標98%で未達）
  ・在庫回転率: 9.5回（目標10回）

各部門への展開:
  製造部（生産・品質確保・コスト）
  品質部（品質管理・顧客対応・ISOの維持）
  購買部（仕入れコスト・調達安定・在庫）

出力形式:
・「部門別KPI目標一覧（部門×指標×前年実績×今年目標×根拠）」
・「各部門のKPI目標の設定根拠（なぜその数値を目標にするか）」
・「部門間のKPI連携（製造部の目標が品質部・購買部にどう影響するか）」
・「KPI未達成時のエスカレーションルール（何を超えたら社長報告か）」
・「月次KPI振り返り会議のアジェンダフォーマット（部門別5分以内）」
```
