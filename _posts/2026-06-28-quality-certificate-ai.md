自動車・電子機器・医療機器などの部品サプライヤーは、出荷品に対して「品質成績書」「検査成績書」または「CoC（Certificate of Conformance：適合宣言書）」を添付することを客先から求められる場合があります。AIを使うと、測定データや検査結果から品質成績書の文章部分を整理したり、社内フォームの項目を統一したりする作業を効率化できます。

> **注意**: 品質成績書・検査成績書の数値データは、実際の測定値・検査結果に基づくものです。AIは「文章の整理・フォーマットの統一・コメントの文章化」を補助するものであり、測定値・判定値の生成はAIに任せないでください。

## 品質成績書・検査成績書の種類

| 種類 | 内容 | 主な用途 |
|---|---|---|
| 品質成績書 | 寸法・材質・特性の測定結果 | 自動車部品・精密機器 |
| 検査成績書 | 検査項目・合否判定の記録 | 一般的な受託加工品 |
| CoC（適合宣言書） | 規格・仕様への適合を宣言する書類 | 海外取引・ISO規格品 |
| ミルシート（材料成績書） | 材料の化学成分・機械的性質 | 鉄鋼・アルミ材料 |

---

## 品質成績書の文章部分（特記事項・備考欄）を整理するプロンプト

数値は別で記録し、文章コメント・特記事項の整理に使えます。

```
以下の情報をもとに、品質成績書の「特記事項・備考欄」に
記載する文章を整理してください。

【製品・品番】
（例：プレス加工品 品番: ABC-001、材質: SPCC、数量: 200個）

【検査の実施概要】
（例：
・全数外観検査（目視）: 合格200個/200個（不良0件）
・抜き取り寸法検査（5%、10個）: 全項目合格
・ミルシート確認: 材質・板厚が図面仕様と一致を確認
）

【特記事項・今回の特別な注意点】
（例：
・客先指示により、表面処理（三価クロメート）後の外観に
  「ざらつき感が出やすい」との事前連絡があった
・今回のロットで目視確認を強化し、10個を抜き取りサンプルとして別保管
・上記サンプルは客先要請があれば提出可能
）

出力形式:
・品質成績書の「備考・特記事項」欄に記載できる簡潔な文章（200字以内）
・客観的な事実の記述を優先し、不確実な表現（「〜と思われる」等）は避ける
```

## 検査成績書の確認項目・合否判定欄を整理するプロンプト

社内の検査記録をもとに、客先提出用の検査成績書フォームの確認項目を整理します。

```
以下の情報をもとに、検査成績書の確認項目一覧を作成してください。

【製品・図面情報】
品番・品名: （例：旋盤加工品 品番XY-200）
図面規格（主要寸法の例）:
（例：
・外径: φ50 ±0.02mm
・内径: φ20 H7（+0.021/0）
・長さ: 100 ±0.1mm
・表面粗さ: 外径Ra1.6以下
・外観: 傷・バリなし
）

【客先から指定されている検査規格・基準】
（例：JIS B 0401（はめあい方式）準拠）

出力形式:
・「検査項目 / 規格値（上下限） / 測定方法 / 測定値（記入欄） / 合否（記入欄）」
  の5列で表形式
・測定値・合否欄はプレースホルダー（空欄）にする
・最後に「総合判定（合格・不合格）/ 検査実施者 / 検査日 / 承認者」欄を付ける
```

## CoC（適合宣言書）の文章を整理するプロンプト

規格・仕様への適合を宣言するCoC文書の文章部分を作ります。

```
以下の情報をもとに、CoC（Certificate of Conformance: 適合宣言書）の
文章を作成してください。

【宣言する内容】
製品名・品番: （例：精密プレス部品 Part No. A-1234）
納入数量: （例：500個）
ロット番号: （例：LOT-2026-0720）
適合する規格・仕様: （例：図面 Rev.C に記載の寸法・材質・表面処理仕様）
材質: （例：SUS304 ステンレス鋼板（ミルシートNo.○○に準拠））
製造者情報: （例：○○製作所株式会社 品質保証部）

【実施した検査・確認の概要】
（例：
・社内検査基準書（QI-001）に基づく寸法検査（全数）
・外観検査（全数）
・材料証明書（ミルシート）による材質確認
）

出力形式:
・英文と日本語の両方（英文先・日本語後）
・「We hereby certify that...」で始まる標準的なCoC英文形式
・署名欄・日付欄・会社名・担当者名欄（プレースホルダー）
・「この文書はAIを補助ツールとして作成した下書きです。
  最終確認は品質保証責任者が行ってください。」の注記を末尾に付ける
```

## ミルシート（材料成績書）の読み方・整理コメントを作るプロンプト

仕入先から受け取ったミルシートの内容を、社内や客先にわかりやすく要約します。

```
以下のミルシートの情報をもとに、材料成績書の要点まとめコメントを
作成してください。

【ミルシートに記載されている主な情報】
材料名・規格: （例：冷間圧延鋼板 SPCC（JIS G3141））
板厚・寸法: （例：1.0mm × 1,000mm × 2,000mm）
ヒート番号（熔製ロット）: （例：H2026-0701）
化学成分: （例：C: 0.12%, Mn: 0.50%, P: 0.030%, S: 0.030%以下（実績値は上限を下回ることを確認））
機械的性質: （例：引張強度 314MPa、降伏点 175MPa、伸び 40%以上（規格値内を確認））

【確認したこと】
（例：図面指定のSPCCに合致しており、全項目JIS規格範囲内。問題なし。）

出力形式:
・社内担当者向けの「材料確認コメント」として
・材料規格・ロット情報・主要特性値の要点を箇条書きで
・「規格範囲内であることを確認した」旨を明記
・ミルシート保管場所・ファイル番号の記入欄
```

## AIを使う際の注意

- 品質成績書・検査成績書の数値（寸法測定値・試験結果）は実際の測定値のみを記載し、AIが生成した数値は絶対に使用しないでください
- CoC（適合宣言書）は法的・契約上の宣言文書です。日本語・英語問わず、最終版は必ず品質保証責任者が確認・署名してください
- 品質成績書の電子署名・押印の要否は取引先の要求に従ってください
