ISO 9001やその他の品質マネジメントシステムでは、「品質目標の設定・展開・達成状況の評価」が求められます。年初の品質目標設定書の作成から月次・四半期の進捗管理レポートまで、AIで文書化を効率化できます。

---

## 年間品質目標設定書を整理するプロンプト

前年の実績を踏まえた今年の品質目標を文書化します。

```
以下の情報をもとに、年間品質目標設定書を作成してください。

【前年実績（参考値）】
不良率: （例：前年実績 0.8%（社内基準 0.5%以下に対して未達））
クレーム件数: （例：前年 3件（前々年5件から改善）)
外注先不良率: （例：前年 1.2%）
検査コスト: （例：対売上比 2.1%）

【今年の品質方針・重点事項】
（例：
・不良品流出ゼロを最優先課題とする
・外観検査の判断基準を明確化し、担当者間のばらつきを解消する
・外注先品質の向上を支援する（協力会社への技術支援）
）

【今年の品質目標（案）】
- 社内不良率: 0.5%以下（前年0.8%からの改善）
- クレーム件数: 1件以下（前年3件）
- 外注先不良率: 0.8%以下

出力形式:
・年間品質目標設定書のフォーマット
・目標ごとに「目標値 / 評価指標 / 測定方法 / 測定頻度 / 担当」の表
・「重点施策（目標達成のために何をするか）」を各目標に2〜3点
・承認欄（品質保証部長・経営者）
・期中見直し時期（例：半期に1回）の記載
```

## 品質目標の部門別展開シートを作るプロンプト

会社の品質目標を各部門に展開する文書を作ります。

```
以下の情報をもとに、品質目標の部門別展開シートを作成してください。

【会社全体の品質目標】
（例：社内不良率 0.5%以下 / クレーム1件以下 / 外注先不良率 0.8%以下）

【製造部への展開（対象：製造部）】
部門の役割: 生産工程における品質確保
展開目標:
- 工程内不良率: 1.0%以下（製造部内での発生を抑制）
- 初回品質合格率（初物確認）: 90%以上
- 是正措置の完了率: 指摘後2週間以内に100%完了

【品質保証部への展開（対象：品質保証部）】
展開目標:
- 流出不良件数: 0件
- 検査実施率: 100%（基準書に従って）
- SDS・外観基準書の更新遅延: 0件

出力形式:
・部門別展開シートのフォーマット（部門名 / 展開目標 / 目標値 / 測定方法 / 担当者）
・会社目標との連携が一目でわかる形式（左列に会社目標・右に部門展開）
・月次確認のチェック欄（簡易な進捗記録ができるよう）
```

## 品質目標の進捗報告（月次）を整理するプロンプト

月次の品質目標達成状況を集計・報告します。

```
以下のデータをもとに、品質目標の月次進捗報告を作成してください。

【対象月】（例：2026年7月度）

【目標と実績】
- 社内不良率:
  年間目標 0.5%以下 / 今月実績 0.3% / 前月実績 0.6% / 判定: ○（達成）
- クレーム件数:
  年間目標 1件以下 / 今月実績 0件 / 前月実績 1件 / 判定: ○（達成）
- 外注先不良率:
  年間目標 0.8%以下 / 今月実績 1.1% / 前月実績 0.9% / 判定: △（未達）

【外注先不良率が目標未達の原因】
（例：特定の外注先（○○加工）から1件の不良があった。寸法外れ。是正対応中。）

出力形式:
・「目標 / 今月実績 / 前月比 / 達成状況（○△×）/ コメント」の表形式
・未達項目には「原因と対応状況」を追記
・「総合評価」（全体的な品質状況の1〜2行コメント）
・翌月の重点活動（1〜2点）
```

## 品質管理レビュー（MR）向けサマリーを作るプロンプト

ISO 9001で求められるマネジメントレビューの品質報告サマリーを作ります。

```
以下のデータをもとに、ISO 9001マネジメントレビュー向けの
品質報告サマリーを作成してください。

【対象期間】（例：2026年1月〜6月（上半期））

【品質指標の実績】
- 社内不良率: 上半期平均 0.42%（年目標 0.5%以下 → 達成）
- クレーム件数: 上半期 1件（年目標 1件以下 → 達成 / 是正措置完了）
- 外注先不良率: 上半期平均 0.95%（年目標 0.8%以下 → 未達）
- 内部監査: 上半期1回実施 / 不適合 2件（是正完了）

【主な改善活動】
（例：外観検査基準書を7品番分改訂。新入検査員OJT教育を3回実施。）

【課題・下半期の方針】
（例：外注先不良率の改善が最重点課題。○○加工への技術支援と
  受入検査強化を下半期に実施予定。）

出力形式:
・MR報告資料として使えるA4 1ページのサマリー
・「上半期の成果と課題」を見開きで整理
・「経営者へのアクション要求（あれば）」の欄
・次回MR（下半期）の予定日
```
