製造部門の月次報告書は、生産実績・品質・設備稼働状況・課題と対策を経営層や上位部門に伝える重要な書類です。「毎月同じような書類を一から作っている」「数字を集めてから文章にまとめるのに時間がかかる」という担当者向けに、AIを使って効率よく仕上げる方法をまとめます。

## AIが助けてくれる場面

月次生産報告書の作成で、AIが特に役立つのは以下の場面です。

- **数値データを文章に変換する**（実績数字を箇条書きで渡すと、読みやすい文章にまとめてくれる）
- **前月比・計画比のコメントを生成する**（差分の数値から「なぜ」を説明する文章の下書き）
- **課題と対策の文章を整える**（箇条書きメモを報告書向けの文体に変換）

数値の計算ミスや事実確認はAIに任せず、必ず人が確認してください。

## 基本プロンプト（月次報告書の本文作成）

---

```
以下の生産実績データをもとに、上司・経営層向けの月次生産報告書の本文を作成してください。

【対象月】
2026年○月

【生産実績】
・計画生産数量: ○○○個
・実績生産数量: ○○○個（達成率 ○○%）
・主な未達/超過の理由: （例：設備トラブル2日間停止、受注追加で生産追加等）

【品質実績】
・不良率: ○○%（前月 ○○%）
・主な不良内容: （例：外観不良 ○件、寸法不良 ○件）
・クレーム件数: ○件

【設備稼働状況】
・稼働率: ○○%（計画 ○○%）
・主なトラブル・停止内容: （例：プレス機油漏れで4時間停止）

【来月の重点課題・対策】
・（例：不良率低減のため検査工程を追加、設備予防保全の見直し）

【出力形式】
・A4 1〜2枚程度の分量
・「今月のまとめ」→「生産実績」→「品質」→「設備」→「来月の重点課題」の順で構成
・数値は正確に、コメントは簡潔に
```

**2026年○月 生産実績報告書**

**今月のまとめ**

今月の生産実績は○○○個（計画比 ○○%）でした。月初に発生したプレス機の油漏れトラブル（4時間停止）が影響し、計画を下回る結果となりました。品質面では不良率が○○%と前月（○○%）から改善しており、先月導入した検査工程追加の効果が現れています。来月は設備の予防保全見直しを最優先課題として取り組みます。

## 「なぜ未達だったか」のコメントを作るプロンプト

月次報告で難しいのは、数値の差分に対する「原因分析のコメント」です。箇条書きのメモからAIに文章を作らせると、報告書向けの表現に整えてくれます。

```
以下の状況を報告書向けの文章にまとめてください。

状況:
・7月に設備Aが2日間停止（油漏れが原因）
・その間、他ラインに振り替えたが能力が50%程度
・結果として月間計画の8%未達

報告書向けの分析コメントとして：
・未達の事実と数量（簡潔に）
・主因（設備トラブル）
・部分的な対応（ライン振り替え）
・今後の対策（設備保全の見直し）
の4点を2〜3段落にまとめてください。
```

## 前月比コメントを自動で作る

毎月の「前月比」コメントも、数値さえ渡せばAIが下書きしてくれます。

```
以下の数値をもとに、月次報告書向けの「前月比コメント」を作成してください。

不良率: 今月 1.2% / 先月 1.8%
主な改善施策: 今月から検査工程を追加（最終検査の目視確認を2人から3人に）

コメントの条件:
・改善の事実（数値で示す）
・主な要因（施策との紐づけ）
・今後の課題または目標
の3点を2〜3文で。
```

## データを集める前の準備にも使える

月末に数値データを集める前の段階でも、「どんなデータを準備すればよいか」をAIに確認できます。

```
製造部門の月次報告書を作るために、どんなデータを事前に集めておく必要があるか、
チェックリスト形式でまとめてください。
（生産数量・品質・設備・人員・コスト・来月予定の観点で）
```
