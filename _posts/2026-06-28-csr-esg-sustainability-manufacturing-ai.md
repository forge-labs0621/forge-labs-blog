製造業では顧客（特に大手メーカー・上場企業）から「ESG・CSR対応状況の開示」を求められるケースが増えています。AIを使ってCSR活動の整理・ESG情報の文書化・サステナビリティレポートの作成を効率化できます。

---

## CSR活動報告書の骨子を整理するプロンプト

取引先・ステークホルダー向けのCSR活動報告書を作ります。

```
以下の情報をもとに、製造業のCSR活動報告書の
骨子を作成してください。

【自社のCSR活動の実績（例）】
環境への取り組み:
  ・CO2排出量の削減（2020年比○%削減 / 再生可能エネルギーの一部導入）
  ・廃棄物の削減（製品廃棄物リサイクル率○%、産廃ゼロエミッション目標）
  ・グリーン調達の推進（VOC・有害物質の使用削減）

社会への取り組み:
  ・地域雇用への貢献（従業員○名うち地元採用○%）
  ・障がい者雇用（法定雇用率○%、実績○%）
  ・地域清掃活動・工場見学（年○回）

労働環境への取り組み:
  ・安全衛生（休業災害ゼロ継続○年）
  ・育児休業取得率（○%）
  ・時間外労働の削減（働き方改革）

ガバナンス:
  ・内部統制・コンプライアンス研修の実施
  ・情報セキュリティ対応

出力形式:
・「CSR活動報告書の構成（目次と各章の要点）」
・「自社のCSR活動の強み・特徴を引き出す表現の整理」
・「顧客・取引先からのCSRアンケートに対応する標準回答文の骨子」
・「SDGs（持続可能な開発目標）との対応付け一覧（自社の取り組みと
  SDGsゴールの関係整理）」
・「CSR活動のKPI（測定指標）設定と年次モニタリング計画」
```

## 取引先からのESGアンケート回答書類を作るプロンプト

顧客から送られてくるESGサプライチェーン調査票に回答します。

```
以下の情報をもとに、取引先からのESG・サプライチェーン
調査票への回答を整理してください。

【調査票の主な質問項目（例）】
① 環境（E）:
  ・CO2排出量の把握・削減目標の有無（Scope1/Scope2）
  ・再生可能エネルギーの使用比率
  ・廃棄物削減・リサイクルの取り組み
  ・化学物質・有害物質の管理（RoHS/REACH対応）

② 社会（S）:
  ・労働安全衛生の取り組み（労災発生件数）
  ・ハラスメント防止・コンプライアンス教育
  ・人権デューデリジェンスの実施状況
  ・サプライチェーン管理（仕入先の行動規範）

③ ガバナンス（G）:
  ・法令遵守体制・コンプライアンス方針
  ・情報セキュリティ対策（ISMS・Pマーク等）
  ・腐敗防止・汚職防止の取り組み

【自社の状況（回答の根拠）】
CO2排出量: 電力使用量から換算・把握はしているが削減目標は未設定
労災: 直近3年連続ゼロ
ISO 9001認証: 取得済み
情報セキュリティ: 社内ルールはあるが文書化できていない

出力形式:
・「ESGアンケート各設問への回答文テンプレート」
  （「実施している」「実施していない・対応検討中」の両パターン）
・「正直に「未対応」と回答しながら、改善意欲を示す表現の例」
・「ESGアンケート対応のために今すぐ整備すべき書類の優先リスト」
・「ESGアンケートへの対応を継続的に管理するためのフォーマット」
```

## カーボンフットプリント（CF）計算の準備をするプロンプト

製品のライフサイクルでのCO2排出量（CFP）の計算準備をします。

```
以下の情報をもとに、製品のカーボンフットプリント（CFP）の
計算準備と報告書類を整理してください。

【背景】
顧客（大手自動車メーカー）から「2027年度より製品ごとの
カーボンフットプリント（CFP）を開示してほしい」という
要請を受けた。自社ではCO2排出量の計算を行ったことがない。

【自社の主な排出源（概算）】
Scope 1（直接排出）:
  ・燃焼設備（ボイラー・フォークリフトの燃料）: ガスA重油 月○L
  ・工程ガス: なし

Scope 2（間接排出: 電力）:
  ・電力使用量: 月○kWh（電力会社: ○○電力）
  ・購入電力の排出係数: ○kg-CO2/kWh（電力会社の報告値）

Scope 3（その他の間接排出・把握困難な部分）:
  ・材料の調達（購入した鋼板・部品の製造に伴うCO2）
  ・製品の輸送（配送業者への委託）

出力形式:
・「CFP計算の進め方ガイド（自社でできる範囲と専門機関が必要な範囲）」
・「Scope 1・Scope 2のCO2排出量の計算フォーマット」
・「顧客向けの「CFP開示対応状況報告書」の骨子」
・「CFP計算に必要なデータの収集方法と担当部署の整理」
・「LCA（ライフサイクルアセスメント）ツールの活用検討のポイント」
```
