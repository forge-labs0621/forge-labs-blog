製造業の現場で取り組む5S活動（整理・整頓・清掃・清潔・躾）。「チェックリストが形骸化している」「毎月の巡回チェックに使えるシートを作りたい」という現場担当者向けに、AIを使って5Sチェックリストを素早く作る方法をまとめます。

## 5Sチェックリストに必要な項目

5S活動のチェックリストに入れるべき観点は、大きく以下の5つです。

| S | 内容 | チェックの例 |
|---|------|------------|
| 整理（Seiri） | 必要なものと不要なものを分ける | 不要物が放置されていないか |
| 整頓（Seiton） | 必要なものを使いやすい場所に置く | 工具・材料の置き場所が決まっているか |
| 清掃（Seiso） | 汚れ・ゴミを取り除き点検する | 設備・床の清掃状態、オイル漏れなど |
| 清潔（Seiketsu） | 整理・整頓・清掃の状態を維持する | 掲示物・表示が最新の状態か |
| 躾（Shitsuke） | ルールを習慣として守り続ける | 安全装備の着用、手順書通りの作業 |

これをゼロから作ると時間がかかりますが、AIに条件を渡せば数分でたたき台が出てきます。

## 5SチェックリストをAIで作るプロンプト

以下をChatGPTやClaudeに貼り付けて、条件部分だけ書き換えてください。

---

```
以下の条件で、製造現場の5S活動巡回チェックリストを作成してください。

【条件】
対象エリア: （例：機械加工エリア、組立ライン、倉庫、事務所など）
チェックする人: （例：班長、品質担当、安全衛生委員）
チェック頻度: （例：毎日始業前、週1回、月1回）
特に問題になりやすい点: （例：工具の置き忘れ、通路の荷物、油漏れ）

【出力形式】
5S（整理・整頓・清掃・清潔・躾）それぞれ3〜5項目を一覧にしてください。
各項目に「OK/NG/対応中」の評価欄と「コメント欄」をつけてください。
現場の作業者でも迷わず確認できる、具体的な文言にしてください。
```

## 実際の出力例（機械加工エリア・週1回巡回）

上のプロンプトで「機械加工エリア・班長・週1回・工具の置き忘れと油漏れ」という条件を渡すと、以下のような内容が出てきます。

---

### 5S巡回チェックリスト（機械加工エリア）

**■ 整理**

| No | チェック項目 | OK / NG / 対応中 | コメント |
|----|------------|-----------------|---------|
| 1 | 使用中の材料・仕掛品以外の物が加工エリア内に放置されていないか | | |
| 2 | 不要な治工具・材料が機械の上や足元に残っていないか | | |
| 3 | 廃材・スクラップが所定の場所以外に積まれていないか | | |

**■ 整頓**

| No | チェック項目 | OK / NG / 対応中 | コメント |
|----|------------|-----------------|---------|
| 1 | 工具・測定器が定位置（場所の表示あり）に戻っているか | | |
| 2 | 通路・非常口の前に荷物が置かれていないか | | |
| 3 | 使用中の材料に品番・ロット番号の表示があるか | | |

**■ 清掃**

| No | チェック項目 | OK / NG / 対応中 | コメント |
|----|------------|-----------------|---------|
| 1 | 床面に油・切り粉・ゴミが残っていないか | | |
| 2 | 機械本体・カバー周辺のオイル漏れ・にじみがないか | | |
| 3 | 集塵機・フィルターに目詰まりがないか（週次確認） | | |

**■ 清潔**

| No | チェック項目 | OK / NG / 対応中 | コメント |
|----|------------|-----------------|---------|
| 1 | 安全表示・禁止表示が汚れ・破損なく読める状態か | | |
| 2 | 作業手順書・標準書が所定の場所に最新版が掲示されているか | | |

**■ 躾**

| No | チェック項目 | OK / NG / 対応中 | コメント |
|----|------------|-----------------|---------|
| 1 | 作業者が保護具（安全靴・保護メガネ等）を正しく着用しているか | | |
| 2 | 前回の指摘事項が改善・再発防止されているか | | |

---

AIが出したものをそのまま使うのではなく、現場の実情に合わせて項目を追加・削除してください。

## エリアごとにカスタマイズする

5Sチェックリストはエリアによって重点が異なります。以下のように条件を変えて、エリア別のリストを作ると使いやすくなります。

**倉庫・資材置き場向け**

```
対象エリア: 原材料倉庫・仕掛品ストック場所
特に問題になりやすい点: 期限切れ材料の混入、入出庫記録の未記入、
通路への荷崩れ、フォークリフト通路の確保
```

**事務所向け**

```
対象エリア: 製造管理事務所
特に問題になりやすい点: 書類・図面の散乱、不要ファイルの保管、
パソコン周りの配線整理、個人情報書類の施錠管理
```

## 月次巡回記録に活用する

毎月の5S巡回結果を記録に残すには、こんなプロンプトも使えます。

```
以下の5S巡回チェック結果をもとに、月次報告用の5S活動記録をまとめてください。

【今月の巡回結果メモ】
（ここにチェックリストの結果や気づきを箇条書きで貼る）

【出力内容】
・今月の良かった点（2〜3点）
・今月の課題（2〜3点）
・来月の重点改善テーマ（1〜2点）
・担当者・確認者・日付欄
```
