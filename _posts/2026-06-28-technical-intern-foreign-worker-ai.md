製造業では技能実習生・特定技能外国人の活用が増えています。関連する書類（技能評価記録・生活指導記録・監理団体報告・在留資格更新）の整備と、多言語コミュニケーションの補助にAIを活用できます。

---

## 技能実習計画・指導記録のフォーマットを作るプロンプト

技能実習法に基づく技能実習計画と日々の指導記録を整理します。

```
以下の情報をもとに、技能実習生の管理書類のフォーマットを
作成してください。

【実習生の基本情報（記録管理対象）】
在籍: （例: 技能実習2号 / ベトナム国籍 / 3名 / 機械加工職種）
実習計画: 監理団体（○○協同組合）を通じて認定済み（3年間）
実習内容: 旋盤・フライス盤操作による機械加工

【整備したい書類】
① 技能実習日誌（月次記録）
② 技能習得状況評価シート（中間評価・修了評価）
③ 生活指導記録（月次: 生活相談・健康確認）
④ 監理団体向け実習実施記録（四半期）

出力形式:
・上記4つの書類フォーマット（日付・指導内容・担当者・評価欄を含む）
・「技能実習法で求められる記録保存期間と管理のポイント」
・「技能評価（基礎級・随時3級等）の受験前チェックリスト」
・「実習生が実習を中断・帰国した場合の書類手続きフロー」
・「監理団体の定期訪問（3か月に1回）に備えて準備する書類一覧」
```

## 技能実習生への多言語作業指導書を作るプロンプト

日本語が不十分な実習生に伝わる作業指示書・安全教育資料を作ります。

```
以下の情報をもとに、技能実習生・特定技能外国人向けの
多言語作業指示書を作成してください。

【対象作業】
作業名: （例: 旋盤加工（外径切削）の基本手順）
対象者: ベトナム人技能実習生（日本語能力: N4程度）

【日本語版の作業手順（元の作業指示書）】
1. 材料（丸棒φ50mm・長さ200mm）を確認し、傷・錆のないことを確認する
2. チャックに材料をセットし、芯出しを行う（振れ0.05mm以内）
3. バイトを取り付け、切削条件を設定する（S:800rpm、F:0.1mm/rev、切込み:0.5mm）
4. 乾式切削で外径をφ48mmに仕上げる（公差: ±0.05mm）
5. ノギスで寸法を確認し、作業日報に記録する

【多言語化の要件】
対象言語: ベトナム語（日本語ルビつきで残す）
表現: できるだけ簡単な表現・短文に直す
重要注意事項: 赤字・太字で強調する（チャックの確実な締め付け・
  回転中は刃物台に手を触れない等の安全ルール）

出力形式:
・多言語作業指示書（日本語ルビ + ベトナム語対訳 + 写真説明の場所を示す注記）
・「安全注意事項の多言語版（絵・アイコンで補足できる形式）」
・「日本語と現地語が混在した指示書を作る際の注意点」
・「実習生が理解しているか確認するためのチェック手順」
```

## 在留資格更新・行政手続きの事前整理プロンプト

技能実習・特定技能の在留資格更新に必要な社内書類を整理します。

```
以下の情報をもとに、在留資格更新・変更申請の際に
会社が準備する書類を整理してください。

【在留資格更新の種別】
（例: 技能実習2号から技能実習3号への移行申請
  / 申請期限: ○か月後）

【申請時に会社が準備する主な書類（例）】
・実習実施計画（技能実習3号）: 監理団体と連携して作成
・技能実習評価試験合格証の写し（随時3級または技能検定3級等）
・法令に基づく技能実習生の待遇（賃金・宿舎等）の説明書類
・役員の住民票・登記事項証明書等

【整理したい情報】
・書類準備のタイムライン（申請○か月前から何を準備するか）
・監理団体・出入国在留管理庁とのやり取りで確認が必要な事項
・申請書類の不備が多いポイント（よくある失敗）

出力形式:
・在留資格申請書類準備チェックリスト（書類名・準備主体・期限）
・「申請タイムライン（3か月前から申請日までのスケジュール）」
・「よくある不備・申請遅延の原因と防止策」
・「特定技能（製造業3分野）に切り替えを検討する場合の比較」の骨子
・「入管法改正・技能実習制度廃止・育成就労制度への移行（2024年改正）の
  現時点の影響整理」（注: AIの知識限界があるため最新情報の確認を推奨）
```
