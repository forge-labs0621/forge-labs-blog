製造業のプレス加工・研削・鍛造工場では騒音・振動による健康被害のリスクがあります。AIを使って作業環境測定記録・騒音対策計画・振動障害防止書類を効率的に整備できます。

---

## 作業環境測定（騒音）の記録を整理するプロンプト

法定測定を漏れなく記録します。

```
以下の情報をもとに、騒音の作業環境測定
結果記録と事後措置書類を整理してください。

【当社の騒音測定状況】
対象工程: プレス加工工場（250トンプレス × 2台・100トン × 3台）
騒音測定の法的根拠: 労働安全衛生法 / 騒音障害防止のためのガイドライン
測定頻度: 年1回以上（6ヶ月に1回推奨）

最新の作業環境測定結果（2026年5月実施）:
  測定場所A（250トンプレス 作業位置）: 98dB(A) [第3管理区分]
  測定場所B（100トンプレス 作業位置）: 92dB(A) [第3管理区分]
  測定場所C（段取り・材料置き場）: 84dB(A) [第2管理区分]

管理区分の基準:
  第1管理区分（良好）: 85dB未満
  第2管理区分（改善努力要）: 85〜90dB未満
  第3管理区分（対策義務）: 90dB以上

現在の保護具: 耳栓（100dB対応）を全員配布

出力形式:
・「作業環境測定結果記録（法定書式に準拠した記録フォーマット）」
  （測定日時 / 測定場所 / 使用機器 / 測定値 / 管理区分 / 測定者）
・「第3管理区分に対して必要な法定措置の一覧」
  （医師による健康診断 / 継続的な保護具着用 / 改善計画の策定）
・「騒音対策計画書の骨子（設備対策 + 管理的対策の組み合わせ）」
  （発生源対策: プレス設備の防音 / 受音側: 耳栓の適切使用 / 作業時間制限）
・「耳栓の正しい使い方・管理方法の教育資料骨子
  （挿入方法 / 種類の選び方 / 耳栓の衛生管理 / 効果の確認方法）」
・「騒音性難聴（職業性難聴）の健康診断項目と結果の管理方法」
```

## 振動障害防止管理書類を整理するプロンプト

チェーンソー・ハンドグラインダー使用の管理書類を整備します。

```
以下の情報をもとに、振動工具の使用管理と
振動障害防止のための書類を整理してください。

【振動工具の使用実態】
使用する振動工具:
  ①ハンドグラインダー（研磨・バリ取り）: 4名が使用
    1日の使用時間: 平均4〜6時間（継続使用）
  ②エアチッパー（バリ取り・スラグ除去）: 2名が使用
    1日の使用時間: 平均2〜3時間
  ③インパクトレンチ（締結作業）: 随時

法的根拠: 振動障害防止対策指針（2006年）/ 労働安全衛生法

現在の問題:
  ①振動工具の日振動ばく露量（A（8）値）を計算していない
  ②使用時間の記録がない（継続使用しすぎている可能性）
  ③3年以上使用のハンドグラインダーで、振動レベルが上昇している可能性

出力形式:
・「振動工具の使用記録シートのフォーマット
  （工具種別 / 使用者 / 使用時間 / 累積使用時間）」
・「日振動ばく露量（A（8）値）の計算方法の説明と計算例」
・「振動障害防止のための作業管理ルールの整理
  （連続使用の上限時間 / 休憩頻度 / 保護手袋の使用基準）」
・「振動障害（白ろう病）の早期発見のための健康診断項目と
  作業者への自覚症状のセルフチェックシート」
・「振動工具の定期点検・交換基準（振動が大きくなった工具の判断方法）」
```
