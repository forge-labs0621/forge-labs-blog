原材料費・エネルギーコストの上昇が続く中、「価格改定（値上げ）」を取引先に申し入れることは多くの製造業にとって重要な課題です。この申し入れは数字を根拠に丁寧に伝えることが大切で、「文章の構成・丁寧な言い回し」の部分をAIが手伝えます。

> **注意**: 価格改定の具体的な数値・交渉条件は、社内の原価計算・経営判断に基づいてください。AIは文章の整理・コメントの清書を補助するものです。

---

## 価格改定申請書（正式文書）を作るプロンプト

取引先への正式な価格改定依頼書を作ります。

```
以下の情報をもとに、取引先への価格改定申請書を作成してください。

【送付先】
宛先: （例：○○自動車工業 株式会社 購買部 ○○様）
自社: （例：○○製作所 代表取締役社長 ○○ / 営業部 ○○）

【価格改定の内容】
対象品番・品名: （例：品番A-1234 プレス加工品）
現行単価: （例：¥○○円）
申請単価: （例：¥○○円（○%増）」）
申請の適用希望時期: （例：2026年10月1日出荷分から）

【価格改定の根拠（コスト上昇の状況）】
材料費の状況:
  （例：SPCC（冷間圧延鋼板）の仕入れ単価が2024年比で○%上昇。
   弊社購入単価: 2024年 ¥○○/t → 現在 ¥○○/t（○%増））
エネルギーコスト:
  （例：電力単価が2023年比で○%上昇。月間電力コストが¥○○万円増加）
輸送費:
  （例：配送委託費が前年比○%上昇）

【対象品番での原価への影響】
（例：上記材料費上昇だけで、品番A-1234の製造原価が1個あたり
 約¥○○円（○%）増加している）

出力形式:
・正式なビジネス文書（会社の文書様式に相当する体裁）
・「謹啓 時下ますますご清祥のこととお喜び申し上げます」から始まる丁寧な書き出し
・価格改定の理由を段落で（コスト削減努力・限界に達した旨）
・価格改定の内容（現行→申請・適用時期）を表形式で
・「ご検討をお願い申し上げます」の締め
・返答期限の明示（例：○月○日まで）
・添付資料の案内（コスト根拠資料があれば別添する旨）
```

## 取引先への価格改定の事前相談メールを作るプロンプト

正式申請の前に、担当者レベルで相談する際のメールです。

```
以下の情報をもとに、価格改定の事前相談メールを作成してください。

【相談の背景】
（例：原材料費の上昇が止まらず、弊社の製造コストが大幅に上昇している。
 正式申請前に購買担当者と話し合いの機会を設けたい。）

【伝えたいこと】
- コスト上昇の状況（概要のみ）
- 正式な価格改定申請を近日中に提出したい
- まず話し合いの機会を設けていただけないか

出力形式:
・担当者レベルの率直なビジネスメール（軽めの文体でOK）
・感謝と関係性への配慮を最初に
・「詳細は添付（または別途送付）の資料でご説明します」の記載
・面談希望日の候補（プレースホルダー）
・急かすような表現を避け、協力関係を大切にするトーン
```

## コスト上昇の根拠資料サマリーを作るプロンプト

価格改定申請に添付する根拠資料のサマリー文章を作ります。

```
以下のコストデータをもとに、価格改定申請の根拠資料サマリーを
作成してください。

【弊社のコスト増加状況（根拠データ）】
材料費:
  - SPCC購入単価変化: 2024年1月 ¥○○/kg → 2026年7月 ¥○○/kg（○%増）
  - 品番A-1234 1個あたりの材料費: ¥○○ → ¥○○（+¥○○/個）

エネルギーコスト:
  - 月間電力料金: 2024年 ¥○○万 → 2026年 ¥○○万（○%増）
  - 製品1個あたり電力コスト換算: +¥○○/個

コスト削減の取り組み（自助努力の実績）:
  - （例：作業効率化で年間¥○○万のコスト削減を実施済み
    （人件費上昇分の一部を吸収））

出力形式:
・A4 1枚の「価格改定申請根拠書」
・現在の価格に対するコスト増加分の割合を明示
・「弊社として削減できる限界に達している」という事実をデータで示す
・グラフ挿入のプレースホルダー（「ここに材料費推移グラフを挿入」）
```

## 価格改定交渉後の確認書（合意内容）を作るプロンプト

取引先と価格改定で合意した後、内容を書面で確認する文書を作ります。

```
以下の合意内容をもとに、価格改定の確認書を作成してください。

【合意の内容】
協議日: （例：2026年8月20日）
協議出席者: 
  自社側: （例：営業部長 ○○・営業担当 ○○）
  客先側: （例：購買部長 ○○・購買担当 ○○）

合意した価格改定:
  品番A-1234: ¥○○/個 → ¥○○/個（○%増）
  品番B-5678: 今回は現行価格を維持（次回改定時に再協議）
  適用時期: 2026年10月1日出荷分から

付帯合意:
  （例：来年6月に原材料費の動向を確認し再協議することを確認）

出力形式:
・「価格改定合意書（確認書）」のフォーマット
・合意内容を表形式で
・双方の署名欄（自社・客先、プレースホルダー）
・付帯合意事項の記載
・「本書面をもって口頭での協議内容を確認します」の文言
```
