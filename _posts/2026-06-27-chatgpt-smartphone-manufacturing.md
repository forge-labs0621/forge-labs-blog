「AIを使ってみたいけど、工場のパソコンは作業の合間にすぐ使えない」「スマートフォンでも同じことができるの？」という現場担当者向けに、スマートフォンのChatGPTアプリを使ってAIを活用する方法をまとめます。

## スマートフォンでChatGPTを使う準備

### ステップ1: アプリのインストール

- **iPhone（iOS）**: App Storeで「ChatGPT」を検索 → OpenAIのアプリをインストール
- **Android**: Google Playで「ChatGPT」を検索 → OpenAIのアプリをインストール

無料版で使えます。アカウント登録（メールアドレスまたはGoogleアカウント）が必要です。

**注意:** 「ChatGPT」の名前が似た偽アプリがあります。OpenAI社が提供しているものを必ずインストールしてください（開発者欄が「OpenAI」になっているか確認）。

### ステップ2: アカウント登録

1. アプリを開く
2. 「Sign up」をタップ
3. メールアドレスまたはGoogleアカウントで登録
4. 無料プランのまま「Continue with free plan」を選ぶ

登録が完了すれば、すぐに使い始めることができます。

## スマートフォンで使いやすい製造業の活用場面

スマートフォンは入力が遅いため、「長い文章を書かせる」よりも「短いメモから整えてもらう」用途に向いています。

### 作業日報を書く（おすすめ）

帰り際や休憩中に、以下のような短いメモをChatGPTに送るだけで日報の下書きができます。

**入力の例（箇条書き）:**
```
6/27 旋盤A 9時から16時
45個完了 目標50個
刃物の摩耗早め 明日確認必要
安全問題なし
```

**ChatGPTへの指示:**
```
このメモをもとに作業日報を作ってください。
項目: 作業内容・完了数量・品質・安全・申し送り
```

これだけで整った日報の下書きが出てきます。

---

### ヒヤリハット報告書（音声入力と組み合わせる）

スマートフォンには音声入力機能があります。発生直後に声で状況を入力し、AIで文書化する方法が特に現場で使いやすいです。

**手順:**
1. スマートフォンのキーボードにある「マイクボタン」をタップ
2. 「さっき旋盤の刃物交換中に主軸が完全に止まっていなかった気がした。幸い事故にはならなかった」と声で入力
3. ChatGPTに「これをヒヤリハット報告書にしてください」と送る

```
以下の音声メモをヒヤリハット報告書の形式にしてください。
発生日時・場所・内容・危険の可能性・対策の欄で。

（音声で入力したテキストをここに貼る）
```

### 取引先へのメール文を作る

外出先でも取引先への連絡メールをすぐに作れます。

**例:**
```
外注先に「今月の纳期を3日遅らせてほしい」と伝えるメールを作ってください。
・理由: 設備トラブルで2日間生産が止まった
・対応: 残業で追いつける見込み
・文体: 丁寧、謝罪の気持ちを込めて
会社名・担当者名は「○○様」のプレースホルダーで
```

## スマートフォンならではの便利な使い方

### 短いメモから文章を作る

打ち込む文字数が少なくて済む入力方法を工夫すると、スマートフォンでも使いやすくなります。

- **箇条書きで入力**: 文章ではなく単語・短文の箇条書きをAIが整えてくれる
- **「簡潔に」を付ける**: 「A4 1枚以内で」「箇条書き3点で」など文量を指定する
- **前回の出力を使い回す**: 「前回と同じ形式で、今日の数量と日付だけ変えて」と指示する

### パソコンで使ったプロンプトを保存しておく

よく使うプロンプトをスマートフォンのメモアプリ（標準のメモ、Googleキープ等）に保存しておくと、毎回入力する手間を省けます。

保存の例:
```
[日報プロンプト]
このメモをもとに作業日報を作ってください。
項目: 作業内容・完了数量・品質・安全・申し送り

（上の行にメモを貼ってから送る）
```

## スマートフォンで使う際の注意事項

### 情報入力に注意する

スマートフォンで使うと、ついうっかり実名・会社名・詳細な数値を入力してしまうことがあります。

入力しないでください:
- 取引先の会社名・担当者名
- 具体的な品番や価格
- 顧客情報・個人情報

入力しても問題ない:
- 作業内容の概要（「旋盤で金属加工」程度）
- 一般的な数量・時間
- 「外注先A社」「品番B」のような仮称

### Wi-Fiのない環境では注意

工場の中でWi-Fiや4G/5Gの電波が届かない場合、アプリが動作しません。事前にプロンプトをメモアプリに準備しておき、電波のある場所（休憩室、駐車場等）でAIに送るという使い方が現実的です。

### バッテリー消費に注意

AIアプリは通常のテキスト入力より通信量・バッテリー消費が多いです。仕事用に使う場合は充電ケーブルを準備しておくことをおすすめします。

## まとめ

スマートフォンのChatGPTアプリは、以下の場面で特に役立ちます：

- 帰り際・休憩中に作業日報の下書きを作る
- 現場でヒヤリハットが発生した直後に音声で記録する
- 急なメール返信・連絡文を外出先で作る

「毎回ゼロから書いている書類がある」という方は、まず1種類だけスマートフォンで試してみてください。
