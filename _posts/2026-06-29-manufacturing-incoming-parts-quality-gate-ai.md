購入部品の品質不良は工程内不良よりも対応コストが大きく、サプライヤー別のリスク管理と受入検査の効率化が重要です。AIを使って受入検査計画・スキップロット判定・サプライヤー品質レポートを体系化できます。

---

## 受入検査計画・スキップロット判定書類を整理するプロンプト

```
以下の情報をもとに、受入検査計画書とスキップロット判定基準を
整理してください。

【購入部品の受入検査体制】
品目数: 120品目（月間受入件数: 380件）
検査員: 2名（検査が追いつかずスキップが増えている）
現状の課題: 
  ・全数全項目検査が原則だが、処理しきれずなし崩しでスキップ
  ・スキップの判断基準が検査員任せになっている
  ・サプライヤーの品質実績を受入判定に活かせていない

サプライヤーの品質実績（過去6ヶ月）:
  A社（主力サプライヤー）: 受入不良率0.03% → 安定
  B社（コスト重視）: 受入不良率1.82% → 問題あり
  C社（新規）: データ不足（受入3ロットのみ）
  D社（専門部品）: 受入不良率0.15% → 概ね安定

出力形式:
・「サプライヤー別スキップロット判定基準（Aランク/Bランク/Cランク）の設計」
  （ランク基準: 不良率・クレーム履歴・ISO認証有無・直近ロット実績）
・「受入検査計画書のフォーマット
  （品目別: 検査項目 / 検査方法 / サンプリング数 / 判定基準 / スキップ可否）」
・「B社向け受入強化措置の書類（全数検査復帰・改善要求書）」
・「C社（新規サプライヤー）の初期ロット承認プロセスと承認書フォーマット」
・「受入検査工数削減試算（スキップロット導入後の検査工数削減効果）」
```

## サプライヤー別品質報告書を整理するプロンプト

```
購入部品のサプライヤー別品質報告書（月次）のフォーマットと
不良品が発生した場合の仕入先対応書類を整理してください。

【今月の受入品質状況】
B社から納入された部品で重大不良が発生した。

不良内容:
  品目: 金属プレス部品（型番XX-2847）
  受入日: 2026-06-15
  受入数量: 500個
  検査で発見された不良: 穴径オーバーサイズ（規格: φ10.0±0.05mm）
  実測: φ10.18mm（全数測定で300/500個が不良）
  不良率: 60%
  影響: 工程内停滞・後工程への流出0個（早期発見で食い止め）

サプライヤーへの対応:
  ・全数返品を指示
  ・改善要求書の発行
  ・是正回答期限: 2週間

出力形式:
・「月次受入品質報告書（サプライヤー別不良率・ランク・特記事項）」
・「B社への不良品返品・改善要求書（不良内容 / 影響範囲 / 要求事項 / 回答期限）」
・「B社の改善回答確認書フォーマット（是正内容 / 再発防止策 / 有効性確認方法）」
・「B社のサプライヤーランク引き下げ判断基準と通知書フォーマット」
・「購入部品不良による損害金額の試算（検査工数・返品物流・ライン停止損失）」
```
