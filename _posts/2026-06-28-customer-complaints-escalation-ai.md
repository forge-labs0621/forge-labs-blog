顧客からクレームが入ったとき、迅速かつ誠実な対応が信頼維持の鍵です。AIを使って初動連絡・謝罪文・原因分析報告書・再発防止策を素早く整理することで、顧客との関係を守れます。

---

## クレーム初動対応・謝罪文を整理するプロンプト

発生直後の24時間以内の連絡文を作ります。

```
以下の情報をもとに、顧客クレームへの
初動対応の連絡文と謝罪メールを整理してください。

【クレームの概要】
発生日時: 2026年6月25日（木）午後2時頃
顧客からの連絡内容（電話）:
  「御社から納品されたプレスブラケット（品番P-205・
   ロットL2506-003・500個）のうち、当社の組み立てラインで
   15個の穴径が規格外（φ8.5mm以上で規格上限φ8.2mm超過）
   であることが判明した。品質管理は何をしているのか」

被害状況（顧客の報告）:
  ・問題の15個は既に組み立てラインに投入済み
  ・組み立て完了品3台への影響を確認中
  ・ライン停止: 現時点ではしていないが、残りの485個の
    使用可否を至急確認してほしい

当社の第一報の対応状況:
  ・電話を受けたのは営業担当・山本（15時頃）
  ・即座に品質部・工場長に報告済み
  ・出荷前の検査記録を確認中（調査に1〜2時間かかる見込み）

出力形式:
・「電話での初動対応スクリプト
  （謝罪→事実確認→当社での対応方針の伝え方→次の連絡タイミングの約束）」
・「1〜2時間後に送る第一報メールの文案
  （謝罪・現状確認・当社の暫定対応・次の報告時刻の明示）」
・「社内の緊急連絡フロー（誰に・いつまでに・何を報告するか）」
・「顧客の「残り485個の使用可否」への回答までのアクションプラン」
・「クレーム対応記録の開始（何を・いつ・誰が対応したかの記録フォーマット）」
```

## クレーム原因分析・再発防止報告書プロンプト

顧客に提出する正式な報告書を作成します。

```
以下の調査結果をもとに、顧客へ提出する
クレーム報告書（原因分析・再発防止策）を整理してください。

【調査結果（クレーム発生から48時間後）】
問題の内容:
  品番P-205の穴径（φ8.0+0.2/-0mm規格）が
  φ8.35〜φ8.48mmと規格上限を超過していた

原因調査の結果（なぜなぜ分析）:
  直接原因: 穴明け用ドリル（φ8.0）が摩耗していた
  なぜ①: ドリルの交換基準（500個/本）を超えて使用していた
  なぜ②: ドリルの交換記録が正しく管理されていなかった
  なぜ③: 交換記録は自己申告制だったが、確認する仕組みがなかった
  根本原因: 消耗工具の使用量管理が個人任せで、
           管理基準が現場で形骸化していた

対象ロット範囲の確認:
  L2506-003（500個）のみ影響。他ロットは確認済みで問題なし。
  L2506-003の出荷前検査: 穴径は抜き取り5個（全て合格）
  → 抜き取り検査では今回の不良を捉えられなかった

再発防止策（決定事項）:
  即時: ①問題ロット全数の穴径全検（当社費用で実施・完了）
       ②他の消耗工具（ドリル・エンドミル）の全数点検・交換
  短期（1ヶ月以内）: ③工具使用量管理システムの導入
    （ドリル本数×使用個数を電子管理・交換タイミングをアラート）
  中期（3ヶ月以内）: ④穴径の検査を全数測定に変更（抜き取り廃止）

出力形式:
・「顧客提出用クレーム報告書の文案（A4・2〜3ページ）」
  （件名・発生状況・原因分析・対策・恒久対策・
  今後の品質保証の取り組み・締めの言葉）
・「報告書の提出時の添付資料リスト（再発防止策の証拠書類）」
・「口頭報告（顧客訪問）時の説明スクリプト（15分版）」
・「顧客から「補償はどうなるか」と聞かれた場合の答え方
  （ライン停止・手直し費用の扱いについて）」
・「社内クレーム記録の最終版の記入（管理台帳への反映）」
```

## 重大クレームのエスカレーション手順プロンプト

経営層が動くべきクレームの基準と動き方を整理します。

```
以下の情報をもとに、製造業の
クレームエスカレーション基準と手順を整理してください。

【当社の現状の課題】
  ・軽微なクレームも重大なクレームも、同じ対応フローで動いている
  ・「これは社長に報告すべきか」の判断が担当者に委ねられている
  ・過去に重大クレームの報告が遅れて、顧客から
    「なぜもっと早く社長が来なかったのか」と叱責されたことがある

【エスカレーション基準に含めたい要素】
  ①クレームの深刻度（人身事故・生産停止・リコールの可能性）
  ②財務的インパクト（推定賠償額・代替品費用）
  ③顧客への影響（ライン停止・最終製品の品質への影響）
  ④メディア・法的リスク（報道・訴訟の可能性）
  ⑤複数顧客への影響（同一部品を複数顧客に納入している場合）

出力形式:
・「クレームのエスカレーションレベルの定義（3段階）」
  Level 1: 担当者・品質部長で対応
  Level 2: 工場長・営業部長が関与
  Level 3: 社長・取締役が関与（顧客訪問含む）
・「各レベルの具体的な判断基準（例: 賠償額¥●円以上はLevel2等）」
・「エスカレーション連絡のフロー（担当者→部長→工場長→社長の
  誰に・いつ・何を報告するかの一覧）」
・「Level 3クレーム時の社長が最初にすべき5つのアクション」
・「クレーム対応状況の社内共有フォーマット（日次報告書）」
```
