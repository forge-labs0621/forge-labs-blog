製造業で製品の欠陥・安全上の問題が発覚した場合、迅速なリコール対応・製品回収が必要です。AIを使って告知文・行政報告書・原因究明報告・再発防止計画の書類を整理できます。

---

## リコール告知文・消費者向け案内を作るプロンプト

問題が発覚した製品の告知文と回収案内を作ります。

```
以下の情報をもとに、製品リコール・回収に関する
消費者向け告知文と対応案内を作成してください。

【製品と問題の概要】
製品名: （例: 電動工具用バッテリーパック / 型番: ○○-○○）
販売期間: （例: ○○年○月〜○○年○月 / 販売台数: 約○○台）
問題の内容: （例: バッテリーセルの膨張・発煙・発火の恐れがある
  事象が○件報告されている。設計上の問題が原因と判明）
確認された事故件数: ○件（うち火災: ○件、けが: ○件、財産損害: ○件）
行政への報告:
  ・消費生活用製品安全法に基づく重大製品事故報告済み
  ・消費者庁・経産省に報告、リコールを実施する旨を届出済み

【回収の対応方針】
対象者への通知方法: 販売店・販売記録からのDM + ウェブ告知 + PR
返金・交換: 無償交換または全額返金（選択制）
問い合わせ先: 専用コールセンター設置（平日○〜○時）
使用中止の依頼: 即時使用を中止し、充電しないよう告知

出力形式:
・「消費者向けリコール告知文（ウェブ・プレスリリース用）」
  （誠実・明確・行動促進を重視した文章）
・「お客様への個別通知レター（DM用）」
・「専用コールセンターのQ&A（よくある質問と回答）」10問
・「販売店・流通業者向けの回収協力依頼文」
・「社内向けリコール対応マニュアル（担当部門・作業フロー）の骨子」
```

## 行政報告書・原因究明報告書を作るプロンプト

消費者庁・経済産業省への報告書類と社内の原因究明書類を整理します。

```
以下の情報をもとに、製品事故の行政報告書類と
原因究明報告書を整理してください。

【製品事故の詳細（例）】
事故発生日時: ○年○月○日（最初の報告）
事故内容: バッテリーパックが充電中に発煙（その後発火）
被害: 財産損害（充電器周辺が焦げた / 外壁一部損傷）
被害者: 個人購入者（性別・年齢: 確認中）
事故発生場所: 屋内（一般住宅・充電スペース）
自社への報告日: ○年○月○日（販売店からの連絡）

【原因究明の結果（例）】
根本原因: セルメーカーからの特定ロット（○〜○）のバッテリーセルに
  過充電保護回路の設定値の不具合があり、過充電状態になると
  熱暴走が発生するリスクがあることが判明
対象ロット: ○○〜○○（出荷数: ○○個 / 回収済み: ○個 / 未回収: ○個）

出力形式:
・「消費者庁への重大製品事故報告書の記載項目と記入例」
  （消費生活用製品安全法 第35条に基づく報告）
・「原因究明報告書の構成（事故概要・原因分析・是正措置・
  再発防止策・今後の対応）」
・「なぜなぜ分析または特性要因図を使った原因分析の記録フォーマット」
・「対象ロットの特定と未回収製品の追跡管理フォーマット」
・「記者会見・メディア対応時の想定Q&A」5〜10問
```

## 再発防止計画書と品質保証体制の見直し書類を作るプロンプト

リコール後の品質保証体制の強化計画を整理します。

```
以下の情報をもとに、リコール後の再発防止計画書と
品質保証体制強化計画を作成してください。

【再発防止の取り組み内容（例）】
① 仕入先管理の強化:
  ・バッテリーセルの仕入先評価基準を改訂（受入試験項目を追加）
  ・特定の安全試験（過充電試験・短絡試験）を受入検査に追加
  ・仕入先の生産工程の変更通知義務を契約書に明記

② 設計プロセスの変更:
  ・保護回路の設定値の計算プロセスにダブルチェック（第三者確認）を追加
  ・試作評価に第三者認証機関による安全試験を義務化

③ 市場でのモニタリング強化:
  ・顧客からの問い合わせに「バッテリー関連」キーワードを
    自動フラグ付けして品質部門に通知

【行政・顧客への対応の締めくくり】
・消費者庁への最終報告書の提出
・再発防止策の実施状況の定期報告（○か月後）

出力形式:
・「再発防止計画書」（行動計画・担当・期限の一覧表形式）
・「品質保証体制強化計画の骨子（経営会議向けレポート）」
・「取引先・購入者への再発防止策の案内文」
・「製品安全に関する社内ルール（変更管理・仕入先管理）の見直し箇所」
・「PL保険の加入確認・保険で対応できる費用の整理」のコメント
```
