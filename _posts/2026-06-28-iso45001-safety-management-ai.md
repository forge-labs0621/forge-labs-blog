ISO 45001は労働安全衛生に関する国際規格で、OHSAS 18001の後継規格です。製造業では「ゼロ災害」「労災を出さない経営」のために取得・維持する企業が増えています。AIを使って書類整備・危険源特定・審査準備を効率化できます。

---

## 危険源の特定・リスクアセスメントを整理するプロンプト

ISO 45001の中核要件である「危険源の特定とリスクアセスメント」を整理します。

```
以下の情報をもとに、製造現場の危険源特定・リスクアセスメントの
記録を整理してください。

【対象の職場・作業】
職場: （例: プレス加工工場 / 対象作業: プレス機運転・金型交換）
作業環境: （例: 夜間作業あり / 高騰する環境温度 / 騒音あり）
作業者の特性: （例: 高齢者が全体の30% / 外国人技能実習生5名）

【既に特定している危険源の例】
・プレス機の可動部への挟まれ・巻き込まれ
・金型搬送時の落下・つまずき転倒
・金属粉じんの吸入（グラインダー作業）
・夜間作業中の眠気による作業エラー
・フォークリフトと歩行者の接触

【各危険源のリスク評価（現状）】
プレス機挟まれ: 発生頻度: 低（インターロックあり）/ 重篤度: 最大
金型搬送転倒: 発生頻度: 中 / 重篤度: 中（骨折リスク）
金属粉じん: 発生頻度: 高（日常的）/ 重篤度: 長期影響あり

出力形式:
・リスクアセスメント表のフォーマット（危険源・発生頻度・重篤度・
  リスクレベル・現在の管理策・追加対策・残留リスク評価）
・「リスクの優先順位付け（高リスクから対処する順位）」
・「労働安全衛生目標（年間）の設定案」（数値目標つき）
・「作業者参加型の危険源特定（ToolBox Meeting活用）の進め方」
・ISO 45001審査で「リスクアセスメントはどう実施していますか？」
  と聞かれた際の回答骨子
```

## 安全衛生目標・年間活動計画を作るプロンプト

労働安全衛生目標と年間の取り組み計画を文書化します。

```
以下の情報をもとに、ISO 45001に準拠した
労働安全衛生目標と年間活動計画書を作成してください。

【自社の安全衛生実績（過去1年）】
労働災害発生件数: （例: 休業災害1件（手の挟まれ・休業5日）
  不休業災害3件 / ヒヤリハット報告: 42件）
強度率: ○（業種平均との比較）
安全衛生委員会: 月1回実施・議事録保管
教育実績: 新入社員教育○回、定期安全教育○回

【今年度の課題と改善テーマ】
（例:
  課題1: ヒヤリハット報告が減少傾向（自主的な報告文化の醸成が必要）
  課題2: 高年齢作業者の転倒・転落リスク対策が不十分
  課題3: 夜勤帯のリスクが昼間帯より高い（要員・点検頻度の格差）
）

出力形式:
・「労働安全衛生目標（年間）」の文書（定量目標・改善テーマ・
  担当・期限を含む表形式）
・「月次の安全衛生活動計画」（KY活動・設備点検・委員会・
  教育訓練のスケジュール骨子）
・「目標達成状況の月次モニタリングシート」
・「経営会議向けの安全衛生報告フォーマット」
・ISO 45001の「安全衛生目標設定に必要な要件の解説（箇条6.2）」
```

## ISO 45001の内部監査・マネジメントレビュー記録を作るプロンプト

審査準備の証跡となる内部監査報告書とマネジメントレビュー記録を整理します。

```
以下の情報をもとに、ISO 45001の内部監査報告書と
マネジメントレビューの記録を整理してください。

【内部監査の実施状況】
対象: （例: 生産部門の安全衛生活動 / 安全巡視・教育訓練・
  危険源特定・緊急事態への準備）
監査員: （例: 安全担当者（内部認定資格者） / 製造部長）
前回指摘事項:
  ・不適合: 緊急避難手順書の最新版が一部作業エリアに掲示されていない
  ・観察事項: 転落防止手すりの点検記録が3か月分空白

【マネジメントレビューへのインプット情報】
・前年度の安全衛生目標の達成状況
・労働災害・ヒヤリハットの傾向分析
・法令要求事項の変更（改正労働安全衛生法の対応状況）
・前回の第三者審査・内部監査での指摘事項の是正状況
・労働者からのフィードバック（安全委員会での意見等）

出力形式:
・内部監査報告書のフォーマット（不適合・観察事項・是正処置の記録）
・「是正処置の有効性確認フォーマット」
・マネジメントレビューの議題リストと記録フォーマット
・「ISO 45001審査で確認される証跡一覧（証拠の準備チェックリスト）」
・「審査員から高く評価される取り組みの例」3点
```
