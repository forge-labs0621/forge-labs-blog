仕入先・外注先の品質管理レベルを直接確認する第二者監査は、供給リスク管理の要です。AIを使って監査計画・チェックリスト・報告書・改善フォローアップを体系化できます。

---

## 仕入先監査計画書・チェックリストを整理するプロンプト

```
以下の情報をもとに、仕入先監査（第二者監査）の計画書と
チェックリストを整理してください。

【今回の監査概要】
監査対象: 株式会社〇〇機工（精密切削加工の外注先）
監査目的: 
  ①品質管理体制の確認（ISO 9001取得なし）
  ②受入不良率が直近3ヶ月で0.8%→1.5%→2.3%と悪化中
  ③新規品目の外注委託前の事前確認
監査日: 2026-07-10（終日）
監査員: 品質部長＋購買担当（2名）

監査の重点ポイント:
  ・工程管理の実態（作業標準書の整備・遵守状況）
  ・設備管理と工具管理の状況
  ・検査体制（測定器・検査員の技能）
  ・不良品発生時の処置方法

出力形式:
・「監査実施計画書（目的 / 日程 / 監査員 / 対象工程 / 準備書類一覧）」
・「監査チェックリスト（カテゴリ別・100項目・5段階評価）
  カテゴリ: 品質管理体制 / 工程管理 / 設備保全 / 検査・測定 /
  文書管理 / 不適合品管理 / 人材育成」
・「事前提出依頼書（監査前に提出してもらう書類一覧と送付依頼文）」
・「監査当日のタイムスケジュール（挨拶→工場視察→書類確認→評価→フィードバック）」
```

## 監査報告書・改善要求書を整理するプロンプト

```
仕入先監査を実施した結果をもとに、監査報告書と
改善要求書を整理してください。

【監査結果の概要（〇〇機工 2026-07-10実施）】
総合評価スコア: 61点 / 100点（合格基準: 70点以上）

主な発見事項（不適合）:
  重大不適合（Aランク）:
    ・測定器の校正期限が切れたまま使用中（5台中3台）
    ・不適合品の識別・隔離管理ができていない（合否混在保管）
  
  軽微不適合（Bランク）:
    ・作業標準書が最新でない（前回改訂から3年間更新なし）
    ・加工条件の記録が口頭伝達のみで帳票化されていない
  
  観察事項（Cランク・改善推奨）:
    ・5S活動が停滞気味
    ・検査員の技能認定の仕組みがない

総合判断: 現状では新規品目の外注委託は保留
          既存品目は継続するが改善期限を設定

出力形式:
・「監査報告書（サマリー / 発見事項詳細 / 総合評価 / 判定結果）」
・「改善要求書（重大不適合2件: 是正期限30日・改善計画書提出要求）」
・「軽微不適合の改善要望書（期限60日・改善内容報告要求）」
・「改善フォローアップスケジュール（30日後の是正確認 / 3ヶ月後の再監査計画）」
・「社内向け報告書（調達リスクの評価と今後の対応方針）」
```
