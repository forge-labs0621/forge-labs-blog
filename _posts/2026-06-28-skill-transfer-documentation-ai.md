製造業では、ベテランが退職する前に技術・技能を次世代に伝える「技能伝承」が重要な経営課題になっています。「長年の勘でわかる」「コツは教えにくい」という熟練技術も、AIが整理の補助をすることで文書化しやすくなります。

---

## 熟練技術インタビューの整理プロンプト

ベテランへのヒアリング内容を作業マニュアルに整理します。

```
以下のベテラン作業者へのインタビューメモをもとに、
後継者向けの作業マニュアル（技能伝承シート）を作成してください。

【作業の概要】
作業名: （例：旋盤加工 / 切削条件の調整）
ベテランの経験: （例：旋盤歴35年、現在65歳）

【インタビューで聞いた「コツ」「勘」（メモ）】
音・振動の聞き方:
  （例：「材料を削り始めたとき、刃物の当たる音が低いと
   切り込みすぎ。軽い金属音が正常。振動が大きくなったら
   切削速度を落とすサイン」）
工具の判断:
  （例：「バイトの刃先を光に当てて見ると、光の反射が
   均一でなくなったら交換タイミング。数字で言うと難しいが
   加工面に光沢がなくなったら確実に交換」）
不良が起きる前のサイン:
  （例：「加工中に切り粉の色が変わってきたら危険。通常は
   銀色だが、青っぽくなってきたら温度上昇のサイン」）

出力形式:
・「状況・観察ポイント → 判断基準 → 対応行動」の3列で表形式
・ベテランの言葉を活かしながら、初心者にもわかる言葉に整理
・「経験年数の目安（この判断ができるようになるまでの期間）」を各項目に
・「よくある失敗と回避方法」のコラム（インタビューで出てきた内容から）
```

## 「経験則」を手順書に変換するプロンプト

ベテランの「感覚的な判断」を数値や確認手順に変換します。

```
以下の「経験則」を、後継者が再現できる「手順・確認方法」に
変換してください。

【変換したい経験則】
（例：
1. 「研削のときに、砥石が焼けた臭いがしたら焼けすぎ」
   → 具体的な確認方法に変換してほしい

2. 「板金を折り曲げるとき、材料が少し温かい感じの日は
   割れやすい。寒い日は慎重に曲げる」
   → 温度管理として具体化してほしい

3. 「仕上げ磨きは力を均等に、リズムよく。腕の疲れを感じたら
   品質が下がっているサイン」
   → 品質確認の手順に変換してほしい
）

出力形式:
・各経験則について「変換前（ベテランの言葉）」→「変換後（手順・基準）」の対比
・変換後は「具体的な観察項目・確認タイミング・合否の基準」を含める
・「感覚に依存せざるを得ない部分は何か（AI変換の限界）」も明記する
・後継者が最初の6か月で習得できる内容と、それ以上かかる内容を分ける
```

## 技能伝承計画書を作るプロンプト

ベテランから後継者への技術移転の計画書を作ります。

```
以下の情報をもとに、技能伝承計画書を作成してください。

【基本情報】
伝承者（ベテラン）: （例：○○ ○○（旋盤担当・65歳・勤続35年）
  退職予定: 2027年3月）
後継者: （例：○○ ○○（30歳・旋盤経験3年））
伝承期間: （例：2026年10月〜2027年3月（6か月））

【伝承が必要な技術・スキル】
高難度技術（1〜2年以上の経験が必要なもの）:
  （例：薄肉部品の加工（変形させずに削る技術）/ 難削材の切削条件判断）
中難度技術（6か月〜1年で習得可能なもの）:
  （例：工具の選定・交換判断 / 切削条件の最適化）
基礎技術（3〜6か月で習得可能なもの）:
  （例：段取り・チャック調整 / 通常品の加工）

出力形式:
・技能伝承計画書のフォーマット（月別スケジュール・習得目標・確認方法）
・月ごとの「この月で習得すること・評価基準」の表
・「伝承者の関与度（毎日同行→週2回→週1回→相談対応）」の段階設定
・最終確認（退職前の技術評価）の実施計画
```

## 技術ノウハウのQ&A集を作るプロンプト

現場でよくある質問と答えを整理したQ&A集を作ります。

```
以下のベテランからよく聞く「後継者からの質問とその回答」を
整理してQ&A集にまとめてください。

【よくある質問と回答のメモ】
Q: 「切削速度はどうやって決めるの？」
A: 「材料によって違う。SUS系は遅め、快削鋼は速めでOK。
   まず教科書通りの推奨値から始めて、加工面の光沢と
   切り粉の形状を見ながら調整する」

Q: 「工具の寿命はいつ判断するの？」
A: 「加工面が荒れてきたとき、切り粉の色が変わってきたとき、
   異音がしてきたとき。どれかが出たら交換。待ちすぎると
   製品に傷がつく」

Q: 「バイトが折れる前に気づけるの？」
A: 「折れる前に必ず振動が増える。ビビリが出てきたら要注意。
   振動が大きくなったら一度止めてチェックする」

出力形式:
・「Q / A（要点） / 実践上の注意点」の表形式
・各QAに「習熟度（初級・中級・上級）」を付ける
・「この質問が出てきたらこのQ&Aで補足説明する」という使い方ガイドを冒頭に
```
