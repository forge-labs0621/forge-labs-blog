製造業の経営者にとって、メインバンクへの業績説明と良好な関係維持は資金調達の根幹です。AIを使って試算表の説明資料・資金繰り計画・事業報告書を分かりやすく整理できます。

---

## 銀行向け業績説明資料を整理するプロンプト

試算表をもとに銀行が理解しやすい説明資料を作ります。

```
以下の財務データをもとに、メインバンクへの
業績説明資料（上期・6ヶ月分）を整理してください。

【当社の基本情報】
業種: 金属プレス加工業（受託製造）
従業員: 45名 / 期中売上高（上期）: ¥185M（前年同期¥172M）

【試算表の主要数値（上期6ヶ月）】
売上高: ¥185,000,000（前年比+7.6%）
売上原価: ¥138,000,000（売上比74.6%・前年76.3%）
売上総利益: ¥47,000,000（粗利率25.4%・前年23.7%）
販管費: ¥31,500,000（前年比+3.2%）
営業利益: ¥15,500,000（前年同期比+34.8%）
経常利益: ¥14,800,000（前年同期比+33.3%）

【数値変化の主な理由】
  ・売上増加: 主要顧客X社からの受注増（新規部品の量産移行）
  ・粗利改善: 材料費の変動に対する価格転嫁が進んだ
  ・販管費増加: 採用強化（中途採用2名）と研修費増

資金繰り状況:
  現金・預金（6月末）: ¥28,000,000
  借入残高: ¥65,000,000（長期借入¥55M・短期¥10M）
  次回返済（8月）: 長期借入の元金¥2,000,000

出力形式:
・「銀行への業績説明資料の構成（A4・3〜4ページ）」
  1ページ: 会社概要と上期ハイライト
  2ページ: 損益の推移と改善要因の説明
  3ページ: 資金繰り状況と下期見通し
・「粗利率改善（+1.7%）の説明（銀行担当者が理解しやすい言葉で）」
・「下期（7〜12月）の売上・利益予測の根拠の説明の仕方」
・「銀行から来そうな質問と回答案
  （「主要顧客X社への依存度は？」「設備の老朽化は？」等）」
・「次回融資（設備更新のための¥15M）に向けた
  今のうちから信頼を積む話の進め方」
```

## 資金繰り計画書を整理するプロンプト

3〜6ヶ月先の資金を見える化します。

```
以下の情報をもとに、今後6ヶ月の
資金繰り計画書を整理してください。

【資金繰りの前提条件】
現在の現金・預金（7月初）: ¥28,000,000

売上・回収の予定（月次）:
  7月: 売上¥32M → 8月末回収（売掛金サイト30日）
  8月: 売上¥30M → 9月末回収
  9月: 売上¥35M（9月は繁忙期） → 10月末回収
  10〜12月: 各月¥38M（年度末に向けて増加）→ 翌月末回収

主要な支払い予定:
  毎月: 材料費¥14M・人件費¥12M・その他経費¥5M（合計¥31M/月）
  8月: 長期借入返済¥2M・ボーナス支払¥8M（夏季賞与）
  10月: 新型コンプレッサー購入代金¥6M（設備更新）
  12月: 長期借入返済¥2M・ボーナス支払¥10M（冬季賞与）

出力形式:
・「月次資金繰り表（月初残高→収入→支出→月末残高の6ヶ月分）」
・「資金不足リスクが発生する月の特定と必要な対策額」
  （8月のボーナス+借入返済が重なる時期が要注意）
・「資金繰りの改善策（3案）」
  ①売掛金の回収サイト短縮交渉
  ②在庫の適正化による資金圧縮
  ③短期融資（当座貸越・つなぎ融資）の事前設定
・「銀行に提出する資金繰り計画書のフォーマット」
・「当座貸越の設定を銀行に申し込む際の説明の仕方」
```

## 銀行担当者との関係管理プロンプト

日常的なバンキング関係を強化します。

```
以下の状況をもとに、メインバンクとの関係を
強化するための活動計画を整理してください。

【当社とメインバンクの現状】
メインバンク: ○○信用金庫（取引歴20年）
担当者: 鈴木支店長代理（40歳・就任2年目）
現在の取引: 長期借入¥55M・短期¥10M・口座・給与振込

良い関係を維持できている点:
  ・毎月試算表を提出（期中に情報提供ができている）
  ・過去の返済は一度も遅延なし
  ・会社見学に担当者を招待したことがある

課題:
  ・下期に設備更新で¥15Mの新規融資を申込む予定
  ・担当者が「最近の業況は？」と聞いてくることが増えた
    （不安を持っているかもしれない）

出力形式:
・「メインバンクとの良好な関係を維持するための
  年間コミュニケーション計画（月次・四半期・年次の接触ポイント）」
・「毎月の試算表提出時に添付する「1枚サマリー」の
  フォーマット（数字+コメント）」
・「新規融資申込の3ヶ月前から始める準備（書類・訪問・
  タイミングの計画）」
・「担当者の不安を払拭するための説明のポイント
  （現在業況が好調であることを具体的な数字で伝える）」
・「サブバンク（第2銀行）との取引を始めることの
  メリット・デメリットの整理」
```
