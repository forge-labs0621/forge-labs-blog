人手不足と品質安定化を両立するために、産業用ロボット・FA設備の導入を検討する製造業が増えています。AIを使って導入検討書・ROI分析・補助金申請・稟議書を効率化できます。

---

## ロボット導入の候補工程選定プロンプト

自社のどの工程にロボットが向いているか評価します。

```
以下の情報をもとに、産業用ロボット導入に
適した工程の評価と優先順位を整理してください。

【当社の工程一覧と現状の課題】
工程名 / 作業内容 / 月次作業時間 / 作業難度 / 現状課題:
  ①プレス後バリ取り / 手工具でのバリ除去 / 320時間 / 低（単純反復）/
    作業者の疲労・手首への負担が大きく離職者が出ている
  ②部品洗浄・乾燥 / 洗浄機への投入・取り出し / 160時間 / 低（繰り返し）/
    深夜でも稼働したいが人員確保が困難
  ③ねじ締め（製品組立） / 電動ドライバーでのネジ締め4本 / 240時間 / 低（反復）/
    締め忘れの品質トラブルが年3〜4件発生
  ④外観検査（目視） / 目視での傷・変形の検査 / 200時間 / 中（判断が必要）/
    判定基準が人によってばらつく問題
  ⑤溶接（アーク溶接） / 治具にセットして溶接 / 280時間 / 高（熟練必要）/
    熟練者不足・溶接品質のばらつき

ロボット導入の前提条件:
  ・予算: 初期投資¥10M以内（1台目）
  ・目標: 3〜5年でROI回収

出力形式:
・「各工程のロボット化適性スコア（繰り返し性・安全性・
  ROI期待・技術的難易度の4軸評価）」
・「ロボット化の優先順位（1〜5位）と理由」
・「各工程に適したロボットの種類の概要
  （垂直多関節・スカラ・協働ロボット・画像検査AIの使い分け）」
・「最優先工程（ねじ締め/バリ取り等）の概算投資費用と期待効果」
・「ロボット導入の一般的なスケジュール（検討→選定→
  設置→試運転→量産移行まで）」
```

## ロボット導入のROI分析プロンプト

投資対効果を試算します。

```
以下の情報をもとに、協働ロボット（コボット）の
ねじ締め工程への導入ROI分析を整理してください。

【導入対象工程】
工程: 製品組立ラインのねじ締め（M4ねじ×4本/製品）
現在の作業体制: 専任作業者1名（月160時間/月・時給¥1,800）
現在の生産量: 2,000個/月
現在の不良（締め忘れ・トルク不足）: 年12件
  → 手直し費用: ¥3,000/件 × 12件 = ¥36,000/年

【導入候補ロボットの仕様（メーカー見積もりより）】
機種: 協働ロボット（UR5e + ねじ締めユニット）
導入費用: 本体¥4,200,000 + 治具・SIer費¥1,800,000 = 合計¥6,000,000
設置スペース: 1m × 1m（作業者と共存可能）
サイクルタイム: 15秒/個（現在の作業者: 18秒/個）
稼働時間: 1日16時間稼働可能（2交代分）
保守費用: ¥150,000/年
耐用年数: 10〜15年

【ロボット導入後の想定】
  ・作業者は別工程に異動（削減ではなく配置転換）
  ・不良ゼロを目標（トルク管理の自動化）
  ・生産量増加可能: 最大2,600個/月（30%増）

出力形式:
・「現在のコスト（人件費+不良コスト）の計算」
・「ロボット導入後のコスト（保守費+電気代）の計算」
・「年間コスト削減額とROI計算（回収年数・IRR）」
・「生産能力増加の収益貢献試算」
・「ロボット導入の費用対効果まとめ（経営者向け1ページ資料の骨子）」
```

## ロボット導入補助金・稟議書作成プロンプト

資金調達と社内承認を取り付けます。

```
以下の情報をもとに、ロボット導入のための
補助金申請書と社内稟議書を整理してください。

【利用を検討している補助金】
補助金名: ものづくり補助金（省力化（オーダーメイド）枠）
補助率: 1/2（中小企業）
補助上限: ¥750万（従業員20人以下）

当社の従業員数: 25名
→ 補助上限: ¥1,000万（従業員21〜50名の区分）

導入総費用: ¥6,000,000
補助金申請額（上限内）: ¥3,000,000（補助率1/2）
自己負担: ¥3,000,000

【補助金申請に必要な事業計画書の骨子】
  ・現在の課題: 人手不足・品質不安定
  ・解決策: 協働ロボット導入
  ・期待される成果: 生産性20%向上・不良率ゼロ
  ・賃上げ計画: 配置転換した作業者の賃金維持

出力形式:
・「ものづくり補助金の事業計画書の骨子
  （1.事業の背景、2.解決策、3.期待効果、4.賃上げ計画）」
・「補助金申請に向けた準備スケジュール
  （公募期間の確認→書類準備→申請→採択→設備発注の流れ）」
・「社内稟議書の骨子
  （目的・現状課題・提案内容・費用・ROI・リスクと対策・承認要求）」
・「ロボットメーカー・SIer選定時のRFP（提案依頼書）に含める
  評価項目（価格・技術力・アフターサポート・実績）」
・「導入後の効果測定計画（何を、いつ、誰が測定するか）」
```
