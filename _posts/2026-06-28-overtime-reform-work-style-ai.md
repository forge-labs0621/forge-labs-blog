2024年4月から製造業にも時間外労働の上限規制が完全適用されました（月45時間・年360時間、特例でも月100時間未満・年720時間）。AIを使って36協定の理解・社内ルール整備・残業管理の書類を整えることができます。

---

## 36協定（時間外・休日労働に関する協定届）の概要整理プロンプト

36協定の内容と自社の現状確認を整理します。

```
以下の情報をもとに、36協定（時間外・休日労働に関する協定届）の
内容確認と社内管理書類を整理してください。

【自社の36協定の現状】
締結状況: （例: 毎年更新し、労働基準監督署に届出済み）
現在の協定内容:
  ・通常時の時間外上限: 月45時間・年360時間
  ・特別条項（繁忙期）: 月80時間・年720時間
    ※ 特別条項を使用する月数: 年6か月以内
  
現在の課題:
  ・繁忙期に月80時間を超えかねない部署がある
  ・36協定の内容を一般従業員が把握していない
  ・勤務管理システムがなく、月末にエクセルで集計している

出力形式:
・「36協定で定める時間外労働の上限と法的ペナルティの整理」
  （何時間超えると何の罰則か）
・「繁忙期に特別条項を発動するための手順と記録フォーマット」
・「従業員向けの『36協定とは何か』の説明文（A4 1枚）」
・「毎月の時間外労働管理シート（部署別・従業員別の月次集計）」
・「年間の時間外労働総量管理の方法（年720時間を超えないための
  月別の使用状況見える化）」
```

## 残業削減・業務効率化計画書を作るプロンプト

長時間労働の削減に向けた社内計画書と取り組み記録を整理します。

```
以下の情報をもとに、時間外労働削減に向けた
業務改善計画書を作成してください。

【現状の課題】
・製造部の平均残業時間: 月○時間（目標の月○時間を上回っている）
・残業の主な原因（現場ヒアリング結果）:
  ① 月末の出荷集中（受注の7割が月末納期）
  ② 設備トラブルによるライン停止とリカバリー
  ③ 品質問題（クレーム対応・手直し）による追加作業
  ④ 引き継ぎ・申し送りに時間がかかる（情報共有の非効率）

・管理職の残業: 法的上限の適用対象外だが月○時間は問題

【削減目標】
6か月後: 月○時間/人（現状から○時間削減 / ○%削減）
1年後: 月○時間/人

出力形式:
・「残業削減アクションプラン（原因別の対策と担当・期限の一覧）」
・「月末出荷集中の平準化対策（顧客交渉・生産前倒し等の選択肢）」
・「業務改善進捗の月次モニタリングレポートのフォーマット」
・「経営層・労働組合（もしあれば）向けの計画説明文の骨子」
・「働き方改革の取り組みを採用活動に活かすための表現方法」
```

## 勤務間インターバル制度の導入案内を作るプロンプト

退勤から次の出勤まで一定時間を確保する取り組みを整理します。

```
以下の情報をもとに、勤務間インターバル制度の導入に関する
社内案内文と管理フォーマットを作成してください。

【制度の概要と自社の方針】
勤務間インターバル: 前日の退勤から当日の出勤まで最低○時間を確保
  （例: 11時間のインターバルを目安に設定）
  ※ 労働時間等設定改善法に基づく努力義務（現時点）

自社の現状:
  ・夜勤明け（例: 翌02:00退勤）後に通常日勤（08:00出勤）になるケースが
    月に数件発生している
  ・インターバルが確保できない場合の振替・対応ルールがない

導入の目的:
  ・睡眠時間を確保し、疲労回復・健康管理を促進する
  ・ヒューマンエラーによる品質問題・労働災害を予防する

出力形式:
・「勤務間インターバル制度導入の社内案内文（全従業員向け）」
・「インターバルが確保できなかった場合の対応ルール（振替休日・
  出勤時間繰り下げ・賃金の取り扱い）」の整理
・「勤務スケジュール作成時のインターバル確認チェックリスト」
・「制度導入の効果測定指標（健康診断データ・残業時間・
  インシデント件数との相関）」
```
