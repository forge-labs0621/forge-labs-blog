製造業では在庫が多すぎると資金が寝てしまい、少なすぎると製造の停止につながります。適正な安全在庫・発注点の設定と在庫管理ルールの整備をAIで補助できます。

---

## 安全在庫・発注点の計算プロンプト

適正な安全在庫量と発注点を計算・整理します。

```
以下の情報をもとに、安全在庫量・発注点を計算し、
在庫管理の推奨値をまとめてください。

【部品情報】
部品番号・品名: （例：RM-A001 SPCC板材 t1.6×1000×2000）
使用部門: （例：プレス工程）
月間使用量（平均）: （例：200枚/月）
月間使用量のばらつき（標準偏差）: （例：±30枚/月）

【調達リードタイム】
通常納期: （例：7日）
最長納期（遅延最大値）: （例：14日）
納期のばらつき（標準偏差）: （例：±3日）

【在庫管理の前提】
欠品許容率: （例：5%以下（サービスレベル95%）/ 安全係数Z ≒ 1.65）
発注単位: （例：50枚単位）
現在の在庫水準: （例：300枚）

【確認してほしいこと】
・安全在庫の計算（σ×Z×√LT の計算）
・発注点（ROP: Reorder Point）の計算
・「今の300枚在庫は適正か」の評価
・発注サイクル（何日ごとに発注するのが最適か）の提案

出力形式:
・安全在庫・発注点の計算式と結果
・「安全在庫 = ○枚 / 発注点 = ○枚」の結論
・現在の在庫水準への評価（過剰・適正・不足）
・「在庫削減できる余地（枚数と金額換算）」の試算
```

## 在庫管理ルール・運用手順を作るプロンプト

部品・材料の在庫管理ルールを整理した手順書を作ります。

```
以下の情報をもとに、製造業の材料・部品在庫管理の
運用手順書を作成してください。

【自社の在庫管理の現状】
管理対象品目数: （例：材料100品目 / 購入部品50品目）
現在の管理方法: （例：Excelの在庫表で手入力管理 / 倉庫担当者が毎日更新）
課題:
  ・入出庫の記録もれが月に数件発生している
  ・在庫表と実棚卸の差が毎月1〜3%出る
  ・ベテランしか在庫状況を把握しておらず、属人化している
  ・「どの棚のどこにある」がわかる人間が限られている

【整備したい在庫管理ルール】
・入庫ルール（仕入品の検品・登録の手順）
・出庫ルール（製造への払い出し・記録の手順）
・在庫確認（定期棚卸・サイクルカウント）
・定位置管理（5S連動で棚の場所を決める）
・補充発注ルール（発注点に達したら誰が・どうやって発注するか）

出力形式:
・在庫管理手順書（誰でも実行できるSTEP形式）
・「在庫管理のKPI（在庫回転率・欠品件数・差異率）」の測定方法
・「入出庫記録のテンプレート（記入例付き）」
・「定期棚卸の実施手順（月次・四半期・年次）」の説明
```

## 滞留在庫・不動在庫の整理プロンプト

長期間動かない「死に在庫」の整理方針を作ります。

```
以下の情報をもとに、滞留在庫・不動在庫の整理計画を
作成してください。

【滞留在庫の状況】
（例: 在庫の棚卸を実施したところ、以下の品目が6か月以上出庫されていない）

不動在庫リスト（例）:
品番: RM-X099 / 品名: 特殊ステンレス板材（SUS304 t2.0mm）
在庫数: 50枚 / 単価: ¥12,000/枚 / 在庫金額: ¥600,000
不動の理由: 3年前に廃番になった製品の残材。使い道がない。

品番: PC-Z045 / 品名: 旧型コンベアのベルト交換部品
在庫数: 3本 / 単価: ¥85,000/本 / 在庫金額: ¥255,000
不動の理由: そのコンベアを2年前に廃棄した。

総不動在庫金額: ¥2,800,000

【取りうる対応策】
（例: 転用（他製品に使えないか）/ 仕入先への返品交渉 / スクラップ売却）

出力形式:
・品目ごとの「処分方針案（転用・返品・売却・廃棄）と優先度」の表
・処分方針の根拠・判断基準
・「仕入先への返品交渉文（例: RM-X099について）」のメール案
・不動在庫の発生防止策（なぜ不動になったか・再発防止）
```
