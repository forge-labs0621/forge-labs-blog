製造設備は適切なタイミングで更新しないと、故障増加・生産性低下・品質リスクが高まります。AIを使って設備の更新タイミング判断・廃棄費用試算・稟議書・固定資産台帳の更新を効率化できます。

---

## 設備更新の必要性判断と更新計画プロンプト

老朽設備をいつ更新すべきか判断します。

```
以下の情報をもとに、老朽化している
製造設備の更新判断と更新計画を整理してください。

【更新検討対象の設備】
設備名: コンプレッサー（空気圧縮機）
メーカー: ○○製作所
型番: AC-75W
設置年: 2001年（現在25年使用）
取得原価: ¥3,500,000
現在の帳簿価額: ¥0（既に減価償却完了）

【現状の問題点】
  ・月1〜2回の修理が発生（修理費用: 平均¥80,000/回）
  ・圧力の立ち上がりが遅く、ライン停止が月3〜4回発生
  ・電力効率が新型機と比べ約30%悪い（電力代年間¥350,000の差）
  ・部品の入手が困難（製造中止品）

【更新候補機種】
新型機: ○○製 AC-75W-NEO
  取得価格: ¥2,800,000（工事費込み）
  耐用年数: 15年
  電力消費: 現状比-30%（年間¥350,000節約）
  メンテナンス: 3年目からの定期点検費用¥150,000/年

出力形式:
・「現状維持 vs 更新の総コスト比較（5年・10年のライフサイクルコスト）」
  （修理費・停止損失・電力費・購入費・メンテ費の合計比較）
・「設備更新の推奨タイミングと根拠」
・「設備更新稟議書の骨子
  （現状の問題・更新の必要性・候補機種・費用・ROI・承認依頼）」
・「設備廃棄時の固定資産台帳の除却処理の手順
  （廃棄日・廃棄理由・廃棄費用・仕訳の確認）」
・「新設備導入のスケジュール（見積→承認→発注→設置→試運転）」
```

## 設備廃棄処分の書類を整理するプロンプト

不用設備の廃棄処理と行政手続きを整理します。

```
以下の情報をもとに、製造設備の廃棄処分に
必要な書類と手続きを整理してください。

【廃棄する設備】
設備① コンプレッサー（上記の老朽機）
  重量: 約320kg / 材質: 鋼板・アルミ・電気部品
  処分方法: 産業廃棄物業者による回収

設備② 旧型CNCマシニングセンタ（2000年製）
  重量: 約2,500kg / クーラント液（廃液）残量: 約50L
  処分方法: 中古機械業者への売却（¥200,000の買取見積あり）

設備③ 蛍光灯照明器具（LED化に伴い撤去）
  数量: 48台 / PCB含有有無: 確認が必要（1990年以前設置の可能性）

【廃棄処分の際の法的事項】
  ・産業廃棄物の処理: マニフェスト（産廃管理票）の作成・保管
  ・クーラント廃液: 廃油・廃液の適切処理（水質汚濁防止法）
  ・PCB含有機器: 特別管理産業廃棄物（PCB廃棄物特措法）

出力形式:
・「廃棄設備ごとの処分方法・必要書類・法的手続きの整理表」
・「産業廃棄物処理業者への依頼書の骨子
  （廃棄物の種類・数量・処理方法・費用・運搬日）」
・「設備売却時の手続き（売買契約書・代金受取・固定資産の除却）」
・「PCB含有機器の調査と対応（設置年・製造メーカーへの問い合わせ方）」
・「廃棄後の固定資産台帳の更新と仕訳
  （除却損の計上・廃棄費用の処理）」
```

## 設備の中長期更新計画を整理するプロンプト

10年後を見据えた設備投資ロードマップを作ります。

```
以下の情報をもとに、製造設備の
中長期更新計画（10年ロードマップ）を整理してください。

【現有設備の概要（主要設備のみ）】
設備名 / 設置年 / 耐用年数 / 次回更新推奨年 / 取得原価:
  プレス機①（200t）: 2005年設置 / 15年 / 2020年〜（超過中）/ ¥8,000,000
  プレス機②（80t）: 2012年設置 / 15年 / 2027年〜 / ¥3,500,000
  レーザー切断機: 2018年設置 / 10年 / 2028年〜 / ¥15,000,000
  CNCマシニングセンタ①: 2001年設置 / 15年 / 2016年〜（超過中） / ¥12,000,000
  CNCマシニングセンタ②: 2022年設置 / 15年 / 2037年〜 / ¥18,000,000
  コンプレッサー: 2001年設置 / 15年 / 今回更新予定 / ¥3,500,000
  クレーン（天井走行式）: 2010年設置 / 20年 / 2030年〜 / ¥5,000,000

今後5年間の設備投資予算の目安: 年間¥5,000,000以内

出力形式:
・「設備ごとの経過年数・超過年数・緊急度（高/中/低）の整理」
・「今後10年間の更新優先順位と推奨スケジュール（年別）」
・「各年の必要投資額と予算制約との照合」
・「緊急度「高」の設備（耐用年数超過）への対応方針
  （更新優先 / 精密点検で延命 / 暫定修繕）」
・「設備投資計画書の骨子（10年ロードマップの経営報告書フォーマット）」
```
