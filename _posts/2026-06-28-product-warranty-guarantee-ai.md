製造業では取引先（特に自動車メーカー・1次サプライヤー）から製品保証に関する書類の提出を求められることがあります。品質保証の範囲・保証期間・免責事項を明確にした製品保証書や、PL法（製造物責任法）対応のドキュメントをAIで下書きできます。

> **注意**: 製品保証書は法的拘束力を持つ文書です。最終的な内容は必ず法務部門または弁護士に確認してください。

---

## 製品保証書（社外向け）の骨子を作るプロンプト

取引先に提出する製品保証書の骨子・文面を整理します。

```
以下の情報をもとに、取引先への製品保証書の骨子を作成してください。

【製品情報】
製品名・品番: （例：プレスブラケット A-1234）
用途・使用環境: （例：自動車ボディ補強部品 / 屋外・振動環境）
主要材料・表面処理: （例：SPCC t1.6mm / 三価クロムめっき）

【保証の内容】
保証する品質基準:
  （例：JIS規格○○・当社図面寸法公差・納入仕様書No.○○に適合）
保証期間: （例：納品後2年間、または走行距離○万km（先に到来する方）／
  ※取引先の要求仕様に合わせて記入）
保証の範囲（当社責任と認められる不具合）:
  （例：材料不良・加工不良・当社起因の外観不良）

【免責事項（保証対象外）】
（例：
・当社仕様以外の用途・環境での使用
・お客様側の誤った取り付けまたは改造
・天災・交通事故等の外的要因による損傷
・通常の使用による経年劣化
）

出力形式:
・製品保証書の文面（正式な証書形式）
・「保証する範囲」と「保証の対象外（免責）」を明確に分けて記載
・「本書の解釈について疑義が生じた場合は、両者協議の上解決する」の条項
・発行日・発行者（会社名・代表者名）の記載欄
・※法的妥当性の最終確認は法務・弁護士が行うこと明記
```

## 品質保証協定書（取引先との取り決め）の確認プロンプト

取引先から提示された品質保証協定書のポイントを整理します。

```
以下の品質保証協定書（案）の主要条項を整理し、
サプライヤー側として注意すべきポイントを確認してください。

【協定書の内容（主要条項を抜粋）】
第○条（品質基準）:
  （例：「サプライヤーは、本協定別紙に定める仕様書・図面・
  規格書に適合した製品を納入しなければならない」）

第○条（品質記録の保存）:
  （例：「サプライヤーは製品の検査記録・トレーサビリティ情報を
  納入日より15年間保存しなければならない」）

第○条（クレーム対応）:
  （例：「クレーム発生時は48時間以内に暫定対策を報告し、
  30日以内に恒久対策を報告すること」）

第○条（損害賠償）:
  （例：「市場回収・リコールが発生した場合、当該製品の
  品質不具合に起因する費用はサプライヤーが負担する」）

確認してほしいこと:
・サプライヤー側として対応が難しそうな条項はどれか
・「15年保存」「48時間以内報告」など期限・義務の整理
・損害賠償の範囲について一般的な商慣習と比較

出力形式:
・各条項について「対応可能 / 要交渉 / 要確認」の3分類
・「このまま調印する前に確認したい点」のリスト
・サプライヤー側が修正提案する際の典型的な文言例
・※法的な判断は弁護士に確認することを明記
```

## PL法（製造物責任法）対応の社内チェックリストを作るプロンプト

製造物責任法（PL法）への備えを確認する社内チェックリストを作ります。

```
以下の視点で、製造業中小企業のPL法（製造物責任法）対応状況を
確認するためのチェックリストを作成してください。

【確認したい項目】
(1) 製品の安全設計・品質管理
・設計段階でのリスクアセスメント（想定される誤使用を含む）
・品質基準・検査基準書の整備状況
・製品のトレーサビリティ（どのロットにどの材料を使ったか追跡できるか）

(2) 製品への情報提供
・取扱説明書・注意書きの整備（誤使用防止のための警告）
・製品への自社ロゴ・製造年月の表示

(3) 事故・クレーム発生時の対応体制
・社内のクレーム窓口と責任者の明確化
・事故情報の収集・分析の仕組み
・製品回収（リコール）を判断するための基準

(4) 保険・リスク対策
・PL保険（製造物賠償責任保険）の加入状況
・契約書での免責条項の確認

出力形式:
・「対応済 / 未対応 / 要確認」の3択チェックボックス形式
・未対応の場合の「優先度（高/中/低）」と「対応のヒント」を各項目に
・「PL法対応で最も重要な3つのポイント」のコラム
```

## 市場クレーム発生時の調査報告書（対外向け）を作るプロンプト

取引先からのクレームに対し、原因調査と再発防止策を報告する文書を作ります。

```
以下の情報をもとに、取引先への品質クレーム調査報告書を作成してください。

【クレームの概要】
発生日・連絡日: （例：2026年6月15日に取引先から連絡を受けた）
クレームの内容: （例：当社が納入したブラケット（品番A-1234）に
  寸法不良が発見された。取引先の組立工程で発覚。該当ロットは
  2026年5月20日納入分（ロット番号○○）。不良数量: ○個確認）
不良の内容: （例：穴径が指定公差±0.05に対して+0.12mmオーバー）

【原因調査の結果】
直接原因: （例：パンチ工具の摩耗が進んでいたが、交換タイミングの
  管理が不十分で、そのまま使い続けた）
根本原因: （例：工具管理台帳の「交換基準（打数）」が古い基準値のまま
  更新されておらず、担当者が誤った基準で判断した）

【対策】
即時対応（実施済み）: 
  （例：該当ロットの全数出荷停止・返品受け入れ・全数再検査を実施）
恒久対策:
  （例：工具管理台帳を最新基準値に更新。工具交換判断を2名体制に変更。
  工具摩耗の定期確認チェックシートを追加）

出力形式:
・8D報告書形式（D1〜D8の構成）または自由形式レポート
・「このようなことで大変ご迷惑をおかけしました」の謝罪から始める
・「再発防止の継続確認をお約束します」の締め
・提出日・自社担当者名・押印欄
```
