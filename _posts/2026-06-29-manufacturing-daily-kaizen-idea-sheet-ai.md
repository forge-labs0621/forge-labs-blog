日常改善・小集団活動は現場全員が参加する継続的な改善活動です。AIを使って改善提案シートのデジタル化・活動計画書・発表資料の作成を効率化できます。

---

## 現場改善提案シートの設計と活動仕組みを整理するプロンプト

```
以下の情報をもとに、現場発の日常改善提案の仕組みと
書類フォーマットを整理してください。

【現場の状況】
従業員数: 45名（製造現場37名・管理8名）
現在の改善活動の問題点:
  ・年1回の「改善提案表彰」はあるが、日常的な提案活動が定着していない
  ・改善提案書の記入が面倒（A4紙1ページ・項目が多すぎる）
  ・提案後の返答が遅く（1〜2ヶ月）、提案者がやる気をなくす
  ・実施された改善が少なく（提案の約15%しか実施されない）

目指す姿:
  ・1人月1件の改善提案を目標にしたい
  ・「ちょっとした気づき」でも提案できるハードルの低い文化
  ・提案から2週間以内に回答し、採用・不採用の理由を明示

出力形式:
・「シンプル改善提案シートのフォーマット（A6サイズ・5項目以内）
  必須: ①気づいたこと ②困っていること ③こうすればいい
  任意: ④効果の見込み ⑤担当に相談したい人」
・「改善提案の受付・回答フロー（提案→班長確認→生産技術レビュー→2週間以内回答）」
・「改善提案の採用判断基準（実施コスト / 効果 / 工数 / リスクの評価マトリクス）」
・「月次改善活動レポート（提案数・採用数・実施数・コスト削減効果の集計）」
・「改善提案活性化のための朝礼アナウンス文案と表彰基準」
```

## 小集団活動の計画書・発表資料を整理するプロンプト

```
製造現場の小集団改善活動（3〜6名のグループ活動）の
計画書・活動記録・発表資料を整理してください。

【小集団活動の概要】
グループ名: 「加工ライン改善チーム」（旋盤担当5名）
活動期間: 2026年4月〜9月（6ヶ月）
テーマ: 「段取り替え時間を30%短縮する」
現状の段取り時間: 平均47分
目標: 33分以内（▲30%）

活動の進捗（現時点: 6月末）:
  ・現状分析完了（段取り時間の内訳: 工具探し12分・測定器確認8分・
    プログラム呼出し7分・初品確認20分）
  ・改善案検討完了（工具影絵板設置・段取り手順標準化・
    先行プリセット化・プログラム管理サーバー導入）
  ・試行実施中（影絵板は効果あり・段取り時間4分短縮を確認）
  ・残り改善策: 7〜8月に順次実施予定

出力形式:
・「小集団活動計画書（テーマ / 目標 / スケジュール / 役割分担 / KPI）」
・「中間発表資料の構成（現状→目標→原因分析→改善策→試行結果→今後の計画）」
・「改善効果の数値化方法（時間短縮効果 / 年間コスト削減試算）」
・「活動記録シート（週次ミーティング記録・実施事項・次回の約束）」
・「最終発表に向けたストーリー構成（PDCA展開図・ビフォーアフター・横展開提案）」
```
