「良い提案があっても、社内で通らない」「経営会議でうまく説明できない」という悩みを持つ製造業の方へ。AIを使って説得力のある提案書・プレゼン資料を効率よく作れます。

## 現場改善提案書を経営者に通すプロンプト

現場からの提案を経営判断レベルの文書に変換します。

```
以下の現場の改善アイデアをもとに、
経営者が「やろう」と判断できる提案書を整理してください。

【提案者と背景】
提案者: 製造部 主任・田中
提案の発端: バリ取り工程で毎月20〜30時間の残業が発生しており、
          作業者の疲労と品質バラツキが問題になっている

【提案内容（アイデアベース）】
  ・電動バリ取りツールの導入（現在は手工具のみ）
  ・新ツールの参考型番: ABCグラインダー社のモデルX
  ・見積もり: 1台¥85,000 × 3台 = ¥255,000
  ・期待効果: 作業時間が約35%短縮できると思う

【提案書で伝えたいターゲット読者】
  ・工場長（直属上司）: まず承認を取りたい
  ・社長: 最終承認者・投資金額¥255,000に敏感

出力形式:
・「提案書の構成（A4・2〜3ページ）」
  1. 現状の問題と課題の定量化
  2. 提案内容（何を・なぜこれを選んだか）
  3. 導入費用と期待効果（ROI）
  4. 実施スケジュール
  5. リスクと対応策
・「現状の問題の定量化（残業20〜30時間/月の人件費コスト換算）」
・「工場長への口頭説明の練習用スクリプト（3分版）」
・「社長が「なぜこれじゃないといけないか」を
  納得する説明のポイント（競合製品との比較）」
・「提案が通らなかった場合の次の一手（段階的な試行提案）」
```

## 経営会議向け月次報告プレゼン資料プロンプト

月次の部門業績を経営層に分かりやすく報告します。

```
以下の情報をもとに、経営会議（月次）での
製造部部門報告のプレゼン資料（5分間）を整理してください。

【経営会議の概要】
参加者: 社長・副社長・各部長（製造・品質・営業・経理）
形式: スライド+口頭説明・製造部の持ち時間5分
目的: 今月の実績報告と来月の課題・対策の共有

【今月（6月）の製造部実績データ】
  生産数量: 3,200個（計画3,000個・達成率107%）
  品質:
    不良品率: 0.8%（目標0.5%・未達・先月1.2%から改善中）
    顧客クレーム: 1件（P-205の寸法外れ・原因調査中）
  生産性:
    OEE: 72%（目標80%・未達）
    残業: 月計180時間（計画160時間・超過20時間）
  安全: 労働災害・ヒヤリハット: 0件

来月の重点課題:
  ①不良品率: 0.5%以下への改善（バリ取り工程の改善実施予定）
  ②OEE向上: プレス機のチョコ停対策（原因分析完了・対策着手中）

出力形式:
・「プレゼンスライドの構成（5枚以内）と各スライドのメッセージ」
  スライド1: 今月のサマリー（信号機方式・赤黄緑で一目で分かる）
  スライド2〜4: 主要指標の詳細（生産・品質・安全）
  スライド5: 来月の重点対策
・「各スライドの記載内容（ビジュアル・数字・コメントの指示）」
・「口頭説明の台本（5分・スライドごとの発言内容）」
・「社長から来そうな質問と回答（3問）
  例: 「クレームの原因は特定できたか？」」
・「プレゼン後に提出する1枚サマリーのフォーマット」
```

## 新規事業・横展開の社内提案プロンプト

事業の幅を広げる提案を経営層に通します。

```
以下の情報をもとに、新規事業の社内提案書を整理してください。

【提案する新規事業】
提案者: 営業部長・鈴木（社長に相談したい）
内容: 現在の受託製造（プレス加工）に加えて、
     「設計提案型営業」に転換する提案
     （顧客の製品設計段階から参加し、VA/VE提案で付加価値を出す）

【提案の背景】
  ・現在の受注は顧客からの図面支給・指値受注が中心
  ・値下げ圧力が毎年続いており利益率が低下（5年で粗利-4%）
  ・設計提案型に移行すれば粗利率が20〜30%に改善できると予測
  ・競合他社がすでに取り組んでいる（業界トレンド）

【課題・リスク】
  ・設計者の採用が必要（コスト増）
  ・顧客との関係が「指示を受ける側」から「提案する側」に変わる
  ・移行期間（2〜3年）は収益が安定しない可能性

出力形式:
・「新規事業提案書の構成（A4・4〜5ページ）」
  1. 市場環境の変化と問題意識
  2. 提案内容（設計提案型営業への転換）
  3. 収益モデルの試算（現状 vs 3年後）
  4. 実施ロードマップ（フェーズ1-3）
  5. 必要な投資とリスク・対策
・「経営会議で社長に一番伝えるべきメッセージの
  1文（エレベーターピッチ）」
・「反対論を想定した事前の論点整理
  （「設計者を採用する余裕はない」への答え方等）」
・「まず試す「小さな一歩」の提案（全社移行前のテスト検証案）」
```
