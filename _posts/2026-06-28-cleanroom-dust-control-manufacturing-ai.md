精密部品・半導体・食品製造業ではクリーンルームの環境管理が品質に直結します。AIを使って入室手順書・環境測定記録・異物混入対策書類を体系的に整備できます。

---

## クリーンルーム管理基準書を整理するプロンプト

環境管理の基準を文書化します。

```
以下の情報をもとに、クリーンルームの
管理基準書を整理してください。

【当社クリーンルームの概要】
業種: 精密光学部品（レンズ・プリズム等）の研磨・組み立て
クリーンルームの規模: 200m²（クラス1000相当 / ISO 6相当）
主な管理項目:
  ①パーティクル（粒子数）: ≥0.5μm 35,200個/m³以下
  ②温度: 23±1℃（精密部品の熱膨張対策）
  ③相対湿度: 45±5%（静電気防止・露点管理）
  ④差圧: 室内 > 室外（陽圧管理・外気流入防止）
  ⑤人員制限: 同時入室10名以下

現在の問題:
  ①入室規則（手洗い・更衣・エアシャワー）の遵守が徹底されていない
  ②環境測定（パーティクルカウンター）の記録が紙で散在
  ③新人作業者への入室手順教育が口頭のみ

出力形式:
・「クリーンルーム管理基準書の構成と各章の記載内容」
  （管理目標値 / 入室ルール / 日常点検 / 異常時対応 / 記録の保管）
・「入室手順書（ステップバイステップ形式）」
  手洗い → 更衣 → グローブ着用 → エアシャワー → 入室確認 の詳細手順
・「環境測定記録シートのフォーマット
  （測定日時 / 測定場所 / 各パラメータ値 / 判定 / 異常対応）」
・「違反行為ランキングTOP5（実際によく起きる入室違反）と防止策」
・「クリーンルーム管理の新人教育テキストの骨子
  （なぜ清潔さが必要か / 自分の行動が品質に与える影響）」
```

## 異物混入対策書類を整理するプロンプト

クリーンルームでの異物管理を強化します。

```
以下の情報をもとに、異物混入対策の
管理書類を整理してください。

【異物混入の実態と対策状況】
業種: 電子部品の検査・組み立て（クリーンルーム環境）
発生した異物混入問題（過去1年）:
  ①金属粉（研磨工程の切り粉）の混入: 2件（顧客クレームに発展）
  ②毛髪の混入: 1件（完成品検査で発見・出荷前に回収）
  ③作業グローブのゴム粒子: 3件（内部発見）

現在の対策:
  ・入室時のエアシャワー（10秒）
  ・頭部カバー・フェイスマスク・帽子の着用義務
  ・金属探知機（出荷前 / 1日2回）
  ・ルーペ目視検査（全数 / 抜き取り）

まだ対策できていない部分:
  ・加工工程（研磨）からのオープンスペース移動時の切り粉飛散対策
  ・グローブの交換頻度・交換基準が不明確
  ・毛髪混入の場合、どのロットまで遡及するかの基準がない

出力形式:
・「異物管理計画書（異物の種類 × 発生源 × 対策方法 の一覧表）」
・「異物混入発生時の対応フロー（発見 → 報告 → 原因調査 → 遡及 → 是正）」
・「遡及判断基準書（どの時点まで製品を回収・再検査するか）」
  （作業交代のタイミング / 清掃実施記録 / 最後に異物がなかったことを確認した時点）
・「グローブ管理基準書（種類・交換頻度・廃棄方法・数量管理）」
・「異物混入ゼロ活動の月次モニタリングシート
  （発生件数 / 工程別 / 種類別 の集計表と推移グラフの説明）」
```

## 食品工場の異物混入防止計画（HACCP補完）を整理するプロンプト

食品製造業での異物管理を強化します。

```
以下の情報をもとに、食品工場の
異物混入防止計画書を整理してください。

【当社食品工場の概要と問題】
業種: 食品加工（おにぎり・サンドイッチ・弁当の製造）
生産量: 1日5,000食
主な異物リスク:
  ①金属異物（ステンレス・刃具の欠け片）: 最大リスク
  ②プラスチック異物（梱包フィルム・容器の破片）
  ③毛髪・異物（作業者由来）
  ④虫（外部からの侵入）

現在の検出機器:
  ・金属探知機: 1台（最終工程 / 全数）
  ・X線検査装置: なし

顧客（コンビニ）からの要求:
  ・異物混入事故を年間0件に
  ・月次での異物管理報告書の提出（工程別・種類別の発生件数）
  ・検出機器の日常点検記録の提出

出力形式:
・「異物混入防止計画書の構成と記載内容
  （HACCP補完書類として機能する構成）」
・「金属探知機の日常点検記録シートのフォーマット
  （テストピース確認 / 感度チェック / 異常時対応手順）」
・「異物混入防止のための入室・作業ルール一覧
  （毛髪・アクセサリー・作業服・手洗い等の管理基準）」
・「コンビニ等の顧客向けの月次異物管理報告書のフォーマット
  （発生件数 / 工程 / 原因 / 是正措置 / 再発防止状況）」
・「X線検査装置の導入費用対効果の試算方法
  （検出率向上 × クレーム回避価値 vs 初期投資・ランニングコスト）」
```
