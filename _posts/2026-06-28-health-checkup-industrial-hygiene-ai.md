製造業では有機溶剤・粉塵・騒音・振動等に曝露する作業者の特定業務健診が義務づけられています。AIを使って健診計画・記録管理・事後措置書類を効率的に整備できます。

---

## 年間健康診断実施計画書を整理するプロンプト

法令に基づく健診計画を体系的に立てます。

```
以下の情報をもとに、工場の年間健康診断
実施計画書を整理してください。

【当社の健康診断の対象者と状況】
従業員数: 正社員28名 + パート・派遣20名 = 計48名

健診の種類別の対象者:
  ①定期健康診断（全員・年1回義務）: 48名全員
  ②特定業務健診（特殊健診）:
    - 有機溶剤業務従事者（洗浄工程・3名）: 年2回
    - 粉塵業務従事者（研磨・バリ取り工程・6名）: 年2回
    - 騒音業務従事者（プレス工程・8名）: 年2回
    - 振動工具使用者（ハンドグラインダー・4名）: 年2回
  ③雇入れ時健診（採用時・年間3〜5名）: 随時

前回の健診での問題:
  ①受診勧奨（要精密・要治療）が5名いたが、受診確認フォローが不十分
  ②特定業務健診の対象者が変更（人事異動）になったが記録が更新されていなかった
  ③パート社員の健診実施が遅れ（定期健診は週30時間以上のパートも義務）

出力形式:
・「年間健康診断実施スケジュール表
  （健診種別 / 対象者 / 実施時期 / 実施機関 / 費用概算）」
・「健診の法的義務の整理
  （定期健診 / 特定業務健診 / 雇入れ時健診の違いと対象・頻度）」
・「特定業務健診の対象者リスト管理方法
  （人事異動時の更新ルールと担当部門との連携体制）」
・「パートタイム社員への健診義務の判断基準
  （週30時間以上・1年以上勤務の場合の適用）」
・「健診機関への実施依頼文と対象者名簿の提出フォーマット」
```

## 健康診断結果の事後措置書類を整理するプロンプト

受診後の対応を漏れなく管理します。

```
以下の情報をもとに、定期健康診断の
事後措置に関する書類を整理してください。

【健診結果と事後措置の状況】
2026年度定期健康診断の結果（48名受診）:
  異常なし: 33名
  要注意（経過観察）: 8名（血圧・血糖値・BMI等）
  要受診（精密検査推奨）: 5名（1名は高血圧・2名は血糖値異常・2名は聴力低下）
  治療中: 2名（既に通院中）

産業医からの意見:
  ・要受診5名については、産業医面談を実施のうえ業務内容を確認する必要あり
  ・聴力低下の2名については、騒音作業の継続可否を職場環境の測定結果とあわせて判断
  ・高血圧の社員が深夜シフトを担当しており、担当の見直しを検討してほしい

出力形式:
・「健康診断結果の管理台帳フォーマット
  （氏名 / 実施日 / 各検査項目の判定 / 事後措置 / 確認日）」
・「要受診者への受診勧奨通知文（本人渡し用）
  （プライバシーに配慮しながら受診を促す文章）」
・「産業医意見書の依頼文と意見書の記載様式
  （業務適性の意見・就業上の措置の記入欄）」
・「就業制限・配置転換を実施する場合の対応手順書
  （本人への説明 / 上司への連絡 / 記録の管理）」
・「受診確認のフォローアップ体制
  （誰が・いつ・どうやって受診を確認するかの手順）」
```

## 特定業務健診（有機溶剤健診）の記録を整理するプロンプト

有機溶剤・粉塵作業者の特殊健診を管理します。

```
以下の情報をもとに、有機溶剤業務従事者の
特定業務健診の記録と管理書類を整理してください。

【有機溶剤業務の状況】
使用有機溶剤: イソプロピルアルコール（IPA）/ トルエン（ごく少量）
使用工程: 精密部品の洗浄工程（作業者3名が担当）
作業環境: 局所排気装置あり・防毒マスク着用義務

健診の法的根拠: 有機溶剤中毒予防規則第29条・労働安全衛生法第66条
健診頻度: 年2回（6月・12月）
実施機関: 産業医（嘱託）+ 外部健診機関（採血・尿検査）

特定健診の検査項目（有機溶剤）:
  ・業務歴・自覚症状・他覚症状
  ・白血球数・肝機能（GOT/GPT/γ-GTP）
  ・尿中の有機溶剤代謝物測定
    （トルエンの場合: 尿中馬尿酸・クレアチニン）

出力形式:
・「有機溶剤業務従事者健康診断個人票の記載方法と保管期間
  （安衛法に基づく5年保管）」
・「業務歴・自覚症状の問診票フォーマット
  （業務内容 / 使用物質 / 症状 / 防護具着用状況）」
・「健診結果の異常値が出た場合の対応手順
  （産業医への報告 / 作業環境測定の見直し / 業務変更の検討）」
・「有機溶剤業務の新任担当者への健康教育の資料骨子
  （有機溶剤の危険性 / 正しい防護具の使い方 / 体調変化の早期報告）」
・「特定業務健診の結果を産業医に報告する際の書式と項目」
```
