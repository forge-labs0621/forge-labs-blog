製造業で外注先・協力会社に加工や作業を依頼する際、「指示の伝え方がバラバラ」「口頭で伝えたのに仕上がりが違った」「品質要求事項をうまく文書化できない」という問題が起きやすいです。AIを使って外注先への指示書・依頼書を整えることで、指示漏れや認識ズレを防ぐことができます。

## 外注先管理でAIが役立つ場面

- **加工指示書の作成**: 品番・加工内容・仕様・納期を整理した指示書を素早く作る
- **品質要求事項の文書化**: 口頭で伝えていた品質基準を文書として整理する
- **協力会社への依頼メール**: 丁寧で要点を押さえた依頼文を作る
- **外注先との取り決め確認書**: 口頭合意を記録として残す

## 加工指示書をAIで作るプロンプト

---

```
以下の情報をもとに、外注先への加工指示書を作成してください。

【品名・品番】
（例：カバープレート A-101　※機密の場合は仮称で入力）

【加工内容】
（例：板金プレス加工・穴あけ・バリ取り）

【材料】
（例：SPCC t1.0mm、支給材料 / 外注先が調達）

【寸法・仕様の要点】
（例：外形寸法 200×150mm、穴径 φ8 ±0.1mm、表面粗さ Ra1.6以下）

【品質要求事項】
（例：バリ・カエリなし、錆止め油塗布、梱包は1個ずつ保護材で包む）

【数量・納期】
（例：100個、2026年8月10日）

【特記事項】
（例：初回ロットは試作扱いで10個先行納品、検査後に本数量を発注）

出力形式:
・タイトル「加工指示書」
・品名・品番・数量・納期
・加工内容（工程の流れを含む）
・材料・支給条件
・品質要求事項（箇条書きで明確に）
・特記事項
・発注者・確認者・日付欄
```

## 外注先への品質要求事項を文書化するプロンプト

口頭で伝えてきた品質基準を初めて文書化する際に使えます。

```
製造業の外注先（加工協力会社）向けに、以下の加工に関する品質要求事項を
文書化してください。

【加工の種類】
（例：機械加工（切削・旋盤）、板金加工、溶接　など）

【現在、口頭で伝えていること（箇条書き）】
（例：
・バリは取ること
・寸法は図面通りにしてほしい（トレランスは標準）
・サビが出るから油塗って梱包してほしい
・検査証明書を添付してほしい
）

出力形式:
・「品質要求事項書」または「外注先向け品質基準」として
・各要求事項を「確認事項」「判定基準」「不合格時の対応」の形で整理
・文体は取引先に渡せる正式な文書として
```

## 外注依頼メールの下書き

発注書に添付する依頼メールや、急ぎの外注依頼メールも素早く作れます。

```
外注先（協力会社）への加工依頼メールを作成してください。

【依頼内容の概要】
（例：板金加工100個、来月10日納期、詳細は添付の指示書参照）

【特に伝えたいこと】
（例：初めての品番なので不明点は早めに連絡してほしい /
       今回は急ぎのため優先対応をお願いしたい　など）

【文体】
丁寧・ビジネス標準、要点が明確になるように

注意: 会社名・担当者名はプレースホルダー（○○様、△△株式会社）で出力
```

## 口頭合意を確認書にまとめるプロンプト

外注先と電話・口頭で決めた内容を記録に残す際に使えます。

```
以下の口頭合意内容を、外注先との「取り決め確認書」としてまとめてください。

【合意した内容のメモ】
（例：
・単価: ○○円（税別）→ 数量100個以上の場合は○○円
・納期: 通常5営業日、急ぎの場合は追加費用で3営業日対応可
・梱包: 保護材あり1個ずつ、段ボール箱で納品
・不良品の処理: 当社確認後、代品または返金で対応
・合意日: 2026年7月○日、電話にて
）

出力形式:
・「取引条件確認書」として
・日付・双方の会社名・担当者名（プレースホルダー）
・合意事項を箇条書きで明記
・「上記内容に相違ないことを確認します」の文言と署名欄
```

## 外注先からの納品検査チェックリスト

外注先から納品された品物を受け入れる際のチェックリストも作れます。

```
以下の条件で、外注加工品の受入検査チェックリストを作成してください。

【品名・品番】
【加工の種類】
（例：機械加工、板金加工　など）

【主な検査観点】
（例：数量確認、外観（傷・バリ・錆）、寸法（ノギス確認）、
       梱包状態、納品書・検査証明書の有無　など）

出力形式:
・チェック項目ごとに「確認内容」「OK/NG/対応中」「コメント」欄
・合否判定欄（全合格/一部保留/不合格）
・受入日・担当者・品番・ロット番号欄
```

---

## 外注先管理でAIを使う際の注意

- 外注先の具体的な会社名・単価・機密の技術情報はAIに入力しない
- AIが作った指示書・要求事項の内容は、担当者が内容の正確さを確認してから送付する
- 外注先との正式な契約事項（単価・納期保証等）は別途書面で取り交わすこと
