工程能力指数（Cp・Cpk）は工程が規格を安定して満たせるかを示す指標です。AIを使って工程能力調査・計算・報告書・改善計画の書類を体系化できます。

---

## 工程能力分析と改善計画書を整理するプロンプト

```
以下の情報をもとに、工程能力分析（Cp・Cpk）の
調査報告書と改善計画書を整理してください。

【工程能力調査の概要】
加工品目: 軸受シャフト 外径 φ20h6（-0.013〜0mm）
測定データ（連続25個の外径測定値 μm単位・基準値20,000μmからのずれ）:
  -3, -5, -2, -6, -4, -8, -3, -5, -7, -4,
  -2, -6, -5, -3, -8, -4, -6, -3, -5, -7,
  -4, -2, -6, -5, -3

規格:
  上限規格値（USL）: 0μm（0.000mm）
  下限規格値（LSL）: -13μm（-0.013mm）

出力形式:
・「平均値（x̄）・標準偏差（σ）・Cp・Cpkの計算式と
  上記データを使った計算結果」
・「工程能力調査報告書のフォーマット
  （品番 / 測定条件 / 測定値一覧 / ヒストグラム概要 / Cp/Cpk値 / 判定）」
・「工程能力の判定基準
  （Cpk≥1.67: 十分 / ≥1.33: 良好 / ≥1.00: 合格 / <1.00: 不足）」
・「Cpkが1.33未満だった場合の改善計画書の構成
  （原因分析 / 対策 / 改善目標値 / スケジュール）」
・「顧客へのCpk報告書の書き方（自動車Tier1向けの様式の特徴）」
```
