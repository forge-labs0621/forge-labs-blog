製造現場では、受注のたびに「作業指示書」「製造指示書」「ピッキングリスト」を作成する必要があります。毎回フォームを埋め直す手間は小さいように見えて、件数が多くなると時間がかかります。この記事では、AIを使ってこれらの書類を効率よく作成する方法を解説します。

## 作業指示書・製造指示書の種類

| 書類 | 用途 |
|---|---|
| 作業指示書 | 工程別の作業内容・手順・注意事項を現場に伝える |
| 製造指示書（製造オーダー） | 何を・いつまでに・何個作るかを指示する |
| ピッキングリスト（部品取り出し表） | 使用する部品・材料の品番と数量を一覧化 |
| 検査指示書 | 検査する工程・基準・数量を指示する |

---

## 作業指示書の骨子を作るプロンプト

受注情報・品番・工程を入力すると、作業指示書の骨子を作れます。

```
以下の情報をもとに、製造現場向けの作業指示書を作成してください。

【受注情報】
品番・品名: （例：A-1234 プレス加工品（SPCC、板厚1.6mm））
受注数量: （例：200個）
納期: （例：2026年8月1日 午前中）
客先・品質特記: （例：○○自動車工業、バリ取り後に全数目視確認必要）

【工程順序】
1. 材料確認（ミルシートと一致確認）
2. プレス加工（型番P-003使用）
3. バリ取り（全数手仕上げ）
4. 外観検査（全数目視）
5. 梱包・ラベル貼付

【注意事項（前回のクレーム・特記）】
（例：前回ロットでバリが残った部位: 左下コーナーR部分。今回は要注意。）

出力形式:
・工程番号・工程名・内容・担当・確認方法の表形式
・注意事項を最初に大きく記載（現場で見やすいよう）
・作業者サイン欄・工程完了確認欄（プレースホルダー）
```

## ピッキングリストを作るプロンプト

製品の部品構成（BOM）をもとに、組立ラインのピッキングリストを作れます。

```
以下の情報をもとに、組立作業用のピッキングリストを作成してください。

【製品情報】
製品名・品番: （例：組立品 ASM-002）
製造数量: （例：50セット）

【使用部品一覧（BOMより）】
- 品番A-101 外装パネル × 1個／セット（保管場所: 棚A-3）
- 品番B-205 モータユニット × 1個／セット（保管場所: 棚C-1）
- 品番C-310 締付ボルト M5×20 × 4本／セット（保管場所: 部品棚E-2）
- 品番D-409 ラバーパッド × 2枚／セット（保管場所: 棚B-5）

出力形式:
・品番 / 品名 / 保管場所 / 1セット数量 / 必要総数 / ピッキング数量記入欄 / 担当者サイン欄
・表形式（Markdown）
・ヘッダーに製品名・製造数・作業日・担当者記入欄
・最終確認サイン欄（ピッキング責任者）
```

## 複数工程の「工程札」を作るプロンプト

ロットが工程間を移動する際に使う「工程札（トラベラーカード）」の文書を作れます。

```
以下の情報をもとに、製造ロットの工程管理札（トラベラーカード）を作成してください。

【ロット情報】
ロット番号: （例：LOT-20260801-001）
品番・品名: （例：精密旋盤加工品 P-5678）
数量: （例：100個）
受注先: （例：○○製作所）

【通過工程】
1. 材料受入確認
2. 旋盤加工（○○工程）
3. 熱処理（外注: △△熱処理）
4. 研削加工
5. 最終検査
6. 出荷

出力形式:
・工程順に「工程名 / 担当者 / 実施日 / 数量（合格/不合格) / 備考」の欄
・各工程で押印・サインできるよう欄を確保
・不良が出た場合の「特記・不適合数」記入欄
・最終工程に「出荷OKサイン欄」
```

## 日程変更・特急指示の案内文を作るプロンプト

生産順序が変わったときや、特急品が入ったときに現場に伝える文書です。

```
以下の情報をもとに、製造現場への特急指示・生産順序変更の案内文を
作成してください。

【変更内容】
変更の理由: （例：○○自動車からの緊急増量依頼。納期が3日前倒しになった。）
変更前の指示: （例：品番A-1234 / 通常生産ライン / 来週月曜納品予定）
変更後の指示: （例：品番A-1234 / 特急ライン優先 / 今週金曜午前中納品必要 / 200個）
担当への依頼: （例：今日中に材料確認と段取りを完了し、明日朝から生産開始してほしい）

出力形式:
・現場責任者（班長・チームリーダー）向けの簡潔な案内文
・変更前後の比較が一目でわかるよう表を含める
・何を・いつまでに・誰が対応するかを明確に
・確認サイン欄（受領確認）
```

---

## AI活用のポイント

- 作業指示書の「工程内容・注意事項・特記」は必ず現場担当者が確認・加筆してください
- 品番・数量・納期などの数値データはAIが生成した値ではなく、受注データや設計書から転記してください
- ピッキングリストの「保管場所」は、倉庫レイアウト変更時に毎回更新する必要があります
