製造業の購買部門では調達コストの削減と供給安定性の両立が求められます。AIを使って購買戦略書類・TCO（総所有コスト）分析・バイヤーとサプライヤーの交渉材料を効率的に整備できます。

---

## 購買カテゴリー戦略を整理するプロンプト

品目ごとの購買戦略を体系化します。

```
以下の情報をもとに、購買カテゴリーごとの
戦略と優先施策を整理してください。

【当社の購買品目と現状】
年間購買額: ¥140,000,000（対売上比 約37%）
主要購買品目:
  ①鋼板（フラット材）: ¥45,000,000/年 / 仕入先2社
  ②アルミ材: ¥20,000,000/年 / 仕入先1社（依存度高）
  ③外注加工費（切削・溶接）: ¥35,000,000/年 / 外注先8社
  ④標準購入部品（ネジ・ボルト・軸受等）: ¥15,000,000/年 / 仕入先5社
  ⑤補助材料（切削油・研磨材等）: ¥8,000,000/年 / 仕入先3社
  ⑥梱包材: ¥5,000,000/年 / 仕入先2社
  ⑦その他（事務用品・消耗品等）: ¥12,000,000/年

現在の課題:
  ・アルミ材が1社依存（代替仕入先の開拓が急務）
  ・外注加工費の単価がここ3年で25%上昇
  ・消耗品は相見積もりなしで従来業者に発注している

出力形式:
・「購買カテゴリーの分類（コスト規模 × 調達リスクの2軸マトリクス）」
  戦略購買品（重要・高額） / レバレッジ品（高額・代替可能）
  ボトルネック品（重要・代替困難） / 一般品（低額・低リスク）の分類
・「各カテゴリーの推奨購買戦略
  （アルミ材: 複数社化 / 外注加工: 見直し交渉 / 消耗品: 一括取引等）」
・「購買KPI設定案
  （コスト削減率 / 支払い条件 / 仕入先評価スコア / 代替先確保状況）」
・「今期の購買コスト削減目標の設定（年間¥140,000,000の3%削減 = ¥4,200,000の達成計画）」
・「購買部門の月次報告書フォーマット骨子」
```

## TCO（総所有コスト）分析書類を整理するプロンプト

「単価」だけでない総コストを可視化します。

```
以下の情報をもとに、購入部品・材料の
TCO（Total Cost of Ownership / 総所有コスト）分析を
整理してください。

【比較対象の仕入先2社】
品目: 精密鋼板（SPC）φ1.6mm 定尺1,800×900mm

A社（現在の仕入先）:
  単価: ¥2,200/枚
  リードタイム: 4日
  MOQ（最小発注量）: 100枚
  不良率: 0.5%（過去6ヶ月実績）
  緊急対応: 可（1日対応・追加費用なし）
  代金支払い: 月末締め翌月末払い

B社（新規候補）:
  単価: ¥1,980/枚（A社比 -10%）
  リードタイム: 7日
  MOQ: 200枚
  不良率: 2.0%（B社自己申告）
  緊急対応: 不可
  代金支払い: 前払いまたは納品時払い

当社の購買条件:
  月間使用量: 300枚/月
  不良品発生時のコスト: 不良材1枚あたり後工程での手直し¥3,000

出力形式:
・「TCO計算表（A社 vs B社の比較）」
  単価差 / 不良コスト差 / 在庫保有コスト差 / 資金調達コスト差 / 緊急対応価値の合算
・「隠れたコスト（単価以外の要素）の金額換算
  （不良率差: 1.5% × 300枚/月 × ¥3,000 = ¥13,500/月 等）」
・「TCO総計の比較（A社 vs B社 / 月次・年次）と結論の推薦」
・「B社採用の場合のリスクと対策
  （7日リードタイムへの対応 / 安全在庫の増加 / 不良率の実績確認）」
・「新規仕入先を採用する際の試験期間の設計
  （サンプル受入 → 試行発注 → 本採用 の3段階評価）」
```

## グローバル調達の検討書類を整理するプロンプト

海外仕入先の活用を検討するための書類を整備します。

```
以下の情報をもとに、海外（アジア）からの
調達を検討するための比較分析書類を整理してください。

【グローバル調達の検討背景】
検討理由: 国内仕入先からの価格が年間¥10,000,000超える特定部品について、
海外調達でコスト削減できないか経営から打診
対象品目: 汎用ネジ・ボルト（M3〜M12 / SUS304 / 年間購入量: 2,000,000個）
現行単価: ¥8/個（国内仕入先A社）
目標単価: ¥5/個（海外調達の場合）

過去の海外調達の経験: なし
懸念点:
  ①品質（ISO認証の確認 / 材料証明書の有無）
  ②リードタイム（海上輸送 4〜6週 / 最小発注量 大きい場合あり）
  ③コミュニケーション（言語・時差）
  ④為替リスク（USD建て取引の場合）
  ⑤関税・輸入費用の追加コスト
  ⑥品質クレーム発生時の対応

出力形式:
・「グローバル調達の総コスト試算（単価差 vs 追加コスト）」
  輸送費 / 関税（関税率確認方法） / 在庫増加コスト / 品質リスクコスト / 為替ヘッジコスト
・「海外仕入先の評価・選定チェックリスト
  （ISO認証 / 財務健全性 / 第三者品質監査の可否 / 過去実績）」
・「試験調達（テスト発注）計画書の骨子
  （サンプル受入 → 試験期間6ヶ月 → 本格採用判断）」
・「海外仕入先との基本取引条件（英語でのメール文例）」
・「経営への提案書骨子（海外調達メリット・リスク・推奨方針）」
```
