製造業ではISO 14001の環境マネジメントシステムの一環として、環境内部監査・法令遵守確認が求められます。AIを使って環境内部監査チェックリスト・法令遵守評価・環境目標の実績報告書類を効率的に整理できます。

---

## 環境内部監査チェックリストを作るプロンプト

ISO14001の内部監査で確認する項目を整理します。

```
以下の情報をもとに、ISO 14001の環境内部監査の
チェックリストを作成してください。

【監査の概要】
監査対象: 金属プレス加工工場（ISO 14001: 2015認証取得）
監査員: 品質・環境管理部 ○○さん（内部環境監査員資格取得済み）
監査範囲:
  ・製造工程（プレス・バリ取り・検査）
  ・化学物質管理（切削油・洗浄剤の取り扱い）
  ・廃棄物管理（産業廃棄物・一般廃棄物の分別・管理）
  ・エネルギー管理（電力・LPG）

【監査で確認するISO14001の主要要求事項】
  ・環境側面の特定と評価（重要な環境側面の最新状態）
  ・環境目標と実績（省エネ・廃棄物削減の目標達成状況）
  ・緊急事態への準備と対応（油漏れ・火災等の手順の整備）
  ・法令順守の評価（適用される環境法令の遵守状況）
  ・監視・測定・分析・評価（環境パフォーマンスの測定データ）
  ・是正処置（前回監査の指摘事項への対応）

出力形式:
・「環境内部監査チェックリスト（ISO14001の要求事項別）」
・「監査の進め方（書類確認→現場確認→担当者ヒアリングの手順）」
・「監査発見事項の分類（不適合・観察・良好事例の3分類）」
・「環境内部監査報告書のフォーマット（監査日・範囲・
  発見事項・是正要求・推奨事項）」
・「是正処置フォームのフォーマット（不適合内容・根本原因・
  是正計画・完了確認）」
```

## 環境法令遵守確認（コンプライアンス評価）を整理するプロンプト

適用される環境法令の遵守状況を定期確認します。

```
以下の情報をもとに、製造業に適用される環境法令の
コンプライアンス評価表を整理してください。

【当社が対象となる主な環境法令（金属加工業）】
廃棄物関連:
  ・廃棄物処理法（産業廃棄物の適切な委託処理・マニフェスト）
  ・金属スクラップ・切削くずの処理

水・土壌関連:
  ・水質汚濁防止法（特定施設: 洗浄設備がある場合）
  ・土壌汚染対策法（油類の地下浸透防止）

大気関連:
  ・大気汚染防止法（ばい煙発生施設の届出・測定）
  ・VOC（揮発性有機化合物）の排出規制

化学物質関連:
  ・化管法（PRTR）: 特定化学物質の届出・排出量把握
  ・労働安全衛生法: 有機溶剤・特定化学物質の作業環境測定

エネルギー関連:
  ・省エネ法（エネルギー使用量の定期報告・省エネ目標）

出力形式:
・「環境法令コンプライアンス評価表（法令名・規制内容・
  当社への適用有無・現状の対応状況・評価・担当部署）」
・「未対応・要改善の法令項目の優先対応リスト」
・「法令改正情報の収集方法（環境省・都道府県HP・業界団体）」
・「コンプライアンス評価の実施頻度と記録の保管方法」
・「外部機関（行政・審査機関）からの指摘に備えた書類整備」
```

## 環境目標・環境パフォーマンスの実績報告プロンプト

年間の環境目標の達成状況を経営陣に報告します。

```
以下の情報をもとに、環境マネジメントシステムの
年次環境パフォーマンス報告書を整理してください。

【当期の環境目標と実績】
目標①「電力使用量の削減」:
  目標: 前年比 -2%（480,000kWh → 470,400kWh以下）
  実績: 461,000kWh（前年比 -3.9%）✓ 目標達成

目標②「廃棄物（産廃）の削減」:
  目標: リサイクル率 85%以上
  実績: リサイクル率 82%（金属スクラップ 92%・切削油廃液 65%）
  ✗ 目標未達成（切削油廃液のリサイクルルートが未確立）

目標③「化学物質（有機溶剤）使用量の削減」:
  目標: 前年比 -5%
  実績: -7.2% ✓ 目標達成（脱脂洗浄剤の水系洗浄への切り替え完了）

未達成目標への対応:
  廃棄物リサイクル率が未達成 → 切削油廃液の処理業者の見直し（来期対策）

出力形式:
・「環境パフォーマンス年次報告書の構成（目標・実績・
  達成率・達成要因・未達成の原因と来期対策）」
・「環境目標達成状況のグラフ化イメージ（目標 vs 実績）」
・「経営者レビューで報告するエグゼクティブサマリー
  （1ページ・信号機方式の評価つき）」
・「来期の環境目標の設定方法（継続改善の観点からの目標水準）」
・「環境報告書（取引先・地域社会への開示）の骨子」
```
