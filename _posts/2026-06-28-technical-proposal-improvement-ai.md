取引先から「コスト削減提案をしてほしい」「品質改善の提案がほしい」と依頼されることや、自社からVE（バリューエンジニアリング）提案・改善提案を積極的に行うことで受注拡大を図るケースがあります。技術提案書の「文章の構成・説明の整理」をAIで効率化できます。

---

## コスト削減提案書（VE提案書）を作るプロンプト

製造方法・材料・工程の見直しによるコスト削減を提案する文書を作ります。

```
以下の情報をもとに、取引先へのVE（バリューエンジニアリング）提案書を
作成してください。

【提案の対象】
品番・品名: （例：ブラケット A-1234）
取引先: （例：○○自動車工業 購買部 ○○様）
自社担当: （例：○○製作所 営業部 ○○、技術部 ○○）

【現状と課題】
現在の製造方法: （例：SPCC板材からプレス加工後、別工程でバリ取り・洗浄）
コスト上の課題: （例：バリ取り工程が手仕上げで工数がかかっている）

【提案する改善内容】
提案内容: （例：プレス金型の形状改善により、バリが出にくい構造に変更。
  バリ取り工程を不要または大幅に短縮できる。）
コスト削減効果: （例：1個あたりバリ取り工数を○分削減 → ○円/個のコスト削減）
品質面の効果: （例：バリに起因する外観不良リスクが低減）
実施費用・期間: （例：金型改造費 ¥○○万円 / 工期2か月）
回収期間: （例：月産○個の場合、○か月で回収）

出力形式:
・技術提案書のフォーマット（現状分析→提案内容→効果→費用・回収計画）
・「現状」と「改善後」の比較表を含める
・投資回収の試算を数値で示す
・「ご検討のほど、よろしくお願いいたします」の締め
・添付資料の案内（図面変更案・見積明細等）
```

## 品質改善提案書を作るプロンプト

不良低減・品質向上のための改善提案を取引先に出す文書を作ります。

```
以下の情報をもとに、品質改善提案書を作成してください。

【提案の背景】
（例：貴社製品の検査工程で、○○品番のバリ関連クレームが
 ○か月に○件発生していた。自社での原因分析と改善策を提案したい。）

【現状の問題】
（例：
・プレス加工後のバリ取りが目視確認のみで、基準があいまいだった
・作業者によって確認レベルにばらつきが生じていた
・月間不良率 ○%（業界目標 0.3%以下に対して超過）
）

【提案する改善策】
（例：
1. 外観検査基準書を製品ごとに作成（限度見本付き）
2. 検査員全員への統一教育（○月○日実施予定）
3. バリ高さ測定器（デジタルハイトゲージ）を導入し数値管理に切り替え
）

【期待される効果】
（例：不良率を0.5%→0.2%以下に改善。クレーム件数ゼロを目指す）

【スケジュール】
（例：○月: 基準書作成・○月: 教育実施・○月: 効果測定）

出力形式:
・品質改善提案書のフォーマット
・問題点→原因→対策→効果→スケジュールの論理的な流れ
・改善前後の比較（不良率グラフのプレースホルダー）
・「次回訪問時に詳細をご説明させてください」の締め
```

## 納期短縮提案書を作るプロンプト

現在の納期を短縮できることを取引先に提案する文書を作ります。

```
以下の情報をもとに、納期短縮の技術提案書を作成してください。

【現状の納期と背景】
品番・品名: （例：○○部品）
現行納期: （例：受注後 15営業日）
取引先のニーズ: （例：将来的に10日以内の対応が求められる可能性があると聞いた）

【当社が提案できる改善内容】
（例：
・素材の在庫化: 主材料（SPCC t1.6mm）を最小ロット分を常時在庫化することで、
  材料調達の2〜3日を圧縮
・工程の並行化: 外注（熱処理）の発注を加工開始と同時に行うことで
  待ち時間を削減
・専用ラインへの切り替え（件数が増えた場合）
）

【提案後の納期】
（例：月産○個以上の場合、10営業日への短縮が可能）
（例：ただし素材在庫コストとして月 ¥○○の負担が発生する見込み）

出力形式:
・「現状の課題→改善内容→改善後の納期→条件」の構成
・「この提案のメリット（取引先にとっての価値）」を先に述べる
・コスト面での条件（在庫リスク・追加費用）も正直に記載
・「詳細は別途ご相談させてください」の締め
```

## 提案後のフォローメール（結果確認）を作るプロンプト

提案書を提出した後、一定期間後にフォローするメールを作ります。

```
以下の情報をもとに、技術提案後のフォローメールを作成してください。

【提案の内容】
（例：○月○日に品番A-1234のVE提案書を提出。金型改造による
 バリ取り工程削減を提案した。）

【経過日数・状況】
（例：提出から3週間。特に返答がない状態）

【フォローの目的】
（例：検討状況を確認し、質問があれば回答したい。否定的なら理由を聞きたい。）

出力形式:
・取引先担当者へのビジネスメール
・「その後、ご検討はいかがでしょうか」という自然な聞き方
・「追加資料や説明が必要であればすぐに対応します」と伝える
・催促感がなく、次の面談につなげるトーン
```
