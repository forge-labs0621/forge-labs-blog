ChatGPTを使い始めた製造業の担当者が「もっと精度の高い回答を引き出したい」と感じたときに役立つ、上級プロンプトテクニックを紹介します。

---

## テクニック1: 役割（ペルソナ）を明確に設定する

AIに専門家の立場から回答させることで、専門性の高い回答が得られます。

```
あなたはISO 9001の審査に30年の経験を持つ
品質管理の専門家です。

【背景】
私は金属加工の中小メーカーで品質管理を担当しています。
初めてISO 9001の内部監査員を任されました。

【質問】
製造工程の内部監査では、どの書類を最も
重点的に確認すべきですか？

実務経験に基づいた具体的なアドバイスをお願いします。
```

---

## テクニック2: 出力フォーマットを細かく指定する

回答の形式を指定することで、そのまま使える書類が完成します。

```
以下の条件で、外観検査基準書を作成してください。

【条件】
・A4サイズで印刷できる表形式
・列: 検査項目 / 判定基準 / 測定方法 / 良品例 / 不良品例
・行: 外観5項目以上
・製造現場の作業者が一目でわかる平易な言葉で記載
・専門用語には括弧書きで説明を追記

【製品】
プレス成形品（自動車部品用ブラケット）
```

---

## テクニック3: 段階的に思考させる（Chain of Thought）

複雑な問題は「考え方の手順」を踏ませることで精度が上がります。

```
以下の問題を段階的に考えてください。

【状況】
プレス工程で、先週から寸法不良の発生率が
通常の0.5%から3.2%に増加しています。

考え方の手順:
Step 1: 寸法不良が増加した可能性がある原因を
       「4M（人・機械・材料・方法）」の視点で列挙
Step 2: それぞれの原因の可能性（高/中/低）を評価
Step 3: 優先的に確認すべき原因TOP3を選定し、
       確認方法を具体的に説明
Step 4: 原因特定後に取るべき応急処置と恒久対策を提案
```

---

## テクニック4: 具体的な数値・事例を含めて質問する

抽象的な質問より、数値や具体例を含めた方が実用的な回答が得られます。

**抽象的な質問（効果低）:**
```
原価を下げる方法を教えてください。
```

**具体的な質問（効果高）:**
```
当社の製品（単価¥2,800、材料費¥1,200・加工費¥900・
外注費¥350・管理費¥350）について、
外注費¥350の削減余地を検討してください。

月産500個のバリ取り工程を外注しており、
社内でやる場合の必要な設備・人員・コストと
外注継続した場合の比較を教えてください。
```

---

## テクニック5: 複数の案を比較検討させる

「A or B」ではなく「複数案を提示して比較」させることで、判断が容易になります。

```
以下の課題に対して、3つの解決策（A案・B案・C案）を
それぞれ提案し、「コスト・効果・実現難易度」の
3軸で比較してください。

【課題】
プレス機の型摩耗が早く、型の交換サイクルが
計画（10万ショット）の半分（5万ショット）しか
もっていない。

各案のフォーマット:
・解決策の概要
・初期コスト
・月次の効果（削減額）
・実現に必要な期間
・リスク
```

---

## テクニック6: 「批判的に見直す」を活用する

作成した書類を自己評価させ、品質を高めます。

```
以下の不適合報告書の下書きを批判的に見直し、
改善点を3つ指摘してください。

【下書き】
（自分で作成した不適合報告書を貼り付け）

見直しの観点:
① 根本原因の分析が十分か
② 再発防止策が具体的で実行可能か
③ 読み手（顧客・審査員）が理解しやすい文章か
```

---

## テクニック7: 業界・法令の文脈を明示する

製造業の特有の規制・慣習を前提に含めます。

```
製造業の文脈で回答してください。
具体的には:
・自動車部品サプライヤー（Tier2）として
  顧客からIATF 16949を要求されている
・顧客の変更承認（3変更通知）ルールに従う必要がある
・トレーサビリティは製品番号・ロット・
  製造日まで要求されている

この前提のもとで、溶接材料を変更する際の
社内手続きと顧客への報告書類を整理してください。
```

---

## テクニック8: Few-shot（例示）プロンプトで精度向上

良い例と悪い例を示すことで、AIが期待するスタイルを学びます。

```
以下の「良い例」「悪い例」を参考に、
提示した状況への是正処置文を書いてください。

【悪い例（避けるべき文章）】
「今後は注意して作業します」
（具体性がなく、何をするか不明）

【良い例（目指すべき文章）】
「作業者○○が毎日17:00に記録台帳を確認し、
記録漏れがあった場合は当日中に記入・班長に
報告する。班長は翌朝の朝礼で確認状況を
チームに共有する。」（誰が・何を・いつまでに・
確認方法まで具体的）

【状況】
検査工程で、検査記録の入力忘れが月2〜3回発生している。
```

---

## テクニック9: 要約と詳細の2段階生成

長い文書は要約と詳細を分けて作らせます。

```
製造業の「化学物質管理規程」について、
2段階で作成してください。

Step 1:
まず「目次と各章の要点（1行以内）」だけを作成してください。
（全体像を確認するため）

（Step1の結果を見てOKなら、以下を依頼）

Step 2:
承認した目次に基づいて、各章の本文を
作業者が実際に使えるレベルの詳細で書いてください。
```

---

## テクニック10: 「もし〜だったら」で想定外を検討する

リスク分析や対策立案に有効です。

```
もし以下の状況が発生したとしたら、
それぞれどのように対応しますか？

製品: 自動車部品（シート取付ブラケット）
通常の品質管理体制は整っている前提で:

シナリオA: 出荷後に顧客の組立ラインで
  自社製品の寸法不良が発覚した（出荷数 500個）

シナリオB: 自社製品が原因で顧客の自動車に
  リコールが発生したと顧客から通知があった

シナリオC: 製造工程で使用していた溶剤が
  有害物質規制の対象になったと判明した

各シナリオについて:
  ・最初の30分でやること
  ・24時間以内にやること
  ・1週間以内に整備すべき書類・体制
```
