製造業では「溶接」「熱処理」「表面処理（めっき・塗装）」などの「特殊工程」は、結果を後から全数検査で確認できないため、ISO 9001などで工程のコントロールと記録が特に求められます。AIを使ってこれらの記録・管理文書を効率よく整理できます。

---

## 溶接工程管理記録（WPS/PQR）の整理に使えるプロンプト

溶接作業指示書（WPS）の骨子や溶接記録の整理に使えます。

```
以下の情報をもとに、溶接工程管理記録（作業記録）の文章を整理してください。

【溶接の基本情報】
溶接方法: （例：被覆アーク溶接（SMAW）/ MIG溶接（GMAW））
母材種類: （例：SS400 / SUS304 / A5052（アルミ））
板厚・形状: （例：SS400 t9mm / T字継手）
溶接材料: （例：軟鋼用溶接棒 JIS Z 3211 E4316）

【作業記録（現場が記録した内容）】
作業日: （例：2026年7月15日）
溶接士: （例：○○（JIS Z 3801 専門級資格保有））
溶接条件: （例：電流: 130A、電圧: 24V、溶接速度: 約20cm/分）
予熱・後熱: （例：予熱なし（母材温度15℃以上確認）、後熱なし）
外観確認: （例：全溶接線を目視確認。溶け込み良好、アンダーカットなし）

出力形式:
・溶接作業記録シートのフォーマット（日付・担当者・条件・確認結果）
・「溶接士の資格番号・有効期限確認欄」を含める
・外観確認の結果を合否で記載（合格 / 要手直し / 不合格）
・承認者欄
```

## 熱処理管理記録を整理するプロンプト

焼入れ・焼戻しなどの熱処理作業の記録を整理します。

```
以下の情報をもとに、熱処理管理記録の文章を整理してください。

【熱処理の基本情報】
処理種類: （例：焼入れ・焼戻し（QT処理）/ 浸炭焼入れ / 焼鈍し）
対象製品・品番: （例：軸部品 品番SH-2345、材質SCM435）
要求仕様: （例：硬さ HRC 45〜52、表面処理なし）

【処理の記録（外注先または自社）】
処理実施日: （例：2026年7月20日）
処理設備: （例：バッチ式焼入れ炉 FUR-001）
加熱温度・保持時間: （例：830℃ × 30分）
焼戻し: （例：180℃ × 90分）
冷却方法: （例：油冷）
処理後の確認: （例：抜き取り3個の硬さ測定: HRC 47 / 49 / 48 → 全数規格内）

出力形式:
・熱処理記録表フォーマット
・「温度チャート添付」のプレースホルダー欄（炉内温度の記録を添付する運用）
・硬さ測定結果を表形式で（測定位置・測定値・判定）
・外注先（外注で実施の場合）の証明書添付チェック欄
```

## 表面処理（めっき・塗装）の指示書・記録を整理するプロンプト

めっき・塗装工程の作業指示と処理証明の文書化に使えます。

```
以下の情報をもとに、表面処理指示書または処理証明書の文章を整理してください。

【表面処理の基本情報】
処理種類: （例：三価クロメート処理（亜鉛めっき後） / ユニクロ / 無電解ニッケルめっき）
対象製品・品番: （例：プレス部品 品番A-1234）
要求仕様: （例：RoHS2.0対応品。めっき厚さ 8μm以上、腐食試験 SST 72時間以上）
外注先: （例：○○めっき工業）

【処理結果の確認（外注先の証明書内容）】
- めっき厚さ測定: 3点平均 12μm（規格8μm以上 → 合格）
- 塩水噴霧試験: 96時間後に白錆発生なし（規格72時間以上 → 合格）
- RoHS適合: 六価クロム不検出（証明書番号: RoHS-2026-○○）

出力形式:
・社内保管用の「表面処理確認記録」フォーマット
・外注先の証明書番号・発行日の記入欄
・確認項目と合否の表（めっき厚さ / 腐食試験 / 有害物質）
・受入確認者サイン・確認日
```

## 特殊工程作業員の資格管理台帳を作るプロンプト

溶接士・非破壊検査員などの技能資格の管理台帳を整理します。

```
以下の情報をもとに、特殊工程作業員の資格管理台帳の記載内容を整理してください。

【対象の資格・作業員情報】
氏名: （例：○○ ○○）
取得資格:
- JIS Z 3801 溶接技能者（被覆アーク溶接 専門級）
  取得日: 2022年4月、有効期限: 2025年4月（→ 更新が必要）
- JIS Z 3841 半自動溶接技能者（MIG溶接 基本級）
  取得日: 2023年10月、有効期限: 2026年10月

出力形式:
・資格管理台帳の1レコード（台帳に追記できる形式）
・「氏名 / 資格名 / 取得日 / 有効期限 / 更新予定日 / 担当工程 / 備考」の欄
・有効期限が3か月以内に迫っているものに「★更新要確認」の注記
・資格の喪失・更新の手続き注記
```

## AI活用の注意

- 溶接記録・熱処理記録の数値（温度・時間・電流値・測定値）は実際の機器計測値を使用してください
- ISO 9001 での特殊工程管理は「工程パラメータの継続的モニタリング」が求められます。AIはその記録の文章整理には使えますが、パラメータ管理そのものは設備・人の管理で行ってください
- 外注先の処理証明書は正規の書類であることを確認してから保管してください
