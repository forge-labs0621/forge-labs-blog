製造業の購買・調達担当者にとって、外注先や仕入先への発注書・注文書の作成は日常業務のひとつです。「毎回似たような文書を作り直している」「文言のバラつきを統一したい」という方向けに、AIを使って下書きを素早く仕上げる方法をまとめます。

## 発注書・注文書でAIが使いやすい場面

AIは以下のような作業が得意です。

- **定型的な文書の下書き作成**（品番・数量・納期をもとにした発注書の文面）
- **丁寧な言い回しへの変換**（箇条書きのメモを取引先向けの文章に整える）
- **過去の文書パターンの横展開**（別の品番・納期・数量でほぼ同じ文書を作る）

機密情報（取引先の名前・価格・仕様の詳細）は入力しないように注意してください。使い方はプレースホルダーを活用して、文面の型だけAIに作らせるのが安全です。

## 基本のプロンプト（発注書の文面作成）

---

```
以下の条件で、外注先・仕入先への発注書の本文を作成してください。

【条件】
品目: （例：部品名・品番 ※機密の場合は「部品A」など仮称で入力）
数量: （例：100個）
希望納期: （例：2026年7月15日）
納品場所: （例：自社工場の受入口）
特記事項: （例：梱包仕様は別途図面に従うこと、検査証明書を添付すること）
差出人（発注元）: （例：製造部 購買担当）

【出力形式】
・タイトル「発注書」
・発注番号・日付・先方名の欄（プレースホルダーでOK）
・品目・数量・単価（空欄）・納期・納品場所を表形式で
・備考・特記事項欄
・発注元の署名欄
```

AIが作ったものを土台に、実際の品番・価格・取引先名をExcelやWordに入力すれば完成です。

## 外注先へのメール文

発注書に添付するメールの文面も、AIで下書きできます。

```
以下の条件で、外注先への発注メールの本文を作成してください。

【条件】
目的: 部品の製作を依頼する
添付書類: 発注書・仕様図面
確認してほしい点: 納期が確保できるかどうか
回答期限: （例：3営業日以内）
文体: 丁寧・ビジネス標準

【注意】
会社名・担当者名はすべてプレースホルダー（○○様、△△株式会社等）で出力してください。
```

出力の例：

---

件名：部品製作のご依頼について（発注書添付）

○○株式会社
○○様

お世話になっております。△△株式会社、製造部の□□でございます。

今回、下記の件につきまして、製作のご依頼をさせていただきたく、ご連絡いたしました。

添付の発注書・仕様図面をご確認のうえ、**○月○日（△曜日）までに**納期のご確認をいただけますでしょうか。

ご不明な点がございましたら、お気軽にお申し付けください。
何卒よろしくお願いいたします。

## 急ぎの発注で納期を交渉するメール

納期が厳しいときのメール文も、プロンプトで素早く作れます。

```
取引先に対して、通常より短い納期での対応をお願いするメールを作成してください。

状況: 急な受注追加で、通常より2週間早い納期になってしまった
依頼内容: 対応可否と、可能な場合の費用・条件の確認
文体: 丁寧・恐縮気味に、でも要件は明確に
```

## 注意事項

- 発注書の最終確認・押印・送付は必ず担当者が行ってください
- 単価・合計金額などの数字はAIに頼らず、社内の見積もりデータから正確に入力してください
- 取引先との合意内容や契約条件はAIの文面で上書きしないよう注意してください
