製造業の現場で「品質管理チェックリストを作りたいけど、何を入れればいいかわからない」「毎回ゼロから作っていて時間がかかる」という担当者向けに、AIを使ってチェックリストを素早く作る方法をまとめます。

## AIでチェックリストを作るメリット

手書きやExcelでチェックリストを1から作ろうとすると、項目の抜け漏れが出たり、文章の整合性を取るのに時間がかかったりします。AIに作成を任せると：

- **項目の抜け漏れを減らせる**（AIが一般的な点検観点を網羅してくれる）
- **現場の状況に合わせてすぐカスタマイズできる**
- **文章の統一感が出る**

ただし、AIが出したものをそのまま使うのではなく、必ず自社の実情に合わせて人が確認・修正することが前提です。

## 基本的なプロンプトの使い方

### ステップ1: 作りたいチェックリストの条件を整理する

まず以下を決めます。

- **対象工程**（例：機械加工、溶接、塗装、組立など）
- **チェックのタイミング**（作業前・作業中・作業後、シフト切り替え時など）
- **チェックする人**（作業者本人、リーダー、品質担当など）

### ステップ2: AIにプロンプトを渡す

以下のプロンプトをChatGPTやClaudeに貼り付けて、条件部分を書き換えてください。

---

```
以下の条件で、製造現場の品質管理チェックリストを作成してください。

【条件】
対象工程: （例：機械加工・旋盤）
チェックタイミング: （例：作業開始前・作業終了後）
チェックする人: （例：作業者本人）
特に確認したいポイント: （例：寸法精度・工具の状態・安全）

【出力形式】
チェック項目を10〜15個、「確認内容」「OK/NG」「備考」の3列で表形式にまとめてください。
現場の作業者が迷わず確認できる、具体的な文言にしてください。
```

---

### ステップ3: 出力を確認・調整する

AIが出したリストを印刷して現場のベテランに見せ、「抜けているもの・実情に合わないもの」を確認します。2〜3回フィードバックを繰り返すと、自社に合った形に仕上がります。

## 実際の出力例（機械加工・作業前チェック）

上のプロンプトで機械加工・作業前という条件を渡すと、以下のようなリストが出てきます（簡略版）。

---

| No | 確認内容 | OK/NG | 備考 |
|----|---------|-------|------|
| 1 | 加工プログラム番号が指示書と一致しているか | | |
| 2 | 工具の種類・刃径が加工仕様と合っているか | | |
| 3 | 工具の摩耗・欠けがないか目視確認 | | |
| 4 | ワークの固定（チャック・クランプ）が確実か | | |
| 5 | 材料の品番・ロット番号が指示書と一致しているか | | |
| 6 | 冷却液の量・濃度が適正か | | |
| 7 | 安全カバー・防護板が正しく設置されているか | | |
| 8 | 前工程からの引き継ぎ事項の確認 | | |

---

これを自社の品番・管理基準に合わせて修正すれば、すぐ使えるチェックリストになります。

## チェックリストの種類別プロンプト例

目的に合わせてプロンプトをカスタマイズすることで、様々な種類のリストを作れます。

### 最終検査用チェックリスト

```
完成品の最終検査で使うチェックリストを作ってください。
製品: （品名・品番）
検査項目: 外観、寸法、機能、梱包
出荷先で問題になりやすいポイント: （例：キズ、寸法公差、附属品の入れ忘れ）
```

### 設備点検チェックリスト

```
プレス機の始業点検チェックリストを作ってください。
確認観点: 安全装置、油脂類、異音・振動、清掃状態
点検者: オペレーター（資格不要で確認できる内容のみ）
```

### 工程引き継ぎチェックリスト

```
シフト交代時の工程引き継ぎチェックリストを作ってください。
引き継ぐ情報: 生産進捗、品質異常、設備の状態、残材の処理、翌シフトへの申し送り
```

## AIを使う際の注意点

- 出力はあくまで**たたき台**です。最終的には現場の実情に詳しい人が確認・承認してください
- **機密情報（顧客名、図面番号、工法の詳細）はAIに入力しないでください**
- 定期的に見直しを行い、現場の変化に合わせて更新することが重要です
