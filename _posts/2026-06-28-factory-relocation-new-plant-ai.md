工場移転・新工場立ち上げは製造業にとって大きな経営イベントです。移転先の選定から顧客への報告・許認可取得・設備移設・生産の立ち上げまで、多くの書類が必要になります。AIを使って計画書類の文書化を効率化できます。

---

## 工場移転計画書の骨子を整理するプロンプト

経営会議・取引先・金融機関に提示する移転計画書を整理します。

```
以下の情報をもとに、工場移転計画書の骨子を
作成してください。

【移転の背景と目的】
移転理由: （例: 現工場の老朽化・耐震性不足 / 受注増加に伴う
  生産能力拡張 / 人員確保のため立地を変更）
移転先: （例: 同一市内の工業団地 / 隣接市の民間貸工場）
移転時期: （例: 2027年○月に仮操業開始 → 2027年○月に完全移転）

【移転のスコープ】
設備: 工作機械○台・プレス機○台・搬送設備一式
人員: 従業員○名（うち移転に伴う異動対象: ○名）
営業中断: 最小化を目指す（移転期間中の生産継続方法を検討中）

【移転に必要な許認可・届出（主な例）】
・建築確認申請（新工場の建設の場合）
・消防署への防火管理者届出・消防設備の設置届
・工場立地法の届出（特定工場の新設・変更）
・環境関連の届出（大気汚染防止法・水質汚濁防止法）
・雇用保険・社会保険の事業所移転届（年金事務所・ハローワーク）

出力形式:
・「工場移転計画書の構成（目的・背景・スケジュール・体制・
  コスト・リスク対応・成功の定義）」
・「移転プロジェクトのWBS（主要タスクの洗い出し）」の骨子
・「移転に関する許認可・届出チェックリスト（時系列）」
・「移転コスト試算フォーマット（設備移設・内装・許認可・
  生産停止コスト等の項目一覧）」
・「金融機関（設備資金融資）への移転計画説明の骨子」
```

## 顧客・取引先への工場移転報告文を作るプロンプト

移転に伴い顧客・仕入先に送る報告文と品質保証の説明を整理します。

```
以下の情報をもとに、工場移転に関する顧客・取引先への
報告文を作成してください。

【報告内容の概要】
移転先住所: （例: ○○県○○市○○ ○番地）
移転時期: （例: 2027年○月○日（仮操業）→ 2027年○月○日（完全移転））
新住所での生産開始: （例: 2027年○月○日）
電話番号・FAX: 変更なし（転送設定あり）

【顧客への懸念事項への対応】
納期への影響: （例: 移転期間中（○月〜○月）は在庫を事前に
  積み増ししてリスクを最小化。万一影響が出る場合は事前に個別連絡）
品質への影響:
  ・工作機械は専門業者による移設・精度確認後に生産再開
  ・量産再開前に品質確認品（サンプル）を提出する予定
顧客への承認:
  ・製造場所の変更として4M変更（場所変更）扱いになる場合は
    事前に顧客の変更承認を取得する

ISO認証・品質認証:
  ・ISO 9001・IATF 16949は移転後も維持の予定（認証機関に変更申請中）

出力形式:
・「顧客への工場移転のご報告（正式通知文）」（社長名で発信する文章）
・「品質保証・納期保証の説明補足資料の骨子」
・「4M変更（場所変更）の顧客承認申請書のフォーマット」
・「仕入先・外注先への移転通知文」
・「移転に伴う名刺・封筒・会社案内等の変更チェックリスト」
```

## 新工場立ち上げの生産準備チェックリストを作るプロンプト

新工場での生産を開始するまでの準備事項を整理します。

```
以下の情報をもとに、新工場（または移転先工場）での
生産立ち上げに向けた準備チェックリストを作成してください。

【立ち上げの概要】
生産開始予定: ○年○月
主要品目: （例: 精密金属プレス加工品 / 月産○○万個）
新工場の特徴: 既存設備の移設 + 新規設備2台の導入

【立ち上げまでに必要な主な作業】
設備面:
  ・設備の搬入・据付・精度確認
  ・電源工事・エアー配管工事の完了
  ・ユーティリティ（電気・ガス・水道）の供給確認

品質面:
  ・工程フロー・QC工程表の新工場版への改訂
  ・初物検査・トライアル生産の実施
  ・顧客承認サンプルの提出

人員面:
  ・作業者の配置と役割の確定
  ・新工場のレイアウト・動線に慣れるための事前訓練
  ・安全教育（新工場の緊急避難経路・消火器の場所等）

行政・法令面:
  ・消防署への新設届の完了確認
  ・使用前検査（クレーン等）の実施
  ・地元行政・近隣住民への説明（振動・騒音等の配慮）

出力形式:
・「新工場立ち上げ準備チェックリスト（工程別・担当別）」
・「試作・トライアル生産の評価基準と合格判定フォーマット」
・「生産開始判定会議のアジェンダと判定基準（Go/NoGoの定義）」
・「立ち上げ後3か月間の重点モニタリング指標」
・「地域コミュニティへの配慮事項（騒音・振動・交通量増加等）の
  整理と対応策」
```
