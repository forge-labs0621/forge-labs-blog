製造業は廃棄物処理法（廃棄物の処理及び清掃に関する法律）に基づき、産業廃棄物の適正処理と管理票（マニフェスト）の保存義務があります。AIを使って廃棄物管理書類の整備と記録管理を効率化できます。

---

## 産業廃棄物の種類と処理業者の管理一覧を作るプロンプト

自社から排出される産業廃棄物と処理業者の契約状況を整理します。

```
以下の情報をもとに、自社の産業廃棄物管理一覧表を
作成してください。

【自社工場から排出される主な廃棄物】
（例:
  ① 廃プラスチック類: プレス加工の端材・使用済みパッキン
     月間排出量: 約○kg / 処理方法: 中間処理（焼却）→ 業者A
  ② 廃油: 切削油・潤滑油の廃液
     月間排出量: 約○L / 処理方法: 再生業者B（再資源化）
  ③ 金属くず: プレス端材・切粉
     月間排出量: 約○t / 処理方法: 再生業者C（有価売却）
  ④ 廃酸: 洗浄・脱脂処理後の廃液（特別管理産業廃棄物）
     月間排出量: 約○L / 処理業者: 許可業者D
  ⑤ 汚泥: 廃水処理から発生する汚泥
     月間排出量: 約○kg / 処理方法: 中間処理（脱水）→ 業者E
）

【処理業者の管理状況】
・業者Aの産業廃棄物収集運搬許可証: 最終確認日 ○○年○月
・業者Bの産業廃棄物処分業許可証: 最終確認日 ○○年○月
（許可証の更新確認を年1回実施しているが、期限管理が属人的）

出力形式:
・廃棄物管理一覧表のフォーマット（廃棄物種類・廃棄物コード・
  特別管理産業廃棄物の有無・排出量・処理業者・許可証期限）
・「処理業者の許可証更新確認スケジュール」の整理方法
・「産業廃棄物の品目と廃棄物コード（産廃コード）の対応表」の骨子
・「違法な廃棄物処理に自社が巻き込まれないためのチェックポイント」
```

## 産業廃棄物管理票（マニフェスト）の管理記録を整理するプロンプト

法令で義務付けられた管理票（マニフェスト）の保存・照合を整理します。

```
以下の情報をもとに、産業廃棄物管理票（マニフェスト）の
管理・保存体制を整理してください。

【現在のマニフェスト管理方法】
使用区分: （例: 紙マニフェスト使用（電子マニフェストは未導入））
保管方法: （例: 月別にファイリング。5年間保存の義務あり）
課題:
  ・D票（処理完了報告）が戻ってこない業者がある
    （90日以内に返送が義務。超過時は都道府県に報告義務）
  ・E票の管理がバラバラで、年度末の集計に時間がかかる
  ・マニフェストの発行漏れが過去にあった（小口廃棄物の運搬時）

【管理したい書類】
A票（排出事業者控): 5年間保存
B2票（運搬業者→排出事業者に返送）
D票（処分業者→排出事業者に返送: 90日以内）
E票（最終処分完了報告: 180日以内に返送）

出力形式:
・マニフェスト管理台帳のフォーマット（発行番号・廃棄物種類・
  数量・処理業者・各票の返送期日・返送確認日）
・「D票・E票の返送期限アラート管理方法」（超過前に気づく仕組み）
・「マニフェスト発行漏れを防ぐ確認フロー」
・「年間の廃棄物処理実績の集計フォーマット（行政報告用）」
・「電子マニフェストへの移行を検討する際の比較ポイント」
```

## 廃棄物処理に関する行政報告書を整理するプロンプト

年1回の産業廃棄物処理状況の報告義務（多量排出事業者）を整理します。

```
以下の情報をもとに、産業廃棄物処理に関する行政報告書類を
整理してください。

【自社の廃棄物排出規模】
（例:
  年間排出量: 廃油○t・廃プラ○t・金属くず○t
  合計: ○t/年
  ※多量排出事業者（特定産業廃棄物: 50t以上）に該当する
    かどうか要確認）

【行政手続きの状況】
・産業廃棄物管理票交付等状況報告書: 毎年6月末までに都道府県知事へ
  （前年1〜12月の排出・処理状況を集計して報告）
  現状: （例: 担当者が集計に3日かかっている）

・保管場所の届出（改正廃棄物処理法）:
  一定量以上を工場内に保管する場合、届出が必要
  現状: （例: 未確認）

出力形式:
・「産業廃棄物管理票交付等状況報告書」の記入要領と
  集計表のフォーマット
・「年間廃棄物排出量の月次集計シート（報告書作成を楽にする）」
・「廃棄物保管場所の現状確認チェックリスト（届出要否の判定）」
・「廃棄物処理法の主な義務と違反した場合のリスク（罰則）」の要点
・「廃棄物管理担当者の引き継ぎ資料のフォーマット骨子」
```
