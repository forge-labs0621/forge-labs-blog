製造業では、新入社員や異動してきた社員へのOJT（On-the-Job Training）が重要です。しかし、「指導はできても記録を書く時間が取れない」「評価の文章を毎回ゼロから考えるのが面倒」という現場担当者は少なくありません。AIを使ってOJT指導記録・教育訓練記録の下書きを効率よく作る方法を紹介します。

## OJT記録でAIが役立つ場面

- **訓練計画書の作成**（対象スキル・習得目標・期間を渡すと、計画表の下書きを作ってくれる）
- **評価コメントの下書き**（「まだ一人では難しい」などのメモを評価表向けの文章に変換）
- **指導履歴の文章化**（日々のメモを月次のOJT報告書にまとめる）

ISO 9001やIATF 16949などの品質マネジメント規格では、教育訓練の計画・実施・効果確認の記録が求められます。AIで記録の文章部分の手間を減らすことで、記録の習慣化につなげることができます。

## 教育訓練計画書を作るプロンプト

---

```
以下の条件で、製造現場の新人向けOJT教育訓練計画書を作成してください。

【対象者】
（例：入社○年目の新人、異動者、特定工程の未経験者）

【習得させたいスキル・工程】
（例：旋盤の基本操作、品質検査の目視判定、梱包・出荷作業）

【訓練期間】
（例：入社後1ヶ月間、2026年7月1日〜7月31日）

【指導担当者の経験】
（例：担当工程10年のベテラン社員が1名指導）

【出力形式】
・週ごとの習得目標と実施内容
・評価基準（一人で問題なく作業できる/監督のもとで作業できる/未習得）
・最終評価の確認欄（指導担当者・受講者・上長のサイン欄）
```

## 出力例（旋盤基本操作 1ヶ月計画）

---

**教育訓練計画書**

対象者：○○ / 担当工程：機械加工（旋盤） / 期間：2026年7月1日〜31日

| 週 | 習得目標 | 主な実施内容 |
|----|---------|------------|
| 第1週 | 安全知識・機械の基本構造の理解 | 安全教育、機械各部名称の学習、起動・停止手順 |
| 第2週 | 基本的な旋削作業（指導のもとで） | 外径旋削、端面加工の実施（指導員立会い） |
| 第3週 | 計測・品質確認の習得 | ノギス・マイクロメーターの使い方、自主検査の実施 |
| 第4週 | 一人作業と振り返り | 単独作業トライアル、評価・フィードバック |

評価基準：
- ◎ 一人で確実に作業できる
- ○ 確認しながら作業できる
- △ 補助が必要
- × 未習得

## 日々の指導記録を書くプロンプト

毎日の指導内容を短いメモに残しておき、まとめてAIで文章化する方法です。

```
以下の指導メモをもとに、OJT指導記録書（1日分）を作成してください。

【日付】
2026年7月○日

【指導メモ（箇条書き）】
・今日は旋盤の外径旋削を1時間一緒にやった
・切削条件の意味（回転数・送り速度）を説明した
・最初は工具の取り付けに手間取ったが2回目は一人でできた
・安全確認（保護具・カバー）は毎回できている

【出力形式】
・実施した指導内容（2〜3文）
・習熟度の評価コメント（良かった点・課題点）
・翌日の指導予定
```

## 月次のOJT報告書をまとめるプロンプト

月末に1ヶ月分の指導記録をまとめるときにも使えます。

```
以下の1ヶ月分のOJT指導記録メモをもとに、月次OJT報告書を作成してください。

【対象者】○○（入社○ヶ月目）
【対象工程】旋盤加工

【1ヶ月の記録概要】
・1週目: 安全教育、機械の構造学習。安全意識は高い。
・2週目: 旋削作業開始。工具取り付けに時間がかかる。
・3週目: 計測ができるようになった。自主確認の習慣も定着。
・4週目: 単独作業で外径旋削は概ねOK。端面は要追加練習。

【出力内容】
・今月の習熟度のまとめ（全体評価）
・良かった点・伸びた点
・来月の課題・継続指導が必要な点
・評価者コメント欄
```

## 評価コメントが書けないときのプロンプト

「評価は○△×でつけたけど、コメントが書けない」という場合も、AIに手伝ってもらえます。

```
以下の評価をもとに、OJT評価シートのコメント欄に入れる評価文を作成してください。

評価: 旋盤外径旋削 ○（確認しながら作業できる）
状況: 基本手順は覚えているが、加工寸法に誤差が出ることがある。焦らず丁寧に
    確認すれば問題ない水準。本人の安全意識は高く、周囲への声かけもできている。

コメントの形式: 2〜3文、ポジティブな表現を基本にしながら課題も明記する。
```
