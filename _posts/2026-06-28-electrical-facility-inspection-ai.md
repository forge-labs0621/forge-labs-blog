製造業の工場では、電気設備（自家用電気工作物）の定期点検・消防設備の定期点検が法令で義務付けられています。AIを使って点検記録・報告書・点検スケジュールの管理書類を効率化できます。

---

## 電気設備定期点検の記録フォーマットを作るプロンプト

電気主任技術者が実施する電気設備の定期点検記録を整理します。

```
以下の情報をもとに、自家用電気工作物の定期点検記録と
年次点検計画書を作成してください。

【自社の電気設備の概要】
受電電圧: （例: 6,600V（高圧受電））
主な電気設備:
  ・変圧器（6,600V/200V）: 300kVA × 2台
  ・高圧配電盤・低圧配電盤: 各1面
  ・動力設備（プレス機・工作機械等）: 合計○kW
  ・UPS（無停電電源装置）: ○kVA
電気主任技術者: （例: 社内選任 / 外部委託（電気保安協会等））

【点検の種類と頻度（法令要件）】
月次点検（自主点検）:
  ・外観点検（漏電・異音・異臭）
  ・電流・電圧の計測確認
年次点検（停電点検）:
  ・年1回、停電させて実施
  ・絶縁抵抗測定・接地抵抗測定・保護継電器試験等

出力形式:
・月次点検記録のフォーマット（点検日・点検者・設備別の点検項目・
  判定・異常事項の記録欄）
・年次点検計画書のフォーマット（停電日時・作業内容・担当・
  立会い者・報告先）
・「年次点検結果報告書」の骨子（絶縁抵抗値等の記録・評価・
  改善事項）
・「電気設備の台帳管理（製造年・容量・更新時期の見える化）」の
  フォーマット提案
・「電気設備の異常（漏電ブレーカー作動・絶縁低下等）が発生した
  場合の初動対応フロー」
```

## 消防設備点検報告書を整理するプロンプト

消防法に基づく工場の消防設備点検と報告書を整理します。

```
以下の情報をもとに、製造業工場の消防設備点検管理書類を
整理してください。

【工場の消防設備の概要】
延床面積: （例: 2,500m2）
主な設備:
  ・自動火災報知設備（感知器○個・発信機○個・受信機1台）
  ・屋内消火栓設備（消火栓○箇所）
  ・消火器（ABC粉末20型 ○本）
  ・誘導灯（避難口・通路誘導灯 ○台）
  ・スプリンクラー設備: なし
防火管理者: 選任済み（○○ / 甲種防火管理者）

【点検の種類と実施状況】
機器点検（6か月ごと）:
  消防設備士または消防設備点検資格者が実施
  直近実施: ○○年○月（業者委託）

総合点検（1年ごと）:
  機器点検 + 作動点検（実際に作動させて確認）

消防署への報告:
  特定防火対象物（飲食店・病院等）: 毎年報告
  非特定防火対象物（工場）: 3年ごとに消防長・消防署長へ報告

出力形式:
・消防設備点検スケジュール管理表（設備種別・点検頻度・
  次回点検予定日・報告期限）
・「点検業者委託時に確認すること（資格・業者選定基準）」
・消防署への「消防用設備等点検結果報告書」の記入の流れ説明
・「消防訓練計画書・報告書のフォーマット」（年2回の訓練義務）
・「消防設備の不備・指摘事項が出た場合の是正対応フロー」
```

## 工場の設備法定点検スケジュール管理表を作るプロンプト

電気・消防・ボイラー・クレーン等、複数の法定点検を一元管理します。

```
以下の情報をもとに、製造業工場の法定点検スケジュール
一元管理表を作成してください。

【主な法定点検の種類（例）】
法令: 電気事業法
  設備: 自家用電気工作物（高圧受電設備）
  点検頻度: 月次（自主）/ 年次（停電点検）
  担当: 電気主任技術者・外部委託業者

法令: 消防法
  設備: 消防設備一式
  点検頻度: 6か月ごと（機器点検）/ 1年ごと（総合点検）
  報告: 3年ごとに消防署へ報告

法令: クレーン等安全規則
  設備: クレーン（5t）
  点検頻度: 月次点検 / 年次定期自主検査（1年ごと）
  報告: 特定機械は設置時・変更時に労働基準監督署へ届出

法令: ボイラー及び圧力容器安全規則
  設備: 小型ボイラー
  点検頻度: 月次点検 / 年次定期自主検査
  報告: ボイラー技士（免許所持者）による運転管理

出力形式:
・法定点検一元管理表（法令名・設備名・点検種別・頻度・
  担当者・次回実施予定・記録保管場所）
・「法定点検の実施漏れを防ぐ年間カレンダー形式のスケジュール」
・「法定点検の記録保管期間と保管方法のルール」
・「点検を怠った場合の法的リスク（罰則・保険への影響等）」の要点
・「法定点検管理の担当者引き継ぎ資料フォーマット」
```
