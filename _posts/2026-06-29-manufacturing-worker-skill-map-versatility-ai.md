## スキルマップ・多能工化計画書類を整理するプロンプト

```
以下の情報をもとに、作業者のスキルマップと
多能工化推進計画書を整理してください。

【現在の工程と作業者の状況】
工程数: 8工程（A旋盤 / B旋盤 / フライス / MC1 / MC2 / 溶接 / バリ取り / 検査）

作業者（正社員8名 + パート2名）のスキル状況:
  田中（ベテラン）: A旋盤◎ B旋盤◎ フライス○ MC1△ 溶接✕ バリ取り○ 検査○
  鈴木: A旋盤✕ B旋盤✕ フライス○ MC1◎ MC2◎ 溶接✕ バリ取り○ 検査△
  佐藤: 全工程△〜✕（入社2年目・育成中）
  山本: A旋盤○ B旋盤○ フライス○ 溶接◎ バリ取り◎ 検査○（MC未経験）
  木村: 検査◎ バリ取り◎ 他全て✕（検査専任）
  その他3名: 各工程に専任が多く融通が利かない
  パート2名: バリ取り・検査のみ可能

問題:
  ・田中さんが休むとA旋盤が止まる
  ・MC系は鈴木さんだけが対応できる
  ・来年度2名の定年退職予定（田中・別1名）

出力形式:
・「スキルマップ（作業者×工程の評価マトリクス）」
  （◎=一人立ち / ○=指導下なら可 / △=練習中 / ✕=未経験）
・「リスク評価（単独対応者の工程と業務停止リスク）」
・「多能工化優先順位マトリクス（緊急度×習得難易度）」
・「今後2年間の多能工化ロードマップ（誰が・いつ・どの工程を習得）」
・「OJT計画書（ペア指定・習得目標・評価タイミング）」
・「来年度定年退職対応の技能伝承計画（田中さんからの知識移転リスト）」
```
