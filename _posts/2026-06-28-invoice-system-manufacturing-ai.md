インボイス制度（適格請求書等保存方式）は2023年10月にスタートし、製造業の受発注・請求業務に大きな影響を与えています。取引先との請求書フォーマット変更・社内処理ルールの整備・免税事業者との取引への対応など、さまざまな書類作業をAIで整理できます。

> **注意**: 税務・法律に関わる判断は必ず税理士・専門家に確認してください。このページはAIプロンプトの活用例であり、税務アドバイスではありません。

---

## 適格請求書（インボイス）のフォーマット確認プロンプト

自社発行の請求書がインボイスの記載要件を満たしているか確認します。

```
以下の請求書の記載内容が、インボイス制度（適格請求書等保存方式）の
要件を満たしているか確認してください。

【現在の請求書の記載内容】
（例：
・発行者の氏名または名称: 記載あり（○○製作所）
・登録番号: 記載あり（T1234567890123）
・取引年月日: 記載あり
・取引内容: 記載あり（品番・品名・数量・単価）
・税率ごとに区分した合計額: 記載あり（10%対象 ¥○○○）
・税額（または税率ごとの税額）: 記載あり
・請求書の宛名（相手方の名称）: 記載あり
）

確認してほしいこと:
・上記の記載で適格請求書の必須要件を満たしているか
・不足・不備があれば何を追加・修正すれば良いか
・複数税率（10%・8%）が混在する場合の記載方法

出力形式:
・「記載OK / 不足・要確認」を各項目でチェック
・不備がある場合は「この箇所をこう修正する」を具体的に
・なお、最終判断は税理士に確認することを明記
```

## 取引先への登録番号確認依頼メールを作るプロンプト

仕入先・外注先のインボイス登録番号を確認するメールを作ります。

```
以下の情報をもとに、取引先（仕入先・外注先）への
インボイス登録番号の確認メールを作成してください。

【背景・目的】
インボイス制度の仕入税額控除を適用するために、
仕入先が適格請求書発行事業者かどうかを確認する必要がある。

【取引先の状況】
（例：複数の外注先のうち、登録番号をまだ確認できていない業者が5社ある。
  一部は個人事業主や小規模事業者のため、免税事業者の可能性がある。）

出力形式:
・丁寧なビジネスメール（押しつけがましくない表現）
・「インボイス登録番号を教えてください」の依頼
・「登録していない場合は、その旨もお知らせいただけると助かります」の文言
・免税事業者との取引継続の意思は別途判断することを示唆する締め
```

## 免税事業者との取引方針を社内説明するプロンプト

インボイス未登録の取引先への対応方針を社内に周知する文書を作ります。

```
以下の方針をもとに、インボイス未登録の取引先への対応方針を
社内（経理・購買担当者向け）に説明する文書を作成してください。

【自社の対応方針（例）】
・2023年10月〜2026年9月: 経過措置期間。仕入税額の80%は控除可能。
  → 現時点では取引継続（経理処理で経過措置を適用する）
・2026年10月以降: 仕入税額の50%のみ控除可能。
  → 取引単価・条件の見直しを検討（個別に判断）
・免税事業者との新規取引: 代替先があれば課税事業者を優先する

【確認してほしいこと】
・上記方針の社内向け説明文（購買・経理担当者が判断できる基準）
・「免税事業者から請求書が来た場合の経理処理の流れ」を付記

出力形式:
・社内通達または業務手順のフォーマット
・「判断に迷った場合は経理部門に確認する」のエスカレーション先を明示
・※税務判断は税理士に確認済みの内容を反映する形で作成
```

## インボイス対応の社内チェックリストを作るプロンプト

インボイス制度対応状況を点検する社内チェックリストを作ります。

```
以下の視点で、製造業中小企業のインボイス制度対応状況を
確認するためのチェックリストを作成してください。

【確認したい項目】
(1) 発行側（自社が請求書を発行する側）
・自社の登録番号を請求書に記載しているか
・請求書に税率区分・税額を記載しているか
・電子データで請求書を保存しているか（電子帳簿保存法との関係）

(2) 受領側（自社が仕入・外注費を支払う側）
・仕入先リストに登録番号が入力されているか
・未確認の取引先はリストアップできているか
・経過措置期間の終了タイミングを把握しているか

(3) システム・運用
・会計ソフト（freee / マネーフォワード等）の対応設定ができているか
・免税事業者からの請求書に「80%控除」の経理処理ができているか

出力形式:
・「確認済 / 未対応 / 要確認」の3択チェックボックス形式
・各項目に「対応が必要な場合の参照先・担当部門」を付ける
・チェックリストのタイトルに「インボイス制度対応状況確認表（○○年度版）」
```
