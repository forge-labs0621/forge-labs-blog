## 見積書、毎回ゼロから書いていませんか

中小製造業の営業・事務担当者にとって、見積書づくりは「金額を決める」こと
より「文面を整える」ことに地味に時間がかかる作業です。宛名・前書き・
項目の表組み・支払条件・結びの一文…型は毎回ほぼ同じなのに、コピペ元の
過去ファイルを探すところから始めている方も多いのではないでしょうか。

金額や数量の計算はAIに任せるとミスが怖いですが、「確定した金額を、整った
見積書の文面に整形する」作業はAIが得意な領域です。

## 試してみた例

営業担当が決めた内容をメモにして、AIに渡します。

```
取引先名: 株式会社サンプル工業
件名・対象製品/工程: シャフト加工 100本
見積項目と金額: シャフト加工費 @800円×100本=80,000円／検査費 5,000円
納期: 2026年7月15日
支払条件: 月末締め翌月末払い
有効期限: 発行日より30日間
特記事項: 材料はお客様支給品とする
```

「金額・数量は入力された値をそのまま使い、計算は変更しないこと」という
条件を指示と一緒に渡すと、次のような見積書の文面が返ってきます。

```
件名: シャフト加工に関するお見積り

株式会社サンプル工業 御中

見積項目一覧
| 項目 | 数量 | 単価 | 金額 |
|---|---|---|---|
| シャフト加工費 | 100本 | 800円 | 80,000円 |
| 検査費 | - | - | 5,000円 |
| 合計 | | | 85,000円 |

納期: 2026年7月15日
支払条件: 月末締め翌月末払い
有効期限: 発行日より30日間
特記事項: 材料はお客様支給品とします。

ご検討の上、何卒よろしくお願い申し上げます。
```

金額はメモにある数字のまま、文面だけが整った状態で返ってきています。

## 安全に使うための1つのコツ

ポイントは「金額の計算はAIにさせない」ことです。指示の中で「入力された
金額をそのまま使う、計算や変更は一切行わない」と明記しておくことで、
AIが誤った金額を作ってしまうリスクを避けられます。文面整形と金額計算を
分けて考えるのが安全に使うコツです。
