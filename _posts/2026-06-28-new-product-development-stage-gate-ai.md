新製品開発は「商品企画 → 基本設計 → 詳細設計 → 試作 → 量産化」のステージを経て完成します。AIを使って各ステージの管理書類・デザインレビュー資料・開発計画書・量産判定書類を効率的に整理できます。

---

## 新製品開発計画書を作るプロンプト

開発プロジェクトの全体計画を整理します。

```
以下の情報をもとに、新製品開発の開発計画書（プロジェクト計画書）の
骨子を作成してください。

【新製品の概要】
製品名（仮称）: （例: 次世代型プレシジョンブラケット PB-X）
開発背景:
  ・主要顧客から「既存品より20%軽量化・精度±0.01mmを達成した製品」の
    開発を要請された
  ・現行製品（PB-5）は重量1.2kg、精度±0.03mm

開発目標（スペック目標）:
  ・重量: 1.0kg以下（20%軽量化）
  ・寸法精度: ±0.01mm
  ・コスト目標: 現行品 ¥3,500 → ¥3,200以下（8.5%削減）
  ・量産開始目標日: ○年○月

開発体制:
  ・プロジェクトリーダー: 技術部 ○○さん
  ・参加部門: 技術・製造・品質・購買・営業

【開発ステージ設計（ステージゲート法）】
S1: 企画（アイデア具体化・先行投資判断）
S2: 基本設計（DR1: 基本設計審査）
S3: 詳細設計・試作（DR2: 試作評価審査）
S4: 量産試作・検証（DR3: 量産移行判定）
S5: 量産開始

出力形式:
・「新製品開発計画書の構成（プロジェクト目的・開発スコープ・
  マイルストーン・体制・予算・リスクの各章）」
・「開発スケジュール（ガントチャート形式のWBS骨子）」
・「開発予算の構成（試作費・治具費・設備改造費・外注費の内訳）」
・「ステージゲート判定基準（各DR・ゲートで確認すべき項目）」
・「開発リスクの洗い出し（技術リスク・コストリスク・スケジュールリスク）」
```

## DR（デザインレビュー）資料の準備をするプロンプト

設計審査会の議題と評価チェックリストを作ります。

```
以下の情報をもとに、新製品開発のDR（デザインレビュー）会議の
準備資料と審査チェックリストを作成してください。

【DRの段階と目的】
DR実施段階: DR2（詳細設計完了・試作前評価）
目的:
  ・詳細設計の妥当性を確認する
  ・量産性・コスト・品質の観点で問題がないかを判定する
  ・試作に進む前にリスクを排除する

【審査する設計内容】
製品: 新型プレシジョンブラケット PB-X
設計変更ポイント:
  ・材質をSS400 → A5052（アルミ合金）に変更（軽量化対応）
  ・加工方法: 切削加工に変更（複雑形状のため）
  ・寸法許容差: ±0.01mm（現行品の3倍の精度要求）

検討済みの問題点:
  ・アルミの表面処理（アルマイト vs 塗装）の選定
  ・精度±0.01mmを安定して保証できるか（工程能力検討が未完）

出力形式:
・「DR準備資料の目次と各章の骨子」
・「DR審査チェックリスト（機能性・量産性・コスト・安全性・
  法規制対応の観点）」
・「想定されるDR指摘事項（よくある設計上の問題と確認ポイント）」
・「DR議事録フォーマット（審査項目・審査結果・指摘事項・
  対応期限・担当者）」
・「量産移行可否の判定基準（GoNo-Goの基準設定）」
```

## 量産移行判定書類を整理するプロンプト

試作評価完了後、量産開始を承認するための書類を作ります。

```
以下の情報をもとに、新製品の量産移行を承認するための
「量産移行判定書類」を整理してください。

【量産移行のための試作結果サマリー】
製品: PB-X プレシジョンブラケット
試作数量: 50個（量産条件に近い設備・工程で製造）

品質確認結果:
  ・主要寸法（外径・孔径・平面度）: 全数合格（CPK ≥ 1.33）
  ・表面処理: アルマイト処理 塗装密着性 合格
  ・顧客評価: 試作品提出 → 顧客よりOK通知受領

工程・設備確認:
  ・加工治具の完成と精度確認: 完了
  ・作業標準書の作成: 完了
  ・全作業者の教育訓練: 完了（3名）
  ・測定器の校正確認: 完了

量産コスト確認:
  ・実際の製造原価（試作時の実績）: ¥3,150/個
  ・目標コスト: ¥3,200/個 → 目標クリア

残課題（量産開始後に継続確認）:
  ・初期ロット5ロット分の工程能力（Cpk）の継続モニタリング

出力形式:
・「量産移行判定書のフォーマットと記入例」
・「量産移行の承認項目（品質・量産性・コスト・書類の4観点）」
・「量産開始後の初期流動管理計画（何ロット・何個測定するか）」
・「初期流動管理終了判定の基準と手続き」
・「量産移行承認書（設計・製造・品質・経営者の承認フロー）」
```
