製造業では海外顧客への輸出・海外サプライヤーからの調達・海外工場の設立など、グローバルな取引が増えています。AIを使って英文書類の作成・貿易書類の整理・海外パートナーとの契約書類の準備を効率化できます。

---

## 海外顧客向け英文見積書・提案書を作るプロンプト

海外顧客向けの提案書類を英語で整理します。

```
以下の情報をもとに、海外顧客向けの英文見積書と
技術提案書の骨子を作成してください。

【取引の概要】
顧客: （例: ドイツの機械メーカー Kunde GmbH）
問い合わせ内容: プレス成形品のサンプル発注 + 量産見積

【見積内容】
品番: PB-EX-001
品名: Press Formed Bracket（プレスブラケット）
材質: SPCC t2.0mm（Cold rolled steel）
サンプル数量: 50個
量産数量（予定）: 月産 3,000個

【当社の強み】
  ・ISO 9001認証取得
  ・IATF 16949認証あり（自動車向け実績）
  ・Tolerance: ±0.1mm 対応可能
  ・納期: サンプル 3週間 / 量産 FOB横浜

出力形式:
・「英文見積書（Quotation）のフォーマット（品番・品名・
  数量・単価・納期・支払い条件・有効期限）」
・「英文技術提案書の骨子（会社概要・品質認証・技術能力・
  生産実績・設備一覧）」
・「インコタームズ（Incoterms）の選択説明
  （FCA・FOB・CIF・DDPの違いと推奨）」
・「支払い条件の表記（T/T before shipment / L/C at sightの表記）」
・「海外顧客への最初のメール文（自社紹介・見積提出の申し出）」
```

## 海外サプライヤーからの調達管理書類を整理するプロンプト

海外調達のリスク管理と書類管理を整理します。

```
以下の情報をもとに、海外サプライヤーからの
調達管理書類を整理してください。

【調達の背景】
調達先: （例: 中国・広東省の部品メーカー ABC Technology Co., Ltd.）
調達品目: 電子部品（センサーユニット）
現在の調達状況: 月産1,000個 / 単価 USD 4.50

【調達リスクと課題】
  ・品質: 受入不良率が約2.5%（目標1%以下）
  ・納期: 注文〜納品まで45日（国内調達の2倍）
  ・為替リスク: ドル建て契約 / 円安時にコスト増
  ・物流: 船便遅延リスク（コロナ禍以降、2〜3週間の遅延が発生）
  ・情報共有: サプライヤーからの進捗報告が不定期

出力形式:
・「海外サプライヤー評価シート（品質・納期・価格・対応力・
  財務安定性の評価項目）」
・「Purchase Order（発注書）の英文フォーマット（品番・数量・
  単価・納期・検査条件・支払い条件）」
・「輸入時の必要書類チェックリスト（インボイス・パッキングリスト・
  船荷証券・原産地証明書）」
・「為替リスク管理の選択肢（円建て交渉 / ドル預金 / 為替予約の
  概要説明）」
・「海外サプライヤーへの品質改善要求書（英文: Quality Improvement Request）」
```

## 海外工場設立・海外子会社管理を整理するプロンプト

海外拠点の設立・運営に必要な書類を整理します。

```
以下の情報をもとに、海外工場設立（または合弁会社設立）に
向けた準備書類と管理フォーマットを整理してください。

【検討している海外展開の概要】
検討先: タイ（チョンブリ県の工業団地）
進出形態: 現地法人100%子会社（Wholly Owned Subsidiary）
事業内容: 日本向け部品の現地生産（コスト削減・ASEAN販売も検討）
投資規模: 初期投資 USD 1,000,000（設備・土地賃借・初期運転資金）

【進出を検討するに至った背景】
  ・主要顧客からの「コスト20%削減要求」への対応
  ・ASEAN市場の将来性（タイ・インドネシアでの現地販売を検討）
  ・日本人従業員の高齢化・採用難への対応

【現在の不明点（CEOへの承認依頼前に調査が必要）】
  ・タイの法人設立手続き・費用・期間
  ・タイの労働法・最低賃金
  ・工業団地の賃料・インフラ・優遇措置
  ・タイ人従業員の採用方法・給与水準

出力形式:
・「海外進出可能性調査（フィジビリティスタディ）の骨子
  （市場・コスト・リスク・資金の4項目）」
・「現地法人設立の一般的な手順と必要書類の概要
  （タイを念頭に）」
・「海外子会社管理の主要書類（月次レポート・予算・
  移転価格（Transfer Pricing）の概念）」
・「海外進出リスク一覧（政治リスク・為替・品質管理・
  人材育成・撤退コスト）」
・「経営判断のための海外進出Go/No-Go判断基準のフレームワーク」
```
