大規模地震・水害・感染症拡大・取引先の工場停止など、製造業では様々なリスクが事業を脅かします。BCP（Business Continuity Plan：事業継続計画）は「いざというとき、どう動くか」を事前に決めておく計画書ですが、「作り方がわからない」「ひな形はあるが自社向けにアレンジできない」という声をよく聞きます。AIを使うと、自社の状況をもとにBCPの骨子を素早く整理できます。

> **重要**: BCPは一度作ったら終わりではなく、定期的な見直し・訓練が必要です。AIが作成した骨子をたたき台として、経営者・部門責任者が実態に合わせて必ず修正・確認してください。

## AIが役立つBCP・防災計画の整理場面

| 文書の種類 | AIでできること |
|---|---|
| BCPの骨子・構成案 | 自社の状況をもとに文書の構成を提案 |
| リスク一覧（リスクアセスメント） | 業種・立地・取引先などのリスクを整理 |
| 緊急時の連絡体制図のテキスト化 | 口頭で決めている連絡順序を文書に |
| 代替調達先の整理 | 主要資材の代替仕入先リストのひな形 |
| 防災マニュアルの骨子 | 地震・水害等の避難・対応手順の骨子 |

## 製造業向けBCPの骨子を作るプロンプト

自社の状況を入力すると、BCPの目次・各項目の要点を整理できます。

```
以下の情報をもとに、中小製造業向けBCP（事業継続計画）の骨子を
作成してください。

【自社の基本情報】
業種・主な製品: （例：金属プレス加工業。自動車部品メーカー向けプレス部品が売上の70%）
従業員数: （例：35名）
工場の立地・特徴: （例：愛知県。海抜10m地点。河川から1km）
主な設備: （例：プレス機5台、NC旋盤3台、溶接機4台）

【想定する主なリスク（複数選択）】
□ 大規模地震（工場の建物損壊・設備倒壊）
□ 水害（浸水・洪水）
□ 主要取引先・仕入先の工場停止（サプライチェーン寸断）
□ 感染症の拡大（従業員の大量欠勤）
□ 停電・ライフライン停止
□ 火災・爆発

【自社で最も影響が大きいと思うリスク】
（例：
・大規模地震（設備の損壊・工場の使用不能）
・主要仕入先（鉄鋼メーカー）の供給停止
）

出力形式:
・BCP文書の目次（大項目8〜10個）と各項目の要点
・各リスクへの基本的な対応方針（概要）
・次のステップ（実際にBCPを作るために必要な作業リスト）
・「このBCPは○年○月○日時点の情報をもとに作成。定期的な見直しが必要」の注記
```

## 緊急時連絡体制の文書化プロンプト

口頭で決まっている緊急時の連絡順序を文書にします。

```
以下の情報をもとに、緊急時（工場での大きな事故・火災・地震等）の
連絡体制マニュアルを作成してください。

【緊急時に最初にすること（優先順位順）】
（例：
1. けが人の確認・応急処置・救急車の要請（119番）
2. 火災の場合は消火器対応 or 避難誘導（担当: 各フロアの安全衛生担当）
3. 設備の緊急停止（担当: 工場長または代理者）
4. 社内の人員確認（全員の安否を工場長に報告）
5. 社長・管理職への第一報（どの回線で・誰に・何を伝えるか）
）

【緊急時の連絡先（役職と優先順位）】
（例：
1. 工場長 田中○○（携帯: ○○○-○○○-○○○○）
2. 社長 山田○○（携帯: ○○○-○○○-○○○○）
3. 副工場長 鈴木○○（上記2名が連絡取れない場合）
4. 主要取引先（出荷に影響が出る場合）
5. 保険会社（建物・設備への損害が出た場合）
）

【通常勤務時間外（夜間・休日）の初動担当者】
（例：守衛1名が常駐。緊急事態は工場長へ直接連絡）

出力形式:
・「緊急時対応・連絡体制マニュアル」として
・発生直後の初動手順（フローチャートを文字で表現）
・連絡先一覧（役職・氏名・連絡先のプレースホルダー付き表）
・「まず自分の身の安全を確保することが最優先」の注記を冒頭に
```

## サプライチェーン寸断時の代替対応表を作るプロンプト

主要な仕入先・材料が調達できなくなった場合の対応表です。

```
以下の情報をもとに、サプライチェーン寸断時の対応計画表を作成してください。

【主要な仕入先・材料と依存度】
（例：
・A社（鋼板）: 月間使用量の80%を依存。代替仕入先は現在未検討。
・B社（切削油）: 複数社から調達可能。依存度低い。
・C社（外注加工：熱処理）: 近隣では1社のみ。代替手段なし。
）

【自社生産に影響が出るまでの在庫日数】
（例：
・鋼板: 約10日分の在庫
・切削油: 約30日分の在庫
・熱処理は都度外注のため在庫なし
）

出力形式:
・「サプライチェーンリスク対応表」として
・主要調達品目 / 主要仕入先 / 影響が出るまでの日数 / 
  代替手段（現状と検討すべき対策） の表形式
・「今後調査・検討すべき事項」のアクションリストを最後に追加
```

## 地震・水害発生後の初動対応チェックリストを作るプロンプト

発生直後に何をすべきか確認するためのチェックリストです。

```
以下の条件で、地震（震度6以上を想定）発生後の初動対応チェックリストを
作成してください。

【自社の状況】
立地・工場の種類: （例：鉄骨造 2階建て工場。製造1階・事務2階）
主な危険源: （例：プレス機・溶接機・高圧ガスボンベ）
勤務形態: （例：昼間のみ、日曜・祝日は休業）

出力形式:
・「地震発生後 初動対応チェックリスト」として
・時系列で（発生直後 → 5分後 → 30分後 → 数時間後）の行動を整理
・各項目に「担当者（役職）」と「チェックボックス（□）」を付ける
・ライフライン・設備・人員・情報連絡 それぞれの確認項目を含める
・「このリストは事前に全従業員が確認しておくこと」の注記を末尾に
```

## AIを使う際の注意

- BCPは「紙の計画書を作ること」が目的ではなく、「実際に機能すること」が目的です。AIが整理した骨子を、実際の訓練や見直しのベースとして活用してください
- 連絡先・担当者名などの個人情報は、外部AIサービスに入力する際は架空名や役職名のみを使うことを推奨します。実際の個人情報はローカルで作業した文書に後から記入してください
- BCPに記載した代替仕入先・連絡先は、実際に取引・連絡が可能かどうかを事前に確認してください。「計画書には書いてあるが実際には機能しない」という状態を避けることが重要です
