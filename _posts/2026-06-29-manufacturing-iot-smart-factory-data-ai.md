IoT・スマートファクトリーへの投資判断には、現状分析から費用対効果の試算まで体系的な書類整理が必要です。AIを使って導入計画書・ROI評価・運用ルール設計を効率化できます。

---

## IoT導入計画書・ROI評価書を整理するプロンプト

```
以下の情報をもとに、IoT設備稼働管理システムの
導入計画書と投資対効果評価書を整理してください。

【現状の課題と導入背景】
工場: 金属加工工場（従業員55名・設備15台）
現在の問題:
  ・設備稼働状況が「見えない」（稼働・停止・チョコ停の把握ができていない）
  ・OEEを計算したいが、データが手書きで集計に丸1日かかる
  ・設備の異常予兆に気づくのが遅れ、突発故障が月平均2〜3件発生
  ・どの設備が生産ボトルネックになっているか分析できていない

検討中のIoTシステム:
  システム名: 〇〇スマートモニター（中堅IoTベンダー）
  概要: 電流センサーを設備に取付けて稼働/停止を自動判別・クラウドで可視化
  導入費用: 
    初期費用: 設備15台分センサー＋クラウド初期設定 = 約180万円
    月額費用: クラウド利用料・保守 = 約3万円/月
  主な機能: リアルタイム稼働監視・OEE自動計算・異常アラート・レポート自動生成

出力形式:
・「IoT導入の費用対効果試算（5年間TCOと期待効果の試算表）」
  （削減効果: 突発故障削減 / 集計工数削減 / OEE改善 / 品質改善の金額換算）
・「IoT導入計画書（目的 / 対象設備 / 導入フェーズ / スケジュール / 体制）」
・「Phase1（パイロット）計画書（選定設備3台で3ヶ月試行・効果測定の方法）」
・「設備稼働データの運用ルール設計（誰がいつどのデータを見るか・アラート対応者）」
・「経営層向け稟議書（投資額 / ROI / 回収期間 / リスク / 承認を求める事項）」
```

## スマートファクトリー化の段階計画を整理するプロンプト

```
中小製造業がスマートファクトリー化を段階的に進めるための
ロードマップと各フェーズの計画書を整理してください。

【工場の現状デジタル化レベル（2026年6月時点）】
  Level 1（紙・手書き）: 作業日報・点検記録・出荷記録 → まだ紙
  Level 2（Excel/手動入力）: 生産計画・在庫管理・品質記録 → Excel運用
  Level 3（デジタル自動収集）: 未導入
  Level 4（AI・最適化）: 未導入

目標: 3年以内にLevel 3の主要部分を達成

出力形式:
・「DXロードマップ（3年間・12のマイルストーン）」
  Year1: 稼働見える化・ペーパーレス化
  Year2: 工程データ連携・品質データ自動収集
  Year3: 予知保全・生産計画最適化
・「Year1の詳細計画書（具体的なシステム・予算・KPI）」
・「DX推進体制の設計（社内DX推進担当の役割・ITベンダーとの連携方法）」
・「補助金活用計画（IT導入補助金・ものづくり補助金の対象性確認と申請スケジュール）」
```
