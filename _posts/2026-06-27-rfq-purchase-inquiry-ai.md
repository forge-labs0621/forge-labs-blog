製造業の購買・調達担当者が日常的に作成する「仕入先への見積依頼書」や「外注先への問い合わせメール」。条件を整理して正確に伝えるのに手間がかかる書類ですが、AIを使うと素早く下書きを作れます。

> **注意**: 顧客に送る「見積書」とは別の書類です。この記事は自社が**仕入先・外注先に送る見積依頼側**の文書作成を解説します。

## AIが役立つ購買・調達の場面

| 場面 | AIでできること |
|---|---|
| 見積依頼書（RFQ） | 仕入先への見積依頼書の下書き |
| 外注依頼・仕様確認 | 外注先への仕様・条件確認メール |
| 新規取引先への初回問い合わせ | 資料請求・取引条件確認の連絡文 |
| 発注条件の変更連絡 | 数量変更・納期変更の連絡文 |
| 複数社比較のための問い合わせ統一 | 同一内容を複数社に送る文面の統一 |

## 仕入先への見積依頼書を作るプロンプト

材料・部品を複数社に見積もり依頼する際の依頼書です。

```
以下の情報をもとに、仕入先への見積依頼書を作成してください。

【自社情報】
会社名: （例：○○製作所株式会社）
担当者名・部署: （例：購買部 山田○○）
問い合わせ日: （例：2026年7月15日）
見積回答期限: （例：2026年7月25日）

【依頼する品目の情報】
品目名・品番: （例：冷間圧延鋼板 SPCC 1.0mm × 1,000mm × 2,000mm）
数量: （例：200枚）
使用目的・用途（おおよそで構いません）: （例：プレス加工用原材料）
希望納期: （例：2026年8月10日頃）
梱包・荷姿: （例：ジャストパック梱包、1t/パレット）
特別な要求事項（あれば）: （例：ミルシート（材料成績書）添付必須）

【見積に含めてほしい内容】
□ 単価（税別）
□ 最低発注数量
□ 在庫状況
□ 納期（最短・標準）
□ 送料・梱包費の扱い

出力形式:
・「見積依頼書」として
・表題・宛先（プレースホルダー）・日付・自社情報
・依頼品目を表形式で整理
・見積条件（回答期限・含めてほしい内容）
・担当者連絡先（プレースホルダー）
```

## 外注先への加工依頼・仕様確認メールを作るプロンプト

外注先（加工業者）に加工を依頼する際の仕様確認・見積依頼メールです。

```
以下の情報をもとに、外注先への加工依頼・仕様確認メールの
下書きを作成してください。

【外注先情報（宛先）】
会社名: （例：△△精工株式会社）
担当者名: （例：営業部 鈴木様）

【依頼する加工内容】
加工の種類: （例：旋盤加工（外径仕上げ・内径加工））
対象素材・ブランク: （例：SCM415 丸棒 φ60mm × 150mm）
加工後の仕様（概要）: （例：外径φ50±0.02、内径φ20H7、
                      表面粗さ Ra1.6、ただし詳細は図面を添付します）
数量: （例：試作5個、量産は月100個程度の予定）
希望納期（試作）: （例：2026年8月5日）
図面: （例：PDFを添付）

【確認したい事項】
（例：
・この仕様で加工可能かどうか（製作可否）
・単価（試作・量産それぞれ）
・試作の納期
・表面処理（三価クロメート）の対応可否
）

出力形式:
・件名（例：「加工依頼 ご確認のお願い」）
・メール本文（丁寧だが要点を明確に）
・確認事項を箇条書きで整理
・回答期限の明示（例：7月20日までにご回答いただけますと幸いです）
```

## 新規仕入先への初回問い合わせメールを作るプロンプト

展示会・紹介等で知った新しい仕入先・外注先に初めて連絡する際のメール下書きです。

```
以下の情報をもとに、新規仕入先への初回問い合わせメールを
作成してください。

【自社情報】
会社名・業種: （例：○○製作所株式会社、プレス加工業）
担当者名: （例：購買部 田中○○）

【問い合わせ先（相手）】
会社名: （例：□□商事株式会社）
問い合わせのきっかけ: （例：展示会で製品カタログをいただいた）

【問い合わせたい内容】
（例：
・主に使用しているアルミ板材（A5052）の取り扱い可否と価格帯
・最低発注量・発注単位
・配送エリアと送料の目安
・サンプル提供の可否
）

出力形式:
・丁寧な初回問い合わせのビジネスメール
・自社の簡単な紹介（業種・規模）を冒頭に
・問い合わせ事項を箇条書きで整理
・資料・カタログの送付をお願いする一文も入れる
```

## 発注条件変更（数量・納期）の連絡文を作るプロンプト

すでに発注している内容の数量変更・納期変更を依頼する際の連絡文です。

```
以下の状況で、仕入先への発注変更連絡文（メール）を作成してください。

【変更前の発注内容】
発注番号: （例：PO-2026-0712）
品目: （例：冷間圧延鋼板 SPCC 1.0mm 200枚）
当初の納期: （例：2026年8月10日）
当初の数量: （例：200枚）

【変更したい内容と理由】
□ 数量変更: （例：200枚 → 250枚（受注増加のため50枚追加）)
□ 納期変更: （例：8月10日 → 8月5日（客先要請で前倒し））
□ 両方変更

変更の理由: （例：得意先からの追加受注が入り、
  数量と納期の両方を見直す必要が生じました）

出力形式:
・発注変更依頼として
・変更前後を「変更前 → 変更後」で明確に対比
・変更依頼の理由を丁寧に一文で説明
・対応の可否と回答期限の確認を依頼する一文を含める
```

## AIを使う際の注意

- 見積依頼書・発注文書は取引の根拠となる書類です。AIが作成した下書きの数量・仕様・日程・品番に誤りがないか、**必ず担当者が確認してから送付**してください
- 仕様書・図面を外部AIサービスに入力する場合、機密情報の漏洩に注意してください（顧客の図面を使う場合は特に注意）
- 初回取引先への問い合わせは会社の印象を左右します。送付前に上長の確認を推奨します
