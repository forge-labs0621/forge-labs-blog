製造業の管理部門・経営企画・総務担当者は、月次の業績データを集めて「社長報告」「取締役会報告」「月次経営会議資料」などの報告書を作成する必要があります。数値の集計は別途行い、数値の「解説・コメント・課題と対策の文章」をAIで効率化できます。

---

## 月次業績報告サマリーを作るプロンプト

売上・受注・コストのデータをもとに経営報告の文章を整理します。

```
以下の月次業績データをもとに、社長・経営会議向けの
月次業績報告サマリーを作成してください。

【対象月】（例：2026年7月度）

【売上・受注】
売上高: ¥○○百万円（計画比 ○%・前年比 ○%）
受注高: ¥○○百万円（計画比 ○%・前年比 ○%）
受注残（月末時点）: ¥○○百万円

【コスト・利益】
製造原価率: ○%（計画 ○%・前月 ○%）
営業利益（概算）: ¥○○万円（計画比 ○%）

【今月の特記事項】
（例：
・大口受注（○○社 ¥○○万）が確定した
・原材料費が前月比○%上昇（スチール価格上昇の影響）
・設備ダウンタイムにより稼働率が○%低下した
）

【次月の見通し・重点事項】
（例：○○向けの量産開始で来月売上増見込み。原材料費上昇対策を検討中）

出力形式:
・経営会議向けA4 1枚サマリー
・「良かった点」「課題」「翌月の対応方針」の3構造で
・数値はそのまま使用（AIはコメントのみ追加）
・経営判断を促す「課題に対する選択肢」が1〜2点あれば提示
```

## 製造部門の月次報告コメントを作るプロンプト

製造部長や生産管理担当が上位報告に添付するコメントを整理します。

```
以下のデータをもとに、製造部門の月次報告コメントを作成してください。

【生産実績】
生産数量: ○○個（計画 ○○個・達成率 ○%）
生産金額: ¥○○万円
稼働率: ○%（前月比 ○%）

【品質実績】
社内不良率: ○%（前月 ○%）
クレーム件数: ○件
主な不良内容:（例：バリ発生が増加傾向（プレス工程の刃具交換タイミング要見直し））

【安全実績】
労災件数: ○件（今月 / 年累計）
ヒヤリハット報告数: ○件

【今月の取り組み・特記事項】
（例：新製品○○の量産立ち上げを完了。初月生産数○個。不良率は想定内。）

出力形式:
・部門報告として上位会議（経営会議/月次MTG）に出せる文章
・「生産 / 品質 / 安全」の3テーマで各2〜4行
・懸念事項には「対応策と期限」を添える
・全体で400字以内
```

## 来月の生産計画概要を整理するプロンプト

翌月の生産計画・販売計画の概要をまとめます。

```
以下の情報をもとに、来月の生産計画・見通しの概要コメントを
作成してください。

【翌月の受注・生産計画】
受注見込み: ¥○○万円（今月比 ○%）
生産計画数量: ○○個

【来月の主要な受注・生産品目】
（例：
・○○社向け A-1234 × 500個（通常品）
・○○社向け B-5678 × 100個（特急品、8月5日納品）
・新規受注 C-0099 × 50個（初回量産、8月末）
）

【来月の課題・懸念点】
（例：
・特急品B-5678の優先生産で、他品番のライン調整が必要
・新製品C-0099の初回量産でリードタイム不確定
・材料（SECC）の納期が3週間かかる見込み
）

出力形式:
・経営会議・部門会議で口頭説明のサポートになる資料コメント
・「計画の概要 / 重点品目 / 課題と対応」の構成で
・関係者が翌月のリソース配分を考えられる情報量
```

## 年度末・半期の業績振り返りサマリーを作るプロンプト

年度末や半期決算時に業績をまとめる報告文章を整理します。

```
以下の情報をもとに、上半期（または年度）の業績振り返りサマリーを
作成してください。

【対象期間】（例：2026年1〜6月（上半期））

【主な業績指標の実績と計画比】
売上高: 計画¥○○百万 / 実績¥○○百万 / 達成率○%
受注高: 計画¥○○百万 / 実績¥○○百万 / 達成率○%
製造原価率: 計画○% / 実績○%

【上半期の主なトピック（良かった点・悪かった点）】
（例：
良かった点:
・○○社への新規受注を獲得（¥○○万/月の継続受注）
・不良率が前年比○%改善（是正活動の効果）

課題:
・原材料費上昇で原価率が計画比○%悪化
・特定工程のボトルネックにより特急対応のコストが増加
）

【下半期の重点方針】
（例：原材料費上昇への価格転嫁交渉を優先。工程改善でリードタイム短縮。）

出力形式:
・取締役会・経営会議向けの半期振り返りサマリー（A4 1〜2枚）
・「達成できたこと・改善が必要なこと・下半期の方針」の3構造
・数値と定性評価をバランスよく
```
