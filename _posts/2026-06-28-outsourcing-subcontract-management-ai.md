繁忙期の外注活用・専門加工の委託・コスト最適化のために協力工場を使いこなす製造業は多くあります。AIを使って外注先の管理書類・委託契約・品質連携・コスト管理を効率化できます。

---

## 外注先への加工委託仕様書を整理するプロンプト

外注先に作業内容を正確に伝える書類を作ります。

```
以下の情報をもとに、外注先（協力工場）への
加工委託仕様書を整理してください。

【委託する加工の概要】
委託先: ○○板金工業（既存の協力工場）
委託内容: アルミ部品のアルマイト処理（表面処理加工）
品番: AL-3305 / 品名: アルミブラケット
数量: 500個 / 月（毎月発注予定）

【加工仕様】
素材: A5052-H32 板厚2.0mm
加工前の状態: 当社でプレス加工済み（表面処理前の状態）
処理内容: 硫酸アルマイト処理（陽極酸化皮膜）
膜厚: 15〜20μm（JIS H 8601 2種規格）
色: クリア（自然色）
禁止事項: 内径穴へのアルマイト膜付着は不可（マスキング必要）

【品質基準】
外観: 傷・打痕・色ムラなし（SUS製の外観検査基準書を事前送付）
膜厚確認: 抜き取り（5個/ロット）・当社で受入検査実施
検査成績書: 納品時に毎回提出（膜厚測定値・外観確認の記録）

出力形式:
・「外注加工委託仕様書の構成と各項目の記載内容
  （品番・仕様・品質基準・梱包方法・納期・
  価格・不良品対応・情報管理の各項目）」
・「マスキング箇所の指示方法（図面への追記の仕方）」
・「外注先への初回打ち合わせ資料のポイント
  （品質基準・NG例の写真・検査成績書フォーマットの事前共有）」
・「委託加工の受発注フロー
  （発注書→加工→出荷→受入検査→支払いの流れ）」
・「不良品が発生した場合の連絡・対応手順」
```

## 外注先の定期評価と管理計画プロンプト

協力工場を年次で評価して関係を管理します。

```
以下の情報をもとに、外注先（協力工場）の
年次評価と管理計画を整理してください。

【評価対象の外注先】
外注先名: ○○板金工業
取引内容: アルマイト処理・溶接加工（2種の委託加工）
年間取引額: 約¥8,400,000
取引開始: 2018年（8年取引）

【今年度の実績データ】
品質:
  ・不良品発生: 3件（不良率0.6%）
  ・検査成績書の遅延: 2回（書類不備）

納期:
  ・年間24回の発注のうち、3回納期遅延（最大2日遅れ）

コスト:
  ・アルマイト単価: ¥1,680/個（昨年から5%値上げを受け入れ済み）
  ・他社見積もりとの比較: 競合A社の見積¥1,550/個（同等品質）

対応力:
  ・急ぎ対応（スポット発注）: 2回とも対応可能だった
  ・担当者の連絡レスポンス: 良好（メール翌日以内に返信）

出力形式:
・「外注先評価シート（品質20点・納期20点・コスト20点・
  対応力20点・情報管理20点の100点満点評価）」
・「今年度の評価スコアの計算と評価コメント」
・「改善を求める事項のフィードバック文（丁寧な言い方）」
  （不良率と書類遅延の2点を改善してほしい旨）
・「コスト競争力の問題（他社より¥130/個高い）への
  対応方針（価格交渉 vs 多社購買 vs 現状維持）」
・「外注先評価の年次サイクル（いつ・誰が・
  どのプロセスで評価するか）」
```

## 外注費の管理と予算対比分析プロンプト

外注コストを月次で把握します。

```
以下の情報をもとに、外注費の
月次管理と予算対比レポートを整理してください。

【外注費の予算と実績（今年度上期）】
外注先 / 予算（上期合計） / 実績（上期合計） / 差異:
  ○○板金工業（アルマイト）: ¥4,200,000 / ¥4,580,000 / +¥380,000（超過）
  △△機械工業（旋盤加工）: ¥2,400,000 / ¥2,210,000 / ▲¥190,000（節減）
  ××溶接技研（溶接）: ¥1,800,000 / ¥1,920,000 / +¥120,000（超過）
  合計: ¥8,400,000 / ¥8,710,000 / +¥310,000（3.7%超過）

○○板金工業が超過した原因:
  ・3月の急受注対応でスポット発注（割増料金）が発生
  ・5月の値上げ申請を受け入れた（+¥80,000の影響）

出力形式:
・「外注費管理レポートのフォーマット
  （外注先別・月別・予算/実績/差異/差異率の表）」
・「超過原因の分析と今後の対策
  （スポット発注の削減方法・価格交渉の進め方）」
・「下期の外注費予算の修正案（上期実績を踏まえた見直し）」
・「外注費削減のための具体的な施策案（3案）」
  （例: 複数社見積もりの実施・発注ロットの統合・内製化の検討）
・「外注費の月次報告を経理部に提出する際のサマリーフォーマット」
```
