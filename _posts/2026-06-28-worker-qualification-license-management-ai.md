製造業では、フォークリフト・クレーン・玉掛け・溶接・危険物取扱などの法定資格が必要な作業があります。有資格者の管理・資格の更新期限の把握・法令遵守の記録をAIで整理できます。

---

## 有資格者管理台帳のフォーマットを作るプロンプト

工場内の全資格・全有資格者を一覧管理する台帳を作ります。

```
以下の情報をもとに、製造業の有資格者管理台帳の
フォーマットと登録例を作成してください。

【自社で必要な主な資格・免許（例）】
法定資格（必須）:
  ・フォークリフト運転技能講習修了証: 講習修了（有効期限なし・資格所持で運転可）
  ・クレーン運転士免許（1t以上）: 都道府県労働局長発行
  ・玉掛け技能講習修了証: 講習修了（有効期限なし）
  ・危険物取扱者（乙種4類）: 国家試験合格（3年ごとに保安講習が必要）
  ・特別教育（グラインダー・低圧電気等）: 社内教育記録

会社独自の技能認定:
  ・溶接技能者（自社認定: WPS適合確認済みの作業者）: 年次確認
  ・外観検査員（社内認定）: 認定試験合格（2年更新）
  ・フォークリフト社内安全運転認定: 年次確認

【管理台帳に含めたい項目】
・社員番号・氏名・部署
・資格名・資格の種類（国家資格・技能講習・特別教育・社内認定）
・資格取得日・有効期限・更新期日
・資格証のコピーの保管場所
・配置可能な作業（この資格でできる作業）

出力形式:
・有資格者管理台帳のフォーマット（表形式）
・「更新期限が近い資格（3か月以内）のリストアップ方法」
・「法的に無資格者が作業した場合のリスク」の説明コメント
・「社内認定の定義（どの資格をどのように管理するか）」の整理
```

## 作業者配置・資格確認書類を作るプロンプト

特定の作業に就かせる前に有資格者であることを確認する書類を作ります。

```
以下の情報をもとに、有資格者の作業配置確認書類を
作成してください。

【目的・背景】
（例：法律で有資格者のみが行える作業（フォークリフト・クレーン等）に
作業者を配置する際、有資格者であることを事前確認するための書類が
必要。新規採用者・異動者が来た場合も確認を確実にしたい。）

【書類の内容】
・確認する作業の種類（例: フォークリフト操作・クレーン吊り作業）
・確認する資格・技能講習の種類と番号
・確認者（班長・安全担当）のサイン欄
・確認日

出力形式:
・作業配置前の有資格者確認書のフォーマット
・「法令名と根拠条文」の参照（何の法律で資格が必要か）
・「確認書の保管方法・期間」の提案
・「外国人作業者の資格確認時の注意点」のコメント（日本語での資格証の確認等）
```

## 資格取得奨励制度の案内文を作るプロンプト

従業員の資格取得を奨励するための社内案内文を作ります。

```
以下の情報をもとに、資格取得奨励制度の社内案内文を
作成してください。

【制度の内容】
対象資格:
  ・フォークリフト・玉掛け・クレーン等の法定資格
  ・危険物取扱者（乙種各類）
  ・第二種電気工事士、等
支援内容:
  ・受験料・講習費用の全額会社負担
  ・取得後に祝い金を支給（国家資格: ¥5,000〜¥20,000）
  ・多くの資格を持つ作業者は昇給査定でプラス評価

目的:
  ・会社として作業できる範囲を広げる
  ・個人のスキルアップ・キャリアアップを応援する

出力形式:
・全従業員向けの案内文（A4 1枚）
・「取得を推奨する資格リスト」と「取得時の支援内容」の表
・「資格を活かして社内で活躍した先輩の例（架空のケース）」
・「受けたい人は総務部○○まで」の手続き案内
```
