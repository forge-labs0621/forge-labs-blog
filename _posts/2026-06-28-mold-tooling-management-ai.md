プレス加工・射出成形・ダイカスト・鍛造等の製造業では、金型・治具・工具の管理が生産品質と原価管理の要です。AIを使って金型管理台帳・型費回収計画・修理予防保全計画を効率化できます。

---

## 保有金型・治具の管理台帳を作るプロンプト

社内保有の全金型・治具を一元管理する台帳を整理します。

```
以下の情報をもとに、製造業の金型・治具管理台帳の
フォーマットを作成してください。

【保有している金型・治具の概要（例）】
保有金型: プレス金型 ○型、射出成形型 ○型
各金型の情報:
  ・金型No.: ○○-○○（社内管理番号）
  ・品番・品名: （生産品）
  ・顧客型 / 自社型の区別（顧客型は顧客資産の預かりもの）
  ・型種: 順送型 / 複合型 / 単型
  ・金型重量: ○kg / 保管場所: ○番棚
  ・制作年月・制作業者: ○年○月 / 金型メーカー○○
  ・使用機械: プレス機○号機（トン数: ○t）

ショット数管理:
  ・現在のショット数: ○万ショット
  ・推奨最大ショット数: ○万ショット（交換または大修理の目安）
  ・前回修理内容: ○年○月 / パンチ交換・刃研ぎ

出力形式:
・「金型管理台帳のフォーマット（上記全項目を含む）」
・「ショット数と金型寿命管理の方法（更新・修理時期の目安管理）」
・「顧客型の管理における注意事項（資産管理・保険・廃棄の手続き）」
・「金型保管状況の棚卸し確認チェックリスト（年1回程度）」
・「金型損傷・破損時の対応フロー（緊急修理の手配・顧客報告）」
```

## 型費分割回収計画書を作るプロンプト

新規金型を発注する際の型費の分割回収スケジュールを整理します。

```
以下の情報をもとに、金型費用（型費）の
分割回収計画書と管理フォーマットを作成してください。

【型費の概要】
新規金型: （例: プレス順送型 / 制作費: ○百万円）
顧客負担: （例: 型費は顧客持ち・制作費を立替払いし、製品単価に
  上乗せして分割回収する）

型費回収の条件:
  ・分割回収額: ○円/個（製品単価に上乗せ）
  ・分割回収期間: （例: ○万個生産するまで）
  ・回収完了見込み: ○年○月（月産○個として計算）
  ・回収完了後: 型費の上乗せ分を製品単価から差し引く（単価改定）

リスク:
  ・受注が計画を下回った場合、型費の回収が遅れる
  ・顧客が他社に切り替えた場合、未回収の型費が発生する可能性

出力形式:
・「型費分割回収計画表（月次: 回収累計額・回収残高・回収完了予定）」
・「型費の会計処理の整理（仕掛品としての計上・回収に伴う収益認識）」
  の骨子（注: 税理士確認を推奨）
・「顧客との型費回収に関する合意事項を契約書・注文書に明記する際の
  チェックポイント」
・「型費未回収リスクの評価と対策（担保・保証金の検討）」
・「複数顧客の型費回収状況を管理する月次集計フォーマット」
```

## 金型予防保全計画・修理記録を作るプロンプト

金型の予防保全スケジュールと修理記録台帳を整理します。

```
以下の情報をもとに、金型の予防保全計画と
修理記録台帳を作成してください。

【予防保全の概要】
管理方針: ショット数・使用期間に基づいた予防保全
  （不具合が出てから修理するのではなく、事前に計画的にメンテナンス）

主な予防保全の項目（例）:
  ・刃研ぎ（パンチ・ダイの研削）: ○万ショットごと
  ・潤滑・グリスアップ: 月1回（または○万ショットごと）
  ・ガイドポスト・ブッシュの点検・交換: ○万ショットごと
  ・ボルト・ネジの増し締め: 月次点検時

修理記録の管理:
  ・過去の修理記録が現場の紙ファイルに保管されているが、
    金型単位での集計ができていない
  ・どの金型が修理費用を多く使っているかが分からない

出力形式:
・「金型予防保全スケジュール表（金型別・作業種別・次回実施時期）」
・「金型修理記録台帳のフォーマット（修理日・内容・費用・
  担当・ショット数・修理後の品質確認）」
・「修理費用集計レポート（金型別・月次・累計）」
・「金型の廃棄・更新の判断基準（修理コストが新規制作費の何%に
  達したら更新を検討するか）」の整理
・「金型保全作業の外注先管理（金型修理業者の評価・管理台帳）」
```
