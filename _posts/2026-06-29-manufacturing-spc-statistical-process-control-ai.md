「Cp/Cpkの計算方法は知っているが、どの工程に適用すればいいか判断できない」「管理図を作ったが、何が「異常」かの判定基準を社員が理解していない」——SPC（統計的工程管理）は、データで工程の安定性を監視し、問題が大きくなる前に手を打つ仕組みです。AIを使って管理図の設計・工程能力評価・SPC導入計画書類を整備できます。

## SPC（統計的工程管理）・管理図活用書類を整理するプロンプト

```
以下の情報をもとに、精密加工工程へのSPC導入の
書類と管理体制を整備してください。

【SPC導入の対象工程と品質特性】
対象: CNC旋盤工程（外径加工）
品質特性: 外径寸法 φ25.000±0.005mm（公差幅: 0.010mm）
現状: 全数検査で対応・出荷後不良が月1〜2件発生
目的: 全数から抜き取り検査への転換（コスト削減）＋工程の安定化

【基礎データ（初期工程能力調査 n=25サブグループ×m=4個の計測結果）】
平均値（X̄）: 25.0003mm
標準偏差（σ）: 0.00165mm
工程能力指数: 
  Cp = 公差幅 / (6σ) = 0.010 / (6×0.00165) = 1.010
  Cpk = min[(USL-X̄)/3σ, (X̄-LSL)/3σ]
      = min[(25.005-25.0003)/0.00495, (25.0003-24.995)/0.00495]
      = min[0.95, 1.07] = 0.95

→ Cp=1.01（ギリギリ1以上）・Cpk=0.95（1未満 → 工程能力不足）
→ 現状では抜き取り検査への転換は困難・工程改善が必要

【改善の方向性（検討中）】
①工具補正のルール見直し（補正タイミングの明確化）
②段取り後の確認測定の追加（標準化）
③目標: Cpk ≥ 1.33（抜き取り検査移行の条件）

出力形式:
・「X̄-R管理図の設計書（サブグループサイズ・測定頻度・管理限界値の設定）」
・「工程能力評価書（Cp・Cpkの計算結果・判定・改善方針）」
・「SPC異常判定ルール（8つのウエスタンエレクトリックルールの説明と適用手順）」
・「SPC管理シート（現場作業者が記録・プロットするフォーマット）」
・「工程能力改善計画（Cpk 0.95 → 1.33達成のための改善ステップ）」
```
