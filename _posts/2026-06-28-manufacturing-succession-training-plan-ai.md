製造業の経営者にとって「次の世代に工場を任せられる人材を育てる」ことは最重要課題の一つです。AIを使って後継者候補の選抜基準・育成計画・キャリアパスを体系的に整理できます。

---

## 幹部候補の選抜基準を整理するプロンプト

次世代リーダーを選ぶ基準を明文化します。

```
以下の情報をもとに、製造業の幹部候補（次世代
工場長・部長候補）の選抜基準と評価方法を整理してください。

【当社の経営状況と課題】
会社規模: 従業員55名・年商¥500M・受託製造業
経営者（社長）: 68歳・後継者を3年以内に決めたい
現在の幹部: 工場長（62歳）・品質部長（59歳）・営業部長（55歳）
後継者候補として注目されている人材:
  ・田中主任（38歳）: 製造部・技術力が高い・工程管理に強い
  ・鈴木係長（42歳）: 品質部・顧客対応が得意・管理能力あり
  ・山田主任（35歳）: 営業部・コミュニケーション力高い・数字に強い

後継者に求められる能力（社長の考え）:
  ・製造の現場力（技術・品質・コスト管理）
  ・人をまとめるリーダーシップ
  ・経営・数字への理解
  ・取引先との良好な関係

出力形式:
・「幹部候補の評価基準（スキル・リーダーシップ・
  経営センス・人間力の4軸で各5段階の評価シート）」
・「3名（田中・鈴木・山田）の現在の強み・弱みの評価
  （評価基準に当てはめた仮の評価）」
・「後継者育成に必要な「不足スキルの特定」と
  各候補に対して優先的に強化すべき能力」
・「最終的な後継者選定の判断軸
  （いつ・どのプロセスで決定するか）」
・「現経営者が後継者に伝えるべき「見えない資産」
  （暗黙知・人脈・判断基準）の棚卸しシートの設計」
```

## 後継者育成計画（3年間）を整理するプロンプト

段階的に権限と経験を移譲していきます。

```
以下の情報をもとに、後継者候補（田中主任・38歳）の
3年間の育成計画を整理してください。

【田中主任の現状】
  ・現職: 製造部主任（班長3名をまとめる）
  ・強み: 現場改善（TPM・5S推進が得意）・技術力
  ・弱み: 財務・経営数字の理解が浅い・営業・顧客対応は未経験
  ・性格: 真面目・内向き・自己主張が少ない

【3年後の目標状態】
  ・工場長代理として工場運営全体を担える
  ・月次経営会議で部門の結果を報告・説明できる
  ・主要顧客3社との定期会議を単独で担当できる
  ・幹部会議で意見を発信できるリーダーシップがある

育成期間:
  Year 1（2026年）: 現場リーダーとしての実力定着
  Year 2（2027年）: 経営視点の習得・他部門への越境
  Year 3（2028年）: 工場長代理として実地経験

出力形式:
・「3年間の育成ロードマップ（年度別の目標・施策・評価の一覧）」
・「Year 1の具体的な育成施策
  （業務アサイン・研修・メンタリングの計画）」
・「財務・経営知識の習得プログラム案
  （外部研修・自社OJT・社長との勉強会）」
・「顧客対応経験の積み方（段階的な同行→補助→単独担当）」
・「田中主任への現状フィードバック面談の準備
  （どう話すか・どう育成への意欲を引き出すか）」
```

## キャリアパス制度を整理するプロンプト

会社全体のキャリアステップを見える化します。

```
以下の情報をもとに、製造業の
キャリアパス制度の設計と文書化を整理してください。

【当社の職位体系（現状）】
  一般社員 → 熟練社員 → 班長 → 主任 → 係長 → 部長 / 工場長
  ※昇進基準が明文化されておらず、社長の判断のみで決まっている
  ※若手から「どうすれば上に行けるか分からない」という声がある

【キャリアパスに組み込みたい要素】
  ①技術系キャリア（現場のスペシャリスト）
    例: プレス加工のエキスパート → 技能検定取得 → 技術指導員
  ②管理系キャリア（チームリーダー・管理職）
    例: 班長 → 主任 → 工場長補佐
  ③両方の要素を持つハイブリッドキャリア

制度導入の目的:
  ・若手の定着率向上（入社3年以内の離職率: 現在18%）
  ・目標を持って働ける職場づくり
  ・後継者育成との連動

出力形式:
・「キャリアパス制度の設計（2系統のキャリアルートと
  各段階の要件・目安年数）」
・「各ステップの昇格要件の明文化
  （技術スキル・業績・態度・資格の条件）」
・「制度発表時の全社説明資料の骨子
  （なぜ今この制度を作るか・自分はどうなれるか）」
・「入社1〜3年目の社員向けキャリア面談の質問シート」
・「キャリアパス制度と給与体系の連動の考え方
  （賃金テーブルの設計方針）」
```
