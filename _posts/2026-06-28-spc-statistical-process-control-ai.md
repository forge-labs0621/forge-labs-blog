統計的工程管理（SPC）は、製造工程のデータを統計的に分析して異常を検知し、品質を安定化させる手法です。AIを使って管理図の見方・工程能力指数の解釈・異常発生時の対応手順・報告書の作成を効率化できます。

---

## 工程能力指数（Cp・Cpk）の計算と解釈をするプロンプト

工程の品質能力を数値で評価します。

```
以下の測定データをもとに、工程能力指数（Cp・Cpk）を
計算し、品質改善の方針を整理してください。

【製品・工程の情報】
品目: （例: 精密旋削品 / 外径寸法）
品質特性: 外径寸法 φ50.0 ±0.05mm
  上限規格（USL）: 50.05mm
  下限規格（LSL）: 49.95mm
  規格幅（公差幅）: 0.10mm

【測定データ（直近25ロット、各ロット5個測定）】
（例: Excelデータや測定値一覧を貼り付けて計算依頼）
測定値の平均（X-bar）: 50.012mm
測定値の標準偏差（σ）: 0.018mm

【計算式】
Cp（工程能力指数）= （USL - LSL）/（6σ）
Cpk（偏心を考慮した工程能力指数）= min（CPU, CPL）
  CPU = （USL - X-bar）/（3σ）
  CPL = （X-bar - LSL）/（3σ）

出力形式:
・「Cp・CpkおよびCPU・CPLの計算結果」
・「工程能力の評価（1.67以上: 十分→1.33-1.67: 良好→
  1.00-1.33: 普通→1.00未満: 不十分の判定）」
・「工程の偏り（センタリング）の評価（平均が規格中央から
  どれだけずれているか）」
・「工程能力改善のための提案
  （散布を減らすか・平均を中央に近づけるか）」
・「品質改善優先アクション（設備・工具・段取り・測定の観点）」
```

## X-R管理図（シューハートコントロールチャート）の解釈プロンプト

管理図の異常パターンを特定し、原因を絞り込みます。

```
以下の管理図データをもとに、X-R管理図の
異常パターンを特定し、原因と対策を整理してください。

【管理図の情報】
品質特性: （例: プレス品のフランジ高さ）
UCL（管理上限）: 25.3mm
CL（中心線）: 25.0mm
LCL（管理下限）: 24.7mm

【観察された異常パターン（以下から該当するものを選択・記述）】
□ 1点が管理限界の外に出た（ルール1）
□ 連続9点が中心線より同じ側にある（ルール2: 連）
□ 連続6点が増加または減少している（ルール3: 傾向）
□ 連続14点が交互に増減している（ルール4: 上下動）
□ 連続3点中2点が管理限界の近くにある（ルール5）

【発生時の状況（メモ）】
・管理図で連続して上昇傾向が見られた
・発生前日に工具（刃具）の交換があった
・材料ロットが変わった

出力形式:
・「観察されたパターンの異常判定（どのルールに該当するか）」
・「各異常パターンに対応する典型的な原因の一覧（4M観点）」
・「原因追跡のための確認項目（チェックリスト）」
・「応急処置と再発防止策の整理」
・「管理図の記録・報告書テンプレート（異常発生→原因→対策→確認）」
```

## サンプリング検査計画を整理するプロンプト

ロット検査のサンプリング方法と合否判定基準を決めます。

```
以下の情報をもとに、受入検査・出荷検査の
サンプリング検査計画を整理してください。

【検査の背景】
検査種別: 受入検査（仕入先からの購入部品）
対象品目: （例: 外径寸法管理が必要なシャフト部品）
ロットサイズ: 500個/ロット

【品質要求】
不良品率の許容水準:
  ・AQL（合格品質水準）: 1.0%
  ・過検出リスク（生産者危険）は小さくしたい
  ・検査の効率化も重視（全数検査は時間がかかりすぎる）

【現状の検査方法】
現在は10個/ロットのサンプリングで、0個不良なら合格としている。
（この方法で十分かどうか確認したい）

出力形式:
・「MIL-STD-1916 / JIS Z 9015に基づくサンプリング計画の推奨値
  （サンプルサイズ・合格判定個数・不合格判定個数）」
・「現行の10個サンプリング方法のリスク評価（OC曲線の概念説明）」
・「検査コストと品質リスクのバランスを考えたサンプル数の推奨」
・「サンプリング検査記録フォーマット（ロット番号・検査日・
  結果・判定・担当者）」
・「緩和検査・厳格化検査の切替基準（継続合格/不合格で切り替える方法）」
```
