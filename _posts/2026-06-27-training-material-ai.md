製造業の社内研修テキストは「作ること自体が大変」「ベテランが作るべきだが時間がない」という悩みをよく聞きます。AIを使うと、既存の知識・メモをもとに研修テキストの構成案・本文・設問を素早く作れます。

## AIで作れる研修テキストの種類

- **新人入職研修**: 会社の概要・基本ルール・施設案内・緊急時対応
- **安全衛生教育**: 法定の安全教育（雇入れ時・作業変更時）の資料
- **技能研修テキスト**: 作業手順・設備操作・品質管理の知識
- **マナー・コンプライアンス研修**: 職場のルール・情報管理・ハラスメント防止

## 新人入職研修テキストを作るプロンプト

---

```
以下の情報をもとに、製造業（工場）の新人入職研修テキストの骨子を作成してください。

【会社・工場の概要】
（例：金属部品の受託加工業。従業員40名。愛知県○○市。取引先は自動車部品メーカー中心）

【新人に最初に伝えたいこと（優先度が高いもの）】
（例：
・挨拶・報告・連絡・相談（ほうれんそう）の徹底
・安全の基本（保護具着用・通路の歩行・立入禁止区域）
・工場内での携帯電話の取り扱いルール
・不良品を発見したときの報告先と手順
・トイレ・食堂・更衣室の場所と使い方
・緊急時（怪我・火災）の対応手順
）

【研修の時間】（例：初日の午前2時間）

出力形式:
・研修テキストの目次（大項目6〜8個）
・各項目の要点を3〜5つの箇条書きで
・研修後の確認テスト（○×クイズ 10問）も最後に含める
```

## 雇入れ時安全衛生教育のテキストを作るプロンプト

労働安全衛生法では、全ての新入社員に「雇入れ時の安全衛生教育」を実施することが義務付けられています。

```
以下の条件で、雇入れ時安全衛生教育（労働安全衛生法第59条）の
研修テキストを作成してください。

【業種・作業の種類】
（例：機械加工業（切削・旋盤・フライス加工）、製品の組立・検査）

【教育すべき事項（労働安全衛生法施行規則第35条）に沿って】
1. 機械等・原材料等の危険性・有害性
2. 安全装置・有害物抑制装置・保護具の使用方法
3. 作業手順
4. 作業開始時の点検
5. 当該業務に関して発生しやすい疾病の原因・予防
6. 整理・整頓・清潔の保持
7. 事故時の応急措置・退避
8. その他当該業務に関する安全・衛生のために必要な事項

各事項について、機械加工業で特に重要な具体例を含めて
テキストにまとめてください。

出力形式:
・各教育事項を見出しにし、実際に起きやすい事故例・対策を2〜3例含める
・新人でも理解できる平易な文章
・教育実施記録欄（日付・担当者・受講者署名欄のひな形）を最後に付ける
```

## 技能研修テキスト（機械操作編）の骨子を作るプロンプト

特定の設備・工程の操作研修テキストを作る際に使えます。

```
以下の情報をもとに、製造現場の技能研修テキストの骨子を作成してください。

【研修対象の設備・作業】
（例：NC旋盤の基本操作研修）

【受講者のレベル】
（例：製造未経験の新入社員（高校新卒））

【研修で習得してほしいこと】
（例：
・NCプログラムの選択と起動手順
・ワーク（加工物）の取り付け・固定方法
・試し加工と寸法確認の手順
・工具の交換タイミングの判断
・加工中の異常（異音・振動）への対応
）

【研修期間・形式】
（例：3日間（座学1日＋実技2日））

出力形式:
・研修テキストの目次と各章の要点
・実技練習の順番（スモールステップで）
・習熟度チェックリスト（「できた」「まだできない」「上司確認済み」の3段階）
・研修修了の判定基準
```

## コンプライアンス・マナー研修テキストを作るプロンプト

職場のルール・情報管理・ハラスメント防止の研修にも使えます。

```
以下の内容を、製造業の職場向けコンプライアンス研修テキストにまとめてください。

【研修で取り扱いたいテーマ（複数選択可）】
□ 職場のルール・服務規程の基本
□ 情報管理（顧客情報・設計情報の取り扱い）
□ SNS・スマートフォンの使用ルール
□ ハラスメント防止（パワハラ・セクハラ）
□ 5S（整理・整頓・清掃・清潔・躾）の実践
□ その他（自由記入）

【研修時間】（例：30分〜1時間）

出力形式:
・各テーマを見出し別に
・「してはいけないこと」は具体例（ケーススタディ）で示す
・Q&A形式のセクションを最後に3〜5問入れる
・受講確認サインシート（プレースホルダー）を付ける
```

## 研修効果の確認テストを作るプロンプト

研修終了後に理解度を確認するテストも作れます。

```
以下の研修内容に対する確認テストを作成してください。

【研修の主なテーマ・カバーした内容】
（例：雇入れ時安全衛生教育・機械加工の基本・5Sの実践）

【テストの形式】
□ ○×問題（10問）
□ 選択問題（4択、10問）
□ 記述問題（3問）
□ 混合形式

出力形式:
・問題と正解・解説（採点者用）
・受験者用シート（正解なし版）
・合格基準の目安（例：70点以上を合格）
```

---

## AIを使う際の注意

- 法定の安全衛生教育（雇入れ時・作業変更時）は法令で定められた事項を網羅する必要があります。AIが作成したテキストで法定教育事項が漏れていないか、必ず確認してください
- 技能研修テキストは「機械操作の手順」を含むため、**ベテラン・管理者が内容を確認してから使用**することが必須です
- 研修実施の記録（日時・受講者・内容・実施者）は法令上の保存義務がある場合があります（安全衛生教育の記録は3年保存が推奨）
