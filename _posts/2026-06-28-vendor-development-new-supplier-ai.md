製造業では既存サプライヤーへの依存リスク低減・コスト削減・品質向上のために、新規仕入先の開拓（ベンダー開発）が重要です。AIを使って仕入先候補の評価・訪問調査・認定プロセスの書類を効率化できます。

---

## 新規仕入先の候補評価シートを作るプロンプト

候補仕入先を多角的に評価するフォーマットを作ります。

```
以下の情報をもとに、新規仕入先候補の
評価シートとスクリーニング基準を整理してください。

【調達品目の概要】
調達品目: （例: 精密切削加工品（旋削・フライス））
月産予定: 500個/月
品質要求: ISO 9001取得または同等の品質管理体制
希望単価: 現行単価の90%以内

【候補仕入先の概要（3社）】
候補A: A精機（株）
  ・従業員 35名 / 設立25年 / 設備: CNC旋盤10台・マシニング5台
  ・ISO 9001取得済み / 主要顧客: 自動車部品メーカー
  ・現在の取引なし

候補B: B加工（有）
  ・従業員 12名 / 設立10年 / 設備: CNC旋盤5台
  ・ISO 9001未取得（社長が「取得検討中」と言っている）
  ・過去に1回スポットで発注したことがある

候補C: C精密工業（株）
  ・従業員 60名 / 設立40年 / 設備: CNC旋盤20台・他多数
  ・IATF 16949取得済み / 自動車Tier1サプライヤー
  ・単価は現行より5%高い見積もりが来ている

出力形式:
・「仕入先評価シート（品質・技術・価格・納期・財務・
  コンプライアンスの6軸評価）」
・「各候補の評価（A/B/C評価）と総合スコアリング」
・「各候補の強み・弱み・リスクの整理」
・「最終候補の絞り込み推奨（なぜその候補を優先するか）」
・「追加情報収集が必要な項目（次のアクションリスト）」
```

## 新規仕入先訪問調査チェックリストを作るプロンプト

工場訪問時に確認すべき項目を整理します。

```
以下の情報をもとに、新規仕入先への工場訪問調査の
チェックリストを作成してください。

【訪問調査の目的】
候補仕入先A精機（株）の工場を訪問し、
当社の要求品質を継続的に満たせるか確認する

【確認したい主な項目】
品質管理体制:
  ・ISO 9001の運用状況（文書化・記録の管理）
  ・検査体制（設備・人員・記録）
  ・不良品発生時の対応フロー

生産能力・設備:
  ・当社要求品目の加工能力（機械・工具・治具）
  ・設備の稼働状況と余力（当社への対応余力）
  ・設備の保全状況（老朽化度合い）

現場の状態:
  ・5Sの状況（整理・整頓・清掃・清潔・躾）
  ・材料・仕掛品の管理状況
  ・作業手順書の整備・活用状況

経営・財務:
  ・経営者の品質・顧客対応への姿勢
  ・会社の財務安定性（主要顧客・売上規模）
  ・後継者問題・経営リスク

出力形式:
・「工場訪問調査チェックリスト（確認項目・確認方法・
  評価基準（○/△/×）の一覧表）」
・「訪問調査報告書のフォーマット（確認結果・良い点・
  懸念点・総合評価・推奨アクション）」
・「訪問前に準備すべき資料（当社の品質要求書・NDA等）」
・「訪問時の質問リスト（経営者・製造担当・品質担当別）」
・「サンプル発注の判断基準（訪問後のGoNo-Go基準）」
```

## 仕入先認定書類・取引開始手続きを整理するプロンプト

評価完了後の認定・取引開始手続きを整理します。

```
以下の情報をもとに、新規仕入先の認定と
取引開始に必要な書類・手続きを整理してください。

【認定に向けた経緯】
A精機（株）への訪問調査が完了。
総合評価: 品質・設備ともに良好、価格も条件内
サンプル発注: 試作品50個 → 全数合格（Cpk≥1.33）
経営者の品質への姿勢: 良好

認定に際して確認・整備が必要な事項:
  ① 仕入先登録申請書（先方に記入してもらう会社情報）
  ② 品質保証契約書の締結
  ③ NDA（機密保持契約書）の締結
  ④ 取引基本契約書の締結
  ⑤ 振込先口座情報の登録
  ⑥ 社内の仕入先台帳への登録（購買・会計システムへの登録）

今後の取引方針:
  ・初年度は月産100個でスタート（リスク分散のため）
  ・問題がなければ翌年度から月産500個に拡大
  ・6ヶ月後に定期評価を実施

出力形式:
・「仕入先認定書類チェックリスト（取得すべき書類一覧）」
・「仕入先登録申請書のフォーマット（先方に記入してもらう項目）」
・「新規仕入先への最初の発注書の記載ポイント（初回特有の注意点）」
・「定期評価の実施計画（6ヶ月後の評価フォーマット）」
・「社内の購買部門・経理部門・品質部門への新規仕入先通知文」
```
