5S活動の定着には定期的な診断と見える化が不可欠です。AIを使って5S診断シート・採点基準・月次巡回記録・改善計画を体系化できます。

---

## 5S診断シートと月次巡回評価書類を整理するプロンプト

```
以下の情報をもとに、製造現場の5S診断シートと
月次巡回評価書類を整理してください。

【5S活動の現状と目標】
対象エリア: 機械加工工場全体（加工エリア / 部品倉庫 / 工具室 / 通路）
現状レベル: 自己評価で「整頓」「清潔」が弱い
  ・工具が決まった場所に戻されていないことが多い
  ・清掃は毎日しているが「きれいに見える清掃」止まりで、
    機械の内部・奥まった箇所が汚れたまま
診断頻度: 月1回（製造部長が巡回）

5Sの採点基準（レベル設定の希望）:
  各Sを 0〜5点で採点（5点満点 × 5項目 = 合計25点満点）
  目標スコア: 20点以上

出力形式:
・「5S診断チェックシートのフォーマット
  （整理 / 整頓 / 清掃 / 清潔 / 躾 各5項目の採点基準を記述した表形式）」
  各採点基準は「1点:〇〇の状態」「3点:〇〇」「5点:〇〇」の3段階を明示
・「エリア別5S月次スコアシート（加工 / 倉庫 / 工具室 / 通路 別に採点）」
・「5S巡回記録フォーマット（日時 / エリア / 良かった点 / 問題点 / 改善期限）」
・「5S改善指示書（問題エリアの写真添付欄 / 改善期限 / 担当者 のフォーマット）」
・「5Sスコアの月次推移グラフの説明文と「5S掲示板」への反映方法」
```
