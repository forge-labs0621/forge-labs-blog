製造業における在庫の正確な把握と評価は月次損益計算の基礎です。棚卸差異の原因分析・在庫評価方法の選択・月末棚卸作業の標準化をAIで効率化できます。

---

## 月末棚卸作業の計画と手順書プロンプト

棚卸作業をスムーズに実施するための準備を整えます。

```
以下の情報をもとに、月末棚卸作業の
計画・手順書・チェックリストを整理してください。

【当社の在庫の概要】
種別と保管場所:
  ①原材料倉庫: SPCC・アルミ板・SUS材 / 品目数: 約60種
  ②仕掛品: 各工程中の半製品 / プレス加工後→バリ取り前等
  ③製品倉庫: 完成品（出荷待ち） / 品目数: 約150種
  ④副資材: 切削油・グリス・消耗品 / 品目数: 約40種

棚卸実施タイミング: 毎月末（翌月1日に経理へ連絡）
前月の棚卸差異: 製品倉庫で品番P-205の数量が帳簿-実数=▲30個（不足）

【人員と役割】
  棚卸責任者: 倉庫担当・橋本
  カウント担当: 各現場リーダー3名（原材料・仕掛品・製品それぞれ）
  立会い: 経理部・田村（翌日に実数を確認）

出力形式:
・「月末棚卸の作業スケジュール（棚卸前日〜翌日の経理報告まで）」
・「棚卸作業手順書（カウント方法・記録方法・ダブルカウント防止）」
・「棚卸カウントシートのフォーマット（品番・品名・保管場所・
  帳簿数量・実数・差異）」
・「仕掛品の棚卸方法（工程中の半製品をどう数えるか）」
・「棚卸差異▲30個（P-205）の原因調査のアプローチ
  （出荷記録・生産記録・廃棄記録の確認順序）」
```

## 棚卸差異の原因分析と対策プロンプト

差異が出た原因を特定し再発防止策を立てます。

```
以下の情報をもとに、棚卸差異の
原因分析と再発防止策を整理してください。

【今月の棚卸差異の内訳】
製品倉庫（差異あり）:
  品番P-205 / 帳簿1,050個 / 実数1,020個 / 差異▲30個（不足）
  品番P-301 / 帳簿560個 / 実数577個 / 差異+17個（過剰）

原材料倉庫:
  SPCC 1.6mm / 帳簿1,200kg / 実数1,180kg / 差異▲20kg
  差異金額（材料単価¥120/kg × 20kg = ▲¥2,400）

【差異原因の調査結果（聞き取り・記録確認）】
P-205 ▲30個:
  ・出荷記録を確認したところ、6月28日の出荷30個分が
    在庫システムに未入力のまま放置されていた（入力漏れ）

P-301 +17個:
  ・生産実績で18個余分に製造されていたが、15日の不良品
    1個の廃棄記録が未入力のまま → 差異+17個

SPCC ▲20kg:
  ・切断くずの実際の発生量が、標準の損率3%に対して
    4.5%と高かった（切断条件の問題）

出力形式:
・「差異原因の分類（入力漏れ・廃棄記録漏れ・材料歩留差異の3種類）」
・「各差異の正しい修正仕訳と在庫修正の手順」
・「再発防止策（入力漏れへの対策・廃棄記録の徹底方法）」
・「材料歩留差異（SPCC）の原価への影響額の計算」
・「棚卸差異管理台帳のフォーマット（差異品番・金額・原因・対策・
  完了確認の月次記録）」
```

## 在庫評価方法（総平均法・先入先出法）を整理するプロンプト

原価計算に使う在庫評価の方法を理解・適用します。

```
以下の情報をもとに、在庫評価の計算方法と
月次原価計算への適用を整理してください。

【当社の在庫評価方法】
現在の方法: 総平均法（移動平均法）
適用範囲: 主要原材料（SPCC・アルミ板・SUS材）

【今月（6月）の材料受払い記録（SPCC 1.6mm）】
  期首在庫: 1,000kg / ¥120,000（@¥120/kg）
  仕入①: 500kg / ¥65,000（@¥130/kg）← 仕入単価が上昇
  仕入②: 300kg / ¥39,000（@¥130/kg）
  払出: 1,380kg（製造現場への投入）
  期末在庫: 420kg（実棚卸確認済み）

出力形式:
・「総平均法による月平均単価の計算
  （全受入額 ÷ 全受入数量 = 月末の在庫評価単価）」
・「今月の払出金額（製造原価に算入される材料費）の計算」
・「期末在庫の評価額の計算」
・「移動平均法との違い（計算タイミング・特徴の比較）」
・「材料費の月次原価レポートのフォーマット
  （材料別の受払い・評価単価・払出金額・期末在庫額の一覧）」
```
