製造現場で設備トラブルや機械故障が発生したとき、対応と並行して報告書を書く必要があります。「復旧に追われて報告書を書く時間がない」「原因分析のコメントをどう書けばいいか迷う」という現場担当者向けに、AIを使って報告書の下書きを素早く仕上げる方法を紹介します。

## 設備トラブル報告書で必要な情報

設備トラブル・故障報告書には、基本的に以下の情報が必要です。

1. **トラブルの概要**（いつ・どの設備で・何が起きたか）
2. **発見のきっかけ**（警告音、異音、品質異常、担当者が気づいた等）
3. **生産への影響**（停止時間、影響数量、後工程への影響）
4. **原因の推定**（なぜ起きたか・なぜ気づかなかったか）
5. **対応内容**（応急処置、恒久対策）
6. **再発防止策**

これらを現場の短いメモから、AIが報告書向けの文章に整えてくれます。

## 基本プロンプト（設備トラブル報告書）

---

```
以下のメモをもとに、上司・保全担当者向けの設備トラブル報告書を作成してください。

【発生日時】
（例：2026年7月○日 10:30頃）

【設備・機械名】
（例：プレス機No.3、CNC旋盤2号機）

【トラブル内容メモ】
（例：異音がして自動停止した、油圧が下がって動かなくなった、
加工中に製品にキズが入り出した等）

【生産への影響】
（例：2時間停止、○○個の加工が遅延）

【応急処置メモ】
（例：設備を停止し保全に連絡、油圧ポンプ交換で復旧）

【出力形式】
・タイトル「設備トラブル報告書」
・発生日時・設備名・発生場所
・トラブル内容（具体的・簡潔に）
・生産への影響
・原因の推定（直接原因・背後要因）
・対応内容（応急処置・恒久対策）
・再発防止策
・報告者・確認者欄
```

## 出力例

---

**設備トラブル報告書**

発生日時：2026年7月○日 10:30  
設備名：プレス機 No.3（機械加工エリア）  
報告者：○○ / 確認者：□□

**1. トラブル内容**

2026年7月○日 10:30頃、プレス機No.3の稼働中に異音（金属打音）が発生し、安全装置が作動して自動停止した。確認したところ、油圧ラインの継手部分に緩みが生じており、わずかな油漏れが確認された。

**2. 生産への影響**

停止時間：約2時間（10:30〜12:30）  
影響数量：当日生産計画の約20%（○○個）が後ズレ

**3. 原因の推定**

直接原因：油圧ラインの継手部分の締め付け緩み  
背後要因：前月の定期点検で継手部の増し締め確認が記録に残っていなかった

**4. 対応内容**

応急処置：設備停止・油漏れ箇所の清掃・継手の締め直し  
恒久対策：保全担当による全継手部の増し締め確認（実施日：7月○日）

**5. 再発防止策**

月次定期点検に「油圧ライン継手の増し締め確認」を点検項目として追加し、記録に残す。

## 原因分析コメントに迷ったとき

「なぜ起きたか」を掘り下げる「なぜなぜ分析」の下書きもAIに任せられます。

```
以下のトラブルについて、なぜなぜ分析の下書きを作成してください。

トラブル: プレス機の油圧継手の緩みによる停止

1段階目の「なぜ」から始めて、4〜5回の「なぜ」で根本原因まで掘り下げてください。
結果は「なぜ1→なぜ2→なぜ3→...→根本原因」の形で出力してください。
```

出力例（抜粋）：

- **なぜ1**: 油圧継手が緩んでいた
- **なぜ2**: 定期点検で継手部の増し締めが実施されていなかった
- **なぜ3**: 点検手順書に継手確認の項目が明記されていなかった
- **なぜ4**: 手順書の作成時に油圧系の経年劣化を考慮した項目が盛り込まれていなかった
- **根本原因**: 点検手順書のレビューに設備経験者が関与していなかった

→ 対策: 点検手順書の改訂に保全担当者が参加する仕組みをつくる

## 緊急報告メール（上長向け）

トラブル発生直後に上長へ速報を送る際のメール文も、AIで素早く作れます。

```
設備トラブル発生直後に上司へ送る緊急報告メールを作成してください。

状況: プレス機が停止、原因調査中、復旧見込みは1〜2時間後
連絡内容: 事実の報告・対応中であること・詳細は後で報告すること
文体: 簡潔・緊急感あり・丁寧

注意: 会社名・氏名はすべてプレースホルダーで出力
```

## トラブル履歴の記録に活用する

同じ設備の過去のトラブルを記録として残しておくと、再発防止と保全計画の見直しに役立ちます。

```
以下の設備トラブル報告書3件のメモをまとめて、
「設備No.3 トラブル履歴レポート」として整理してください。

発生傾向・共通する原因・推奨される予防保全の見直し点を最後にまとめてください。

【報告書1】...
【報告書2】...
【報告書3】...
```
