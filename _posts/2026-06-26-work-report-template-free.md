製造業の現場で毎日書く作業日報。「書く内容はわかっているのに、文章にまとめるのに時間がかかる」という方向けに、AIで素早く作るためのテンプレートを公開します。

## 基本の作業日報テンプレート

まず、項目を埋めるだけで完成する基本フォーマットです。

---

```
【作業日報】

日付：
作業者氏名：
所属：

■ 本日の作業内容
・作業名：
・対象品番/品名：
・作業時間：　　時〜　　時（実働　　時間）
・完了数量：　　個 / 目標　　個

■ 品質・異常
・不良・異常の有無：　あり ／ なし
・内容（あった場合）：

■ 安全
・ヒヤリハット・異常の有無：　あり ／ なし
・内容（あった場合）：

■ 翌日への申し送り

■ その他・連絡事項
```

---

このフォーマットに手書きやExcelで入力しても使えますし、このまま印刷して紙で運用することもできます。

## AIを使って日報を素早く作る方法

上のテンプレートの各項目を**箇条書きのメモとして**ChatGPTやClaudeに渡すと、整った文章の日報に変換してくれます。

### ステップ1: 現場でメモを取る

作業しながら、以下のようなメモを残します。文章でなくてかまいません。

```
6/26
旋盤 A-102
9時から16時
45個完了（目標50個）
刃物の摩耗早め 明日確認
問題なし（安全）
```

### ステップ2: AIに渡すプロンプト

上のメモをそのまま以下のプロンプトに貼り付けてAIに送ります。

---

```
以下のメモをもとに、製造業の作業日報を作成してください。
読みやすく整えてください。

【メモ】
（ここにメモを貼り付ける）

【出力形式】
・作業内容：
・完了数量：
・品質・異常：
・安全：
・申し送り：
```

---

### ステップ3: 出力を確認・修正する

AIが出した文章を確認して、数字や内容が正確かどうかチェックします。AIの出力はあくまで下書きなので、必ず人が最終確認してから提出してください。

## よくある使い方の工夫

### 複数の作業をまとめる場合

午前・午後で違う作業をした場合は、それぞれのメモをまとめて渡します。

```
午前：旋盤 A-102 25個
午後：検査 B-55 全数（45個）異常なし
```

これを一つのプロンプトに入れれば、まとめた日報が出てきます。

### 毎日同じ作業の場合

同じ品番・作業を毎日繰り返している場合は、「数量と異常の有無だけ毎日変える」形にすると時短になります。

```
昨日と同じ内容で、数量を50個・異常なしに変えて日報を作ってください。
```

## 注意事項

- AIが出力した内容は必ず人が確認してから提出してください
- 取引先名や機密情報はAIに入力しないようにしましょう
- 出力はあくまで下書きです。責任のある報告書は人が確認することが前提です
