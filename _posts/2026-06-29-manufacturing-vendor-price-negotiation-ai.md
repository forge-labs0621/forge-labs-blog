仕入先との価格交渉は根拠を数値で示すことが重要です。AIを使って原価分析・コストダウン要請書・値上げ交渉への対応書類・合意後の覚書を体系化できます。

---

## 仕入先へのコストダウン要請書を整理するプロンプト

```
以下の情報をもとに、仕入先へのコストダウン要請書と
交渉根拠資料を整理してください。

【交渉背景】
対象仕入先: 〇〇樹脂工業（プラスチック部品の成型外注先）
現在の購入価格: 1,850円/個（3年間据え置き）
月間購入量: 2,000個（年間 2,400万円の仕入れ）
要請内容: 現行価格から10%コストダウン（目標: 1,665円/個）
要請期限: 2026年10月1日〜

コストダウンの根拠:
  ①当社の購入量が3年前の1,500個/月→2,000個/月に増加（数量効果）
  ②樹脂材料相場が1年前比で▲8%低下（市場データあり）
  ③競合他社の見積: 1,720円/個（参考見積を取得済み）
  ④長期安定発注のコミットメント（3年契約を提案）

出力形式:
・「コストダウン要請書（公式文書・理由・根拠・要請金額・期限・協議申し入れ）」
・「根拠資料の整理（数量増加効果の試算 / 材料市況の変化 / 競合見積との比較）」
・「交渉シナリオ設計（最大要求 / 着地目標 / 譲れないライン）」
・「コストダウン交渉のQ&A対応集（仕入先の反論と当社の回答案）」
・「合意後の覚書（新単価 / 適用開始日 / 数量保証 / 品質条件）」
```

## 仕入先からの値上げ申請への対応書類を整理するプロンプト

```
仕入先から原材料高騰を理由とした価格改定申請が来た際の
対応書類を整理してください。

【状況】
申請元: 〇〇金属工業（精密プレス部品の外注先）
申請内容: 鋼材価格高騰を理由に現行価格から15%値上げ要請
  現行: 3,200円/個 → 要請価格: 3,680円/個（+480円）
  適用希望日: 2026年8月1日
月間影響額: +48万円（月購入量1,000個×480円）

自社の状況:
  ・顧客への価格転嫁交渉は別途実施予定だが、すぐには無理
  ・〇〇金属工業は長期取引先で品質・納期は安定している
  ・代替仕入先への切り替えは技術的に半年以上かかる見込み

出力形式:
・「値上げ申請の受領確認書と分析依頼（根拠資料・コスト構造の提出要求）」
・「値上げ妥当性の検証チェックリスト（材料比率 / 市況根拠 / 他社比較）」
・「当社回答書のパターン案
  ①全面拒否: 理由と代替案の提示
  ②部分受容: 5%受容・残りはコスト改善努力を要請
  ③条件付き受容: 数量増加・支払条件改善を条件に一部受容」
・「段階的値上げ交渉（一括ではなく複数回に分けて受け入れる提案）」
・「顧客への価格改定交渉資料（材料高騰の説明と転嫁申請書）」
```
