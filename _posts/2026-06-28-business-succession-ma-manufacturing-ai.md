中小製造業では経営者の高齢化が進み、事業承継は喫緊の課題です。AIを使って事業承継計画書の骨子整理・現状分析・後継者への引き継ぎ書類の文書化を効率化できます（法的・税務的な手続きは専門家が必要）。

---

## 事業承継計画書の骨子を整理するプロンプト

経営者交代に向けた計画書の全体構成を整理します。

```
以下の情報をもとに、製造業の事業承継計画書の骨子を
作成してください。

【自社の状況】
創業: ○○年（創業○年）
事業内容: （例: 金属プレス加工・受託製造業）
売上高: ○百万円 / 従業員: ○名
現経営者: （例: 代表取締役 ○○（○歳）/ 創業○年・会長職移行を検討）

【承継の方針】
承継先候補: （例: 息子（35歳・製造部マネージャー / 入社10年）
              ただし経営経験・財務知識が不足）
目標時期: 3〜5年後をめどに実質的な経営権を移転
承継方法: 株式の生前贈与・相続の組み合わせを検討

【承継前に整備すべき課題（現状認識）】
・自社の強み（技術力・顧客関係）が経営者個人に依存
・取引先との関係が代表者の個人的な信頼に基づいているものが多い
・財務内容の透明化（役員報酬・経費等の見直しが必要）
・後継者の経営スキル・リーダーシップの育成

出力形式:
・「事業承継計画書の構成（目次と各章の要点）」
・「経営者から後継者への引き継ぎ事項一覧（技術・人脈・取引先・
  財務・経営ノウハウ）」のフォーマット
・「後継者育成の3〜5年間ロードマップ」
・「事業承継税制（贈与税・相続税の特例）の確認ポイント（専門家へ
  相談すべき事項のチェックリスト）」
・「中小企業庁の事業承継支援（事業承継・引継ぎ補助金等）の活用検討」
```

## 自社の強み・ビジネスモデルを後継者に引き継ぐ書類を作るプロンプト

経営者の頭の中にある暗黙知を文書化します。

```
以下の情報をもとに、経営者の「暗黙知・経営ノウハウ」を
後継者に引き継ぐための文書を作成してください。

【引き継ぐ内容の例】
自社の強みと差別化:
  ・なぜ顧客から選ばれているか（技術力・納期・品質・価格・対応力）
  ・競合他社ができない・やっていないことは何か

主要取引先との関係:
  ・顧客A（○○㈱）との関係の歴史: ○年前のクレーム対応で信頼を得た
    重要なキーパーソン: 購買部長の○○氏（○歳）
    注意事項: 価格の話は決裁者に直接するよりまず担当者への根回しが重要
  ・主要仕入先Bとの関係: 融通してもらえる理由・関係維持の秘訣

社内の人間関係・組織の機微:
  ・製造部のベテラン○○さんは技術の要。退職すると技術が失われるリスク
  ・○○さんと○○さんの間に関係上の課題がある（注意が必要）

経営判断の哲学・基準:
  ・受注判断のルール（断る案件の基準・受ける基準）
  ・値下げ交渉への対応方針
  ・リスクをとるか守るかの判断基準

出力形式:
・「経営者の経験・ノウハウを聞き出すための質問リスト30問」
・「主要取引先・仕入先関係マップのフォーマット（キーパーソン・
  関係のポイント・注意事項）」
・「経営判断の基準書（受注基準・値下げ対応・リスク判断の原則）」の骨子
・「引き継ぎミーティングの議題フォーマット（月次・四半期ごと）」
```

## 第三者承継（M&A）に向けた会社概要・自社PR資料を作るプロンプト

M&A仲介業者や買い手候補に提示する会社紹介資料を整理します。

```
以下の情報をもとに、M&A（第三者承継）の際に
仲介業者や買い手候補に提示するノンネームシート・
企業概要書（IM）の骨子を作成してください。

【自社の基本情報（例）】
業種: 精密部品加工（切削・プレス・溶接）
設立: ○○年 / 従業員: ○名 / 本社・工場所在地: ○県
売上高: ○百万円（直近3期）
営業利益: ○百万円（EBITDA: ○百万円）

自社の強み:
  ・ISO 9001・IATF 16949認証取得（自動車Tier1取引あり）
  ・精密部品加工の実績（公差±0.01mmクラス）
  ・顧客基盤: Tier1メーカー3社との長期安定取引（15〜20年）
  ・技術者○名（うち1級技能士○名）

資産・設備:
  ・工場: 自社所有 ○㎡ / 設備: 工作機械○台・プレス機○台
  ・土地建物の簿価: ○百万円
  ・純資産: ○百万円

M&Aを検討する理由: 後継者不在 / 規模拡大による競争力強化

出力形式:
・ノンネームシート（企業を特定されないレベルの1枚概要書）の文面
・企業概要書（IM）の目次と各章の記載内容
・「M&A仲介業者に相談する際に準備する書類一覧」
・「M&Aの流れ（検討〜成約〜統合）の概要説明」
・「譲渡価格の算定方式（純資産法・DCF法・EV/EBITDA倍率）の
  簡易説明と自社での目安の算出方法」
```
