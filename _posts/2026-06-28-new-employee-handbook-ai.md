製造業では毎年4月や随時採用で新入社員が入社します。工場のルール・安全規則・基本業務の流れを伝えるハンドブックや入社初日の案内資料をAIで効率よく作成できます。

---

## 新入社員ハンドブックの骨子を作るプロンプト

工場の新入社員向けハンドブックの構成を整理します。

```
以下の情報をもとに、製造業（工場勤務）の新入社員向け
ハンドブックの骨子（章立てと各章の内容）を作成してください。

【自社の概要】
業種・規模: （例：プレス加工・溶接・組立の受託製造業 / 従業員75名）
勤務体系: （例：日勤のみ（7:45〜16:45）/ 繁忙期のみ残業あり）
新入社員の想定: （例：高校新卒・工業高校からの採用 / 未経験者が多い）

【ハンドブックに含めたい内容】
会社のこと:
  ・会社の概要・事業内容（何を作っているか）
  ・会社のルール・方針（5S、安全第一、品質最優先）
  ・組織図・部署の紹介
工場のルール:
  ・安全規則（保護具の着用・立入禁止エリア・緊急時の対応）
  ・5S活動の基本（整理・整頓・清掃・清潔・躾）
  ・報告・連絡・相談（ホウレンソウ）の重要性
  ・持ち込み禁止品・撮影禁止区域
働くためのルール:
  ・勤怠管理（出退勤・遅刻・欠勤の届け出方法）
  ・給与・昇給・賞与の仕組み
  ・福利厚生（各種手当・健康保険等）
  ・研修・資格取得制度

出力形式:
・ハンドブックの目次（章立て）と各章の内容概要
・「新入社員が最初に読むべき」優先度の高いセクション
・読みやすさの工夫（箇条書き・チェックリスト形式を活用）の提案
```

## 工場安全ルール説明書を作るプロンプト

新入社員向けに工場の安全ルールを説明する文書を作ります。

```
以下の情報をもとに、新入社員向けの工場安全ルール説明書を
作成してください。

【自社工場の特徴と主な危険】
（例：
・プレス機・溶接機・フォークリフトが稼働している
・床に切り粉・廃油が落ちることがある
・重量物の移動がある
・一部エリアに騒音・高温がある
）

【必ず守らせたいルール】
（例：
・作業エリアでは必ず安全靴・保護メガネ・ヘルメットを着用する
・機械の起動・停止は担当者のみ行う（勝手に動かさない）
・フォークリフトの走行ルートには絶対に立ち入らない
・怪我・ヒヤリハットは必ず班長に報告する
・立入禁止区域には入らない（必要なら班長に同行を依頼）
・緊急停止ボタンの場所を必ず確認する
）

【緊急時の対応】
（例：
・けが: 直ちに班長へ報告 → 応急処置（救急箱の場所）→ 安全衛生担当へ
・火災: 非常ベル → 避難 → 消火器（場所を説明）/ 119番は班長が行う
・設備異常: 機械を止める → 班長へ報告 → 自分で修理しない
）

出力形式:
・A4 2枚程度の説明書（新入社員が1人で読んで理解できる分量）
・重要なルールは太字・赤字表示のプレースホルダーで強調
・「こんな場合どうする？」Q&A形式のコラムを3〜5件
・「ルールを守ることは自分と仲間を守ること」というメッセージ
```

## 入社初日のオリエンテーション進行表を作るプロンプト

新入社員の入社初日の受け入れプログラムを整理します。

```
以下の情報をもとに、新入社員の入社初日のオリエンテーション
進行表と、担当者向けの進行スクリプトを作成してください。

【入社初日に行う予定の内容】
（例：
・9:00 受付・出迎え（人事担当）
・9:15 社長挨拶・会社説明（15分）
・9:30 各部門長・先輩社員との顔合わせ
・10:00 ハンドブック読み合わせ（安全・ルール説明）
・11:00 工場見学（担当: 製造部長）
・12:00 昼食（先輩社員と一緒に食堂へ）
・13:00 各種手続き（保険・給与振込先・緊急連絡先の記入）
・14:00 配属部署でのOJT開始
）

【新入社員の不安に配慮したいポイント】
（例：未経験者・高校卒業したての若者が多い。
  工場・機械は初めての環境。緊張するのが自然なので、
  歓迎の雰囲気を大切にしたい。）

出力形式:
・入社初日の進行表（時間・内容・担当者・場所）の表
・受け入れ担当者向けの「一言声がけ例文」（何を言うか具体的に）
・「入社初日に渡す書類・グッズのチェックリスト」
  （ハンドブック・名札・更衣室案内・緊急連絡先カード等）
・「入社初日を終えた後にするフォロー（班長が翌日確認すること）」
```

## 新入社員が質問しやすい環境を作るプロンプト

新入社員が困ったときに相談しやすい雰囲気を伝える文書を作ります。

```
以下の状況をもとに、新入社員へのウェルカムメッセージと
「質問・相談の仕方ガイド」を作成してください。

【伝えたいメッセージ】
・「わからないことは何でも聞いていい」という文化を根付かせたい
・「先輩も最初は同じだった」という安心感を伝えたい
・「勝手にやって失敗するよりも、確認してから行動する」を習慣化させたい
・「困ったときの相談先（直属の班長 / 人事 / 先輩○○さん）を明確にしたい」

【よくある新入社員の悩み】
（例：
・機械の操作が分からないがどう聞けばいいか分からない
・ミスをしてしまったが怒られるのが怖くて報告できない
・体調不良や個人的な相談はどこにすればいいか分からない
）

出力形式:
・新入社員向けのウェルカムメッセージ（2〜3段落）
・「相談の仕方・質問の仕方」の具体的な例（「○○さん、今お時間よろしいですか」の例文）
・「こんなときは誰に相談する？」シーン別の相談先一覧
```
