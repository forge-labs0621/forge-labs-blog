製造指図書と現品票は生産指示から出荷までをつなぐ重要書類です。AIを使って製造指図書のフォーマット設計・ロット追跡の仕組み・トレーサビリティレポートを体系化できます。

---

## 製造指図書・現品票のフォーマット設計を整理するプロンプト

```
以下の情報をもとに、製造指図書と現品票の設計書類を
整理してください。

【製造現場の概要】
製品形態: 受注生産品（1ロット50〜200個）
製造工程: 素材受入 → 前加工 → 熱処理（外注） → 仕上げ加工 → 検査 → 出荷
管理要件: 
  ・顧客から材料証明書・製造ロット番号の提出要求あり
  ・不良品発生時、使用材料ロットまで遡れること
  ・工程ごとに担当者・日時・検査結果を記録すること

現在の課題:
  ・製造指図書が手書きA4紙で、記入漏れ・判読困難が多い
  ・現品票が工程間で外れて行方不明になるケースがある
  ・トレーサビリティを求められた際、調査に2〜3日かかる

出力形式:
・「製造指図書のフォーマット設計
  （必須項目: 受注番号 / 製品番号 / 品名 / ロット番号 / 数量 / 工程別指示内容 /
  完了予定日 / 使用材料番号 / 特記事項）」
・「現品票（工程内タグ）の設計
  （サイズ: A6耐水紙 / 記載: ロット番号・現在工程・数量・前工程完了印）」
・「ロット番号体系の設計例（受注年月日+品目コード+連番）」
・「工程間受け渡し確認書（前工程→後工程の数量・品質確認のサイン手順）」
・「製造指図書の電子化移行計画（Excelテンプレート化→最終的にMESへ）」
```

## ロット追跡調査報告書を整理するプロンプト

```
顧客から不良品のロット追跡調査依頼が来た際の
調査報告書の作成プロンプトを整理してください。

【調査依頼の内容】
依頼元: 株式会社〇〇（自動車部品メーカー）
依頼日: 2026-06-20
対象部品: 精密加工シャフト（型番 SH-4821）
問題内容: 搭載車両の組立工程でシャフトの嵌合不良が発覚
不良品ロット: 当社納入品のロット番号「L2604-SH4821-023」
調査要求: 
  ①当該ロットの使用材料証明書の提出
  ②製造工程の加工条件記録の提出
  ③品質検査成績書の提出
  ④同一材料を使用した他ロットの特定と現品確認
  ⑤不良原因の分析と再発防止策の提出
  回答期限: 2026-06-27（7日後）

出力形式:
・「ロット追跡調査の手順書（どこを・誰が・どう調査するかのフロー）」
・「顧客提出用トレーサビリティ報告書のフォーマット
  （材料証明 / 製造ロット記録 / 検査成績 / 工程条件 / 担当者）」
・「同一材料使用ロットの特定と拡大調査の進め方」
・「顧客への調査状況中間報告書（中間報告5日目時点の文例）」
・「最終調査報告書の構成（サマリー / 事実確認 / 原因分析 / 再発防止策 / 添付書類一覧）」
```
