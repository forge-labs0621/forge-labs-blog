中小製造業の人事総務担当者にとって、給与計算・社会保険手続きは繁忙期に集中する重い作業です。AIを使って書類の下書き・確認リスト・計算チェックを効率化できます（最終確認は社労士等の専門家が必要）。

---

## 社会保険の算定基礎届（定時決定）の準備をするプロンプト

毎年4〜6月に提出が必要な算定基礎届の事前準備を整理します。

```
以下の情報をもとに、社会保険の算定基礎届（定時決定）の
準備作業を整理してください。

【自社の概要】
従業員数: ○名 / うち社会保険被保険者: ○名
事業所所在地: ○都道府県
給与形態: 月給制（固定給 + 残業代 + 諸手当）

【算定基礎届の概要と提出スケジュール】
目的: 毎年9月〜翌年8月の標準報酬月額の決定
対象月: 4月・5月・6月の報酬（支払いベースで集計）
提出期限: 毎年7月10日

【集計が必要な手当の扱い（社会保険算入・除外）】
算入する手当（例）:
  ・基本給
  ・職務手当・役職手当
  ・時間外手当（残業代）
  ・通勤手当（月10万円を超える部分は除外）
  ・住宅手当

除外する手当（例）:
  ・臨時の報酬（慶弔見舞金・皆勤手当で3か月超なし）
  ・年3回以内の賞与

出力形式:
・「算定基礎届の準備チェックリスト（何を・いつまでに・誰が準備するか）」
・「手当の算入・除外判定の確認フォーマット」
・「従業員別の4〜6月報酬集計シートのフォーマット」
・「算定基礎届が初めての担当者向け作業フロー」
・「社労士に確認すべきポイント（自社で判断が難しい手当）」
```

## 賞与支払届・月額変更届の準備を整理するプロンプト

賞与支払後・随時改定時の届出書類を整理します。

```
以下の情報をもとに、社会保険の賞与支払届と
月額変更届（随時改定）の準備を整理してください。

【賞与支払届の概要】
賞与支払日: （例: 2026年7月15日・12月15日の年2回）
賞与対象者: 社会保険被保険者全員（休職者は要確認）
提出期限: 賞与支払後5日以内（年金事務所・健康保険組合へ）

確認が必要なポイント:
  ・育児休業中の従業員の賞与（保険料免除申請との関係）
  ・賞与が高額になった場合の年間上限（573万円）の確認

【月額変更届（随時改定）の確認条件】
次の3つを全て満たした場合、定時決定を待たずに月額を変更する:
  ① 昇給・降給等の固定的賃金の変動がある
  ② 変動月から連続する3か月間の平均標準報酬月額が現在と2等級以上差がある
  ③ 3か月間いずれも支払基礎日数が17日以上

出力形式:
・「賞与支払届の記入確認チェックリスト」
・「月額変更届の要否判定フローチャート（言葉で説明）」
・「昇給・変動のあった従業員をリストアップする際の確認事項」
・「提出ミスが多い記入例と注意点」3点
・「手続きのタイムライン（賞与支払い→届出→保険料天引き反映まで）」
```

## 労働保険年度更新の準備を整理するプロンプト

毎年6月の労働保険年度更新の書類準備を整理します。

```
以下の情報をもとに、労働保険（雇用保険・労災保険）の
年度更新の準備作業を整理してください。

【自社の概要】
業種: （例: 金属製品製造業）
従業員数: ○名（うち雇用保険加入者: ○名 / パート・アルバイト: ○名）

【年度更新の概要】
対象期間: 前年度（4月1日〜3月31日）の賃金総額の確定
確定保険料の計算: 賃金総額 × 料率（雇用保険 / 労災保険）
概算保険料の計算: 当年度（4月1日〜翌3月31日）の見込み賃金 × 料率
提出・納付期限: 毎年6月1日〜7月10日

【集計が必要な賃金の範囲】
含む: 基本給・各手当・賞与・時間外手当（雇用保険は雇用保険料）
除く: 退職金・弔慰金（支払いが1回限りのもの）

出力形式:
・「年度更新の準備チェックリスト（3か月前から何を準備するか）」
・「賃金総額の集計フォーマット（雇用保険対象者・労災対象者別）」
・「業種別の労災保険料率の確認方法（自社の料率を調べる手順）」
・「年度更新で誤りやすいポイントと確認事項」
・「新規雇用者・途中退職者の賃金算入の考え方」
```

---

### 注意事項

給与計算・社会保険手続きは、法令要件が複雑で会社の状況によって異なります。AIで作成したドラフトは社会保険労務士等の専門家に必ず確認してください。特に算入・除外の判断、随時改定の要否、育休・産休中の手続きは要確認です。
