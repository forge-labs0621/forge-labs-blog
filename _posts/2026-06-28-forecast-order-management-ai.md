製造業では取引先から正式な注文書が届く前に「内示（内々示・フォーキャスト）」「仮受注」の情報が届くことが多く、これを生産計画に反映する仕組みが必要です。内示管理・受注管理の書類整理をAIで効率化できます。

---

## 内示（フォーキャスト）の管理表を作るプロンプト

取引先から届く内示情報を一覧管理する表を作ります。

```
以下の情報をもとに、内示（フォーキャスト）管理表のフォーマットと
整理例を作成してください。

【内示の受け取り方と活用】
内示の形式: （例：取引先から毎月20日前後にExcelで届く
  3か月先の生産予定数量（月次単位））
活用方法: （例：内示に基づいて次月の材料発注・要員計画・外注手配を行う）
内示と確定受注の違い:
  ・内示: 予定数量（変更される可能性あり・法的拘束力なし）
  ・確定受注（発注書）: 正式な注文（変更困難・法的拘束力あり）

【管理したい内示情報】
・品番・品名
・取引先名
・内示受取日
・3か月分の内示数量（月別）
・前回内示との差異
・確定受注への更新状況

出力形式:
・内示管理表のフォーマット（Excel的な表形式）
・「前回内示との比較（増減量・増減率）」の計算欄
・「内示急変時のリスク管理」（在庫・材料発注への影響評価）
・「内示精度を定期的に確認するためのKPI」の提案
```

## 内示急変・受注キャンセルへの対応メールを作るプロンプト

取引先から内示・受注の大幅な減少・キャンセルが来た場合の対応を整理します。

```
以下の状況で、取引先へのメールと社内対応方針を整理してください。

【状況】
（例：取引先から「来月の内示を従来比50%に削減したい」という
連絡が来た。当社はすでに材料を先行発注しており、
在庫 ¥800万円分が滞留することになる見込み。）

【自社の立場と確認したいこと】
・なぜ急減したのか（理由の確認）
・已発注済み材料のキャンセルが可能かどうか（仕入先への影響確認）
・補償・調整の可能性があるか（大幅内示変更は取引基本契約上どうなるか）
・来月以降の回復見通しはあるか

出力形式:
・取引先への確認メール（感情的にならず事実確認を優先するトーン）
・「取引基本契約書で内示変更に関する条項があれば確認すること」の注記
・社内向けの緊急対応方針メモ（仕入先への発注キャンセル・在庫の評価・
  要員計画の見直し）
・「在庫の滞留コスト試算と代替処理案（他品番への流用・返品交渉等）」の骨子
```

## 受注管理・出荷計画表を作るプロンプト

確定受注から出荷スケジュールを作成する管理表を整理します。

```
以下の情報をもとに、受注管理・出荷計画表のフォーマットを
作成してください。

【管理したい情報】
・受注番号・取引先名
・品番・品名・受注数量
・受注日・希望納期
・当社の回答納期（製造リードタイム考慮後）
・製造着手日・完了予定日
・検査完了予定日
・出荷予定日・出荷確定日
・出荷数量・残数量（分割納入の場合）
・ステータス（受注済 / 製造中 / 検査中 / 出荷済 / 完了）

出力形式:
・受注管理表のフォーマット（表形式）
・「今週の出荷予定一覧」のフィルタビュー説明
・「納期遅延リスクがある案件（今週・来週の出荷予定で着手が遅れているもの）」
  のハイライト方法
・「月末の出荷集中を避けるための案件分散の考え方」のアドバイス
```
