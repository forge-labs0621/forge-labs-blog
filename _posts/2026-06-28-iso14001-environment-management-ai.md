ISO 14001は環境マネジメントシステムの国際規格です。製造業では取引先（特に大手自動車メーカー等）からISO 14001の取得・維持を求められるケースが増えています。環境目標設定・内部監査・アスペクト評価などの書類整備をAIで効率化できます。

---

## 環境目標・環境管理プログラムを作るプロンプト

ISO 14001の環境目標と、それを達成するための管理プログラムを作ります。

```
以下の情報をもとに、ISO 14001:2015に基づく
環境目標・環境管理プログラムを作成してください。

【自社の概要・環境方針】
業種・規模: （例：精密プレス加工業 / 従業員60名 / 工場面積2,000㎡）
主要な環境側面（自社の事業が環境に与える主な影響）:
  （例：電力使用（CO2排出）/ 廃棄物（金属切り粉・廃油）/ 
  化学物質（洗浄剤・防錆油）/ 排水（冷却水））

【今年度の環境目標（上位3つ）】
（例：
1. 電力使用量を昨年度比5%削減（製造部門）
2. 産業廃棄物（金属切り粉）の再資源化率を90%以上に維持
3. 化学物質の使用量を5%削減（洗浄剤の濃縮タイプへ切り替え）
）

出力形式:
・環境目標シート（目標 / 達成指標（KPI）/ 担当部門 / 期限）の表
・各目標に対する「環境管理プログラム」（具体的な実施施策・スケジュール）
・目標の「測定方法（何を・いつ・誰が測定するか）」の記載
・ISO 14001 6.2条「環境目標及びそれを達成するための計画策定」の要件に対応した構成
```

## 環境アスペクト評価表を作るプロンプト

事業活動の環境への影響（アスペクト）を評価する表を作ります。

```
以下の自社の事業活動をもとに、
ISO 14001:2015の環境アスペクト評価表を作成してください。

【評価対象の活動・設備】
製造プロセス（例）:
  - プレス加工: 電力使用 / 騒音（隣接住宅への影響）/ 金属切り粉発生
  - 塗装工程: 有機溶剤（VOC）排出 / 廃塗料・廃シンナーの発生
  - 洗浄工程: 洗浄液の使用・廃液発生 / 排水処理
補助活動（例）:
  - フォークリフト運転: 排気ガス（CO2）/ 燃料漏洩リスク
  - 事務活動: 用紙使用 / ゴミ（一般廃棄物）

【評価基準（自社で設定しているもの）】
（例: 重要度 = 発生頻度 × 環境への影響規模 × 関連法規制の有無
  合計スコア9以上を「重大な環境アスペクト」として管理強化）

出力形式:
・「活動 / 環境アスペクト / 環境影響 / 通常/緊急の区別 / 
  評価スコア / 重大かどうか」の表
・「重大な環境アスペクト」に対する管理計画の方向性
・緊急時（漏洩・流出等）を想定したアスペクトの追加
```

## ISO 14001内部環境監査チェックリストを作るプロンプト

環境マネジメントシステムの内部監査で確認するチェックリストを作ります。

```
以下の情報をもとに、ISO 14001:2015の内部環境監査
チェックリストを作成してください。

【監査対象部門・確認条項】
対象部門: （例：製造部（プレス・洗浄・塗装工程））
重点確認項目:
  ・6.1.2（環境アスペクト）: アスペクト評価表が最新か
  ・6.2（環境目標）: 目標値の管理・測定が実施されているか
  ・8.1（運用計画及び管理）: 環境に影響する作業の手順書が整備されているか
  ・8.2（緊急事態への準備及び対応）: 化学物質漏洩時の対応手順は整備・周知されているか
  ・7.2（力量）: 環境管理に関する教育・訓練の記録はあるか

出力形式:
・「確認項目 / 確認方法 / ISO条項 / 確認結果（○/×/要確認） / コメント」の表形式
・各確認項目に「確認すべき証拠書類・記録の名称」を添える
・「監査でよく見つかる不適合例」のコラム（例：記録の未記入・目標値の未更新等）
```

## マネジメントレビュー（環境）の準備資料を作るプロンプト

ISO 14001のマネジメントレビュー向けに環境の実績データをまとめます。

```
以下のデータをもとに、ISO 14001のマネジメントレビュー向けの
環境実績報告書を作成してください。

【今年度（2026年度）の環境実績データ】
電力使用量:
  目標: 前年比▲5% / 実績: 前年比▲3.8%（目標未達）
  主な原因: 8月の猛暑で空調使用量が増加
廃棄物:
  目標: 再資源化率90%以上 / 実績: 93.2%（目標達成）
  内訳: 金属切り粉（売却）/ 廃油（再生業者委託）/ 一般廃棄物
化学物質:
  目標: 使用量▲5% / 実績: ▲7.1%（目標達成）
  要因: 洗浄剤の濃縮タイプへの切り替えが完了（5月）
コンプライアンス:
  法規制違反・行政指導: なし
  環境クレーム（近隣・取引先）: なし

【来年度の課題・計画（案）】
・電力削減: 省エネインバータ設備の更新（設備補助金活用予定）

出力形式:
・マネジメントレビュー向けサマリー（A4 1枚以内）
・目標に対する達成率の評価（目標達成 / 未達成 / 理由）
・「ISO 14001システムの有効性の評価」セクション
・来年度への改善計画の提案
```
